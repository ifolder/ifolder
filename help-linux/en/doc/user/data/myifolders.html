3.0 Managing iFolders

An iFolder™ is a local directory used to selectively share and synchronizes files with user-specified members. The iFolder files are accessible to all members via a host service, such as a Novell® iFolder® 3.x server. iFolder files can be modified and read by those with privileges to do so.
