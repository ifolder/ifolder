启动 iFolder 客户程序

当 iFolder 运行时，“iFolder 服务” 图标将显示在任务栏的通知区域上。iFolder 已集成到桌面环境中。当在文件管理器、桌面或 iFolder 浏览器中查看 iFolder 时，iFolder 徽标（绿色的“i”） 会显示在 iFolder 上。加密的 iFolder 由 徽标（锁定的文件夹）表示。

重要说明：必须启用任务栏的通知区域，才能在 iFolder 运行时显示 iFolder 服务图标。

您可以随时对本地 iFolder 中的文件进行本地操作。无论 iFolder 客户程序是否正在运行，它都会记录您在操作期间对本地 iFolder 数据所作的更改。如果您在未运行 iFolder 时对本地 iFolder 数据作了更改，则 iFolder 会在您下次启动它时标识和记录这些差异。

当 iFolder 未在运行时，iFolder 将显示在文件管理器或桌面上。您可以访问 iFolder 本地拷贝中的文件，但不能管理 iFolder。此外，您必须登录到一个帐户才能创建 iFolder 或同步其文件。

注：创建帐户期间，用户通过 SSL 进行登录。当系统提示您是否接受服务器证书时，必须验证并接受该证书，才能成功连接。

有关停止 iFolder 客户程序的信息，请参见退出 iFolder 客户程序。

Linux

  • 登录时自动启动 iFolder

  • 手动启动 iFolder

登录时自动启动 iFolder

若要启用在登录到桌面时自动启动 iFolder，请执行下列操作之一：

  • 打开应用程序菜单，单击“更多应用程序”，找到“iFolder 3”并右击，然后从菜单中选择“添加到启动程序”。

  • 确保注销时 iFolder 正在运行，然后保存会话设置。当您登录到桌面时选择此会话选项。

  • 打开一个终端窗口，以 root 用户身份登录，然后运行 /opt/gnome/bin/gnome-session-properties 可将 iFolder 添加为登录到计算机时要启动的程序。

若要禁止在登录时自动启动 iFolder，请执行下列操作之一：

  • 打开应用程序菜单，找到“iFolder 3”并右击，然后选择“从启动程序中去除”。

  • 确保注销时 iFolder 未运行，然后保存会话设置。当您登录到桌面时选择此会话选项。

手动启动 iFolder

  1. 以创建 iFolder 时要使用的本地 Linux 用户身份登录到计算机。

  2. 使用以下方法之一启动 iFolder：

    • 在任务栏中，打开应用程序菜单，单击“更多应用程序”，通过搜索或滚动可用选项查找“iFolder 3”，右击“iFolder 3”，然后选择“启动 iFolder 3”。

    • 打开终端壳层，然后输入

      /usr/bin/ifolder

Windows

  • 登录时自动启动 iFolder

  • 手动启动 iFolder

登录时自动启动 iFolder

要在登录到桌面时自动启动 iFolder：

  1. 右击“iFolder 服务” 图标，然后选择“首选设置”以打开“iFolder 首选设置”对话框并进入“常规”选项卡。

  2. 选择“登录到桌面时启动 iFolder”，然后单击“应用”。

  3. 单击“确定”关闭“iFolder 首选设置”对话框。

禁止“登录到桌面时自动启动 iFolder”：

  1. 右击 iFolder 服务图标，然后选择“首选设置”以打开“iFolder 首选设置”对话框并进入“一般”选项卡。

  2. 取消选择“登录到桌面时启动 iFolder”，然后单击“应用”。

  3. 单击“确定”关闭“iFolder 首选设置”对话框。

手动启动 iFolder

  1. 使用创建 iFolder 时要使用的本地 Windows 用户身份登录到计算机。

  2. 如果成功登录时 iFolder 未自动启动，则通过单击“开始” > “程序” > “iFolder 3”来启动 iFolder，然后选择“iFolder 3 客户程序”应用程序。

Macintosh

  • 登录时自动启动 iFolder

  • 手动启动 iFolder

登录时自动启动 iFolder

要在登录到 Macintosh 计算机时自动启动 iFolder，应将 iFolder 添加到“启动项目”。

  1. 启动 iFolder。

  2. 从对接台右击 iFolder 图标并选择“登录时打开”选项。

    或

    从对接台按下 Ctrl 键，单击 iFolder 图标，然后选择“登录时打开”。

要禁用自动启动 iFolder，请从“启动项目”中去除 iFolder。

  1. 从对接台右击 iFolder 图标，然后取消选择“登录时打开”。

    或

    从对接台右击或按下 Ctrl 并单击 iFolder 图标，然后取消选择“登录时打开”。

手动启动 iFolder

  1. 使用要在创建 iFolder 时使用的本地 Macintosh 用户身份登录到计算机。

  2. 使用以下方法之一启动 iFolder：

    • 打开 Macintosh HD 上的“应用程序”目录，选择“应用程序”，然后双击 “iFolder 3”。

    • 在聚光灯中键入 iFolder（Apple 按钮 + 空格键或单击右上部的放大镜图标），然后单击“应用程序”下的“iFolder 3”。

    • 打开终端壳层，以根用户身份登录，然后输入 

      /Applications/iFolder 3.app/Contents/MacOS/iFolder 3

有关更多信息，请参见 法律声明.
