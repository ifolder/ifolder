Configuration des préférences iFolder du Client

  1. Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, puis sélectionnez Préférences pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Général.

  2. Spécifiez vos préférences pour les paramètres suivants :

    Paramètre

    Description

    Démarrer iFolder lors du login au bureau (uniquement disponible pour le client Windows)

    Sélectionnez cette option pour lancer iFolder lorsque vous vous connectez avec le nom d'utilisateur local que vous avez défini lors de la création du compte iFolder. Pour les comptes pour lesquels vous avez activé Mémoriser mon mot de passe, la synchronisation s'effectue en arrière-plan à l'intervalle que vous avez spécifié.

    Désélectionnez cette option pour démarrer iFolder manuellement.

    Afficher la boîte de dialogue de confirmation de la création d'un dossier iFolder

    Cochez cette case pour recevoir un message de confirmation à chaque fois que vous créez un dossier iFolder, en plus des indicateurs visuels habituels comme l'apparition du dossier iFolder dans la liste correspondante.

    Désélectionnez cette case pour ne plus recevoir le message de confirmation.

    Masquer la fenêtre principale du dossier iFolder au démarrage

    Désélectionnez cette case à cocher pour afficher la fenêtre iFolder au démarrage.

    Cochez cette case si vous souhaitez que la fenêtre iFolder reste masquée au démarrage.

    Afficher les journaux de synchronisation

    Cochez cette case si vous souhaitez afficher les journaux de synchronisation au démarrage.

    Désélectionnez cette case à cocher si vous souhaitez masquer les journaux de synchronisation au démarrage.

    Afficher des notifications en cas de violations des stratégies

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsqu'une stratégie est violée.

    M'avertir lorsque des dossiers iFolder sont partagés

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsque vous recevez une nouvelle invitation à partager un dossier iFolder.

    Pour afficher une liste des dossiers iFolder pouvant être téléchargés sur votre ordinateur actuel, sélectionnez Afficher les dossiers iFolder disponibles dans le navigateur iFolder lorsque iFolder est en cours d'exécution.

    M'avertir en cas de conflits

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsqu'un conflit se produit lors de la synchronisation des fichiers.

    Désélectionnez-la pour désactiver les messages contextuels en cas de conflits de synchronisation.

    En cas de conflits, l'icône iFolder Erreur s'affiche sur le dossier iFolder lorsque iFolder est en cours d'exécution.

    M'avertir lorsqu'un utilisateur rejoint le domaine iFolder

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsqu'un utilisateur accepte une invitation à partager un dossier iFolder.

    Désélectionnez-la pour désactiver les messages contextuels.

    Vous pouvez afficher une liste complète des utilisateurs iFolder lorsque iFolder est en cours d'exécution, en sélectionnant le dossier iFolder, puis en cliquant sur Partager avec pour ouvrir la boîte de dialogue Propriétés iFolder sous l'onglet Partage.

    Synchronisation

    Synchroniser automatiquement les dossiers iFolder : cochez cette case pour activer la synchronisation de tous les dossiers iFolder sur cet ordinateur sous votre identité de login ou votre login local actuel. Désélectionnez la case pour désactiver la synchronisation.

    Fréquence de synchronisation des dossiers iFolder : indiquez l'intervalle minimum à utiliser pour la synchronisation des dossiers iFolder sur cet ordinateur sous votre identité de login ou votre login local actuel. Spécifiez la valeur et les unités. Si cette option est activée, la valeur par défaut est de 5 minutes. L'intervalle minimum peut être défini sur 5 secondes.

    Si vous utilisez le client iFolder avec un serveur Novell iFolder 3.7 ou version ultérieure, l'intervalle de synchronisation minimal effectif est toujours la valeur la plus élevée des paramètres suivants :

    • La valeur par défaut de la stratégie système est zéro (0), sauf si une stratégie utilisateur est définie. Si c'est le cas, la règle d'utilisateur prévaut sur la règle du système, que la valeur de la règle d'utilisateur soit plus grande ou plus petite que la règle du système.

    • La stratégie de la machine locale ou la configuration sur la machine du client synchronisée avec le serveur.

    • La stratégie (de collection) du dossier iFolder.

  3. Cliquez sur Fermer pour fermer la boîte de dialogue Préférences iFolder.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
