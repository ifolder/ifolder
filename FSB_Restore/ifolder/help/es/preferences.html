Configuración de las preferencias de iFolder del cliente

  1. Haga clic con el botón derecho en el icono Servicios de iFolder en el área de notificaciones y, a continuación, seleccione Preferencias para abrir la pestaña General del recuadro de diálogo Preferencias de iFolder.

  2. Especifique sus preferencias para las siguientes opciones:

    Parámetro

    Descripción

    Iniciar iFolder al entrar en el escritorio (sólo disponible en el Cliente Windows)

    Seleccione esta opción para iniciar iFolder siempre que entre con el nombre de usuario local en el equipo donde ha configurado la cuenta de iFolder. En las cuentas en las que ha habilitado la opción Recordar mi contraseña, la sincronización se producirá en segundo plano según el intervalo de sincronización especificado.

    Deseleccione esta opción para iniciar iFolder manualmente.

    Visualizar recuadro de diálogo de confirmación sobre la creación satisfactoria de iFolder

    Seleccione este recuadro de verificación para recibir un mensaje de confirmación cada vez que cree una carpeta iFolder además de las indicaciones visuales habituales como, por ejemplo, la aparición de la carpeta iFolder en la lista de carpetas iFolder.

    Deseleccione este recuadro de verificación para dejar de recibir el mensaje de confirmación.

    Ocultar ventana principal de iFolder durante el inicio

    Deseleccione este recuadro de verificación para visualizar la ventana de iFolder durante el inicio.

    Seleccione este recuadro de verificación si desea que la ventana de iFolder permanezca oculta durante el inicio.

    Visualizar registros de sincronización

    Seleccione este recuadro de verificación si desea visualizar los registros de sincronización durante el inicio.

    Deseleccione este recuadro de verificación si desea ocultar los registros de sincronización durante el inicio.

    Visualizar notificaciones para infracciones de directivas

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que se infrinja una directiva.

    Notificar cuando se comparten carpetas iFolder

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que reciba una nueva invitación para compartir una carpeta iFolder.

    Para ver una lista de las carpetas iFolder disponibles para su descarga en el equipo, seleccione Mostrar carpetas iFolder disponibles en el navegador de iFolder siempre que iFolder se esté ejecutando.

    Notificar cuando se producen conflictos

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que se produzca un conflicto al sincronizar archivos.

    Deseleccione este recuadro de verificación para inhabilitar los mensajes emergentes en caso de conflicto de sincronización.

    Cuando se produzca algún conflicto, se mostrará el icono Error de iFolder en la carpeta iFolder en cuestión, siempre que se esté ejecutando iFolder.

    Notificar cuando un usuario se une al dominio de iFolder

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que un usuario acepte una invitación para compartir una carpeta iFolder.

    Deseleccione este recuadro de verificación para inhabilitar mensajes emergentes.

    Para ver la lista completa de usuarios de iFolder, siempre que iFolder se esté ejecutando, seleccione la carpeta iFolder y, a continuación, haga clic en Compartir con para abrir la pestaña Compartición del recuadro de diálogo Propiedades de iFolder.

    Sincronización

    Sincronizar carpetas iFolder automáticamente: seleccione esta casilla de verificación para habilitar la sincronización en todas las carpetas iFolder del equipo en cuestión con la identidad de entrada o inicio de sesión local actual. Deseleccione la casilla de verificación para inhabilitar la sincronización.

    Sincronizar carpetas iFolder cada: especifique el intervalo mínimo que desea utilizar para sincronizar carpetas iFolder en este equipo con la identidad de entrada o inicio de sesión local actual. Especifique el valor y las unidades. Si esta opción está habilitada, el tiempo por defecto es 5 minutos. El intervalo mínimo puede fijarse en 5 segundos.

    Si utiliza el Cliente iFolder con un servidor Novell iFolder 3.7 o posterior, el intervalo de sincronización mínimo que se aplicará siempre será el valor más alto de los parámetros siguientes:

    • El valor por defecto de la directiva del sistema es cero (0), a menos que se establezca una directiva de usuario. Si existe una directiva de usuario establecida, ésta anulará la directiva del sistema, tanto si es superior o inferior en valor.

    • La directiva del equipo local o el ajuste del equipo cliente que se sincroniza con el servidor.

    • La directiva de iFolder (recopilación).

  3. Haga clic en Cerrar para cerrar el recuadro de diálogo Preferencias de iFolder.

Para obtener más información, consulte Información legal.
