iFolder 用戶端

iFolder 用戶端整合了作業系統，在您的原生桌上型環境中提供 iFolder 服務。 iFolder 用戶端可讓您管理自己的 iFolder，以及選取要在每個電腦上設定的共用 iFolder。iFolder 支援下列作業系統：

  • SUSE® Linux Enterprise Desktop 10 SP1 與 SP2

  • SUSE® Linux Enterprise Desktop 11

  • Windows Vista SP1

  • Windows XP SP2

  • OpenSUSE 11.1

  • Apple Macintosh® 10.4 和更新版本

有關更詳細的資訊，請參閱 法律聲明.
