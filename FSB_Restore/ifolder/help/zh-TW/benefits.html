iFolder 的優點

iFolder 具有下列優點：

  • iFolder 與原始桌上型電腦環境整合，讓您容易地建立和管理多個 iFolder。

  • iFolder 具有高度的延展性和彈性，對於每位使用者的 iFolder 數量或每個 iFolder 的成員數目不具有實際上的限制。

  • 啟用加密之後，iFolder 可以保護跨網路傳輸並儲存在 iFolder 伺服器上的資料。

  • 使用增強的 Web 存取主控台，您可以不需要使用 iFolder 用戶端而從任何工作站存取 iFolder 伺服器上的檔案，並使用 Web 瀏覽器來執行 iFolder 用戶端所有的作業。

  • 您可以容易地選擇共用個人檔案和商務檔案。

  • iFolder 可以將 iFolder 資料移轉和升級到更新的版本。

  • iFolder 支援 LDAPGroup。您可以與 LDAPGroup 共用 iFolder。LDAPGroup 的成員可以建立、擁有 iFolder 以及與其他個別使用者或其他 LDAPGroup 共用 iFolder。

  • 您可以對本身擁有的 iFolder 成員使用者，或您有控制成員存取權之必要權限的成員使用者，控制他們的存取層級。

  • iFolder 會明顯地更新多個工作站上成員 iFolder 的檔案。

  • iFolder 執行時，它可追蹤並記錄您在離線工作時所做的變更，然後在您連接至伺服器時同步這些變更.

  • iFolder 對存取 iFolder 以同步其資料的成員提供安全的驗證。

  • iFolder 提供透過電子郵件交換檔案的替代方案，避免電子郵件附件檔案大小限制，並降低電子郵件儲存空間的需求。

有關更詳細的資訊，請參閱 法律聲明.
