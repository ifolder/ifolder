Разрегистрация учетной записи iFolder

Если необходимо работать в автономном режиме, разрегистрируйте учетную запись iFolder.

  • Macintosh

  • Linux

  • Windows

Macintosh

  1. Щелкните значок Сервисы iFolder на панели быстрого запуска приложений, затем выберите Учетные записи, чтобы открыть диалоговое окно "Настройки iFolder".

  2. Щелкните значок Учетные записи на панели инструментов для просмотра списка учетных записей в текущем домене iFolder.

  3. Найдите учетную запись, управление которой необходимо осуществить, затем щелкните Разрегистрация для отключения от сервера iFolder.

Linux

  1. В области оповещений щелкните правой кнопкой значок Сервисы iFolder , затем выберите Учетные записи, чтобы открыть диалоговое окно "Настройки iFolder" на вкладке Учетные записи.

  2. Выберите необходимую учетную запись, затем снимите флажок Интерактивный режим рядом с учетной записью, чтобы завершить соединение с сервером iFolder.

  3. Щелкните Закрыть для закрытия диалогового окна "Настройки iFolder".

Windows

  1. В области оповещений щелкните правой кнопкой значок Сервисы iFolder , затем выберите Учетные записи, чтобы открыть диалоговое окно "Настройки iFolder" на вкладкеУчетные записи.

  2. Выберите необходимую учетную запись, затем снимите флажок Интерактивный режим рядом с учетной записью.

  3. Щелкните Применить, чтобы завершить соединение с сервером iFolder, затем щелкните ОК, чтобы закрыть диалоговое окно "Настройки iFolder".

Дополнительную информацию см. в Юридическая информация.
