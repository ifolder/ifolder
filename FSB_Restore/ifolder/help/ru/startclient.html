Запуск клиента iFolder

Если приложение iFolder запущено, значок Сервисы iFolder отображается в области оповещений на панели задач. Приложение iFolder интегрировано в вычислительную среду рабочей станции. Эмблема iFolder (зеленая буква "i") отображается на папках iFolder во время их просмотра в менеджере файлов на рабочей станции или в средстве просмотра iFolder. Зашифрованная папка iFolder обозначается эмблемой (заблокированная папка).

ВАЖНО.Необходимо включить область оповещений на панели задач, чтобы при запуске приложения iFolder на ней отображался значок Сервисы iFolder.

Работать локально с фалами в локальных папках iFolder можно в любое время. Если клиент iFolder запущен, во время работы с папками все изменения локальной папки данных iFolder протоколируются. Если локальные данные папки iFolder изменяются, когда приложение iFolder не запущено, при последующем запуске приложение iFolder определяет и протоколирует все изменения.

Если приложение iFolder не запущено, папки iFolder отображаются в менеджере файлов или на рабочей станции. Можно получить доступ к файлам в локальной копии папки iFolder, но управлять папками iFolder нельзя. Кроме того, необходимо выполнить вход с помощью учетной записи, чтобы иметь возможность создавать папку iFolder или синхронизировать ее файлы.

ПРИМЕЧАНИЕ.Во время создания учетной записи вход пользователя в систему выполняется по протоколу SSL. При получении запроса о том, чтобы принять сертификат сервера, для успешного соединения необходимо проверить сертификат и принять его.

Дополнительную информацию об остановке клиента iFolder см. в разделе Завершение работы с клиентом iFolder.

Linux

  • Автоматический запуск iFolder при входе

  • Запуск iFolder вручную

Автоматический запуск iFolder при входе

Чтобы настроить автоматический запуск iFolder при входе на компьютер, выполните одно из следующих действий:

  • Откройте меню приложений, щелкните More Applications (Приложения), найдите и щелкните правой кнопкой iFolder 3, затем в меню выберите Add to Startup Programs (Добавить в запуск программ).

  • При выходе убедитесь, что приложение iFolder запущено, затем сохраните настройки сеанса. При входе на компьютер выберите этот сеанс.

  • Откройте окно терминала, войдите в качестве пользователя root, затем введите команду /opt/gnome/bin/gnome-session-properties, чтобы добавить iFolder в качестве программы, которая будет загружаться при входе на компьютер.

Чтобы отключить автоматический запуск iFolder при входе, выполните одно из следующих действий:

  • Откройте меню приложений, найдите и щелкните правой кнопкой iFolder 3, затем выберите Remove from Startup Programs (Удалить из запуска программ).

  • При выходе убедитесь, что приложение iFolder не запущено, затем сохраните настройки сеанса. При входе на компьютер столе выберите этот сеанс.

Запуск iFolder вручную

  1. Войдите на компьютер в качестве локального пользователя Linux, которого необходимо использовать для создания папок iFolder.

  2. Для запуска iFolder используйте один из следующих способов:

    • На панели задач откройте меню приложений, щелкните More Applications (Приложения), среди доступных элементов найдите или перейдите к iFolder 3, щелкните правой кнопкой "iFolder 3", затем выберите Start iFolder 3 (Запуск iFolder 3).

    • Откройте оболочку терминала и введите

      /usr/bin/ifolder

Windows

  • Автоматический запуск iFolder при входе

  • Запуск iFolder вручную

Автоматический запуск iFolder при входе

Порядок настройки автоматического запуска iFolder при входе на рабочую станцию:

  1. Щелкните правой кнопкой значок Сервисы iFolder , затем выберите Настройки, чтобы открыть диалоговое окно "Настройки iFolder" на вкладке Общие.

  2. Выберите Запуск iFolder при входе на рабочую станцию , затем щелкните Применить.

  3. Щелкните ОК, чтобы закрыть диалоговое окно "Настройки iFolder".

Порядок отключения автоматического запуска iFolder при входе на рабочую станцию:

  1. Щелкните правой кнопкой значок Сервисы iFolder, затем выберите Настройки, чтобы открыть диалоговое окно "Настройки iFolder" на вкладке Общие.

  2. Отмените выбор параметра Запуск iFolder при входе на рабочую станцию, затем щелкните Применить.

  3. Щелкните ОК, чтобы закрыть диалоговое окно "Настройки iFolder".

Запуск iFolder вручную

  1. Войдите на компьютер в качестве локального пользователя Windows, которого необходимо использовать для создания папок iFolder.

  2. Если после успешной входа в систему приложение iFolder не было запущено автоматически, запустите iFolder, щелкнув Пуск > Программы > > iFolder 3, затем выберите приложение iFolder 3 Client (Клиент iFolder 3).

Macintosh

  • Автоматический запуск iFolder при входе

  • Запуск iFolder вручную

Автоматический запуск iFolder при входе

Для разрешения автоматического запуска клиента iFolder при входе на компьютер Macintosh добавьте iFolder в Startup Items (Автозагрузка).

  1. Запустите iFolder.

  2. На панели быстрого запуска приложений щелкните правой кнопкой значок iFolder и выберите опцию Open at Login (Открывать при входе).

    или

    На панели быстрого запуска приложений нажмите клавишу Ctrl, щелкните значок iFolder, затем выберите Open at Login (Открывать при входе).

Чтобы отменить автоматический запуск приложения iFolder, удалите iFolder из списка Startup Items (Автозагрузка).

  1. В прикреплении щелкните правой кнопкой значок iFolder и отмените выбор параметра Open at Login (Открывать при входе).

    или

    На панели быстрого запуска приложений щелкните правой кнопкой значок iFolder или нажмите клавишу Ctrl и отмените выбор параметра Open at Login (Открывать при входе).

Запуск iFolder вручную

  1. Войдите на компьютер в качестве локального пользователя Macintosh, которого необходимо использовать для создания папок iFolder.

  2. Для запуска iFolder используйте один из следующих способов:

    • Откройте каталог Applications (Приложения) на Macintosh HD, выберите Applications (Приложения), затем дважды щелкните приложение iFolder 3.

    • Введите iFolder в выделенном поле (кнопка Apple + пробел, либо щелкните значок лупы в верхнем правом углу), затем щелкните iFolder 3 в списке приложений.

    • Откройте терминальную оболочку, выполните вход в качестве пользователя root, затем введите 

      /Applications/iFolder 3.app/Contents/MacOS/iFolder 3

Дополнительную информацию см. в Юридическая информация.
