Movendo uma pasta iFolder

Depois que umapasta iFolder é criada, o caminho para o iFolder fica estático. Você não pode mover uma pasta iFolder de sua localização original para outra localização em seu computador local e mantê-la como uma pasta iFolder. A movimentação da pasta iFolder quebra o vínculo entre o banco de dados de sincronização e a pasta que você converteu em uma pasta iFolder.

Para mover uma pasta iFolder no computador onde foi criado:

  1. Converta a pasta iFolder para uma pasta normal.

  2. Mova a pasta.

  3. Converta a pasta em uma pasta iFolder.

  4. (Opcional) Compartilhe o iFolder.

Para mover uma pasta iFolder em um computador participante:

  1. Nas pastas iFolder, reverta a pasta para que seja interrompida a participação no iFolder, mas deixe-a disponível para você.

  2. Selecione a pasta iFolder disponível e, em seguida, clique em Configuração.

  3. Especifique um novo caminho local onde deseja colocar a pasta iFolder.

  4. Permita que a pasta iFolder sincronize arquivos com o servidor empresarial.
