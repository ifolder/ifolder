Resolvendo conflitos de arquivos

Quando acontecem conflitos entre versões de arquivos gravadas ao mesmo tempo ou quando um nome de arquivo difere somente devido ao uso de iniciais maiúsculas e minúsculas, o arquivo é assinalado com um Conflito de Nome e armazenado na área de conflito e um alerta é enviado para sinalizar o arquivo pelo cliente iFolder.


Conflitos de versões

Os conflitos de versões devem ser coordenados com outros membros antes que o conflito seja resolvido. Escolha, então, a versão correta na caixa de diálogo Resolução de Conflito.

  1. Quando ocorre um conflito, uma mensagem pop-up aparece na área de Notificação. Clique no X para ignorar a mensagem.

    Você pode desabilitar o comportamento pop-up configurando a preferência nas Preferências do iFolder > guia Geral.

  2. Na área de Notificação, clique o botão direito do mouse no ícone do iFolder e, em seguida, clique em Pastas iFolder para abrir o browser de pastas iFolder.

  3. Selecione o arquivo em conflito e, em seguida, clique em Resolver para ver a caixa de diálogo Resolver Conflitos.

  4. Veja as informações para o arquivo em sua máquina local e no arquivo do servidor para determinar qual versão do arquivo deseja manter e, em seguida, clique em Gravar esta Versão para manter a versão correta do arquivo.

  5. Quando resolver o conflito, clique em OK.


Conflitos de nomes de arquivos

Os conflitos de nomes de arquivo ocorrem quando os nomes da versão local do arquivo e da versão no servidor do arquivo diferem devido ao uso de iniciais maiúsculas e minúsculas. O conteúdo pode diferir ou pode ser o mesmo. Esses tipos de conflitos podem ser resolvidos renomeando-se os arquivos, mas tome cuidado com o local onde renomeia o arquivo.

  • Se o conteúdo for o mesmo, você poderá renomear um dos arquivos na caixa de diálogo Resolver Conflitos.
  • Se o conteúdo diferir, faça uma cópia do arquivo local e dê a ela um nome diferente e, em seguida, volte para a caixa de diálogo Resolver Conflitos e grave a versão no servidor do arquivo. Essa ação permite que ambas as versões do arquivo existam na pasta iFolder com nomes de arquivos diferentes. Se você renomear uma versão do arquivo dentro da caixa de diálogo Resolver Conflitos, essa versão do arquivo sobregravará a outra e o conteúdo do outro arquivo será destruído.

  1. Quando ocorre um conflito, uma mensagem pop-up aparece na área de Notificação. Clique no X para ignorar a mensagem.

  2. Na área de Notificação, clique o botão direito do mouse no ícone do iFolder e, em seguida, clique em Pastas iFolder para abrir o browser de pastas iFolder.

  3. Selecione o arquivo em conflito e, em seguida, clique em Resolver para ver a caixa de diálogo Resolver Conflitos.

  4. Confirme que é um conflito de nome de arquivo e determine se o conteúdo é o mesmo ou se é diferente.

  5. Execute uma das seguintes ações:

    • Se o conteúdo for o mesmo, renomeie um dos arquivos e, em seguida, clique em Gravar esta Versão e clique em OK para fechar a caixa de diálogo Resolver Conflitos. O procedimento está concluído.
    • Se o conteúdo for diferente, continue com a Etapa 6.

  6. Feche a caixa de diálogo Resolver Conflitos.

  7. Abra um gerenciador de arquivos e navegue até a cópia local do arquivo em conflito.

  8. Copie o arquivo e grave-o com um nome exclusivo na pasta iFolder.

    O iFolder sincroniza o arquivo como um arquivo novo para o servidor e para os computadores dos membros.

  9. Volte para a caixa de diálogo Resolver Conflitos, selecione a versão no servidor do arquivo, clique em Gravar esta Versão e, em seguida, clique em OK.

    É feito o download da versão no servidor do arquivo para a máquina local, sobregravando a cópia local do arquivo com aquele nome.
