Visualizando e modificando configurações da conta do iFolder

  1. Clique o botão direito do mouse no ícone do iFolder na área de Notificação e, em seguida, selecione Contas para abrir Preferências do iFolder na guia Contas.

  2. Selecione a conta que deseja visualizar ou na qual deseja mudar os seguintes parâmetros:

    Parâmetro Descrição

    Servidor

    O nome DNS do servidor empresarial. Por exemplo: svr21.example.com.

    Nome de usuário

    A identidade de usuário para esta conta. O seu administrador do iFolder decide se o formato que você deve usar é o seu ID de usuário na rede (nome comum LDAP) ou o seu endereço de e-mail.

    Por exemplo, se o seu nome for John Smith com o ID de usuário jsmith e o endereço de e-mail john.smith@example.com, o administrador configura uma definição para o servidor que determinará qual dos dois formatos deve ser aceito no campo Nome de Usuário para propósitos de autenticação.

    Você pode criar apenas uma conta por servidor empresarial.

    Senha

    A senha para o seu nome de usuário.

    Lembrar Senha

    Selecione Lembrar Senha para efetuar login automaticamente nesta conta do iFolder sempre que efetuar login no computador. Para obter informações, consulte Efetuando login em uma conta do iFolder.

    Se a sua senha de rede mudar, a autenticação automática falhará normalmente da próxima vez que o iFolder tentar efetuar login na conta do iFolder. O iFolder solicitará que você efetue login com a nova senha e permitirá que você especifique e grave a nova senha.

    Habilitar Conta

    Selecione Habilitar Conta para permitir a sincronização em segundo plano dos convites do iFolder, da lista de usuários e das pastas iFolder para esta conta. Esta opção administra somente funções no cliente.

    A coluna Status da lista Contas mostra o status da conta, isto é, Habilitado ou Desabilitado. Você pode ver rapidamente o status da configuração Conta Habilitada se tiver mais de uma configuração de conta.

    Quando você usa o iFolder com o Novell® iFolder® 3.x, o administrador do sistema iFolder pode desabilitar a sua conta do iFolder (temporariamente, em geral) no lado do servidor. Você pode esperar o seguinte comportamento:

    • Se estiver conectado quando a conta for desabilitada no lado do servidor, você poderá gerenciar suas pastas iFolder, e seus dados do iFolder continuarão a sincronização com o servidor até que você termine a sessão efetuando logout ou desabilitando a sincronização, ou até que o servidor desconecte a sua sessão, o que acontecer primeiro.
    • Se a conta estiver desabilitada no lado do servidor, sempre que tentar efetuar login, você receberá uma mensagem pop-up para informar que a conta está desabilitada. Entre em contato com o administrador do sistema iFolder para obter ajuda.

    Conta Padrão

    Selecione Conta Padrão para tornar esta conta a conta selecionada por padrão na lista suspensa de contas quando você criar pastas iFolder.

  3. Na guia Contas nas Preferências do iFolder, selecione a conta do iFolder e, em seguida, clique em Detalhes para ver as seguintes informações sobre a conta do iFolder.

    Parâmetro Descrição

    Servidor

    O nome de host do servidor empresarial. Por exemplo, se o nome DNS do servidor for svr21.example.com, svr1 será a informação relatada.

    Descrição

    A descrição amigável do servidor empresarial, servidor simples ou computador host de grupo de trabalho, de acordo com o que tiver sido digitado pelo administrador. Se não for digitada uma descrição, o valor será "nenhum".

    Disponível

    O valor total de espaço disponível no momento para as suas pastas iFolder nesta conta do iFolder. O valor é igual à quota menos o espaço utilizado.

    Utilizado

    O valor total do espaço consumido no momento por todas as pastas iFolder de sua propriedade nesta conta.

    Quota

    O valor total de espaço alocado para a sua conta do iFolder no servidor empresarial. Esse valor pode ser a quota padrão para todos os usuários ou uma quota específica do usuário. Se a quota não for definida, o valor será vazio ou N/A (não aplicável).
