Visão geral do iFolder

O iFolderTM é um aplicativo de compartilhamento de arquivos para clientes Linux, Windows* e Macintosh*. Você pode compartilhar arquivos em várias pastas iFolder, cada uma para um grupo diferente de usuários. Você controla quem pode fazer parte de um iFolder e o nível de acesso de cada membro. Além disso, você também pode fazer parte de pastas iFolder que terceiros compartilham com você.
