Revertendo um iFolder para uma pasta normal

Qualquer membro pode reverter um iFolder. Depois que ele é revertido, o relacionamento de compartilhamento colaborativo permanece para todos os membros do iFolder, mas o iFolder não é mais sincronizado com este computador. Esta ação não apaga o iFolder e seu conteúdo das estações de trabalho de membros.

  1. Em um gerenciador de arquivos ou na área de trabalho, localize o iFolder que deseja reverter para uma pasta normal.

  2. Use um dos seguintes métodos:

    • Clique o botão direito do mouse no iFolder e, em seguida, clique em iFolder > Reverter para uma Pasta Normal.
    • Selecione o iFolder e, em seguida, na barra de ferramentas do gerenciador de arquivos, clique em Arquivo > iFolder > Reverter para uma Pasta Normal.

    O ícone do iFolder muda para um ícone de pasta normal. Os dados permanecem intocados em todos os computadores de membros.
