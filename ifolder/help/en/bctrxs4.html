Coexistence of Novell iFolder 2.x and iFolder Clients

The iFolder client for Novell iFolder 3.7 or later versions and the Novell iFolder 2.x client can coexist on the same computer, but they are not integrated and their software should not be installed in the same folder. If you plan to use Novell iFolder 2.x on your computer, you must comply with the following restrictions:

  • Any local directory where you store your Novell iFolder 2.x data should not be converted to an iFolder.

  • Any directory that contains a local directory where you store your Novell iFolder 2.x data should not be converted to an iFolder.
