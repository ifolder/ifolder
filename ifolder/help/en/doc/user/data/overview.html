1.0 Overview of iFolder

iFolder™ is a file-sharing application for Linux, and Windows clients. You can share files in multiple iFolders, each with different users. You control who can participate in an iFolder and the access level for each member. You can also participate in iFolders that others share with you.

iFolder 3.6 also provides you a simple, hassle-free user interface for both client-side migration from iFolder 2.x to iFolder 3.6 and upgrade from iFolder 3.x to iFolder 3.6.
