Enterprise Server Sharing

The iFolder client supports sharing in Enterprise Server mode. Files are synchronized across multiple computers through a central server. It provides the following capabilities:

  • Allows you to share files across computers
  • Allows you to share files with others
  • Allows you to own multiple iFolders
  • Allows you to participate in multiple iFolders that other users share with you
  • Allows you to participate as owners or members of iFolders in multiple accounts
  • Synchronizes files at any time through a central server with improved availability, reliability, and performance over other sharing methods
  • Autoprovisions users from the enterprise LDAP containers, groups, and users and creates a list of users with whom you can share iFolders
  • Authenticates users with enterprise LDAP directory services
  • Backs up local files to a server
  • Allows administrators to back up iFolders to backup media
  • Allows administrators to restore deleted files from backup media
  • Allows all iFolders in the enterprise to be managed centrally with policy-based administration tools
