2.9 Exiting the iFolder Client

  1. Right-click the iFolder Services icon in the Notification area, then select Quit.

Files are synchronized with your iFolder server account only when you are connected to the iFolder server. You can stop synchronization by logging out of an account. iFolder continues to run in the background in offline mode. For information, see Section 2.10, Logging Out of an iFolder Account.

Note that you can have multiple account, each created on different host server (domain).
