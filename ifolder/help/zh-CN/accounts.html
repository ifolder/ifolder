配置 iFolder 帐户

使用 iFolder 帐户创建向导添加并配置新帐户。您必须至少配置一个 iFolder 帐户才能创建 iFolder 或在系统上共享 iFolder。在任何给定时间均可使用一个帐户连接到 iFolder 服务器。但如果要连接到多个 iFolder 服务器，则可以配置多个帐户。您可以分别登录到每个帐户，也可以同时登录多个帐户。

持有不同本地登录身份的多个用户可以在同一台计算机上拥有 iFolder。在您第一次使用当前本地登录身份启动 iFolder 时，iFolder 会提示您设置一个 iFolder 帐户。

重要说明：iFolder 3.7 和更高版本增加了一种称为自动创建帐户的新功能，可帮助 iFolder 管理员为您创建和配置帐户。如果您的 iFolder 管理员已启用了自动帐户创建功能，则不必为您自己配置帐户。当您首次启动 iFolder 时，将出现提示要求您输入口令。输入您的口令并完成帐户配置。

配置新帐户：

  1. 使用下列方法之一，访问 iFolder 帐户创建向导：

    • 在您第一次使用当前本地登录身份启动 iFolder 时，iFolder 会提示您设置一个 iFolder 帐户。单击“向前”直接转到“iFolder 帐户创建向导”。

    • 右击通知区域中的 iFolder 服务 图标，选择“帐户”以打开“iFolder 首选设置”对话框，进入“帐户”选项卡，然后单击“添加”以打开“iFolder 帐户创建向导”。

  2. 遵循屏幕指导指定以下值：

    参数

    描述

    服务器地址

    您帐户所在的 iFolder Enterprise Server 的 DNS 名称或 IP 地址。从 iFolder 管理员处获取此信息。

    例如： svr21.example.com 或 192.168.1.1

    服务器名始终随同其 IP 地址一起显示在 iFolder 用户界面上。例如：iFolder - 192.168.1.1

    重要说明：您不能使用 iFolder 3.7 或更高版本客户程序连接到 iFolder 3.2 服务器。

    将此服务器设置为我的默认服务器

    如果帐户连接有多个服务器，选中“将此服务器设置为我的默认服务器”复选框，将服务器下拉列表中选定的服务器设置为创建或上载 iFolder 时的默认服务器。

    用户名

    指定此帐户对应的用户身份。iFolder 管理员决定您应使用的格式是网络用户 ID（LDAP 常用名）还是电子邮件地址。

    例如，如果您的姓名是 John Smith，用户 ID 是 jsmith，电子邮件地址是 john.smith@example.com，则管理员将为服务器配置一个设置以确定“用户名”字段接受两种格式中的哪一种来进行鉴定。

    口令

    指定用户名的口令。

    通过安全通道 (SSL) 进行鉴定。系统提示您是否验证并接受服务器证书。

    记住口令

    选中此计算机上的“记住口令”复选框，可以在登录到计算机时自动登录到此 iFolder 帐户。

    如果更改了网络口令，则在下一次 iFolder 尝试登录到 iFolder 帐户时，自动鉴定将失败。iFolder 将提示您用新口令登录并且允许您指定和保存新口令。

  3. 校验您输入的信息。要进行任何更改，单击“返回”按钮，重新访问之前的页面。单击“连接” 以继续并验证与服务器的连接。

  4. 如果提示您“接受此服务器的证书”，则单击“细节”查看证书信息，然后单击“确定”接受该证书信息。

    注：创建帐户期间，用户通过 SSL 进行登录。当系统提示您接受服务器证书时，必须验证并接受该证书，才能成功连接。

  5. 如果管理员已启用加密策略，iFolder 会提示您输入通行口令。

    在“通行口令”字段中输入通行口令。

    在“重新键入通行口令”字段中重新输入通行口令。

    选中“记住通行口令”复选框以在当前会话过程中自动访问加密的 iFolder 数据。如果选择此选项，iFolder 可在下次登录过程中记住您的通行口令。此通行口令交换会自动进行。

  6. 从列表中选择所需的恢复代理，然后单击“确定”。

  7. 指定用于创建默认 iFolder 的位置。

    默认情况下，默认 iFolder 存储在<用户主目录>/ifolder/服务器名/用户名。

  8. 选择您希望创建的默认 iFolder 的类型。如果选择“加密”，则系统会提示您提供通行口令。

    重要说明：您只能共享常规 iFolder。无法共享加密的 iFolder。

    如果 iFolder 管理员尚未对您的当前帐户启用加密，您只能创建常规 iFolder，而如果启用加密，则默认情况下，“常规”选项处于选中状态。

  9. 选择“安全同步”可对所创建的 iFolder 中的文件启用安全同步。

    您可以同时对常规 iFolder 和加密 iFolder 启用安全通道。在选中此选项时，iFolder 允许使用安全通道在服务器上的 iFolder 与本地计算机上的 iFolder 间进行同步。

    如果您去除服务器上的默认 iFolder，则下次从任何富客户端配置该帐户时，iFolder 将会提示您再次创建一个默认 iFolder。

    但如果从本地计算机去除默认的 iFolder，则下次配置帐户时 iFolder 会提示您下载默认的 iFolder。

  10. 如果已设置了一个默认 iFolder，那么您可以将其下载到本地计算机。

    下载默认 iFolder，单击“下一步”继续。

  11. 如果系统中已存在与默认 iFolder 同名的文件夹，则您会收到合并两个文件夹或将默认 iFolder 下载至其他位置的提示。

    单击“确定”合并两个文件夹或单击“取消”将默认 iFolder 下载至其他位置

  12. 单击“完成”，关闭 iFolder 帐户创建向导。

    iFolder 同步指定帐户的用户列表，并且标识可用于下载的 iFolder。初次下载可能需要花费一些时间，这取决于帐户的用户列表的大小。

  13. 校验您设置的帐户是否出现在帐户列表中。

    1. 在通知区域中右击“iFolder 服务” 图标，然后从菜单中选择“首选设置”以打开“iFolder 首选设置”对话框并进入“帐户”选项卡。

      当连接到 iFolder 服务器时，将选中“联机”复选框。

    2. 关闭“iFolder 首选设置”对话框

  14. 使用以下两种方法之一为帐户设置 iFolder：

    • 右击通知区域中的 iFolder 服务图标，然后选择“iFolder”以打开 iFolder 浏览器。继续创建和上载 iFolder。

    • 如果此帐户的 iFolder 可从服务器下载，则将在 <iFolder 服务器名> 上 iFolder 窗口的“iFolder”下列出来。下载图标 指示 iFolder 可供下载。继续下载可用的 iFolder。

有关更多信息，请参见 法律声明.
