iFolder 的优点

iFolder 具有以下优点：

  • iFolder 与您的本机桌面环境相集成，从而更易于创建和管理多个 iFolder。

  • iFolder 高度灵活且可伸缩，对于每个用户的 iFolder 数或每个 iFolder 的成员数都没有任何实际限制。

  • 启用加密功能后，iFolder 将在数据跨网移动和储存在 iFolder 服务器上时保护数据的安全。

  • 使用增强的万维网访问控制台，您可从未安装 iFolder 客户程序的任何工作站访问您存放在 iFolder 服务器上的文件，并可使用万维网浏览器执行 iFolder 客户程序的所有操作。

  • 您可轻松选择共享个人文件或业务文件。

  • iFolder 可将您的 iFolder 数据迁移和升级到最新版本。

  • iFolder 支持 LDAP 组。您可与 LDAP 组共享 iFolder。LDAP 组的成员可以创建和拥有 iFolder 并可与其他单个用户或其他 LDAP 组共享 iFolder。

  • 您可控制您拥有的 iFolder 的成员用户的访问级别，或者，如果您有必需的权限，还可控制成员的访问权限。

  • iFolder 透明地将您的文件更新到多个工作站上的成员 iFolder。

  • iFolder 运行时，可跟踪和记录您脱机工作时所作的更改，并在连接到服务器时同步这些更改.

  • iFolder 为访问 iFolder 以同步其数据的成员进行安全鉴定。

  • iFolder 提供了一种通过电子邮件交换文件的替代方法，从而避免了电子邮件附件的文件大小限制并降低了电子邮件储存要求。

有关更多信息，请参见 法律声明.
