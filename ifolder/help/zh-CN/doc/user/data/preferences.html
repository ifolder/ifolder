为客户程序配置 iFolder 自选设置

  1. 在“通知”区域中右击 iFolder 应用程序图标 ，然后选择“自选设置”打开“iFolder 自选设置”对话框的“一般”选项卡。


  2. 指定以下值：

    参数 说明

    当登录桌面时启动 iFolder

    如果选择此选项，那么每当您使用设置了 iFolder 帐户的本地用户名登录时，就会启动 iFolder。对于那些启用了“记住口令”的帐户，系统将在后台按指定的同步间隔进行同步。

    若要手工启动 iFolder，请取消选择此选项。

    创建 iFolder 时显示“确认”对话框

    如果选择此选项，那么每当您创建 iFolder 时，除了收到常规的可视提示（如出现在 iFolder 列表中的 iFolder）外，还会收到一个确认讯息。

    若要停止接收确认讯息，请取消选择此选项。

    共享 iFolder 通知

    如果选择此选项，那么每当您收到共享 iFolder 的新邀请时，都会在“通知”区域显示弹出讯息。

    系统将根据您在“iFolder 自选设置”中指定的“同步间隔”对邀请进行检索。该间隔控制文件、iFolder 共享邀请和用户列表的同步。您还可以单击“刷新”在需要时更新邀请。

    若要禁用为邀请显示弹出讯息的行为，请取消选择此选项。

    冲突通知

    如果选择此选项，那么同步文件时，每当发生冲突，都会在“通知”区域显示弹出讯息。

    若要禁用为同步冲突显示弹出讯息的行为，请取消选择此选项。

    用户加入时通知

    如果选择此选项，那么每当用户接受共享 iFolder 的邀请时，都会在“通知”区域显示弹出讯息。如果用户在他/她自己的计算机上实际地对 iFolder 进行设置，即表示该用户接受了邀请。

    若要禁用为用户确认共享显示弹出讯息的行为，请取消选择此选项。

    同步

    自动同步 iFolder：如果选择此复选框，那么将为此计算机上当前本地登录 ID 之下的所有 iFolder 启用同步。若要禁用同步，则取消选择此复选框。

    同步 iFolder 间隔：指定此计算机上当前本地登录 ID 下用于同步 iFolder 的最短时间间隔。请指定值和单位。

    如果将 iFolder 客户程序与 Novell iFolder 3.x 配合使用，则有效的最短同步间隔始终是以下设置中的最大值：

    • 如果没有设置用户策略，则为系统策略（默认值为零）。如果设置了用户策略，则无论用户策略的值是大于还是小于系统策略，用户策略都将覆盖系统策略。
    • 本地计算机策略，或与服务器同步的客户机上的设置。
    • iFolder（集合）策略。

  3. 完成后，单击“应用”使更改生效。

  4. 单击“确定”关闭“iFolder 自选设置”对话框。
