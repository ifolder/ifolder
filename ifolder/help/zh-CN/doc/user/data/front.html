iFolder 帮助

iFolderTM 是一款用于 Linux*、Windows* 和 Macintosh* 客户程序的、开放源代码的文件共享应用程序。使用 iFolder，您的本地文件将跨多台计算机自动跟随您到任何地方，无论您联机还是脱机都时刻形影不离。您共享多个 iFolder 中的文件，每个 iFolder 具有一组不同的用户。由您控制哪些用户可以参与 iFolder，以及他们对 iFolder 文件的访问权限。您还可以参与其他用户共享给您的 iFolder。

本帮助系统介绍如何使用和管理 iFolder 客户程序。本帮助系统面向用户，分为以下几个部分：

  • iFolder 概述
  • 管理 iFolder 帐户和自选设置
  • 管理 iFolder


文档约定

在 Novell 文档中，大于号 (>) 用于分隔一个步骤中的不同操作，以及一个交叉引用路径中的不同项。

商标符号（®、TM 等）表示 Novell 商标。星号 (*) 表示第三方商标。

如果某个路径名的书写在某些平台上必须使用反斜线，而在另一些平台上必须使用正斜线，则本文档使用反斜线表示该路径名。在需要使用正斜线的平台（如 Linux 或 UNIX*）上，用户应根据软件的要求改用正斜线。
