管理 iFolder

iFolderTM 是一个本地目录，用于与用户指定的成员有选择地共享和同步文件。所有成员都可以访问 iFolder 文件，具有适当权限的成员还可以修改和读取它。

本节讨论以下 iFolder 任务：

  • iFolder 定位准则
  • 不适合同步的文件类型和大小
  • iFolder 及其文件夹和文件的命名约定
  • 创建 iFolder
  • 共享 iFolder
  • 设置可用 iFolder
  • 查看和配置 iFolder 的属性
  • 同步文件
  • 解决文件冲突
  • 将 iFolder 恢复为常规文件夹
  • 移动 iFolder
  • 删除 iFolder
