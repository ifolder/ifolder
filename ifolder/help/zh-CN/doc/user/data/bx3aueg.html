iFolder 共享

通常，当您在多个地点办公或与他人合作时，必须小心地管理文件版本。使用 iFolder，您可以在安装了 iFolder 客户程序并设置了共享 iFolder 的任何计算机上获取文件的最新版本。iFolder 还允许您与其他授权用户共享多个 iFolder 及这些 iFolder 中的独立内容。您可以控制每个 iFolder 的参与者，以及每个成员的访问级别。同样，您可以参与合作环境中其他用户拥有的共享 iFolder。

iFolder 客户程序通过中央计算机同步多台计算机上的文件，从而实现对共享的支持。它允许您执行以下操作：

  • 在多台计算机上共享文件
  • 与其他用户共享文件
  • 拥有多个 iFolder
  • 参与其他用户共享给您的多个 iFolder
  • 通过多个帐户以 iFolder 拥有者或成员的身份参与
