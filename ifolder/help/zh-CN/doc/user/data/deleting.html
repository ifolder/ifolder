删除 iFolder

如果从您的集合中删除某个 iFolder，该 iFolder 的本地副本及其内容都将被删除。此操作还可能影响该 iFolder 的共享关系，具体情况取决于您对该 iFolder 的访问权限。下表概述了根据删除 iFolder 的用户的访问权限，删除 iFolder 对共享关系和成员 iFolder 会有怎样的影响。

删除某个 iFolder 不会从您的工作站卸载 iFolder 客户程序。

  1. 在文件管理器或 iFolder 浏览器中，找到要删除的 iFolder。

  2. 右击该 iFolder，然后单击“删除”。
