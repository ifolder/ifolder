移动 iFolder

创建 iFolder 之后，iFolder 的路径是静态的。如果将 iFolder 从其原位置移动到本地计算机上的另一个位置，则它将不再是 iFolder。移动 iFolder 将破坏同步数据库和已转换为 iFolder 的文件夹之间的链接。

要在创建 iFolder 的计算机上移动 iFolder，请执行以下操作：

  1. 将 iFolder 转换为常规文件夹。

  2. 移动该文件夹。

  3. 将该文件夹转换为 iFolder。

  4. （可选）共享该 iFolder。

要在参与的计算机上移动 iFolder，请执行以下操作：

  1. 在 iFolder 中，将 iFolder 恢复为停止参与 iFolder，但仍然可供您使用。

  2. 选择可用的 iFolder，然后单击“设置”。

  3. 指定用于存放 iFolder 的新的本地路径。

  4. 允许该 iFolder 与 Enterprise Server 同步文件。
