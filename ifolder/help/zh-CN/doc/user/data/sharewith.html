共享 iFolder

您可以与多个用户共享 iFolder。本节介绍以下任务：

  • 了解用户访问权限
  • 将用户添加到 iFolder
  • 修改用户访问权限
  • 从 iFolder 中去除用户

将用户添加到 iFolder


使用文件管理器添加用户

  1. 在文件管理器中，定位到 iFolder，然后选择它。

  2. 右击 iFolder，然后单击“iFolder”>“属性”。

  3. 单击“共享对象”。

  4. 从用户列表中选择一个或多个用户。

  5. 选择一种访问权限：“完全控制”、“读/写”或“只读”。

  6. 单击“应用”。


使用 iFolder 浏览器添加用户

  1. 在“通知”区域中，右击 iFolder 图标 ，然后单击“iFolder”以打开 iFolder 浏览器。

  2. 在 iFolder 列表中，选择 iFolder，然后单击“共享” 以打开“iFolder 属性”窗口的“共享”选项卡。

  3. 单击“添加”打开“选择用户”对话框。

  4. 在用户列表中，通过将一个或多个用户添加到“所选用户”列表来选择用户，然后单击“确定”。

    将邀请同步到服务器要花费几秒钟时间。

  5. 选择一个或多个用户，单击“访问权限”，选择“完全控制”、“读/写”或“只读”访问权限，然后单击“确定”。

  6. 重复第 4 和第 5 步，直到您邀请了所有需要共享该 iFolder 的用户。

  7. 单击“应用”，然后单击“确定”，关闭“iFolder 属性”窗口。


修改用户访问权限

iFolder 成员的工作关系和状态会不断更改，因此您可能需要调整用户对 iFolder 的访问权限。例如，可能要为可信用户授予对 iFolder 的完全控制权限。


使用文件管理器设置用户权限

  1. 在文件管理器中，定位到 iFolder，然后选择它。

  2. 右击 iFolder，然后单击“iFolder”>“属性”。

  3. 单击“共享对象”。

  4. 从用户列表中选择一个或多个用户。

  5. 选择“访问”，然后指定“完全控制”、“读/写”或“只读”权限。

  6. 单击“应用”，然后单击“确定”，关闭“iFolder 属性”窗口。


使用 iFolder 浏览器设置用户权限

  1. 在“通知”区域中，右击 iFolder 图标 ，然后单击“iFolder”以打开 iFolder 浏览器。

  2. 在 iFolder 列表中，选择 iFolder，然后单击“共享对象”。

  3. 从用户列表中选择一个或多个用户。

  4. 选择“访问”，然后指定“完全控制”、“读/写”或“只读”权限。

  5. 单击“应用”，然后单击“确定”，关闭“iFolder 属性”窗口。


从 iFolder 中去除用户

要取消 iFolder 的共享，必须去除充当 iFolder 成员的用户。用户工作站上的 iFolder 将成为常规文件夹。数据的副本仍然留在以前成员的工作站上，但这些文件不再与共享 iFolder 同步。


使用文件管理器去除用户

  1. 在文件管理器中，定位到 iFolder，然后选择它。

  2. 右击 iFolder，然后单击“iFolder”>“属性”。

  3. 单击“共享对象”。

  4. 从用户列表中选择一个或多个用户。

  5. 单击“去除”，再单击“应用”，然后单击“确定”。


使用 iFolder 浏览器去除用户

  1. 在“通知”区域中，右击 iFolder 图标 ，然后单击“iFolder”以打开 iFolder 浏览器。

  2. 在 iFolder 列表中，选择 iFolder，然后单击“共享对象”。

  3. 从用户列表中选择一个或多个用户。

  4. 单击“去除”，再单击“应用”，然后单击“确定”。
