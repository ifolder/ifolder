登录 iFolder 帐户

在设置 iFolder 帐户后，只要想同步本地 iFolder 数据、iFolder 用户列表和您的共享 iFolder 通知，就必须进行登录。您可以同时登录到多个帐户。登录每个帐户时，可使用以下方法之一：

  • 启用“记住口令”，这样您登录桌面或启动 iFolder 客户程序时，系统会自动登录 iFolder 帐户。

    如果您的网络口令发生更改，则在下一次 iFolder 尝试登录 iFolder 帐户时，自动鉴定以正常方式失败。iFolder 将提示您用新口令登录，并且允许您指定和保存新的口令。

  • 禁用“记住口令”，仅当要在 iFolder 中同步文件或管理 iFolder 时，才登录 iFolder。

要对现有 iFolder 帐户启用或禁用“记住口令”，请执行以下操作：

  1. 在“通知”区域中右击 iFolder 图标 ，然后选择“帐户”。

  2. 选择 iFolder 帐户以查看其帐户设置。

  3. 选择（启用）或取消选择（禁用）“记住口令”。

  4. 指定口令，然后单击“确定”或“应用”。

  5. 单击“确定”确认对“帐户”设置的更改，或单击“取消”不进行任何更改。

要登录 iFolder 帐户，请执行以下操作：

  1. 在“通知”区域中右击 iFolder 图标 ，然后选择“帐户”。

  2. 选择 iFolder 帐户以查看其帐户设置。

  3. 指定口令，然后单击“登录”。

    如果登录成功，则帐户的状态将变为“已登录”。

    如果登录失败，则出现错误讯息报告原因（如口令失败）。

  4. 单击“确定”关闭“帐户”对话框。
