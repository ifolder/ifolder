删除 iFolder 帐户

您可以指定是仅从当前计算机去除您拥有的 iFolder，还是将其从服务器一并去除，从而在本质上停止为该帐户设置的共享 iFolder。

如果从您的计算机去除 iFolder 帐户，则 iFolder 的相关本地副本将恢复为常规的文件夹。您可以继续通过其它计算机或万维网访问这些 iFolder。

如果从服务器去除 iFolder 帐户：

  • 您拥有的所有 iFolder 都将取消共享。
  • 成员计算机上 iFolder 的本地副本将恢复为常规的文件夹。
  • 从服务器上去除 iFolder 及其内容。
  • 从其他用户共享给您的 iFolder 中去除您的成员资格。

  1. 在“通知”区域中右击 iFolder 应用程序图标 ，然后选择“帐户”以打开“iFolder 自选设置”对话框的“帐户”选项卡。

  2. 选择要删除的 iFolder 帐户，然后单击“删除”。

  3. 出现一个讯息，提示您指定删除操作的范围。执行以下操作之一：

    • 仅从当前计算机去除 iFolder 帐户：取消选择“从服务器去除我的 iFolder 和文件”，然后单击“是”。iFolder 将只从本地工作站去除所选的帐户。此帐户的本地 iFolder 将恢复为常规的文件夹。如果不需要数据的本地副本，请使用文件管理器定位并删除这些副本。

      您仍然可以通过装有 iFolder 客户程序的其它计算机访问此帐户的 iFolder。如果将 iFolder 客户程序与 Novell® iFolder® 3.x Enterprise Server 配合使用，则可以在任何位置使用 Novell iFolder 3.x 万维网访问来访问这些文件。

    • 从服务器去除 iFolder 帐户以及您的 iFolder 和文件：选择“从服务器去除我的 iFolder 和文件”，然后单击“是”。iFolder 将从本地工作站去除所选的帐户。它还将从服务器去除您在此帐户中拥有的所有 iFolder，并在当前计算机和所有成员计算机上将这些 iFolder 恢复为常规文件夹。最后，此操作将去除您在此帐户中对共享 iFolder 拥有的成员资格。所有本地 iFolder 都将恢复为常规文件夹。如果不需要数据的本地副本，则使用文件管理器定位并删除这些副本。

      如果您希望在此计算机或另一台计算机上再次使用此帐户，则必须重新设置此帐户及其 iFolder，就像对待新帐户一样。

    • 取消删除：单击“否”退出删除操作。您的帐户及其 iFolder 将保持不变。
