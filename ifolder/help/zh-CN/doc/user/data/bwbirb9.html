为 iFolder 交通配置本地病毒扫描程序设置

如果您的计算机上使用病毒扫描软件，请从病毒扫描中排除 ..\simias\WorkArea\ 目录。例如，..\simias\WorkArea\ 目录在 Windows 上的位置是 c:\documents and settings\username\local settings\application data\simias\WorkArea\.simias.wf.guid。

iFolder 在 ..\simias\WorkArea\ 目录中缓存从服务器下载的文件。如果某个 iFolder 文件携带病毒，您的病毒扫描软件扫描到该病毒，而该文件在 ..\simias\WorkArea 目录中，则病毒扫描程序弹出讯息提示找到病毒，将该文件移动到隔离区，然后 iFolder 再次尝试同步文件。

如果不扫描 WorkArea，同步将顺利完成；这样受感染的文件会成功下载并将其移动到本地 iFolder。随后，病毒扫描程序在 iFolder 中检测到受感染的文件并将其从该文件夹移到隔离区中，以保护计算机不受感染。iFolder 检测到文件的删除，下次开始同步时，iFolder 将从服务器和成员计算机中去除该文件。

但是，如果用于存放下载文件的 iFolder 是只读的，则不会强制对服务器和其它成员计算机执行与本地类似的删除。必须在有同步更改的权限的其它位置检测病毒。用户的病毒扫描程序可能会发出警报，提示用户存在病毒，这样该用户才能向其他 iFolder 成员发出病毒警报。
