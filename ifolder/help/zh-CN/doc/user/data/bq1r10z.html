iFolder 的优点

iFolder 具有以下优点：

  • 与您的本机桌面环境集成，使您得以轻松创建和管理多个 iFolder
  • 具有高度的伸缩性和灵活性，对每个用户拥有的 iFolder 数或每个 iFolder 拥有的成员数没有实际限制
  • 使您得以方便、有选择地共享个人和业务文件
  • 对于您所拥有的 iFolder 或您有权控制其成员访问权限的 iFolder，您可以控制成员用户的访问级别
  • 将您的文件以透明的方式更新到多个工作站上的成员 iFolder
  • 跟踪并记录您在脱机工作时进行的更改，并且在您联机后同步这些更改
  • 提供对成员 iFolder 的安全鉴定以便同步数据
  • 为通过电子邮件交换文件提供了另一种选择，采用这种方法，您不必限制电子邮件附件的大小，还可以降低电子邮件储存所需要的空间
