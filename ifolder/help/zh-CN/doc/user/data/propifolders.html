查看和配置 iFolder 的属性

使用“iFolder 属性”页查看和配置有关选定 iFolder 的以下信息。


了解 iFolder 属性

以下参数报告服务器上占用的磁盘空间：

参数 说明

可用空间

选定 iFolder 中的文件或目录当前可以使用的服务器磁盘空间总量。该值等于空间定额减去已用空间。

已用空间

选定 iFolder 中所有文件和目录当前占用的服务器磁盘空间总量。

以下参数之一：

  • 设置定额
  • 定额

  • 设置定额：如果您是 iFolder 拥有者，则可以为 iFolder 指定一个定额。

  • 定额：如果您是成员，该值报告 iFolder 拥有者分配给选定 iFolder 的服务器磁盘空间的总量。如果未设置定额，则此值为空或 N/A（不适用）。

以下参数报告 iFolder 本地副本的同步状态：

参数 说明

上次成功同步

上次成功与 iFolder 服务器同步 iFolder 的本地副本的完成时间。

要同步的文件/文件夹

iFolder 中需要进行同步的储存项（文件和文件夹）总数。

自动同步间隔

iFolder 的同步间隔。指定以分钟、小时或天为单位的值。

如果将 iFolder 客户程序与 Novell iFolder 3.x Enterprise Server 配合使用，该间隔不能覆盖 iFolder 管理员为服务器或您在服务器上的帐户指定的最短时间间隔，但可以覆盖所有客户程序的同步间隔。

立即同步

在需要时单击该按钮可启动同步。


从文件管理器访问 iFolder 属性

  1. 在文件管理器中，定位到 iFolder，然后选择它。

  2. 右击 iFolder，然后单击“iFolder”>“属性”。


从 iFolder 浏览器访问 iFolder 属性

  1. 在“通知”区域中，右击 iFolder 应用程序图标 ，然后单击“iFolder”以打开 iFolder 浏览器。

  2. 在 iFolder 列表中，选择 iFolder，然后单击“属性”。
