设置可用 iFolder

如果其他用户与您共享 iFolder，那么在您的 iFolder 浏览器中，这些 iFolder 旁边会出现“可用 iFolder”图标。要参与当前本地计算机上的共享 iFolder，请设置 iFolder，使其在计算机上处于活动状态。

本地 iFolder 的位置必须符合以下准则：

  • iFolder 定位准则
  • iFolder 及其文件夹和文件的命名约定

要设置 iFolder，请执行以下操作：

  1. 在“通知”区域中，右击 iFolder 图标 ，然后单击“iFolder”以打开 iFolder 浏览器。

  2. 在 iFolder 列表中，选择可用 iFolder，然后单击“设置”。

  3. 如果尚未登录到帐户，请以您的用户名和口令登录，然后单击“确定”。

  4. 浏览到要创建本地 iFolder 的位置。

  5. 单击“确定”两次。
