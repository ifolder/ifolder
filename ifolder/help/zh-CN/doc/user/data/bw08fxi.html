启动 iFolder 客户程序


启动 iFolder for Novell Linux Desktop 9

  1. 使用创建 iFolder 时要使用的用户 ID 登录计算机。

  2. 使用以下方法之一启动 iFolder：

    • 打开终端壳层，然后输入

      /opt/novell/ifolder3/bin/ifolder

    • 如果希望 iFolder 在您登录时自动启动，则在您注销 GNOME 会话时确保 iFolder 处于运行状态，然后保存该会话。您还可以打开终端窗口，以根用户身份登录，然后运行 /opt/gnome/bin/gnome-session-properties，将 iFolder 添加到登录计算机时启动的程序之列。

  3. 如果已对 iFolder 帐户禁用了“记住口令”，则必须通过登录到该帐户来同步“用户”列表、同步 iFolder 并管理 iFolder。

    有关信息，请参见登录 iFolder 帐户。


启动 iFolder for Windows 2000/XP

  1. 使用创建 iFolder 时要使用的 Windows 用户 ID 登录计算机。

  2. 如果 iFolder 在成功登录时没有自动启动，则依次单击“开始”>“程序”>“iFolder”，然后选择 iFolder 应用程序，启动 iFolder。

    如果您在“iFolder 自选设置”中禁用“当登录到桌面时启动 iFolder”设置，则 iFolder 不会自动启动。

  3. 如果已对 iFolder 帐户禁用了“记住口令”，则必须通过登录到帐户来同步“用户”列表、同步 iFolder 并管理 iFolder。

    有关信息，请参见登录 iFolder 帐户。


启动 iFolder for Macintosh OS X 10.3 版和更高版本

  1. 使用创建 iFolder 时要使用的用户 ID 登录计算机。

  2. 使用以下方法之一启动 iFolder：

    • 打开“应用程序”菜单，选择“Internet”子菜单，然后选择“iFolder”。
    • 要从命令行启动 iFolder，请打开终端壳层，然后输入

      /opt/novell/ifolder3/bin/ifolder

    • 如果希望 iFolder 在您登录 Macintosh 计算机时自动启动，则将 iFolder 添加到“启动项”中。

  3. 如果已对 iFolder 帐户禁用了“记住口令”，则必须通过登录到帐户来同步“用户”列表、同步 iFolder 并管理 iFolder。

    有关信息，请参见登录 iFolder 帐户。
