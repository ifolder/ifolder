删除 iFolder 帐户

您可以指定是仅从当前计算机中去除帐户，还是同时从服务器中去除您的 iFolder。

如果仅从计算机中去除 iFolder 帐户，则：

  • 此帐户的本地 iFolder 将恢复为常规文件夹。如果不需要数据的本地拷贝，则使用文件管理器查找并删除它们。

  • 您可以继续从具有 iFolder 客户程序的其他计算机中访问 iFolder。

  • 将 iFolder 客户程序与 Novell iFolder 3.7 或更高版本 Enterprise Server 一起使用时，您还可从支持 Novell iFolder 3.7 或更高版本万维网访问功能的任何地方访问文件。

如果从服务器中去除 iFolder 帐户，则：

  • 对您拥有的所有 iFolder 取消共享。

  • 成员计算机上的 iFolder 本地拷贝将恢复为常规文件夹。如果不需要数据的本地拷贝，则使用文件管理器查找并删除它们。

  • 将从服务器中去除 iFolder 及其内容。

  • 从其他用户共享给您的 iFolder 中去除您的成员资格。

  • 将从本地计算机中去除此帐户。

如果您决定在此计算机或另一台计算机上再次使用此帐户，则必须像对待新帐户那样重新设置该帐户及其 iFolder。

删除 iFolder 帐户：

  1. 在通知区域中右击“iFolder 服务” 图标，然后选择“帐户设置”以打开“iFolder 首选设置”对话框并进入“帐户”选项卡。

  2. 选择要删除的 iFolder 帐户，然后单击“去除”。

  3. 出现讯息提示您确定去除操作的作用范围。

    执行以下操作之一：

    • 仅从当前计算机去除 iFolder 帐户： 确保取消选中从服务器中去除我的 iFolder 和文件，然后单击是。

    • 从服务器去除 iFolder 帐户以及您的 iFolder 和文件： 选中从服务器中去除我的 iFolder 和文件，然后单击是。

    • 取消删除： 单击否可以取消删除操作。您的帐户及其 iFolder 保持不变。

有关更多信息，请参见 法律声明.
