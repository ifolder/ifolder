iFolder 浏览器

您可以使用 iFolder 浏览器来管理 iFolder。下面给出的表格总结了 iFolder 浏览器的元素：

表 2 iFolder 浏览器元素

元素名称

说明

创建 iFolder

用于将本地计算机的文件夹转换为 iFolder，并将其上载至所连接的 iFolder 服务器。

有关详细信息，请参见创建和上载 iFolder

立即同步

用于立即启动本地计算机和服务器间的 iFolder 同步。

共享

用于和其他用户共享您的 iFolder。

有关详细信息，请参见共享 iFolder

恢复为常规文件夹

用于将 iFolder 恢复为常规文件夹。还可从服务器删除 iFolder。有关详细信息，请参见将 iFolder 恢复为常规文件夹

服务器

显示所连接 iFolder 服务器的 IP 地址。

名称

显示选定 iFolder 的名称。

拥有者

显示选定 iFolder 的拥有者名字。

权限

显示 iFolder 拥有者所拥有的权限。

等待同步的文件/文件夹

指出等待同步的文件或文件夹的数量。

上次成功同步时间

指出上次成功同步的时间和日期

下载 iFolder

用于从 iFolder 服务器将 iFolder 下载到本地计算机。

有关详细信息，请参见下载可用的 iFolder

iFolder 类型

指出选定的 iFolder 是经过加密的还是常规的。

iFolder 大小

显示选定 iFolder 的大小。

合并

合并功能可帮助您合并 iFolder 数据并将 iFolder 服务器和客户端之间的数据传输量降至最低。您可将服务器上 iFolder 中的内容与本地计算机上现有文件夹的内容合并在一起。有关详细信息，请参见合并 iFolder

从服务器中删除

用于从服务器中删除选定的 iFolder。

帐户列表

显示帐户配置的服务器列表。

连接/断开连接

用于连接或断开与 iFolder 服务器的连接。如果已与 iFolder 服务器连接，则会显示“断开连接”按钮，如果未连接，则显示“连接”按钮。

解析

用于解决文件冲突。要解决文件版本的冲突，您必须在“冲突解决方案”对话框中选择正确的版本。有关详细信息，请参见解决文件冲突

注：对于 Macintosh 客户机，要解决名称冲突，必须手动更新文件名或文件夹名称，以确保没有包含任何不受支持的字符 ( \ / : * ? " < > | ; )。

用户

显示登录到选定 iFolder 服务器用户的名称。

iFolder 数量

显示选定 iFolder 服务器上与本地计算机上的 iFolder 总数。

磁盘定额

显示“已用空间”和“可用空间”字段的值。

已用空间

显示所选 iFolder 服务器上的所有文件和目录当前所占用的服务器磁盘空间的总量。

可用空间

显示选定的 iFolder 服务器上，当前已由其上所有文件和目录可用的服务器磁盘空间。

自动同步

显示两个连续自动同步周期间的时间间隔

有关更多信息，请参见 法律声明.
