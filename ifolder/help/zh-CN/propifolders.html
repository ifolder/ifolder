使用“iFolders 属性”页查看和配置有关所选 iFolder 的以下信息。

  • 通过文件管理器访问 iFolder 属性

了解 iFolder 属性

以下参数报告服务器上磁盘空间的使用情况：

表 4 储存在 iFolder 服务器上的 iFolder 的储存参数

参数

说明

可用

当前可用于所选 iFolder 中的文件或目录的服务器磁盘空间的总量。此值等于定额减去已使用空间。

已使用

所选 iFolder 中的所有文件和目录当前所占用的服务器磁盘空间的总量。

定额

此值报告 iFolder 拥有者分配给所选 iFolder 的服务器磁盘空间总量。如果未设置定额，则该值为空或为 N/A（不适用）。

以下参数报告 iFolder 的本地拷贝的同步状态：

表 5 本地储存的 iFolder 的同步参数

参数

说明

上次成功同步

iFolder 的本地拷贝与 iFolder 服务器的最近一次成功同步的完成时间。

mm/dd/yyyy hh:mm:ss

要同步的文件/文件夹

需要同步的 iFolder 中的储存项目（文件和文件夹）的总数。

自动同步间隔

iFolder 的同步间隔。以分钟、小时或天为单位指定此值。

iFolder 客户程序与 Novell iFolder 3.6 Enterprise Server 一起使用时，此间隔无法覆盖由 iFolder 管理员为服务器或服务器上的帐户指定的最小间隔，但它可以覆盖客户程序级的同步间隔。

安全同步

用于安全地同步所创建 iFolder 中的文件。您可以同时对常规 iFolder 和加密 iFolder 启用安全通道。如果选择此选项，iFolder 允许使用安全通道在服务器上的 iFolder 与本地计算机上的 iFolder 间进行同步。

立即同步

单击此按钮以根据需要启动同步。

通过文件管理器访问 iFolder 属性

  1. 在文件管理器中，导航至 iFolder，然后选择它。

  2. 右击该 iFolder，然后单击“iFolder” > “属性”。

通过 iFolder 浏览器访问 iFolder 属性

  1. 在通知区域中，右击 iFolder 服务图标，然后单击“iFolder”打开 iFolder 浏览器。

  2. 在 iFolder 列表中，选择该 iFolder，然后单击“属性”。

有关更多信息，请参见 法律声明.
