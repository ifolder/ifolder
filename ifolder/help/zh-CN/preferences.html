为客户程序配置 iFolder 首选设置

  1. 在通知区域中右击“iFolder 服务” 图标，然后选择“首选设置”以打开“iFolder 首选设置”对话框并进入“一般”选项卡。

  2. 指定以下选项的首选设置：

    参数

    说明

    登录到桌面时启动 iFolder（仅在 Windows 客户端中可用）

    选中此选项可在您使用本地用户名登录到设置 iFolder 帐户的位置时启动 iFolder。对于已启用“保存我的口令”的帐户，同步过程将按指定的同步间隔在后台进行。

    取消选中此选项以手动启动 iFolder。

    成功创建 iFolder 时显示确认讯息

    选中此复选框可在每次创建 iFolder 时除了接收常规可视提示（如 iFolder 出现在 iFolder 列表中）外，还接收确认讯息。

    取消选中此复选框可停止接收确认讯息。

    启动时隐藏 iFolder 主窗口

    取消选中此复选框以便在启动时显示 iFolder 窗口。

    如果希望在启动时隐藏 iFolder 窗口，请选中此复选框。

    显示同步日志

    如果希望在启动时显示同步日志，请选中此复选框。

    如果希望在启动时隐藏同步日志，请取消选中此复选框。

    显示违反策略行为的通知

    选中此复选框以便在违反策略时，在通知区域中显示弹出讯息。

    在共享 iFolder 时通知

    选中此复选框以便在接收到新的邀请以共享 iFolder 时，在通知区域中显示弹出讯息。

    要查看当前计算机上可用于下载的 iFolder 的列表，请在 iFolder 运行时在 iFolder 浏览器中选择“显示可用 iFolder”。

    出现冲突时通知

    选中此复选框，以便同步文件时出现冲突时，在通知区域中显示弹出讯息。

    取消选中此复选框以禁用同步冲突的弹出讯息。

    当 iFolder 运行过程中出现冲突时，iFolder 上将显示出 iFolder 错误 图标。

    用户加入 iFolder 域时通知

    选中此复选框以便在用户接受共享 iFolder 的邀请时，在通知区域中显示弹出讯息。

    取消选中此复选框以禁用弹出讯息。

    您可选择一个 iFolder 以在它运行时查看其用户的完整列表，然后单击“共享对象”以打开“iFolder 属性”对话框来进入“共享”选项卡。

    同步

    Automatically Synchronize iFolders: 选中此复选框可为此计算机上当前本地登录或登录身份下的所有 iFolder 启用同步。取消选中此复选框以禁用同步。

    Synchronize iFolders Every: 指定用于同步此计算机上当前本地登录或登录身份下的 iFolder 的最小间隔。指定值和单位。如果启用了此选项，则默认值为 5 分钟。最小时间间隔可设置为 5 秒。

    如果将此 iFolder 客户程序与 Novell iFolder 3.7 或更高版本的服务器一起使用，则有效的最小同步间隔始终是以下设置的最大值：

    • 系统策略默认为零 (0)，除非已设置用户策略。如果设置了用户策略，无论用户策略的值是大于还是小于系统策略的值，此用户策略都将覆盖系统策略。

    • 本地计算机策略或客户机上的设置与服务器同步。

    • iFolder（集合）策略。

  3. 单击“关闭”来关闭“iFolder 首选设置”对话框。

有关更多信息，请参见 法律声明.
