登录 iFolder 帐户

可以随时在本地访问 iFolder 目录中的文件。您必须登录 iFolder 帐户以同步用户列表、同步 iFolder 数据、上载新的 iFolder、下载共享 iFolder、共享 iFolder 以及管理 iFolder。可以分别登录也可以同时登录多个帐户。

对每个帐户使用以下登录方法之一：

  • 自动登录： 如果您希望在 iFolder 客户端启动的时候自动登录，则启用 iFolder 帐户的“记住口令”。当您配置帐户或在以后修改帐户设置时，可以启用“在此计算机上记住口令”。

    只要更改了口令，自动鉴定就会在您下次试图登录到 iFolder 帐户时失败。在 iFolder 提示您登录时，可以输入新口令，且如果需要的话，再次选中“记住口令”复选框。

  • 按需登录： 如果希望仅在需要时登录到 iFolder 帐户，请取消选中“记住口令”复选框。

要对现有 iFolder 帐户启用或禁用“记住口令”功能：

  1. 在通知区域中右击“iFolder 服务” 图标，然后选择“帐户”以打开“iFolder 首选设置”对话框并进入“帐户”选项卡。

  2. 选择 iFolder 帐户，然后单击“属性”以查看其在“属性”对话框中的设置。

  3. 单击“身份”选项卡。

  4. 选择（启用）或取消选择（禁用）“记住口令”。

  5. 在“口令”字段输入口令。

  6. 单击“关闭”即可成功完成。

  7. 单击“iFolder 属性”对话框中的“关闭”保存更改并登录到服务器。

要登录 iFolder 帐户，请执行以下操作：

  1. 在通知区域中右击“iFolder 服务” 图标，然后选择“帐户”。

  2. 选择 iFolder 帐户，然后选择它旁边的“联机”复选框。

  3. 如果提示需要输入口令，则指定口令，然后单击“连接”。

    如果已启用该帐户的“记住口令”，除非您的口令已过期，否则不会提示您输入口令。

    如果登录失败，则出现报告原因的错误讯息，例如口令失败。

  4. 连接成功后，关闭“iFolder 首选设置”对话框。

有关更多信息，请参见 法律声明.
