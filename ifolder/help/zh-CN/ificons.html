了解 iFolder 图标

下表说明了 iFolder 图标及相关任务。

表 1 iFolder 图标的说明

iFolder 图标

说明

在桌面应用程序菜单中显示 iFolder 3 客户程序的 iFolder 服务图标。

运行 iFolder 时，iFolder 服务图标还出现在桌面任务栏的通知区域。右击该图标可获取一个可能任务的菜单，其中包括 iFolder 浏览器、帐户设置、同步日志、首选设置以及帮助。

创建 iFolder 图标显示在 iFolder 浏览器中。

您可以创建新的 iFolder 或将现有文件夹转换为 iFolder。如果您拥有多个帐户，则必须指定要使用的帐户；iFolder 只可以属于一个帐户。有关信息，请参见创建和上载 iFolder。

当指定服务器上有您可用的 iFolder，而您尚未下载至本地计算机时会显示 iFolder 下载图标。

可以将不同帐户下的任何 iFolder 组合下载到本地计算机上。有关信息，请参见下载可用的 iFolder。

加密的 iFolder 图标（锁定的 iFolder）指出该 iFolder 是加密的，此图标显示在 iFolder 浏览器中。

登录时提供的通行口令可让您检索加密的 iFolder 中的文件。有关加密的更多信息，请参见管理加密 iFolder 的通行口令。

iFolder 徽标（绿色的“i”），在文件管理器、桌面或 iFolder 浏览器中查看 iFolder 时，只要运行 iFolder 服务应用程序，都会在 iFolder 上显示该徽标。

iFolder 不运行时，如果在文件浏览器或桌面上查看 iFolder，它显示为常规文件夹。当 iFolder 不运行时，iFolder 浏览器不可用。

在仅包含 iFolder 的单个目录中组织 iFolder 的本地集合是个很好的做法。这样可以在 iFolder 未运行且未提供将它们标记为 iFolder 的图标时轻松记住哪些文件夹为 iFolder。

iFolder 警告图标指出服务器不可用。iFolder 正在运行，但您未连接到 iFolder 服务器上。可能的原因有：

  • 您没有登录到 iFolder 帐户。有关信息，请参见登录 iFolder 帐户。

  • 客户端禁用该帐户；即已禁用“自动连接”参数。有关信息，请参见查看和修改 iFolder 帐户设置中的“自动连接”。

  • 服务器已关闭或已禁用服务器端的帐户。与 iFolder 管理员联系以获取帮助。

  • 防火墙阻止 iFolder 通讯。有关解决此问题的信息，请参见为 iFolder 通讯配置本地防火墙设置。

  • 网络连接已关闭。请检查本地网络设置。

iFolder 冲突图标指示在本地拷贝和服务器拷贝之间存在文件冲突。通过指示要保留文件的哪个版本即可解决该冲突。有关文件冲突的更多信息，请参见解决文件冲突。

iFolder 等待同步图标指出 iFolder 包含要与服务器同步的文件且同步过程尚未开始。

iFolder 每次同步一个 iFolder，并对同步任务进行排队。要确保本地 iFolder 中的文件已与服务器中的文件同步，请等待直到同步开始并成功完成再关闭 iFolder，如图标中的变化所示。

iFolder 正在同步图标指出 iFolder 中的文件当前正在与服务器同步。

iFolder 错误图标指示因文件大小限制或 iFolder 数量限制而产生策略冲突，致使 iFolder 中的一个或多个文件尚未进行同步。

在“共享”对话框中，iFolder 拥有者用户图标指出该用户拥有该共享 iFolder。

在“共享”对话框中，iFolder 当前用户图标指示该用户当前正在管理该 iFolder。如果当前用户是拥有者，则将显示拥有者用户图标。

有关更多信息，请参见 法律声明.
