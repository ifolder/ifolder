Удаление учетной записи iFolder

Вы можете указать, нужно ли удалить только учетную запись с текущего компьютера или необходимо также удалить все папки iFolder, владельцем которых Вы являетесь на сервере.

Если учетная запись iFolder удаляется только с компьютера:

  • Локальные папки iFolder для данной учетной записи будут вновь преобразованы в стандартные папки. С помощью менеджера файлов найдите и удалите локальную копию данных, если они не нужны.

  • Вы можете по-прежнему получать доступ к папкам iFolder с других компьютеров, на которых установлен клиент iFolder.

  • Если клиент iFolder используется с сервером Novell iFolder Enterprise Server 3.7 или более поздней версии, можно получать доступ к файлам с любого компьютера с помощью средства веб-доступа Novell iFolder 3.7 или более поздней версии.

Если учетная запись iFolder удалена с сервера:

  • Совместное использование всех папок iFolder будет отменено.

  • Локальные копии папок iFolder на компьютерах членов будут преобразованы в стандартные папки. С помощью менеджера файлов найдите и удалите локальную копию данных, если они не нужны.

  • Все программные компоненты iFolder будут удалены сервера.

  • Ваша учетная запись будет удалена из папок iFolder, которые совместно с Вами использовали другие пользователи.

  • Учетная запись удаляется с локального компьютера.

Если эту учетную запись потребуется использовать снова на этом или другом компьютере, нужно установить учетную запись и ее папки iFolder заново.

Порядок удаления учетной записи iFolder:

  1. Щелкните правой кнопкой значок Сервисы iFolder в области оповещений, затем выберите Параметры учетной записи, чтобы открыть диалоговое окно "Настройки iFolder" на вкладке Учетные записи.

  2. Выберите учетную запись iFolder, которую необходимо удалить, затем щелкните Удалить.

  3. Появится сообщение с предложением выбрать варианты действия удаления.

    Выполните одно из следующих действий.

    • Удаление учетной записи iFolder только с текущего компьютера. Убедитесь, что флажок Удалить мою папку iFolder и файлы с сервера не установлен, затем щелкните кнопкуДа.

    • Удалите учетную запись iFolder, папки iFolder и файлы с сервера. Установите флажок Удалить мою папку iFolder и файлы с сервера, затем щелкните кнопку Да.

    • Отмена и удаление. Для отказа от операции удаления щелкните кнопку Нет. Ваша учетная запись и папки iFolder не будут изменены.

Дополнительную информацию см. в Юридическая информация.
