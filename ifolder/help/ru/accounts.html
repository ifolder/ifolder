Конфигурирование учетной записи iFolder

Для добавления и настройки новых учетных записей предназначен мастер создания учетных записей iFolder. Прежде чем создавать папки iFolder или совместно использовать их в системе, необходимо настроить хотя бы одну учетную запись iFolder. В каждый момент времени к серверу iFolder может быть подключена только одна учетная запись, однако для установки соединения с несколькими серверами iFolder можно настроить несколько учетных записей. Можно войти в систему отдельно с каждой учетной записью или одновременно с помощью нескольких учетных записей.

В результате на одном компьютере будут папки iFolder нескольких пользователей с разными локальными регистрационными данными. При первом запуске iFolder с текущими локальными регистрационными данными появляется предложение настроить учетную запись iFolder.

ВАЖНО.В iFolder 3.7 и более поздних версий добавлена новая функция автоматического создания учетных записей, которая помогает администратору iFolder создавать и настраивать учетные записи для пользователей. Если администратор iFolder разрешил автоматическое создание учетных записей, пользователям не требуется конфигурировать учетные записи самостоятельно. При первом запуске iFolder появляется запрос на ввод пароля. Введите пароль и завершите настройку учетной записи.

Чтобы настроить новую учетную запись, выполните действия, указанные ниже.

  1. Запустить мастер создания учетных записей можно одним из перечисленных ниже способов.

    • При первом запуске iFolder с текущими локальными регистрационными данными появляется предложение настроить учетную запись iFolder. Щелкните Вперед, чтобы перейти непосредственно к мастеру создания учетных записей iFolder.

    • Щелкните правой кнопкой значок сервисов iFolder в области оповещений, выберите пункт Учетные записи, чтобы открыть диалоговое окно настроек iFolder на вкладке Учетные записи, а затем щелкните Добавить, чтобы запустить мастер создания учетных записей iFolder.

  2. Следуйте инструкциям на экране, чтобы определить следующие значения:

    Параметр

    Описание

    Адрес сервера

    Это DNS-имя или IP-адрес сервера iFolder, на котором у Вас есть учетная запись. Эту информацию можно получить у администратора iFolder.

    Например: svr21.example.com или 192.168.1.1

    В интерфейсе пользователя iFolder имя сервера всегда отображается вместе с IP-адресом. Например: iFolder - 192.168.1.1

    ВАЖНО.С помощью клиента iFolder 3.7 или более поздней версии нельзя подключиться к серверу iFolder 3.2.

    Установить как сервер по умолчанию

    Если с одной учетной записью связано несколько серверов, установите флажок Установить как сервер по умолчанию, чтобы сделать сервер, выбранный в раскрывающемся списке серверов, сервером по умолчанию при создании или отправке папки iFolder.

    Имя пользователя

    Укажите пользовательскую идентификационную информацию для данной учетной записи. Администратор iFolder определяет используемый формат: ИД пользователя сети (общее имя LDAP) или адрес электронной почты.

    Например, для пользователя John Smith с ИД пользователя jsmith и адресом электронной почты john.smith@example.com администратор настраивает параметр для сервера, который определяет, какой из двух форматов будет использоваться в поле Имя пользователя для аутентификации.

    Пароль

    Определите пароль, соответствующий имени пользователя.

    Аутентификация выполняется по защищенному каналу (SSL). Появляется запрос на проверку и принятие сертификата сервера.

    Запомнить пароль

    Установите флажок Запомнить пароль на этом компьютере, чтобы каждый раз автоматически входить на компьютер с данной учетной записью.

    После смены сетевого пароля iFolder при следующей попытке входа с помощью учетной записи iFolder произойдет сбой автоматической аутентификации. Программа iFolder попросит Вас войти с новым паролем и позволит определить и сохранить его.

  3. Проверьте введенные данные. Если необходимо внести изменения, щелкните кнопку Назади вернитесь на предыдущие страницы. Щелкните Установить соединение, чтобы продолжить процедуру и проверить подключение к серверу.

  4. Если появится запрос Принять сертификат от данного сервера, щелкните элемент Подробная информация для просмотра сведений о сертификате, а затем щелкните ОК, чтобы принять сертификат.

    ПРИМЕЧАНИЕ.В ходе создания учетной записи вход выполняется с использованием протокола SSL. При выводе предложения принять сертификат сервера, для успешного соединения необходимо проверить сертификат и принять его.

  5. Если администратор включил политику шифрования, iFolder предлагает ввести контрольную фразу.

    Введите контрольную фразу в поле Контрольная фраза.

    Повторите контрольную фразу в поле Подтверждение.

    Установите флажок Запомнить контрольную фразу для автоматического доступа к зашифрованным данным iFolder в рамках текущего сеанса. Если этот флажок установлен, система iFolder сможет вспомнить вашу контрольную фразу при следующем входе. Этот обмен контрольной фразой выполняется автоматически.

  6. Выберите необходимый агент восстановления в списке и щелкните ОК.

  7. Укажите местоположение для создания папки iFolder по умолчанию.

    По умолчанию папка iFolder по умолчанию хранится в каталоге <личный каталог пользователя>/ifolder/имя_сервера/имя_пользователя.

  8. Выберите тип создаваемой папки iFolder по умолчанию. Если включена функция Шифрование, будет предложено указать контрольную фразу.

    ВАЖНО.Можно совместно использовать только обычные папки iFolder. Совместное использование зашифрованных папок iFolder не поддерживается.

    Если администратор iFolder не включил шифрование для текущей учетной записи, соответствующий пользователь сможет создавать только обычные папки iFolder; если шифрование включено, по умолчанию выбрана опция Обычный.

  9. Выберите Защищенная синхронизация, чтобы включить защищенную синхронизацию файлов в создаваемой папке iFolder.

    Можно включить использование защищенного канала как для зашифрованных, так и для обычных папок iFolder. В последнем случае система iFolder позволяет использовать для синхронизации папки iFolder между сервером и локальным компьютером защищенный канал.

    Если удалить с сервера папку iFolder по умолчанию, при следующей настройке учетной записи с помощью любого "толстого клиента" снова появится предложение создать папку iFolder по умолчанию.

    Однако если удалить с локального компьютера папку iFolder по умолчанию, при следующей настройке учетной записи система iFolder предложит загрузить эту папку.

  10. Если папка iFolder по умолчанию уже настроена, ее можно загрузить на локальный компьютер.

    Загрузите папку iFolder по умолчанию и щелкните Далее, чтобы продолжить.

  11. Если в системе уже есть папка, имя которой совпадает с именем папки iFolder по умолчанию, будет предложено объединить эти две папки либо загрузить папку iFolder по умолчанию в другое место.

    Щелкните ОК, чтобы объединить две папки, или Отмена, чтобы загрузить папку iFolder по умолчанию в другое место.

  12. Щелкните Готово, чтобы закрыть мастер создания учетных записей iFolder.

    iFolder выполняет синхронизацию списка пользователей для указанной учетной записи и определяет папки iFolder, доступные для загрузки. Первоначальная загрузка может занимать от нескольких секунд до нескольких минут, в зависимости от размера списка пользователей для данной учетной записи.

  13. Убедитесь, что настроенная Вами учетная запись появилась в списке учетных записей.

    1. Щелкните правой кнопкой значок Сервисы iFolder в области оповещений, затем в меню выберите Настройки, чтобы открыть диалоговое окно Настройки iFolder на вкладке Учетные записи.

      После подключения к серверу устанавливается флажок Интерактивный режим.

    2. Закройте диалоговое окно "Настройки iFolder".

  14. Настройте папки iFolder для учетной записи с помощью любого из следующих методов:

    • В области оповещений щелкните правой кнопкой значок Сервисы iFolder, затем выберите Папки iFolder, чтобы открыть навигатор iFolder. Перейдите к разделу Создание и отправка папки iFolder.

    • Если для учетной записи есть доступные для загрузки с сервера папки iFolder, они отображаются в окне iFolder под заголовком "Папки iFolder" на сервере <имя_сервера_iFolder>. Значок загрузки показывает, что папка iFolder доступна для загрузки. Перейдите к разделу Загрузка доступной папки iFolder.

Дополнительную информацию см. в Юридическая информация.
