Слияние папок

Функция слияния позволяет консолидировать данные iFolder и уменьшить объем данных, передаваемый между сервером и клиентом iFolder. Можно выполнить слияние содержимого папки iFolder на сервере с содержимым папки, существующей на локальном компьютере.

Слияние папки iFolder и стандартной папки

Можно выполнить слияние папки iFolder и стандартной папки с аналогичным именем. Например, можно выполнить слияние содержимого папки iFolder с именем corporate с содержимым стандартной папки с именем corporate.

Предположим, папка iFolder corporate содержит три подпапки: hr, finance и marketing. Необходимо загрузить содержимое папки iFolder corporate в папку с именем corporate, которая содержит только папки hr и finance. В этом случае нет необходимости загружать все три папки в папку corporate и увеличивать скорость передачи. Вместо этого можно выполнить слияние папки iFolder corporate и папки corporate с помощью функции Выполнить слияние с папкой таким образом, чтобы предотвратить загрузку подпапок hr и finance и загрузить только подпапку marketing. Различие данных папки iFolder corporate и папки corporate приводит к появлению конфликта версий. Эти конфликты можно разрешить вручную. Вручную можно также разрешить конфликт имен файлов при его появлении.

Процедура слияния содержимого папки iFolder на сервере и содержимого папки, существующей на локальном компьютере:

  1. В навигаторе iFolder щелкните правой кнопкой необходимую папку iFolder в списке папок iFolder на выбранном сервере, затем выберите Выполнить слияние с папкой.

  2. Найдите и выберите папку, затем щелкните OK.

Теперь выполняется сравнение содержимого этих двух папок и загрузка из папки iFolder файлов, которые отсутствуют в стандартной папке. Осуществляется передача файлов, которые отсутствуют в папке iFolder, но содержатся в стандартной папке.

Изменение расположения существующих папок iFolder

Процедура перемещения существующей папки iFolder с именем corporate в другое местоположение на клиентском компьютере:

  1. Преобразуйте папку iFolder corporate в стандартную папку.

    ВАЖНО.Не удаляйте папку iFolder на сервере.

    См. дополнительные сведения: Преобразование папки iFolder в стандартную папку.

  2. Переместите папку corporate в другое местоположение.

    ВАЖНО.При перемещении папки не изменяйте ее имя.

  3. В окне навигатора iFolder выберите папку iFolder corporate на сервере, затем щелкните Выполнить слияние с папкой.

  4. Выберите папку corporate на локальном компьютере, затем щелкните OK для слияния содержимого.

    ПРИМЕЧАНИЕ.Если вместо функции Выполнить слияние с папкой используется функция Синхронизировать сейчас (см. раздел Перемещение папки iFolder), выполняется проверка всех данных в папке iFolder на сервере, а затем — их синхронизация. Это приводит к увеличению времени передачи данных. Использование функции Выполнить слияние с папкой, напротив, приводит к уменьшению объема передаваемых данных, так как при этом выполняется только слияние содержимого папки iFolder и стандартной папки.

    Для получения дополнительной информации о слиянии см. подраздел Слияние папки iFolder и стандартной папки.

Слияние перенесенных папок iFolder 2.x с папками iFolder 3.7 или более поздней версии

Перенесенные папки iFolder 2.x можно объединять с папками iFolder, созданными на сервере iFolder 3.7 или более поздней версии. Для получения дополнительной информации о миграции см. подраздел Миграция и обновление Novell iFolder.

После миграции папки iFolder 2.x перемещаются на сервер iFolder 3.7 или более поздней версии. Папки iFolder, перенесенные на сервер 3.7 или более поздней версии, можно загрузить на клиентский компьютер. Однако эти папки iFolder, возможно, уже существуют на клиентском компьютере. В этом случае используйте функцию слияния, чтобы предотвратить передачу ненужных данных с сервера на клиент.

Слияние перенесенных папок iFolder 2.x с папками iFolder 3.7 или более поздней версии

  1. Выполните миграцию папок iFolder 2.x папки iFolder перемещаются на сервер iFolder 3.7 или более поздней версии.

  2. Преобразуйте папки iFolder 2.x на локальном компьютере в стандартные папки.

  3. Выберите копию каждой папки iFolder на сервере и щелкните Выполнить слияние с папкой.

  4. Для слияния содержимого выберите стандартную папку и щелкните OK.

Дополнительную информацию см. в Юридическая информация.
