Общие сведения о свойствах iFolder

Следующие параметры относятся к дисковому пространству, используемому на сервере:

Таблица 4 Параметры хранения папок iFolder на сервере iFolder

Параметр

Описание

Доступен

Общий объем дискового пространства, доступного в данный момент на сервере для файлов или каталогов в выбранной папке iFolder. Это значение равно квоте за вычетом уже использованного пространства.

Используется

Общий объем дискового пространства, потребляемого в данный момент всеми файлами и каталогами выбранной папки iFolder.

Квота

Это значение показывает общий объем дискового пространства, выделенный владельцем iFolder для выбранной папки iFolder. Если квота не установлена, значение является пустым или обозначается как Н/Д(Недоступно).

Следующие параметры определяют состояние синхронизации локальной копии папки iFolder:

Таблица 5 Параметры синхронизации для папок iFolder, хранимых локально

Параметр

Описание

Последняя успешная синхронизация

Время завершения последней успешной синхронизации локальной копии папки iFolder с сервером iFolder.

mm/dd/yyyy hh:mm:ss

Файлов/папок для синхронизации

Общее количество элементов хранения (файлов и папок) iFolder, которые необходимо синхронизировать.

Автоматическая синхронизация через каждые

Интервал синхронизации iFolder. Укажите значение в минутах, часах или днях.

Если клиент iFolder используется с сервером Novell iFolder 3.6 Enterprise Server, этот интервал не может переопределить минимальный интервал, указанный администратором iFolder для сервера или учетной записи на сервере, но он может переопределить интервал синхронизации на уровне клиента.

Защищенная синхронизация

Включение безопасной синхронизации файлов в создаваемой папке iFolder. Можно включить использование защищенного канала и для зашифрованных, и для обычных папок iFolder. Если этот флажок установлен, система iFolder позволяет использовать для синхронизации папки iFolder между сервером и локальным компьютером защищенный канал.

Синхронизировать сейчас

Щелкните кнопку для выполнения синхронизации вручную.

Доступ к свойствам iFolder из менеджера файлов

  1. В менеджере файлов перейдите к папке iFolder и выберите ее.

  2. Щелкните правой кнопкой iFolder, затем щелкните iFolder > Свойства.

Доступ к свойствам iFolder из навигатора iFolder

  1. В области оповещений щелкните правой кнопкой значок сервисов iFolder, затем щелкните Папки iFolder, чтобы открыть навигатор iFolder.

  2. В списке папок iFolder выберите папку iFolder и щелкните Свойства.

Дополнительную информацию см. в Юридическая информация.
