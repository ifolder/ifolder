Конфигурирование параметров локальной антивирусной программы для трафика iFolder

При использовании антивирусного программного обеспечения на компьютере исключите каталог ..\simias\WorkArea\ из списка сканирования антивирусного программного обеспечения. Например в системе Windows каталог ..\simias\WorkArea\ располагается в c:\documents and settings\имя_пользователя\local settings\application data\simias\WorkArea\.simias.wf.guid.

В каталоге ..\simias\WorkArea\ iFolder организует файлы для загрузки с сервера. Если файл iFolder заражен вирусом и антивирусное программное обеспечение обнаруживает его в каталоге ..\simias\WorkArea, программа антивирусной защиты выдает всплывающее сообщение об обнаружении вируса и помещает зараженный файл в карантин. Затем iFolder будет пытаться снова и снова синхронизировать зараженный файл.

Исключение рабочей области WorkArea из списка сканирования позволит синхронизации завершиться. Зараженный файл будет успешно загружен с сервера и помещен в локальную папку iFolder. Затем антивирусная программа найдет этот зараженный файл в папке iFolder и поместит в карантин, защищая компьютер от инфицирования. iFolder обнаружит удаление объекта и при следующей синхронизации удалит файл с сервера и с компьютеров пользователей.

Однако, если папка iFolder, в которую был загружен файл, предназначена только для чтения, удаление будет выполнено лишь локально и не затронет сервер и компьютеры других членов. Вирус должен быть обнаружен во всех остальных местоположениях, где разрешена синхронизация всех изменений. Возможно, антивирусная программа предупредит пользователя о вирусе, после чего он должен обязательно уведомить об этом других членов iFolder.

Дополнительную информацию см. в Юридическая информация.
