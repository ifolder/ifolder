Управление контрольными фразами для зашифрованных папок iFolder

Для зашифрованной папки iFolder существует уникальный ключ шифрования данных, который формируется автоматически при создании этой папки. Ключ шифруется с помощью контрольной фразы, указываемой пользователем, а также с помощью открытого ключа агента восстановления.

Агент восстановления выбирается на этапе создания учетной записи iFolder, если администратор iFolder включил на сервере политику шифрования.

В этом случае при создании папки iFolder появляется диалоговое окно, в котором необходимо ввести контрольную фразу. Система iFolder формирует на базе контрольной фразы уникальный ключ шифрования, который затем используется для шифрования и расшифровки данных iFolder на сервере iFolder. Забыв контрольную фразу, вы потеряете доступ к данным iFolder. Чтобы его восстановить, вам потребуется сбросить контрольную фразу.

Восстановление зашифрованной папки iFolder

Сбросить контрольную фразу и восстановить зашифрованную папку iFolder можно с помощью мастера восстановления контрольной фразы. Чтобы его запустить, воспользуйтесь приведенными ниже инструкциями.

  1. В навигаторе iFolder щелкните Безопасность > Контрольная фраза утеряна, чтобы запустить мастер.

  2. В списке Учетные записи iFolder выберите учетную запись, для которой необходимо сбросить пароль, и щелкните кнопку Далее.

    Если в качестве агента восстановления был выбран вариант Server_Default (по умолчанию для сервера), во время создания учетной записи iFolder откроется страница "Установка новой контрольной фразы". Инструкции по сбросу контрольной фразы см. в разделе Сброс контрольной фразы с агентом восстановления Server_Default.

    Если же был выбран агент восстановления, отличный от Server_Default, при сбросе контрольной фразы руководствуйтесь инструкциями из раздела Сброс контрольной фразы с агентом восстановления, отличным от Server_Default.

Сброс контрольной фразы с агентом восстановления Server_Default

Чтобы установить новую контрольную фразу, выполните указанные ниже действия.

  1. На странице "Установка новой контрольной фразы" введите в поле Новая контрольная фраза новую контрольную фразу.

  2. Еще раз введите новую контрольную фразу в поле Подтверждение > .

  3. Укажите пароль в поле Пароль и нажмите кнопку Далее.

  4. Сбросив контрольную фразу, щелкните Готово, чтобы закрыть мастер.

Сброс контрольной фразы с агентом восстановления, отличным от Server_Default

Чтобы сбросить контрольную фразу в случае, когда выбран агент восстановления, отличный от Server_Default, выполните указанные ниже действия.

  1. На странице приветствия в мастере отображается информация, необходимая для сброса контрольной фразы. Щелкните Далее, чтобы открыть страницу восстановления контрольной фразы.

  2. Сбросить контрольную фразу с помощью мастера можно тремя перечисленными ниже способами.

    • С помощью секретного файла и пароля к нему Сбросить контрольную фразу можно с помощью секретного файла и пароля к нему. Если у вас есть секретный файл и пароль, укажите в мастере расположение этого файла и введите пароль к нему. Инструкции по сбросу контрольной фразы с помощью секретного файла см. в разделе Сброс контрольной фразы с помощью секретного файла.

      ПРИМЕЧАНИЕ.Получить секретный файл и соответствующий пароль можно у администратора iFolder.

    • С помощью нового файла данных Сбросить контрольную фразу также можно с помощью нового файла данных. Получить новый файл данных можно у администратора после экспорта старого файла данных. Инструкции по сбросу контрольной фразы с помощью нового файла данных см. в разделе Сброс контрольной фразы с помощью нового файла данных.

    • Посредством экспорта старого файла данных Если у Вас нет секретного файла или нового файла данных, экспортируйте старый файл данных с помощью мастера, а затем отправьте его администратору по электронной почте. Получив от администратора взамен новый файл данных, сбросьте контрольную фразу с помощью мастера. Инструкции по экспорту старого файла данных см. в разделе Сброс контрольной фразы с помощью экспортированного файла.

Сброс контрольной фразы с помощью секретного файла

Чтобы сбросить контрольную фразу с помощью секретного файла, выполните указанные ниже действия.

  1. Выберите опцию У меня есть секретный файл и пароль к нему и щелкните Далее.

  2. Заполните указанные ниже поля.

    Расположение секретного файла Укажите расположение секретного файла или щелкните кнопку Обзор и найдите его.

    Секретный пароль Укажите пароль к секретному файлу.

    Новая контрольная фраза Укажите новую контрольную фразу.

    Подтверждение Подтвердите контрольную фразу.

  3. Щелкните Далее, чтобы сбросить контрольную фразу.

  4. Нажмите кнопку Готово, чтобы закрыть мастер.

ПРИМЕЧАНИЕ.Получить секретный файл и соответствующий пароль можно у администратора iFolder.

Сброс контрольной фразы с помощью нового файла данных

Чтобы сбросить контрольную фразу с помощью нового файла данных, выполните указанные ниже действия.

  1. Выберите опцию У меня есть новый файл данных, полученный от администратора и щелкните Далее.

  2. Укажите расположение нового файла данных в поле Расположение нового файла данных или нажмите кнопку Обзор, чтобы найти и выбрать этот файл.

  3. Если полученный от администратора новый файл данных зашифрован, установите флажок Указанный выше файл зашифрован?. Установка этого флажка активирует поле Одноразовый пароль.

  4. В поле Одноразовый пароль введите пароль, с помощью которого администратор зашифровал файл данных.

  5. В поле Новая контрольная фраза введите новую контрольную фразу.

  6. Подтвердите новую контрольную фразу в поле Подтверждение.

  7. Щелкните Далее, чтобы сбросить контрольную фразу.

  8. Нажмите кнопку Готово, чтобы закрыть мастер.

Сброс контрольной фразы с помощью экспортированного файла

Чтобы сбросить контрольную фразу без секретного файла и нового файла данных, выполните указанные ниже действия.

  1. Выберите опцию У меня нет ни секретного файла, ни файла данных и щелкните Далее.

  2. В поле Место для сохранения старого файла данных укажите место, куда будет сохранен старый файл данных, или выберите его с помощью кнопки Обзор.

  3. Щелкните Далее, чтобы получить старый файл данных. Появится сообщение, в котором будет указано расположение старого файла данных и адрес электронной почты администратора iFolder. Эта информация понадобится для получения нового файла данных.

  4. Нажмите кнопку Готово, чтобы закрыть мастер.

  5. Отправьте администратору iFolder запрос на получение нового файла данных и приложите к нему старый файл.

  6. Получив новый файл данных, сбросьте контрольную фразу, руководствуясь инструкциями из раздела Сброс контрольной фразы с помощью нового файла данных.

Смена контрольной фразы

Из соображений безопасности необходимо периодически менять контрольную фразу. Сбросить контрольную фразу можно с помощью одноразового пароля, полученного от агента восстановления.

  1. В навигаторе iFolder щелкните Безопасность > Смена контрольной фразы.

  2. В окне Смена контрольной фразы выберите учетную запись iFolder, для которой необходимо сбросить контрольную фразу, и укажите перечисленные ниже данные.

    Ввод контрольной фразы. Введите текущую контрольную фразу.

    Ввод новой контрольной фразы. Введите новую контрольную фразу для текущей учетной записи iFolder.

    Повтор контрольной фразы. Для подтверждения введите новую контрольную фразу еще раз.

    Агент восстановления. Выберите агент восстановления, который был установлен для текущей учетной записи iFolder.

  3. Щелкните Сброс.

Дополнительную информацию см. в Юридическая информация.
