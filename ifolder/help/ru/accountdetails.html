Просмотр и изменение настроек учетной записи iFolder

В диалоговых окнах "Настройки учетной записи" и "Свойства учетной записи" отображаются следующие параметры:

Таблица 1 Параметры и свойства учетной записи

Описание

В сети

Показывает, установлено ли в настоящий момент соединение с сервером iFolder для учетной записи.

Установите флажок Интерактивный режим рядом с учетной записью, чтобы установить соединение с сервером iFolder для данной учетной записи.

Снимите флажок Интерактивный режим рядом с учетной записью, чтобы завершить соединение с сервером iFolder.

Настройки iFolder вкладка > Учетные записи

Да

Имя сервера

Отображение описательного имени сервера iFolder, на котором хранятся папки iFolder для данной учетной записи. Это имя предоставляется сервером и устанавливается администратором iFolder.

Этот параметр не может быть изменен пользователем.

Настройки iFolder вкладка > Учетные записи

Свойства учетной записи > вкладка Сервер

Нет

Имя пользователя

Отображение электронной персоны пользователя для данной учетной записи, например сетевого ИД пользователя (общее имя LDAP) или адреса электронной почты. Этот параметр не может быть изменен.

Имя пользователя указывается при настройке учетной записи в приложении "Помощник по настройке учетных записей iFolder". Оно не может быть изменено для существующей учетной записи.

Для использования другого имени пользователя на том же сервере необходимо удалить существующую учетную запись, а затем создать другую учетную запись с этим именем.

Настройки iFolder вкладка > Учетные записи

Свойства учетной записи > вкладка Идентификация

Нет

Адрес

Отображение DNS-имени или IP-адреса сервера iFolder Еnterprise Server, для которого у Вас есть учетная запись. Эту информацию можно получить у администратора iFolder.

Например: svr21.example.com или 192.168.1.1

Этот адрес можно изменять, но перед внесением изменений необходимо завершить соединение с сервером iFolder.

Свойства учетной записи > вкладка Сервер

Да

Пароль

Позволяет определить пароль, соответствующий Вашему имени пользователя.

Свойства учетной записи > вкладка Идентификация

Да

Запомнить пароль

Показывает, включен ли для пользователя автоматический вход в iFolder при запуске iFolder. Если параметр учетной записи Подключаться автоматически разрешен (по умолчанию), выполняется попытка установить соединение с сохраненным паролем.

Выберите Запомнить пароль, чтобы при каждом входе в компьютере автоматически использовалась данная учетная запись iFolder.

После смены сетевого пароля iFolder при следующей попытке входа с помощью учетной записи iFolder произойдет сбой автоматической аутентификации. Программа iFolder попросит Вас выполнить вход с новым паролем и позволит определить и сохранить его.

Свойства учетной записи > вкладка Идентификация

Да

Квота

Информация об общем количестве пространства, выделенного для учетной записи на сервере Enterprise Server. Данное значение может быть квотой, устанавливаемой по умолчанию для всех пользователей, или квотой для отдельного пользователя. Если квота не установлена, значение является пустым или обозначается как Н/Д(Недоступно). Можно также просмотреть доступное пространство в основном окне iFolder для каждого подключенного сервера.

Свойства учетной записи > вкладка Дисковое пространство

Нет

Используется

Информация об общем количестве пространства, используемого в данный момент всеми папками iFolder, владельцем которых является пользователь данной учетной записи. Здесь не учитывается пространство, занимаемое совместно используемыми папками iFolder, владельцами которых являются другие пользователи.

Свойства учетной записи > вкладка Дисковое пространство

Нет

Доступен

Информация об общем количестве пространства, доступного в данный момент для папок iFolder в этой учетной записи iFolder. Это значение равно квоте за вычетом уже использованного пространства.

Свойства учетной записи > вкладка Дисковое пространство

Нет

Подключаться автоматически

Если это значение выбрано (разрешено), при запуске iFolder автоматически выполняет соединение для данной учетной записи. Отмените выбор этого параметра, чтобы запретить соединение для учетной записи.

Этот параметр только позволяет клиенту разрешать или запрещать учетную запись.

При использовании приложения iFolder с сервером Novell® iFolder 3.7 или более поздней версии администратор системы iFolder может отключить учетную запись iFolder (как правило, временно) на сервере. В этом случае может случиться следующее:

  • Если учетная запись запрещена на сервере после Вашего входа, можно управлять папками iFolder, а данные iFolder будут по-прежнему синхронизироваться с сервером до тех пор, пока Вы не завершите сеанс, пока не будет отключена синхронизация или пока сервер не отключит Ваш сеанс в зависимости от того, что произойдет раньше.

  • Если учетная запись запрещена на сервере, при попытке войти Вы получите всплывающее сообщение, что Ваша учетная запись запрещена. Обратитесь за помощью к администратору iFolder.

Свойства учетной записи, на всех вкладках

Да

Учетная запись по умолчанию

При наличии нескольких учетных записей установите флажок Учетная запись по умолчанию, чтобы выбрать учетную запись, которая будет использоваться по умолчанию при создании и передаче папок iFolder, в раскрывающемся списке.

Это значение не может быть изменено при наличии только одной учетной записи.

Свойства учетной записи, на всех вкладках

Да, при наличии нескольких учетных записей

Параметры шифрования

В меню Безопасность можно устанавливать и сбрасывать параметры шифрования. Для получения дополнительной информации о настройках шифрования см. раздел Управление контрольными фразами для зашифрованных папок iFolder.

Меню Защита

Пользователь может сбросить контрольную фразу

  1. Щелкните правой кнопкой значок Сервисы iFolder в области оповещений, затем выберите Параметры учетной записи, чтобы открыть диалоговое окно "Настройки iFolder" на вкладке Учетные записи.

    В отчете "Учетные записи" для каждой учетной записи можно настраивать следующие параметры:

    • Интерактивный режим (выберите для соединения; отмените выбор для отключения)

    • Имя сервера (только просмотр)

    • Имя пользователя (только просмотр)

  2. Выберите необходимую учетную запись iFolder, затем щелкните Свойства, чтобы открыть диалоговое окно "Свойства учетной записи" на вкладке "Сервер".

    На вкладке Сервер для выбранной учетной записи можно настраивать следующие параметры:

    • Имя (только просмотр)

    • Адрес (доступен для просмотра, только если установлено соединение; может быть изменен, если соединение отключено)

    • Описание сервера (только просмотр; для просмотра всей информации выполните прокрутку)

    • Подключаться автоматически (выберите для разрешения; отмените выбор для запрета)

    • Учетная запись по умолчанию (выберите, чтобы разрешить; параметр доступен только при наличии нескольких учетных записей)

  3. Выберите вкладку Идентичность для настройки следующих параметров выбранной учетной записи:

    • Имя пользователя (только просмотр)

    • Пароль (определите пароль)

    • Запомнить пароль (выберите для разрешения; отмените выбор для запрета))

    • Подключаться автоматически (выберите для разрешения; отмените выбор для запрета)

    • Учетная запись по умолчанию (выберите, чтобы разрешить; параметр доступен только при наличии нескольких учетных записей)

  4. Выберите вкладку Дисковое пространство для настройки следующих параметров выбранной учетной записи:

    • Квота (только просмотр)

    • Используется (только просмотр)

    • Доступен (только просмотр)

    • Подключаться автоматически (выберите для разрешения; отмените выбор для запрета)

    • Учетная запись по умолчанию (выберите, чтобы разрешить; параметр доступен только при наличии нескольких учетных записей)

  5. По завершении закройте диалоговое окно Свойства учетной записи, затем закройте диалоговое окно Настройки iFolder.

Дополнительную информацию см. в Юридическая информация.
