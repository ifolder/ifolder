Загрузка доступной папки iFolder

Когда другие пользователи используют папки iFolder совместно с Вами, эти папки появляются в навигаторе iFolder со значком загрузки папки iFolder в нижней части списка Папки iFolder на <имя_сервера_iFolder> , где имя_сервера_iFolder - имя сервера для данной учетной записи. Для участия в совместном использовании папки iFolder на компьютере необходимо загрузить папку iFolder.

Когда к списку членов совместно используемой папки iFolder присоединяется новый пользователь, в области оповещений отображается всплывающее сообщение iFolder с уведомлением о новом пользователе.

Пользователь совместно используемой папки также получит всплывающее сообщение в области оповещений по окончании загрузки совместно используемой папки iFolder. Содержание сообщения: <Имя пользователя, совместно используемой папки iFolder> присоединился к папке iFolder <имя_папки_iFolder>. После того как новый пользователь присоединился к совместно используемой папке, все остальные члены получают уведомление о новом члене.

  1. Щелкните правой кнопкой значок сервисов iFolder , затем щелкните Папки iFolder, чтобы открыть навигатор iFolder.

  2. В списке папок iFolder выберите доступную папку iFolder и щелкните Загрузка папки iFolder.

  3. Если вход в систему не выполнен, выполните его, указав имя пользователя и пароль, затем щелкните Подключить.

  4. Найдите местоположение, в котором необходимо создать локальную папку iFolder, затем укажите имя, которое необходимо использовать локально для данной папки iFolder.

    Местоположение и имя папки должны соответствовать следующим рекомендациям:

    • Рекомендации по размещению и использованию папок iFolder

    • Соглашения об именовании папки iFolder и содержащихся в ней папок и файлов

  5. Дважды щелкните кнопку OK.

Дополнительную информацию см. в Юридическая информация.
