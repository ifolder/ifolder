Синхронизация файлов

  • Синхронизация файлов по требованию

  • Конфигурирование интервала синхронизации

Синхронизация файлов по требованию

  1. В области оповещений щелкните правой кнопкой значок сервисов iFolder , затем щелкните Папки iFolder, чтобы открыть навигатор iFolder.

  2. В списке папок iFolder выберите папку, затем щелкните Синхронизировать сейчас.

Конфигурирование интервала синхронизации

  1. В области оповещений щелкните правой кнопкой значок сервисов iFolder , затем выберите Настройки, чтобы открыть диалоговое окно "Настройки iFolder" на вкладке Общие.

  2. Выполните одно из следующих действий.

    • Выберите Синхронизировать, чтобы включить фоновую синхронизацию.

    • Отмените выбор Синхронизировать, чтобы отключить синхронизацию.

  3. Если Вы включили синхронизацию, укажите интервал синхронизации файлов, например, каждый час или каждые два дня.

    Когда клиент iFolder используется с Novell iFolder, минимальный системный интервал синхронизации обычно составляет 5 минут и конфигурируется администратором iFolder. Нельзя переопределить минимальное системное значение. Абсолютный минимальный интервал системы составляет 1 минуту.

Дополнительную информацию см. в Юридическая информация.
