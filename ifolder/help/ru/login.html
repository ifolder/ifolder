Вход с помощью учетной записи iFolder

Работать локально с файлами и каталогами iFolder можно в любое время. Чтобы иметь возможность выполнять синхронизацию списка пользователей, синхронизацию данных iFolder, передачи новых папок iFolder, загрузки совместно используемых папок iFolder, совместного использования папок и управления папками iFolder, необходимо выполнить вход с помощью учетной записью iFolder. Можно выполнить вход отдельно или одновременно для нескольких учетных записей.

Для каждой учетной записи используйте один из следующих методов входа:

  • Автоматический вход Включив функцию Запомнить пароль для учетной записи iFolder, вы будете автоматически входить в систему при запуске клиента iFolder. Включить функцию Запомнить пароль на этом компьютере можно на этапе настройки учетной записи или при изменении ее параметров позже.

    При попытке выполнить вход с учетной записью iFolder после изменения пароля автоматическая аутентификация завершается с ошибкой. Когда система iFolder предложит выполнить вход, введите новый пароль и при необходимости снова установите флажок Запомнить пароль.

  • Вход по мере необходимости Если вы планируете выполнять вход с учетной записью iFolder только по мере необходимости, снимите флажок Запомнить пароль.

Чтобы включить или отключить функцию Запомнить пароль для существующей учетной записи iFolder, выполните указанные ниже действия.

  1. В области оповещений щелкните правой кнопкой значок Сервисы iFolder , затем выберите Учетные записи, чтобы открыть диалоговое окно "Настройки iFolder" на вкладке Учетные записи.

  2. Выберите учетную запись iFolder, затем щелкните Свойства, чтобы просмотреть параметр в диалоговом окне Свойства.

  3. Щелкните вкладку Идентификация.

  4. Установите или снимите флажок Запомнить пароль, чтобы соответственно включить или отключить эту функцию.

  5. Введите пароль в поле Пароль.

  6. Щелкните Закрыть, чтобы завершить процедуру.

  7. Щелкните Закрыть в диалоговом окне Свойства iFolder, чтобы сохранить изменения и войти на сервер.

Для входа с помощью учетной записи iFolder выполните указанные ниже действия.

  1. В области оповещений щелкните правой кнопкой значокСервисы iFolder , затем выберите Учетные записи.

  2. Выберите учетную запись iFolder, затем установите флажок Интерактивный режим рядом с учетной записью.

  3. Если появится запрос на ввод пароля, введите пароль, затем щелкните Подключить.

    Если для учетной записи включена функция Запомнить пароль, запрос на ввод пароля не будет выдаваться до тех пор, пока не истечет срок действия пароля.

    Если при входе произошел сбой, появится сообщение об ошибке с описанием его причины, например, что пароль указан неправильно.

  4. После успешного завершения закройте диалоговое окно Настройки iFolder.

Дополнительную информацию см. в Юридическая информация.
