Преобразование папки iFolder в стандартную папку

На локальном компьютере папку iFolder можно преобразовать в стандартную папку. После такого преобразования отношения совместного коллективного использования остаются неизменными для всех членов iFolder, но эта папка iFolder больше не синхронизируется с текущим компьютером. При преобразовании папка iFolder и ее содержимое не удаляется с рабочих станций членов или с сервера.

  1. В менеджере файлов или в навигаторе iFolder найдите папку iFolder, которую необходимо преобразовать в стандартную папку.

  2. Для запуска преобразования папки iFolder используйте один из следующих способов:

    • В менеджере файлов щелкните правой кнопкой папку iFolder, затем щелкните Преобразовать в стандартную папку.

    • В навигаторе iFolder выберите папку iFolder и щелкните iFolder > Преобразовать в стандартную папку.

    В диалоговом окне Преобразовать папку iFolder в стандартную папку? запрашивается подтверждение и отображается пояснение.

  3. Выполните одно из следующих действий.

    • Выберите Нет, чтобы сохранить папку в качестве папки iFolder на локальном компьютере.

    • Выберите Да, чтобы преобразовать папку iFolder в стандартную папку.

      Локальная копия папки iFolder становится стандартной папкой, синхронизация с сервером больше не выполняется. Это действие не влияет на других членов папки iFolder.

    • Если пользователь является владельцем папки iFolder, он может также по своему усмотрению приостановить совместное использование данной папки в качестве папки iFolder. Выберите Удалить эту папку iFolder с сервера и щелкните Да.

      Папка iFolder будет преобразована в стандартную папку на локальном компьютере и на всех компьютерах членов. Копия папки iFolder на сервере удаляется без возможности восстановления.

Дополнительную информацию см. в Юридическая информация.
