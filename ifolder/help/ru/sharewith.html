Совместное использование папки iFolder

Папка iFolder может совместно использоваться несколькими пользователями. В данной разделе рассматриваются следующие задачи:

  • Общие сведения о правах доступа пользователя

  • Доступ к вкладке "Совместное использование"

  • Добавление пользователя iFolder

  • Изменение прав доступа пользователей

  • Удаление пользователей iFolder

  • Передача владения пользователю папки iFolder

В следующей таблице описаны возможности, связанные с каждым уровнем доступа пользователей или группы LDAPGroup.

Доступ к вкладке "Совместное использование"

На вкладке Совместное использование в диалоговом окне "Свойства iFolder" можно управлять членством в папке iFolder.

ВАЖНО.Совместное использование папок iFolder возможно только в том случае, если администратор iFolder включил политику совместного использования для Вашей учетной записи.

  1. Если приложение iFolder не запущено, запустите iFolder.

  2. Выполните одно из указанных ниже действий, чтобы открыть диалоговое окно "Свойства iFolder" на вкладке Совместное использование.

    • В менеджере файлов щелкните правой кнопкой папку iFolder, которая необходима для совместного использования, затем выберите Совместное использование.

    • В навигаторе iFolder выберите папку iFolder, затем выберите Совместное использование.

Добавление пользователя iFolder

  1. На вкладке Совместное использование щелкните Добавить, чтобы открыть диалоговое окно выбора пользователя.

  2. Из списка пользователей выберите одного или нескольких пользователей или группы LDAPGroups, добавляя их в список Добавляемые пользователи, и щелкните ОК.

    Для синхронизации пользователей на сервере потребуется несколько секунд. После синхронизации в нижнем правом углу страницы отображается оповещение.

  3. На вкладке Совместное использование выберите одного или нескольких пользователей, которым необходимо назначить одинаковое право доступа.

  4. Щелкните Права, выберите право доступа (Полный контроль, Чтение/запись или Только чтение) и нажмите ОК.

  5. Повторяйте предыдущие действия, пока все нужные пользователи или группы LDAPGroup не будут приглашены для совместного использования папки iFolder и не будут назначены их права доступа.

  6. Щелкните Закрыть, чтобы вернуться в окно свойств iFolder.

Изменение прав доступа пользователей

Поскольку рабочие взаимосвязи и состояния членов iFolder меняются, может потребоваться изменение права доступа пользователя к папке iFolder. Например Вы можете захотеть дать право полного контроля в папке iFolder пользователю, которому Вы доверяете.

  1. На вкладке Совместное использование выберите одного или нескольких пользователей, которым необходимо назначить одинаковое право доступа.

  2. Щелкните Права и выберите право доступа (Полный контроль, Чтение/запись или Только чтение).

    Если пользователь или группа LDAPGroup приглашается к участию впервые, опция Назначить пользователя владельцем папки iFolder в диалоговом окне "Доступ" отключена (отмечена серым). Если необходимо назначить нового пользователя владельцем, нужно подождать, пока состояние пользователя Приглашенный пользователь изменится на Пользователь iFolder, после чего можно передать владение. При передаче владения группе LDAPGroup все члены данной группы LDAPGroup получают права владения.

  3. Щелкните ОК, затем щелкните OK, чтобы закрыть окно "Свойства iFolder".

Удаление пользователей iFolder

Для отмены совместного использования папки iFolder необходимо удалить пользователя или группу LDAPGroup как члена папки iFolder. Папка iFolder на рабочей станции пользователя или члена группы LDAPGroup преобразуется в стандартную папку. На рабочей станции бывшего участника остается копия данных, однако файлы больше не синхронизируются с совместно используемой папкой iFolder.

  1. На вкладке Совместное использование выберите в списке одного или нескольких пользователей.

  2. Щелкните Удалить, а затем — Да.

Передача владения пользователю папки iFolder

Владелец папки iFolder может передать права владения папкой iFolder другому отдельному члену или группе LDAPGroup с состоянием "Пользователь iFolder". После передачи прав первоначальный пользователь становится членом с правами "Чтение/Запись", а пространство, выделенное для папки iFolder, будет использоваться в соответствии с новым владельцем. Когда группа LDAPGroup становится владельцем папки iFolder, все члены группы получают права владения.

  1. В менеджере файлов или в навигаторе iFolder щелкните правой кнопкой мыши папку iFolder, которую необходимо передать, а затем выберите пункт Совместное использование, чтобы открыть диалоговое окно свойств папки iFolder на вкладке Совместное использование.

  2. Из списка пользователей выберите пользователя iFolder, которого хотите назначить новым владельцем, и щелкните Права.

  3. Выберите Назначить пользователя владельцем папки iFolder.

    Опция Назначить пользователя владельцем папки iFolder в диалоговом окне "Доступ" отключена (серый цвет), если пользователь находится в состоянии Приглашенный пользователь. В этом случае закройте диалоговое окно, подождите, пока состояние пользователя Приглашенный пользователь изменится на Пользователь iFolder, затем повторите предыдущие действия по передаче владения.

  4. Щелкните ОК, затем щелкните OK, чтобы закрыть окно "Свойства iFolder".
