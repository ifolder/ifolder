Перемещение папки iFolder

После создания папки iFolder путь папки iFolder становится постоянным. Чтобы папка осталась папкой iFolder, ее нельзя перемещать с исходного места в другое место локальной рабочей станции. При перемещении папки iFolder теряется связь между базой данных синхронизации и папкой, которую Вы преобразовали в iFolder.

Для перемещения папки iFolder на компьютере, на котором она была создана, выполните следующие действия:

  1. Преобразуйте папку iFolder в обычную папку.

  2. Переместите папку.

  3. Преобразуйте перемещенную папку в папку iFolder.

  4. Установите совместное использование для папки iFolder (необязательно).

Для перемещения папки iFolder на компьютер другого пользователя выполните следующие действия:

  1. В папках iFolder приостановите участие iFolder, но так, чтобы оно оставалась доступным для Вас.

  2. Выберите нужную папку iFolder, затем щелкните "Настройка".

  3. Укажите новый локальный путь, куда нужно поместить папку iFolder.

  4. Разрешите синхронизацию файлов папки iFolder с сервером.
