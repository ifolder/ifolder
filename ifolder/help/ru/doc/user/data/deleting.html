Удаление папки iFolder

При удалении папки iFolder из набора происходит удаление локальной копии папки iFolder и ее содержимого. Удаление папки также может повлиять на совместное использование папки iFolder в зависимости от Ваших прав доступа на эту папку. При удалении папки iFolder программное обеспечение клиента iFolder с рабочей станции не удаляется.

  1. В менеджере файлов или в средстве просмотра папок iFolder найдите папку iFolder, которую нужно удалить.

  2. Щелкните iFolder правой кнопкой, затем щелкните "Удалить".
