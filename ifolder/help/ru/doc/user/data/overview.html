Обзор iFolder

iFolderTM – это приложение для совместного использования файлов клиентами Linux, Windows* и Macintosh*. Вы можете совместно использовать файлы в нескольких папках iFolder, каждая из которых предназначена для определенной группы пользователей. Вы можете управлять пользователями папок iFolder и устанавливать уровень доступа для каждого члена. Кроме того, Вы можете стать пользователем папок iFolder, доступ к которым был предоставлен коллегами.
