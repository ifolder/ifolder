Преобразование папки iFolder в стандартную папку

Любой член может преобразовать папку iFolder на своем компьютере в стандартную папку. После такого преобразования отношения коллективного совместного использования остаются неизменными для всех членов iFolder, но эта папка iFolder больше не синхронизируется с данным компьютером. Это преобразование не удаляет папку iFolder и ее содержимое с рабочих станций членов.

  1. В менеджере файлов или на рабочем столе найдите папку iFolder, которую необходимо преобразовать в стандартную папку.

  2. Воспользуйтесь одним из следующих способов:

    • Щелкните правой кнопкой iFolder, затем щелкните iFolder > Преобразовать в стандартную папку.
    • Выберите папку iFolder, затем на панели инструментов менеджера файлов щелкните Файл > iFolder > Преобразовать в стандартную папку.

    Значок iFolder будет изменен на значок стандартной папки. На всех компьютерах членов данные останутся неизмененными.
