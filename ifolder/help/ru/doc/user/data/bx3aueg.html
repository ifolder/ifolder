Совместное использование папки iFolder

Обычно, если Вы работаете в нескольких местоположениях или совместно работаете с другими пользователями, необходимо постоянное управление версиями файлов. Чтобы получить самую последнюю версию файлов на всех компьютерах, на которых Вы работаете, достаточно установить на них клиент iFolder и настроить совместное использование папки iFolder. iFolder также позволяет совместно использовать несколько папок iFolder и их содержимое с другими авторизованными пользователями. Вы решаете, какие пользователи смогут получать доступ к папкам iFolder, а также устанавливаете права доступа каждого члена. Подобным образом Вы можете стать пользователем совместно используемых папок iFolder, владельцами которых являются другие сотрудники Вашей организации.

Клиент iFolder поддерживает совместное использование файлов путем их синхронизации на различных компьютерах через центральный компьютер. С его помощью можно выполнять следующие задачи:

  • совместно использовать файлы на различных компьютерах;
  • совместно использовать файлы с другими пользователями;
  • владеть несколькими папками iFolder;
  • использовать различные папки iFolder, доступ к которым был предоставлен другими пользователями;
  • быть владельцем или членом папок iFolder с помощью различных учетных записей.
