Запуск клиента iFolder


Запуск клиента iFolder для Novell Linux Desktop 9

  1. Зарегистрируйтесь в компьютере в качестве пользователя, которого необходимо использовать для создания папок iFolder.

  2. Для запуска iFolder используйте один из следующих способов:

    • Откройте оболочку терминала и введите:

      /opt/novell/ifolder3/bin/ifolder

    • Если необходимо, чтобы клиент iFolder автоматически запускался при регистрации, убедитесь в том, что клиент iFolder работает при завершении сеанса GNOME, затем сохраните сеанс. Кроме того, можно открыть окно терминала, зарегистрироваться в качестве пользователя "root", затем ввести команду /opt/gnome/bin/gnome-session-properties, чтобы добавить iFolder в качестве программы, которая будет загружаться при регистрации в компьютере.

  3. Если отключить функцию "Запомнить пароль" для учетной записи iFolder, для синхронизации списка пользователей, папок iFolder, а также управления папками iFolder необходимо будет регистрироваться с помощью учетной записи.

    Для получения дополнительной информации см. раздел Регистрация с помощью учетной записи iFolder.


Запуск iFolder для Windows 2000/XP

  1. Зарегистрируйтесь в компьютере в качестве пользователя Windows, которого необходимо использовать при создании папок iFolder.

  2. Если клиент iFolder не запускается автоматически при успешной регистрации, запустите iFolder, щелкнув "Пуск" > "Программы" > "iFolder", затем выбрав приложение iFolder.

    iFolder не запускается автоматически, если в настройках iFolder отключен параметр "Запуск iFolder при регистрации в рабочей станции".

  3. Если отключить функцию "Запомнить пароль" для учетной записи iFolder, для синхронизации списка пользователей, папок iFolder, а также управления папками iFolder необходимо будет регистрироваться с помощью учетной записи.

    Для получения дополнительной информации см. раздел Регистрация с помощью учетной записи iFolder.


Запуск iFolder для Macintosh OS X v10.3 и более поздних версий

  1. Зарегистрируйтесь в компьютере в качестве пользователя, которого необходимо использовать для создания папок iFolder.

  2. Для запуска iFolder используйте один из следующих способов:

    • Откройте меню "Applications" (Приложения), выберите подменю "Internet" (Интернет), затем выберите "iFolder".
    • Для запуска iFolder из командной строки откройте оболочку терминала и введите команду:

      /opt/novell/ifolder3/bin/ifolder

    • Если необходимо, чтобы клиент iFolder запускался автоматически при регистрации в компьютере Macintosh, добавьте iFolder в "Startup Items" (Автозагрузка).

  3. Если отключить функцию "Запомнить пароль" для учетной записи iFolder, для синхронизации списка пользователей, папок iFolder, а также управления папками iFolder необходимо будет регистрироваться с помощью учетной записи.

    Для получения дополнительной информации см. раздел Регистрация с помощью учетной записи iFolder.
