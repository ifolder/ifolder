Синхронизация файлов


Синхронизация файлов по требованию

  1. В области оповещения щелкните правой кнопкой значок приложения iFolder , затем щелкните "Папки iFolder" для открытия навигатора iFolder.

  2. Выберите папку iFolder из списка папок.

  3. Выполните одно из следующих действий:

    • Щелкните "Синхронизировать" .
    • Выберите iFolder > Синхронизировать сейчас.


Конфигурирование интервала синхронизации

  1. В области оповещения щелкните правой кнопкой значок приложения iFolder , затем щелкните "Настройки" для открытия диалогового окна "Настройки iFolder" на вкладке "Общие".

  2. Выполните одно из следующих действий:

    • Выберите "Синхронизировать" для включения фоновой синхронизации.
    • Отмените выбор параметра "Синхронизировать" для отключения синхронизации.

  3. Если Вы выбрали включение синхронизации, укажите интервал синхронизации файлов, например, каждый час или каждые 2 дня.

    Когда клиент iFolder используется с Novell iFolder 3.x, минимальный системный интервал синхронизации составляет обычно 5 минут и задается администратором iFolder. Вы не можете переопределить этот системный минимум. Абсолютный минимальный интервал системы составляет 1 минуту.
