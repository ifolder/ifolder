Настройка доступной папки iFolder

Когда другие лица используют папки iFolder совместно с Вами, эти папки появляются в навигаторе iFolder со значком доступной папки iFolder. Для участия в совместном использовании папки iFolder на текущем локальном компьютере установите папку iFolder, сделав ее активной на нем.

Локальная папка iFolder должна удовлетворять требованиям, описанным в следующих рекомендациях и соглашениях:

  • Рекомендации по расположению папок iFolder
  • Соглашения об именовании папки iFolder, содержащихся в ней папок и файлов

Для настройки iFolder выполните следующие действия:

  1. В области оповещения щелкните правой кнопкой значок iFolder , затем щелкните "Папки iFolder" для открытия средства просмотра iFolder.

  2. В списке папок iFolder выберите доступную папку iFolder и щелкните "Настройка" .

  3. Если Вы не зарегистрированы с помощью учетной записи, зарегистрируйтесь, используя свое имя пользователя и пароль, затем щелкните кнопку "ОК".

  4. Выберите папку, в которой нужно создать локальную папку iFolder.

  5. Дважды щелкните кнопку "ОК".
