Конфигурирование настроек iFolder для клиента

  1. В области оповещения щелкните правой кнопкой значок приложения iFolder , затем выберите "Настройки" для открытия диалогового окна "Настройки iFolder" на вкладке "Общие".


  2. Укажите следующие значения:

    Параметр Описание

    Запуск iFolder при регистрации в рабочей станции

    Выберите этот параметр, чтобы клиент iFolder запускался при регистрации локального пользователя, под учетной записью которого устанавливался клиент iFolder. Если для учетной записи указан параметр "Запомнить пароль", синхронизация будет происходить в фоновом режиме с указанным интервалом.

    Для запуска iFolder вручную отмените выбор этого параметра.

    Отображение диалогового окна подтверждения при создании папок iFolder

    Выберите этот параметр, чтобы получать сообщение о подтверждении при каждом создании папки iFolder в дополнение к обычным визуальным эффектам, например, появление iFolder в списке папок iFolder.

    Отмените этот параметр, чтобы не получать сообщения о подтверждении.

    Уведомление о совместно используемых папках iFolder

    Выберите данный параметр, чтобы получать всплывающие сообщения в области оповещения при получении новых приглашений на совместное использование iFolder.

    Приглашения поступают в соответствии с интервалом синхронизации, который указан в настройках iFolder. Интервал контролирует синхронизацию файлов, приглашений на совместное использование iFolder и списка пользователей. Можно щелкнуть "Обновить", чтобы обновить информацию о приглашениях вручную.

    Отмените выбор этого параметра, чтобы не получать всплывающие сообщения, относящиеся к приглашениям.

    Уведомлять о конфликтах

    Выберите этот параметр, чтобы разрешить отображение всплывающего сообщения в области оповещения, уведомляющего о возникновении конфликта при синхронизации файлов.

    Отмените выбор этого параметра, чтобы не получать всплывающего сообщения, уведомляющего о конфликтах при синхронизации.

    Уведомление о присоединении пользователей

    Выберите данный параметр, чтобы разрешить отображение всплывающего сообщения в области оповещения при принятии пользователями приглашения на совместное использование папки iFolder. Под принятием подразумевается действительная установка iFolder на пользовательском компьютере.

    Отмените выбор этого параметра, чтобы не получать всплывающего сообщения, уведомляющего о принятии приглашения на совместное использование информации.

    Синхронизация

    Автоматически синхронизировать папки iFolder. Установите этот флажок, чтобы включить синхронизацию всех папок iFolder на данном компьютере при регистрации/входе в систему пользователя. Отмените флажок, чтобы выключить синхронизацию.

    Интервал синхронизации папок iFolder. Укажите минимальный интервал, который будет использоваться для синхронизации папок iFolder на данном компьютере при регистрации/входе в систему пользователя. Укажите значение и единицы измерения.

    Если клиент iFolder используется с Novell iFolder 3.x, эффективным минимальным интервалом синхронизации всегда является самое высокое значение следующих параметров:

    • Системная политика (по умолчанию имеет нулевое значение), если не установлена пользовательская политика. Если установлена пользовательская политика, она переопределяет системную политику, независимо от величины значения.
    • Политика локального компьютера или настройка синхронизации с сервером на клиентской рабочей станции.
    • Политика iFolder (совокупная).

  3. По завершении щелкните "Применить", чтобы изменения вступили в силу.

  4. Щелкните "OK" для закрытия диалогового окна "Настройки iFolder".
