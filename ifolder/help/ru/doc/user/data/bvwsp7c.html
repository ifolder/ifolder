Удаление учетной записи iFolder

Вы можете указать, нужно ли удалить только учетную запись с текущего компьютера или необходимо также удалить все папки iFolder, владельцем которых Вы являетесь на сервере. При этом приостанавливается совместное использование папок iFolder, которые были установлены для данной учетной записи.

При удалении учетной записи iFolder с компьютера относящиеся к ней локальные копии папок iFolder будут преобразованы в стандартные папки. Вы можете по-прежнему получать доступ к папкам iFolder с других компьютеров или через Интернет.

Если учетная запись iFolder удалена с сервера:

  • Совместное использование всех папок iFolder будет отменено.
  • Локальные копии папок iFolder на компьютерах членов будут преобразованы в стандартные папки.
  • Папка iFolder и ее содержание будут удалены с сервера.
  • Ваше членство в папках iFolder, совместно используемых с другими пользователями, удаляется.

  1. В области оповещения щелкните правой кнопкой значок приложения iFolder , затем выберите "Учетные записи" для открытия диалогового окна "Настройки iFolder" на вкладке "Учетные записи".

  2. Выберите учетную запись iFolder, которую необходимо удалить, затем щелкните "Удалить".

  3. Появится сообщение с предложением выбрать варианты действия удаления. Выполните одно из следующих действий:

    • Удалите учетную запись iFolder только с текущего компьютера. Отмените выбор параметра "Удалить мои папки iFolder и файлы с сервера", затем щелкните "Да". Программа iFolder удалит выбранную учетную запись только с локальной рабочей станции. Локальные папки iFolder для данной учетной записи будут вновь преобразованы в стандартные папки. С помощью менеджера файлов найдите и удалите локальные копии данных, если они не нужны.

      Вы можете продолжить использовать папки iFolder для данной учетной записи с других компьютеров с клиентом iFolder. Если клиент iFolder используется с сервером Novell® iFolder® 3.x, можно получить доступ к файлам с любого компьютера с помощью веб-доступа Novell iFolder 3.x.

    • Удалите учетную запись iFolder, папки iFolder и файлы с сервера. Выберите "Удалить мои папки iFolder и файлы с сервера", затем щелкните "Да". Программа iFolder удалит выбранную учетную запись с локальной рабочей станции. С сервера также будут удалены все папки iFolder для данной учетной записи, владельцем которых являетесь Вы, а на текущем компьютере и компьютерах других членов папки iFolder будут преобразованы в стандартные папки. Кроме этого, после выполнения данного действия учетная запись будет удалена из списка членов, совместно использующих папки iFolder. Все локальные папки iFolder будут преобразованы в стандартные папки. С помощью менеджера файлов найдите и удалите локальные копии данных, если они не нужны.

      Если данную учетную запись потребуется использовать снова на этом или другом компьютере, необходимо повторно установить учетную запись и папки iFolder.

    • Отмените удаление. Щелкните "Нет" для отмены удаления. Ваша учетная запись и папки iFolder не будут изменены.
