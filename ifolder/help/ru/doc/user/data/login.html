Регистрация с помощью учетной записи iFolder

После настройки учетной записи iFolder нужно зарегистрироваться, когда необходимо синхронизировать локальные данные iFolder, список пользователей iFolder и совместно используемые оповещения iFolder. Можно зарегистрироваться с помощью нескольких учетных записей одновременно. Для каждой учетной записи используйте один из следующих способов регистрации:

  • Включите функцию "Запомнить пароль", чтобы регистрация с помощью учетной записи iFolder выполнялась автоматически при регистрации в рабочей станции или при запуске клиента iFolder.

    После смены сетевого пароля iFolder при следующей попытке регистрации с помощью учетной записи iFolder произойдет сбой автоматической аутентификации. Программа iFolder попросит Вас зарегистрироваться с новым паролем и позволит определить и сохранить его.

  • Отключите функцию "Запомнить пароль", чтобы регистрироваться с помощью учетной записи iFolder только тогда, когда нужно синхронизировать файлы в соответствующих папках iFolder или управлять папками iFolder.

Для включения или выключения функции "Запомнить пароль" существующей учетной записи iFolder выполните следующие действия:

  1. Щелкните правой кнопкой мыши значок iFolder в области оповещений и выберите "Учетные записи".

  2. Выберите учетную запись iFolder для просмотра ее настроек.

  3. Выберите (включите) функцию "Запомнить пароль" или отмените ее выбор (выключите).

  4. Укажите пароль и нажмите "OK" или "Применить".

  5. Нажмите "OK" для проверки изменений настроек учетной записи или щелкните "Отмена" для отказа от изменений.

Для регистрации с помощью учетной записи iFolder выполните следующие действия:

  1. Щелкните правой кнопкой мыши значок iFolder в области оповещений и выберите "Учетные записи".

  2. Выберите учетную запись iFolder для просмотра ее настроек.

  3. Укажите пароль и щелкните "Регистрация".

    При успешной регистрации состояние учетной записи сменится на "Зарегистрирован".

    Если при регистрации произошел сбой, появится сообщение об ошибке с описанием его причины, например, что пароль указан неправильно.

  4. Щелкните "OK" для закрытия диалогового окна "Учетные записи".
