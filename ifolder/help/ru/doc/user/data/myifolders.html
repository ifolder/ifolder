Управление папками iFolder

Папка iFolderTM – это локальный каталог, который используется для выборочной синхронизации файлов и их совместного использования с определенными пользователями. Файлы в папках iFolder доступны всем пользователям, которые могут изменять и просматривать их в зависимости от привилегий.

В этой главе рассматриваются следующие задачи iFolder:

  • Рекомендации по расположению папок iFolder
  • Рекомендации по типам и размерам не синхронизируемых файлов
  • Соглашения об именовании папки iFolder, содержащихся в ней папок и файлов
  • Создание папки iFolder
  • Совместное использование папки iFolder
  • Настройка доступной папки iFolder
  • Просмотр и конфигурирование свойств папки iFolder
  • Синхронизация файлов
  • Разрешение конфликтов файлов
  • Преобразование папки iFolder в стандартную папку
  • Перемещение папки iFolder
  • Удаление папки iFolder
