Клиент iFolder

Клиент iFolder интегрируется с операционной системой и предоставляет сервисы iFolder в соответствующей вычислительной среде рабочей станции. Продукт iFolder поддерживает следующие операционные системы:

  • Novell Linux Desktop 9
  • Windows 2000/XP
  • Macintosh OS X v10.3 или более поздней версии

Клиент iFolder позволяет управлять собственными папками iFolder и выбрать, какие совместно используемые папки iFolder необходимо установить на каждом компьютере. Дополнительную информацию см. в разделе Управление папками iFolder.


Учетная запись iFolder

Перед созданием папок iFolder необходимо сконфигурировать, как минимум, одну учетную запись сервиса iFolder. Сеанс iFolder начинается при регистрации с помощью учетной записи сервиса iFolder и завершается при разрегистрации или выходе пользователя из клиента iFolder. Файлы в папках iFolder синхронизируются с другими активными папками iFolder только при активном сеансе и при работе в режиме прямого подключения. Доступ к локальным папкам iFolder можно получить в любое время независимо от того, есть регистрация с помощью учетной записи или нет. Для получения дополнительной информации см. раздел Конфигурирование учетной записи iFolder.


Совместное использование папок iFolder

Папка iFolder представляет собой локальный каталог, файлы которого могут совместно использоваться группой пользователей и синхронизироваться через центральный компьютер. Файлы iFolder доступны всем членам папки iFolder и могут изменяться теми пользователями, кто обладает соответствующими правами. Доступ к папкам iFolder можно получить с различных рабочих станций и совместно использовать их с другими пользователями.

Работа с папками iFolder осуществляется непосредственно из менеджера файлов или с помощью средства просмотра iFolder, которое входит в состав клиента. В каталоге iFolder можно создать любую структуру подкаталогов в соответствии с личными или корпоративными потребностями. Структура подкаталогов одинакова для всех членов папок iFolder. При совместном использовании папки iFolder пользователи с правом "Запись" могут изменять структуру каталогов, после чего изменения применяются для всех копий папки iFolder. Каждый пользователь может указать на любом диске собственного компьютера совместно используемую папку iFolder, удовлетворяющую Рекомендациям по расположению папок iFolder.


Права доступа к iFolder

Клиент iFolder поддерживает владельца и три уровня доступа для членов папки iFolder:

  • Владелец. Только один пользователь может быть владельцем папки iFolder. Как правило, это пользователь, который создает папку iFolder. Владелец может использовать клиент iFolder для передачи права на владение другому члену папки iFolder.

    Владелец папки iFolder обладает правом "Полный контроль". Этот пользователь имеет доступ для чтения/записи к папке iFolder, управляет членством и правами доступа пользователей, а также может удалять право "Полный контроль" для любого члена.

  • Полный контроль. Осуществляется членом совместно используемой папки, обладающим правом доступа на полный контроль. Этот член имеет доступ на чтение/запись в папке iFolder, управляет членством и правами доступа всех пользователей, за исключением владельца.

  • Чтение/Запись. Это право получает член совместно используемой папки iFolder для доступа на "Чтение/Запись" к каталогам и файлам в папке iFolder.

  • Только чтение. Это право получает член совместно используемой папки iFolder для доступа "Только чтение" к каталогам и файлам в папке iFolder. Данный член может изменять свой файл, но эти локальные изменения не синхронизируются с копиями других членов папки iFolder.

Любой пользователь iFolder может владеть несколькими папками iFolder и быть членом других папок iFolder. Владелец всегда обладает правом "Полный контроль". Пользователи могут иметь различный уровень доступа к каждой совместно используемой папке iFolder в зависимости от права доступа, устанавливаемого членом с правом "Полный контроль".


Синхронизация файлов и управление данными

При установке учетной записи iFolder пользователь может включить функцию "Запомнить пароль", благодаря которой iFolder может синхронизировать файлы и приглашения iFolder в фоновом режиме во время работы пользователя.

Можно настроить программное обеспечение клиента iFolder для автоматического запуска при каждой регистрации пользователя на рабочей станции. Сеанс выполняется в фоновом режиме, пока пользователь работает со своими файлами в локальной папке iFolder. При этом осуществляется отслеживание и протоколирование всех выполняемых изменений.

Можно указать интервалы синхронизации файлов или синхронизировать их по требованию.
