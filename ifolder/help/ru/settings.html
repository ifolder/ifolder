Параметры iFolder

Вкладку "Параметры iFolder" можно использовать для включения уведомлений.

  1. Чтобы включить уведомления, установите флажок Показывать уведомления.

  2. В списке уведомлений выберите события, для которых необходимо включить уведомления.

Дополнительную информацию см. в Юридическая информация.
