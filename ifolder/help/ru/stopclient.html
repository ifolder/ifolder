Завершение работы с клиентом iFolder

Для выхода из клиента iFolder выполните одну из следующих процедур:

  • В области оповещений щелкните правой кнопкой значокСервисы iFolder , затем выберите Выход.

  • Щелкните iFolder > Выход

    При выборе функции Выход в приложении iFolder отображается сообщение о подтверждении. Для подтверждения действия щелкните Да.

Файлы синхронизируются с серверной учетной записью iFolder, только если соединение с сервером iFolder установлено. Можно остановить синхронизацию, разрегистрировав учетную запись. При переходе в автономное состояние приложение iFolder продолжает работать в фоновом режиме. См. дополнительные сведения: Разрегистрация учетной записи iFolder.

Возможно наличие нескольких учетных записей пользователя, созданных на различных хост-серверах (доменах).

Дополнительную информацию см. в Юридическая информация.
