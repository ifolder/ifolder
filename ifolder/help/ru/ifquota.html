Настройка квоты для папки iFolder

Квоту для собственных папок iFolder установить нельзя. Администратор iFolder устанавливает квоту для Вашей учетной записи пользователя. Администратор может установить квоту на четырех различных уровнях: уровень системы, уровень группы LDAPGroup, уровень пользователя и уровень iFolder.

Дополнительную информацию о просмотре свойств квоты, установленной для вашей учетной записи, см. в указанных ниже разделах.

  • Квота

  • Параметры хранения папок iFolder на сервере iFolder

Если администратором включена квота iFolder, которая меньше общего пространства, занимаемого данными iFolder в текущий момент, синхронизация останавливается до тех пор, пока объем данных не уменьшится и не перестанет превышать ограничение, или значение квоты не будет увеличено и не начнет превышать общий используемый объем пространства.

Дополнительную информацию см. в Юридическая информация.
