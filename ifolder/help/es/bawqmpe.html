Cuenta de iFolder

Se creará una cuenta de iFolder cuando un administrador de servicios de iFolder le asigne como usuario de iFolder en un servidor iFolder. El administrador le proporcionará un nombre de usuario y una contraseña. En el caso de una empresa, podrá tratarse del nombre de usuario o de la dirección de correo electrónico y la contraseña asociada. Antes de crear carpetas iFolder, debe configurar una cuenta de iFolder como mínimo. Para obtener más información, consulte la Configuración de una cuenta de iFolder.

Una sesión de iFolder empieza al entrar en una cuenta de servicios de iFolder y finaliza al salir de la misma, o bien al salir del Cliente iFolder. Las carpetas iFolder sólo sincronizan archivos con otras carpetas iFolder activas cuando la sesión está activa y está funcionando en línea. Puede acceder a los datos de las carpetas iFolder locales en cualquier momento, independientemente de si ha entrado en la cuenta o no. Para obtener más información, consulte la Configuración de una cuenta de iFolder.

Para obtener más información, consulte Información legal.
