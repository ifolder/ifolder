Descripción general de iFolder

Novell iFolder es una aplicación para el uso compartido de archivos destinada a clientes Linux, Windows y Macintosh. Es posible compartir archivos en varias carpetas iFolder, cada una con diferentes usuarios o grupos LDAP. El propietario controla quién puede participar en una carpeta iFolder y el nivel de acceso de cada miembro. También puede participar en carpetas iFolder que otros comparten con usted.

iFolder proporciona una interfaz sencilla y directa para la migración del cliente de iFolder 2.x a iFolder 3.6 y versiones posteriores, así como para la actualización del cliente de iFolder 3.x a iFolder 3.6 y versiones posteriores.
