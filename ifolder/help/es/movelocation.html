Movimiento de una carpeta iFolder

Una vez que se ha creado una carpeta iFolder, la vía a la misma es estática. No es posible mover una carpeta iFolder de la ubicación original a otra ubicación en el equipo local sin que deje de ser una carpeta iFolder. Al mover una carpeta iFolder se rompe el enlace entre la base de datos de sincronización y la carpeta que se ha convertido en una carpeta iFolder.

Para mover una carpeta iFolder en el equipo donde se ha creado:

  1. En un gestor de archivos o navegador iFolder, seleccione la carpeta iFolder que desea mover.

  2. Revierta la carpeta iFolder a una carpeta normal.

  3. En un gestor de archivos, mueva la carpeta a otra ubicación del equipo.

  4. Vuelva a convertir la carpeta en una carpeta iFolder y cárguela en el servidor.

  5. (Opcional) Comparta la carpeta iFolder.

  6. Permita que la carpeta iFolder sincronice archivos con el servidor iFolder.

Para mover una carpeta iFolder en un equipo participante:

  1. En las carpetas iFolder, cambie la carpeta iFolder para que deje de participar en la carpeta iFolder del equipo actual.

  2. Seleccione la carpeta iFolder disponible y, a continuación, haga clic en Descargar una carpeta iFolder.

  3. Especifique una vía local nueva donde desee colocar la carpeta iFolder.

  4. Permita que la carpeta iFolder sincronice archivos con el servidor iFolder.

Para obtener más información, consulte Información legal.
