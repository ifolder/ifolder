Configuración de una cuenta de iFolder

Utilice el Asistente de creación de cuenta de iFolder para añadir y configurar una cuenta nueva. Debe configurar una cuenta de iFolder como mínimo antes de crear o compartir carpetas iFolder en el sistema. Puede conectarse al servidor iFolder con una única cuenta en cualquier momento. Sin embargo, para conectarse a más de un servidor iFolder, puede configurar diferentes cuentas. Puede entrar en cada cuenta por separado o bien entrar simultáneamente en varias de ellas.

Varios usuarios con identidades de entrada locales distintas tendrán carpetas iFolder en el mismo equipo. Cuando inicie iFolder por primera vez con la identidad de entrada local actual, iFolder le solicitará que configure una cuenta de iFolder.

IMPORTANTE:La nueva función de creación automática de cuentas de iFolder 3.7 y versiones posteriores permite al administrador de iFolder crear y configurar una cuenta para el usuario. Si el administrador de iFolder ha habilitado la creación automática de cuentas, no será necesario que el usuario la configure por su cuenta. Al iniciar iFolder por primera vez, el sistema le solicitará que introduzca la contraseña. Introduzca la contraseña para llevar a cabo la configuración de la cuenta.

Para configurar una cuenta nueva:

  1. Acceda al Asistente de creación de cuenta de iFolder a través de uno de los métodos siguientes:

    • Cuando inicie iFolder por primera vez con la identidad de entrada local actual, iFolder le solicitará que configure una cuenta de iFolder. Haga clic en Adelante para acceder directamente al Asistente de creación de cuenta de iFolder.

    • Haga clic con el botón derecho del ratón en el icono Servicios de iFolder del área de notificaciones, seleccione Cuentas para abrir la pestaña Cuentas del recuadro de diálogo Preferencias de iFolder y, a continuación, Añadir para abrir el Asistente de creación de cuenta de iFolder.

  2. Siga las instrucciones que aparecen en pantalla para especificar los valores siguientes:

    Parámetro

    Descripción

    Dirección del servidor

    Nombre DNS o dirección IP del servidor empresarial iFolder en el que tiene una cuenta. Solicite al administrador de iFolder esta información.

    Por ejemplo: svr21.ejemplo.com o 192.168.1.1

    El nombre del servidor se muestra siempre junto con su dirección IP en la interfaz de usuario de iFolder. Por ejemplo: iFolder - 192.168.1.1

    IMPORTANTE:No es posible utilizar un Cliente iFolder 3.7 o posterior para conectarse con el servidor iFolder 3.2.

    Definir esto como mi servidor por defecto

    Si dispone de varios servidores conectados a su cuenta, seleccione el recuadro de verificación Definir esto como mi servidor por defecto para establecer el servidor seleccionado de la lista desplegable de servidores como el servidor por defecto cuando cree o cargue una carpeta iFolder.

    Usuario

    Especifique su identidad de usuario para esta cuenta. El administrador de iFolder decide si el formato que debe utilizarse es el ID de usuario de red (nombre común de LDAP) o la dirección de correo electrónico.

    Por ejemplo, si su nombre es John Smith, con el ID de usuario jsmith y la dirección de correo electrónico john.smith@ejemplo.com, el administrador configura un valor para el servidor que determina cuál de los dos formatos se aceptará en el campo Nombre de usuario con fines de autenticación.

    Contraseña

    Especifique la contraseña correspondiente a su nombre de usuario.

    La autenticación tiene lugar a través de un Canal seguro (SSL). El sistema le solicitará que valide y acepte el certificado de servidor.

    Recordar contraseña

    Seleccione el recuadro de verificación Recordar contraseña en este equipo para entrar automáticamente en la cuenta de iFolder cada vez que inicie sesión en el equipo.

    Si cambia la contraseña de red, la autenticación automática no se realizará correctamente la próxima vez que iFolder intente entrar en la cuenta de iFolder. iFolder le solicitará que entre con la nueva contraseña y le permitirá especificar y guardar la contraseña nueva.

  3. Verifique la información que ha introducido. Para hacer cualquier cambio, haga clic en el botón Atrás y vuelva a visitar las páginas anteriores. Haga clic en Conectar para continuar y validar la conexión con el servidor.

  4. Si se le pregunta si desea Aceptar el certificado de este servidor, haga clic en Detalles para revisar la información del certificado y, a continuación, haga clic en Aceptar para aceptarlo.

    NOTA:Durante la creación de cuentas, la entrada se produce a través de SSL. Cuando el sistema le solicita que acepte el certificado del servidor, debe validar el certificado y aceptarlo para que la conexión se realice correctamente.

  5. Si el administrador ha habilitado la directiva de cifrado, iFolder le solicitará que introduzca la contraseña codificada.

    Introduzca la contraseña codificada en el campo Contraseña codificada.

    Vuelva a introducir la contraseña codificada en el campo Reescribir la contraseña codificada.

    Seleccione el recuadro de verificación Recordar contraseña codificada para acceder automáticamente a los datos cifrados de iFolder durante la sesión actual. Si selecciona esta opción, iFolder puede recordarle la contraseña codificada durante el siguiente proceso de entrada. Este intercambio de contraseña codificada se realiza automáticamente.

  6. Seleccione el agente de recuperación que desea de la lista y haga clic en Aceptar.

  7. Especifique la ubicación en la que desea crear la carpeta iFolder por defecto.

    Por defecto, la carpeta iFolder se almacenará en <directorio_personal_usuarios>/ifolder/nombre_servidor/nombre_usuario

  8. Seleccione el tipo de carpeta iFolder por defecto que desea crear. Si selecciona Cifrado, se le pedirá que indique la contraseña codificada.

    IMPORTANTE:Únicamente es posible compartir las carpetas iFolder normales. No es posible compartir las carpetas iFolder cifradas

    Si su administrador de iFolder no ha habilitado el cifrado para su cuenta actual, sólo podrá crear carpetas iFolder normales y si el cifrado está activado, la opción Normal aparecerá seleccionada por defecto.

  9. Seleccione SecureSync para habilitar la sincronización segura de los archivos en la carpeta iFolder que está creando.

    Es posible habilitar un canal seguro tanto para las carpetas iFolder normales como cifradas. Cuando esta opción se encuentra seleccionada, la carpeta iFolder permite utilizar un canal seguro para establecer una sincronización entre la carpeta iFolder del servidor y el equipo local.

    Si elimina la carpeta iFolder por defecto del servidor, la próxima vez que configure la cuenta desde cualquier thick client (cliente pesado), iFolder le solicitará que vuelva a crear una carpeta iFolder por defecto.

    Sin embargo, si suprime la carpeta iFolder por defecto del equipo local, la próxima vez que configure la cuenta, iFolder le solicitará descargar la carpeta iFolder por defecto.

  10. Si ya ha configurado una carpeta iFolder por defecto, puede descargarla en el equipo local.

    Descargue la carpeta iFolder por defecto y haga clic en Siguiente para continuar.

  11. Si ya existe en el sistema una carpeta con el mismo nombre que la carpeta iFolder por defecto, se le solicitará que fusione las dos carpetas o descargue la carpeta iFolder por defecto en una ubicación diferente.

    Haga clic en Aceptar para fusionar las dos carpetas o haga clic en Cancelar para descargar la carpeta iFolder por defecto en una ubicación diferente.

  12. Haga clic en Finalizar para cerrar el Asistente de creación de cuenta de iFolder.

    iFolder sincroniza la lista de usuarios de la cuenta especificada e identifica las carpetas iFolder disponibles para la descarga. Esta descarga inicial puede durar desde unos segundos hasta varios minutos, en función del tamaño de la lista de usuarios de la cuenta.

  13. Verifique que la cuenta que ha configurado aparece en la lista de cuentas.

    1. Haga clic con el botón derecho del ratón en el icono Servicios de iFolder del área de notificaciones y, a continuación, seleccione Preferencias en el menú para abrir la pestaña Cuentas del recuadro de diálogo Preferencias de iFolder.

      Si está conectado con el servidor iFolder, se seleccionará la casilla de verificación Conectado.

    2. Cierre el recuadro de diálogo Preferencias de iFolder.

  14. Para configurar carpetas iFolder para la cuenta, utilice uno de estos métodos:

    • Haga clic con el botón derecho del ratón en el icono Servicios de iFolder del área de notificaciones y, a continuación, seleccione Carpetas iFolder para abrir el navegador de iFolder. Continúe con la Creación y carga de una carpeta iFolder.

    • Si existen carpetas iFolder de esta cuenta disponibles para descargarlas del servidor, aparecerá una lista en Carpetas iFolder en la ventana iFolder en <nombre_servidor_iFolder>. El icono de descarga indica que la carpeta iFolder está disponible para descargarla. Continúe con la Descarga de una carpeta iFolder disponible.

Para obtener más información, consulte Información legal.
