Creación y carga de una carpeta iFolder

Es posible crear una carpeta iFolder mediante la conversión de una carpeta existente que cumpla con las directrices siguientes:

  • Directrices para la ubicación y utilización de carpetas iFolder

  • Convenciones de denominación de una carpeta iFolder y de las carpetas y archivos que contiene

Si la carpeta no existe, puede examinar una ubicación y crear la carpeta nueva como parte del proceso.

IMPORTANTE:Únicamente es posible compartir las carpetas iFolder normales. No es posible compartir las carpetas iFolder cifradas

En la carpeta de iFolder, es posible configurar cualquier estructura de subdirectorios que se adecue a los hábitos de trabajo personales o de la empresa. La estructura de subdirectorios es constante en todas las carpetas iFolder miembro. Cada estación de trabajo puede especificar un directorio padre diferente para la carpeta iFolder compartida.

Cuando se supera el número límite de carpetas iFolder que un usuario puede crear, iFolder muestra un mensaje de error. Por ejemplo, si el administrador ha establecido la directiva de Número de carpetas por usuario en diez e intenta crear la carpeta iFolder número once, recibirá una notificación acerca de la infracción de la directiva.

Linux

Para crear carpetas iFolder puede utilizar un gestor de archivos o bien el navegador de iFolder, disponible en la aplicación del cliente. Para crear una carpeta iFolder, utilice los métodos siguientes:

  • Creación de carpetas iFolder en el navegador de carpetas iFolder

  • Creación de carpetas iFolder en un administrador de archivos

Creación de carpetas iFolder en el navegador de carpetas iFolder

  1. En el área de notificaciones, utilice uno de los siguientes métodos para abrir el navegador de iFolder:

    • Haga clic en el icono Servicios de iFolder .

    • Haga clic con el botón derecho en el icono Servicios de iFolder y, a continuación, haga clic en Carpetas iFolder.

  2. Haga clic en Crear carpeta iFolder.

  3. En Más opciones, especifique la cuenta de iFolder que desea utilizar para esta carpeta iFolder.

    Si tiene varias cuentas de iFolder, la cuenta por defecto aparecerá en el primer lugar de la lista. Utilice la lista desplegable para seleccionar otra cuenta.

  4. En el navegador, vaya al directorio en el que desea crear una carpeta iFolder.

  5. Seleccione la carpeta normal que desea convertir en una carpeta iFolder mediante uno de estos métodos:

    • Si la carpeta normal ya existe, selecciónela y, a continuación, haga clic en Aceptar.

    • Si la carpeta normal no existe, haga clic en Crear carpeta para crearla; seleccione la carpeta que acaba de crear y, a continuación, haga clic en Añadir.

  6. Seleccione el tipo de carpeta iFolder entre las opciones proporcionadas. Puede decidir si desea crear una carpeta iFolder cifrada o normal.

    IMPORTANTE:Podrá crear carpetas iFolder cifradas sólo si el administrador de iFolder ha definido la directiva de cifrado como Activa. En caso contrario, sólo podrá crear carpetas iFolder normales. Esta opción está seleccionada por defecto. En este caso, las opciones de Tipo están inhabilitadas.

    • Seleccione Cifrado de la contraseña codificada para habilitar el cifrado para la carpeta iFolder seleccionada.

      Si no ha definido la contraseña codificada durante la sesión de entrada y selecciona Cifrado de la contraseña codificada, el sistema le solicitará que introduzca la contraseña codificada. Especifíquela en el recuadro de diálogo Contraseña codificada.

    • Seleccione Normal para habilitar el uso compartido para la carpeta iFolder seleccionada.

      iFolder anuncia la disponibilidad de las carpetas iFolder compartidas mediante un mensaje emergente en el área de notificaciones. Los usuarios compartidos pueden hacer clic en el mensaje para configurar la carpeta iFolder en ese momento o bien pueden cerrarlo para descargar la carpeta iFolder más tarde.

  7. Seleccione SecureSync para habilitar la sincronización segura de los archivos en la carpeta iFolder que está creando.

    Es posible habilitar un canal seguro tanto para las carpetas iFolder normales como cifradas. Cuando esta opción se encuentra seleccionada, la carpeta iFolder permite utilizar un canal seguro para establecer una sincronización entre la carpeta iFolder del servidor y el equipo local.

    NOTA:Es posible habilitar la sincronización segura para una carpeta iFolder seleccionada en su máquina haciendo clic en el botón derecho en la opción iFolder > Propiedades en la ventana principal de iFolder. Para ello, haga clic con el botón derecho en iFolder y seleccione Propiedades en la pestaña General; a continuación, marque la casilla de verificación Secure Sync.

  8. Haga clic en Aceptar.

    La carpeta iFolder aparecerá en la lista Carpetas iFolder en este equipo.

  9. Si aparece el mensaje Carpeta iFolder creada, realice una de las siguientes acciones:

    • Haga clic en Cerrar para salir del mensaje.

    • Seleccione No mostrar este mensaje de nuevo para inhabilitar futuras notificaciones y, a continuación, haga clic en Cerrar para salir del mensaje.

      Si posteriormente decide que desea recibir mensajes de confirmación, vaya a Preferencias de iFolder, seleccione Mostrar recuadro de diálogo de confirmación al crear carpetas de iFolder y, a continuación, haga clic en Aceptar.

Creación de carpetas iFolder en un administrador de archivos

  1. Utilice el administrador de archivos para desplazarse a la carpeta que desea convertir en una carpeta iFolder y, a continuación, seleccione dicha carpeta.

  2. Lleve a cabo uno de estos métodos:

    • Haga clic con el botón derecho en el icono de la carpeta y, a continuación, haga clic en Convertir en una carpeta iFolder.

    • En la barra de herramientas, haga clic en Archivo > Convertir en una carpeta iFolder.

    El icono de carpeta se convertirá en una carpeta con una superposición de emblema de iFolder (la “i” verde).

  3. Si aparece el mensaje Carpeta iFolder creada, realice una de las siguientes acciones:

    • Haga clic en Cerrar para salir del mensaje.

    • Seleccione No mostrar este mensaje de nuevo para inhabilitar futuras notificaciones y, a continuación, haga clic en Cerrar para salir del mensaje.

      Si posteriormente decide que desea recibir mensajes de confirmación, vaya a Preferencias de iFolder, seleccione Mostrar recuadro de diálogo de confirmación al crear carpetas de iFolder y, a continuación, haga clic en Aceptar.

Creación de carpetas iFolder en un punto de montaje FAT32 (Linux)

En Linux, iFolder admite la creación y uso de carpetas iFolder en un punto de montaje de un sistema de archivos FAT32. Antes de crear nuevas carpetas iFolder o de configurar carpetas iFolder compartidas en un volumen FAT32 del equipo Linux, el usuario de la carpeta iFolder debe poseer el punto de montaje FAT32 donde desea almacenar las carpetas iFolder, y el archivo /etc/fstab debe identificar explícitamente el ID de usuario local del usuario de la carpeta iFolder para el punto de montaje.

IMPORTANTE:Esto es necesario porque FAT32 no es compatible con permisos de archivos. Cualquier otro sistema de archivos no compatible con permisos tendrá el mismo problema.

En sistemas de arranque dual, es posible acceder a los datos de las carpetas iFolder del mismo modo que con las carpetas normales del sistema donde la carpeta no se ha configurado como una carpeta iFolder. Si configura la carpeta como una carpeta iFolder en ambas plataformas, el conjunto de datos se almacenará en dos carpetas iFolder diferentes en el servidor, que consume el doble de los recursos de disco del servidor que tiene asignados.

Debido a que el sistema de archivos FAT32 no es compatible con los permisos del sistema de archivos POSIX, asegúrese de que el usuario de iFolder es el propietario local del punto de montaje. Modifique el archivo /etc/fstab para establecer el parámetro de ID de usuario (UID) de la identidad de entrada local del usuario de iFolder que crea carpetas iFolder en dicho punto de montaje.

  1. Entre al equipo como usuario Root (raíz).

  2. Cree el directorio donde desea montar el sistema de archivos FAT32. En un indicador de comandos, introduzca

    mkdir /home/username/fsmount

    Sustituya nombre_usuario por el nombre del usuario que desea utilizar la unidad FAT32 para las carpetas iFolder. Sustituya fsmount por el nombre del directorio que desea utilizar como punto de montaje. Por ejemplo, si el nombre de usuario de una unidad o partición FAT32 es jsmith y el nombre del directorio es fat32mntpt, introduzca

    mkdir /home/jsmith/fat32mntpt

  3. Monte la unidad FAT32 en el punto de montaje deseado. En un indicador de comandos, introduzca

    mount -t vfat /dev/hda1 /home/username/fsmount

    Sustituya /dev/hda1 por el nombre del dispositivo o partición de un dispositivo que no sea Linux. Sustituya nombre_usuario por el nombre del usuario de la carpeta iFolder. Sustituya fsmount por el nombre del directorio que desea utilizar como punto de montaje.

    Para continuar con el ejemplo, si el sistema de archivos FAT32 (vfat) se encuentra en el dispositivo hda4 (/dev/hda4), el nombre de usuario es jsmith y el directorio del punto de montaje es /home/jsmith/fat32mntpt, introduzca

    mount -t vfat /dev/hda4 /home/jsmith/fat32mntpt

  4. Añada el parámetro uid=username del usuario de la carpeta iFolder en la línea que define el punto de montaje FAT32 para editar el archivo /etc/fstab.

    /dev/hda1  /home/username/fsmount  vfat  uid=username

    Para continuar con el ejemplo, añada uid=jsmith al punto de montaje para modificar el archivo /etc/fstab:

    /dev/hda4  /home/jsmith/fat32mntpt  vfat  uid=jsmith
