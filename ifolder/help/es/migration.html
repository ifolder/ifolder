Migración desde iFolder 2.x a iFolder 3.7 o versiones posteriores

En esta sección se analizan los elementos siguientes:

  • Explicación del proceso de migración

  • Procedimiento de migración

Explicación del proceso de migración

iFolder de Novell permite realizar una sencilla migración a partir de iFolder 2.x a versiones de cliente iFolder 3.7 o posteriores. El Asistente de migración incluido en el cliente iFolder le ayuda a migrar el cliente iFolder 2 existente.x rápida y cómodamente.

Durante el inicio, el Asistente de migración detecta la instalación de 2.x y le solicita la migración de los datos. Durante el proceso de migración, le vuelve a solicitar que convierta las carpetas existentes o bien que haga una copia de la carpeta iFolder 2.x de Novell. Para obtener más información al respecto, consulte Paso 6 .

iFolder le consulta acerca de la nueva sincronización de los archivos con el servidor 3.7 o posterior y le pregunta si desea conservar el cliente 2.x el cliente hasta que se complete la sincronización. Durante este proceso sólo se migran los datos y, por tanto, el usuario deberá definir las preferencias en el nuevo cliente.

Procedimiento de migración

  1. Instale el Cliente iFolder de iFolder 3.7 o un servidor posterior y, a continuación, configure la cuenta para el servidor empresarial iFolder 3.7 o posterior.

    Para obtener más información acerca de la instalación del cliente y de la configuración de la cuenta, consulte Gestión de cuentas y preferencias de iFolder

  2. En el navegador de iFolder, cree una o varias carpetas iFolder en el equipo local.

    Puede utilizar cualquier nombre válido para la carpeta iFolder. No cree la carpeta iFolder en la misma vía de directorio que el directorio actual de iFolder 2.x. Para obtener más información, consulte la Creación y carga de una carpeta iFolder.

  3. La carpeta iFolder mostrará un mensaje relativo a la existencia de datos de versiones anteriores y le solicitará que proceda a migrarlos.

    Haga clic en Aceptar para abrir una lista con las carpetas iFolder 2.x disponibles.

  4. Seleccione la carpeta iFolder 2.x carpetas iFolder que desea migrar.

    Haga clic en Migrar para que aparezca el Asistente de migración.

    NOTA:Para abrir el Asistente de migración, también puede seleccionar iFolder > Migración > Migrar desde iFolder 2.x.

    El Asistente de migración le guiará a través del proceso para llevar a cabo la migración de los datos.

  5. Haga clic en Adelante para que aparezcan las opciones de migración.

  6. Seleccione una de las siguientes opciones:

    Migrar la carpeta de iFolder y desconectarla del dominio 2.x: Seleccione esta opción para desconectar la carpeta existente del dominio iFolder 2.x una vez efectuada la migración a la versión 3.7 o posterior.

    Crear una copia de la carpeta iFolder y migrar: Seleccione esta opción para crear una copia de los archivos de iFolder 2.x y convertirlos a iFolder 3.7 o versiones posteriores. Esto no elimina los archivos desde iFolder 2.x el dominio, incluso después de migrar los archivos a iFolder 3.7 o posterior.

    Si selecciona esta opción, también podrá seleccionar la opción Copiar la carpeta padre si desea copiar no sólo el contenido de la carpeta 2.x, sino también la carpeta padre. Si deja esta opción sin seleccionar, sólo se migrará el contenido de la carpeta seleccionada a la versión 3.7 o posterior.

    Haga clic en Examinar para buscar la ubicación en la que desea guardar la carpeta migrada.

    Por ejemplo, c:/Documents and Settings/<nombre de usuario>/ifolder2dir puede ser una carpeta iFolder 2.x, mientras que dir1 y dir2 pueden ser los directorios contenidos en ifolder2dir. Si selecciona la opción Copiar el directorio padre, se creará el directorio ifolder2dir en la nueva ubicación y los directorios dir1 y dir2 se copiarán en él. Si esta opción permanece sin seleccionar, sólo se moverán a la ubicación especificada los directorios dir1 y dir2.

  7. Haga clic en Adelante para seleccionar el dominio y las opciones de seguridad tal como se indica a continuación:

    1. Seleccione el dominio de la lista desplegable.

    2. Seleccione la opción de seguridad deseada:

      • Seleccione Cifrado para cifrar la carpeta iFolder seleccionada.

        Si no ha definido la contraseña codificada durante la entrada, iFolder le solicitará que introduzca una. Si lo desea, puede omitir esta opción y definir la contraseña codificada más tarde.

        Si en estos momentos no se encuentra conectado al dominio iFolder, se le solicitará que se conecte a dicho dominio.

      • Seleccione Compartido para compartir la carpeta iFolder con otros miembros en el dominio iFolder

        Puede migrar una carpeta de iFolder 2.x cifrada a una carpeta compartida. Si intenta migrar una carpeta iFolder 2.x cifrada a otra compartida, se le solicitará que verifique esta acción.

        Haga clic en Aceptar para confirmar la acción.

  8. Haga clic en Adelante para migrar la carpeta iFolder seleccionada.

  9. Verifique que la información introducida es correcta y, si es necesario, haga clic en Atrás para volver a las páginas anteriores y realizar las correcciones oportunas.

    Si la información introducida es correcta, haga clic en Migrar para iniciar el proceso de migración.

    Haga clic en Atrás para volver y realizar los cambios necesarios.

  10. Haga clic en Finalizar para terminar el proceso de migración.

Para obtener más información, consulte Información legal.
