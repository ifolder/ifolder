Explicación de los iconos de iFolder

En la tabla siguiente se describen los iconos de iFolder y las tareas relacionadas con éstos.

Tabla 1 Descripción de los iconos de la carpeta iFolder

Icono de iFolder

Descripción

El icono Servicios de iFolder se muestra con el Cliente iFolder 3 en el menú Aplicaciones del escritorio.

Cuando se está ejecutando iFolder, el icono Servicios de iFolder también aparece en el área de notificaciones de la barra de tareas del escritorio. Al hacer clic con el botón derecho en él, aparecerá un menú con las posibles tareas que puede realizar como, por ejemplo, el navegador de iFolder, los valores de configuración de la cuenta, el registro de sincronización, las preferencias y la ayuda.

Se mostrará el icono crear carpeta iFolder en el navegador de iFolder.

Puede crear una nueva carpeta iFolder o bien convertir una carpeta existente en una carpeta iFolder. Si dispone de varias cuentas, deberá especificar cuál de ellas desea utilizar. Una carpeta iFolder sólo puede pertenecer a una única cuenta. Para obtener más información, consulte la Creación y carga de una carpeta iFolder.

Se mostrará el icono de descarga de iFolder cuando disponga de una carpeta iFolder en el servidor especificado, aunque no la haya descargado en el equipo local.

Es posible descargar cualquier combinación de carpetas iFolder de diferentes cuentas en el equipo local. Para obtener más información, consulte la Descarga de una carpeta iFolder disponible.

El icono de carpeta iFolder cifrada (iFolder con el candado) indica que la carpeta iFolder está cifrada y aparece en el navegador de iFolder.

La contraseña codificada proporcionada en el momento de entrar al sistema le permitirá recuperar los archivos de la carpeta iFolder cifrada. Para obtener más información sobre el cifrado, consulte Gestión de la contraseña codificada para carpetas iFolder cifradas

El emblema de iFolder (la "i" verde) aparece en las carpetas iFolder cuando se ejecuta la aplicación de Servicios de iFolder para las carpetas iFolder que aparecen en el gestor de archivos, el escritorio o el navegador de iFolder.

Si no se ejecuta iFolder, las carpetas iFolder aparecen como carpetas normales cuando se visualizan en el gestor de archivos o en el escritorio. El navegador de iFolder no está disponible cuando no se ejecuta iFolder.

Se recomienda organizar el conjunto local de carpetas iFolder en un directorio aparte que sólo contenga dichas carpetas. De este modo, resultará más sencillo recordar qué carpetas son carpetas iFolder incluso aunque no se ejecute iFolder y no aparezca ningún icono que las identifique como carpetas iFolder.

El icono de advertencia de iFolder indica que el servidor no está disponible. Si bien iFolder se está ejecutando, no está conectado al servidor iFolder. Esto podría ser debido a lo siguiente:

  • No ha entrado en la cuenta de iFolder. Para obtener más información, consulte la Entrada a una cuenta de iFolder.

  • La cuenta está inhabilitada en el cliente, es decir, el parámetro Conectar automáticamente está inhabilitado. Para obtener más información, consulte Conectar automáticamente en Visualización y modificación de los valores de la cuenta de iFolder.

  • El servidor está apagado o bien la cuenta se ha inhabilitado en el servidor. Solicite ayuda al administrador de iFolder.

  • El cortafuegos está bloqueando el tráfico de iFolder. Para obtener más información sobre la solución de este problema, consulte Configuración de los valores del cortafuegos local para el tráfico de iFolder.

  • La conexión a la red está desactivada. Compruebe la configuración de la red local.

El icono Conflicto de iFolder indica un conflicto de archivos entre la copia local y la copia del servidor. Para resolver el conflicto, indique qué versión del archivo desea conservar. Si desea obtener más información sobre el conflicto de archivos, consulte Resolución de conflictos con los archivos.

El icono de iFolder en espera de sincronización indica que la carpeta iFolder contiene archivos que deben sincronizarse con el servidor pero que el proceso de sincronización todavía no ha empezado.

iFolder sincroniza una carpeta iFolder a la vez y organiza las tareas de sincronización en una cola. Para asegurarse de que los archivos de una carpeta iFolder local se sincronizan con los archivos del servidor, no cierre iFolder hasta que la sincronización se haya iniciado y haya terminado correctamente, según muestren los cambios en el aspecto del icono.

El icono de sincronización de iFolder indica que los archivos de la carpeta iFolder se están sincronizando con el servidor.

El icono Error de iFolder indica que actualmente se están sincronizando los archivos de la carpeta iFolder con el servidor.

En el recuadro de diálogo Compartición, el icono de usuario propietario de iFolder indica que el usuario es el propietario de la carpeta iFolder compartida.

En el recuadro de diálogo Compartición, el icono de usuario actual de iFolder indica que el usuario es el que gestiona actualmente la carpeta iFolder. Si el usuario actual es el propietario, el icono de usuario propietario aparecerá en su lugar.

Para obtener más información, consulte Información legal.
