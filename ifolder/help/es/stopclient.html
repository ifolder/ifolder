Salida del Cliente iFolder

Para salir del Cliente iFolder, siga alguna de las opciones siguientes:

  • Haga clic con el botón derecho en el icono Servicios de iFolder del área de notificaciones y, a continuación, seleccione Salir.

  • Haga clic en iFolder > Salir

    Cuando seleccione Salir, iFolder le mostrará un mensaje de confirmación. Haga clic en Sí para confirmar la acción.

Los archivos se sincronizarán con la cuenta del servidor iFolder sólo cuando esté conectado a dicho servidor. Si lo desea, puede salir de una cuenta para detener la sincronización. iFolder se seguirá ejecutando en segundo plano en modo desconectado. Para obtener más información, consulte la Salir de una cuenta de iFolder.

Puede disponer de varias cuentas, cada una de ellas creada en un servidor host (dominio) distinto.

Para obtener más información, consulte Información legal.
