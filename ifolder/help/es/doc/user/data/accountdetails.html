Visualización y modificación de los ajustes de la cuenta de iFolder

  1. Haga clic con el botón derecho en el icono iFolder en el área de notificaciones y, a continuación, seleccione Cuentas para abrir Preferencias de iFolder en la pestaña Cuentas.

  2. Seleccione la cuenta para visualizar o modificar los parámetros siguientes:

    Parámetro Descripción

    Servidor

    Nombre DNS del servidor empresarial. Por ejemplo: svr21.ejemplo.com.

    Usuario

    Identidad del usuario de esta cuenta. El administrador de iFolder decide si el formato que debe utilizarse es el ID de usuario de red (nombre común de LDAP) o la dirección de correo electrónico.

    Por ejemplo, si su nombre es John Smith con el ID de usuario jsmith y la dirección de correo electrónico john.smith@ejemplo.com, el administrador configura el ajuste del servidor que determina cuál de los dos formatos debe aceptarse en el campo Usuario con fines de autenticación.

    Sólo es posible crear una cuenta por servidor empresarial.

    Contraseña

    Contraseña del usuario.

    Recordar contraseña

    Seleccione Recordar contraseña para entrar automáticamente en la cuenta de iFolder cada vez que inicie el equipo. Para obtener información, consulte Entrada a una cuenta de iFolder.

    Si cambia la contraseña de red, la autenticación automática no se realizará correctamente la próxima vez que iFolder intente entrar en la cuenta de iFolder. iFolder le solicitará que entre con la nueva contraseña y le permitirá especificar y guardar la contraseña nueva.

    Enable Account (Habilitar cuenta)

    Seleccione Enable Account (Habilitar cuenta) para permitir la sincronización en segundo plano de las invitaciones de iFolder, la lista de usuarios y las carpetas iFolder de esta cuenta. Esta opción sólo afecta a las funciones del cliente.

    La información del estado Habilitado o Inhabilitado de la cuenta se muestra en la columna Estado de la lista Cuentas. Puede visualizar fácilmente el estado del ajuste Enabled Account (Cuenta habilitada) si tiene más de una cuenta configurada.

    Si utiliza iFolder con iFolder® 3.x de Novell®, el administrador del sistema de iFolder puede inhabilitar su cuenta de iFolder (generalmente, de forma temporal) en el servidor. Puede esperar el comportamiento siguiente:

    • Si ha entrado en el sistema cuando la cuenta se inhabilita desde el servidor, puede administrar las carpetas iFolder y los datos de iFolder seguirán sincronizándose con el servidor hasta que salga o inhabilite la sincronización para finalizar la sesión, o bien hasta que el servidor desconecte la sesión, lo que suceda en primer lugar.
    • Si se inhabilita la cuenta en el servidor, recibirá un mensaje emergente que le informará de que la cuenta se ha inhabilitado cuando intente entrar. Póngase en contacto con el administrador del sistema de iFolder para obtener asistencia.

    Default Account (Cuenta por defecto)

    Seleccione Default Account (Cuenta por defecto) para que se seleccione esta cuenta por defecto en la lista desplegable de cuentas al crear carpetas iFolder.

  3. En la pestaña Cuentas en Preferencias de iFolder, seleccione la cuenta de iFolder y, a continuación, haga clic en Detalles para ver la información siguiente acerca de la cuenta de iFolder.

    Parámetro Descripción

    Servidor

    Nombre de host del servidor empresarial. Por ejemplo, si el nombre DNS del servidor es svr21.ejemplo.com, svr1 es la información que se especifica.

    Descripción

    Descripción sencilla del servidor empresarial, servidor simple o equipo host de grupo de trabajo tal y como la ha introducido el administrador. Si no se ha introducido ninguna descripción, el valor es "ninguno".

    Disponible

    Cantidad total de espacio actualmente disponible para carpetas iFolder en esta cuenta de iFolder. El valor es igual a la cuota menos el espacio utilizado.

    Usado

    Cantidad total de espacio actualmente utilizado por todas las carpetas iFolder que posee en esta cuenta.

    Quota (Cuota)

    Cantidad total de espacio asignado a la cuenta de iFolder en el servidor empresarial. Este valor puede ser la cuota por defecto establecida para todos los usuarios o una cuota específica para un usuario. Si no se ha establecido ninguna cuota, el campo se mostrará vacío o aparecerá N/D (no disponible).
