Inicio del Cliente iFolder


Inicio de iFolder para Novell Linux Desktop 9

  1. Entre en el equipo con la identidad de usuario que desee utilizar al crear carpetas iFolder.

  2. Para iniciar iFolder, utilice uno de los métodos siguientes:

    • Abra un shell de terminal y, a continuación, introduzca

      /opt/novell/ifolder3/bin/ifolder

    • Si desea que iFolder se inicie automáticamente al entrar, asegúrese de que está se ejecutando al salir de la sesión GNOME y, a continuación, guarde la sesión. También puede abrir una ventana de terminal, entrar como usuario raíz y, a continuación, ejecutar /opt/gnome/bin/gnome-session-properties para añadir iFolder como un programa que debe iniciarse al entrar en el equipo.

  3. Si ha inhabilitado la opción Recordar contraseña de la cuenta de iFolder, deberá entrar en la cuenta para sincronizar la lista Usuario, y sincronizar y gestionar carpetas iFolder.

    Para obtener información, consulte Entrada a una cuenta de iFolder.


Inicio de iFolder para Windows 2000/XP

  1. Entre en el equipo con la identidad de usuario de Windows que desea utilizar para crear carpetas iFolder.

  2. Si iFolder no se inicia automáticamente al entrar correctamente, haga clic en Inicio > Programas > iFolder y, a continuación, seleccione la aplicación iFolder.

    iFolder no se inicia automáticamente si se inhabilita el ajuste Start iFolder When Logging In to the Desktop (Iniciar iFolder al entrar en el escritorio) en Preferencias de iFolder.

  3. Si ha inhabilitado la opción Recordar contraseña de la cuenta de iFolder, deberá entrar en la cuenta para sincronizar la lista Usuario, y sincronizar y gestionar carpetas iFolder.

    Para obtener información, consulte Entrada a una cuenta de iFolder.


Inicio de iFolder para Macintosh OS X v10.3 y versiones posteriores

  1. Entre en el equipo con la identidad de usuario que desee utilizar al crear carpetas iFolder.

  2. Para iniciar iFolder, utilice uno de los métodos siguientes:

    • Abra el menú Aplicaciones, seleccione el submenú Internet y, a continuación, seleccione iFolder.
    • Para iniciar iFolder desde la línea de comandos, abra un shell de terminal y, a continuación, introduzca

      /opt/novell/ifolder3/bin/ifolder

    • Si desea que iFolder se inicie automáticamente al entrar en el equipo Macintosh, añada iFolder a los elementos del menú Inicio.

  3. Si ha inhabilitado la opción Recordar contraseña de la cuenta de iFolder, deberá entrar en la cuenta para sincronizar la lista Usuario, y sincronizar y gestionar carpetas iFolder.

    Para obtener información, consulte Entrada a una cuenta de iFolder.
