iFolderクライアント

iFolderクライアントは、ネイティブなデスクトップ環境でiFolderサービスを提供するために、オペレーティングシステムを統合します。iFolderクライアントで、自分が所有するiFolderを管理したり、どの共有iFolderを各コンピュータ上にセットアップするかを選択できます。 iFolderは次のオペレーティングシステムをサポートしています。

  • SUSE Linux Enterprise Desktop 10 SP1、SP2、およびSP3

  • SUSE Linux Enterprise Desktop 11

  • Windows Vista SP1

  • Windows XP SP2およびSP3

  • Windows 7

  • OpenSUSE 11.1

  • OpenSUSE 11.2

  • Apple Macintosh 10.4以降

詳細については、「 保証と著作権.
