iFolderのメリット

iFolderには次のメリットがあります。

  • ネイティブデスクトップ環境と統合することによって、複数のiFolderの作成と管理が容易にできる
  • 拡張性と柔軟性が非常に高く、ユーザごとのiFolderの数やiFolderごとのメンバーの数に実際的な制限がない
  • 私的なファイルや業務上のファイルを選別して、容易に共用できる
  • 自分が所有するiFolderの場合、そのiFolderのメンバーのアクセスレベルを制御でき、iFolderメンバーのアクセス権を制御する権限を付与されている場合も同様に制御できる
  • 複数のワークステーション上にあるメンバーのiFolderへ、ファイルを透過的に更新できる
  • ユーザがオフライン作業中に加えた変更は追跡および記録され、オンライン時に同期される
  • データを同期する際は、メンバーのiFolderに対して安全な認証を行う
  • 電子メールに代わるファイル交換方法を提供するため、電子メールの添付ファイルのサイズを考慮する必要がなくなり、電子メールの保管に必要な容量が削減される
