iFolderアカウントの削除

現在のコンピュータからのみアカウントを削除するか、サーバから所有するiFolderも削除するかを指定できます。原則として、そのアカウントに設定している共有iFolderは停止します。

iFolderアカウントをコンピュータから削除する場合、関連するiFolderのローカルコピーも標準フォルダに戻されます。他のコンピュータまたはWebから、iFolderへのアクセスを継続できます。

iFolderアカウントをサーバから削除する場合

  • 所有するすべてのiFolderは共用を解除されます。
  • メンバーコンピュータ上のiFolderのローカルコピーは標準フォルダに戻されます。
  • iFolderとそのコンテンツはサーバから削除されます。
  • 他のメンバーと共用するiFolderのメンバーから削除されます。

  1. 通知スペースでiFolderアプリケーションアイコンを右クリックし、［アカウント］を選択して［iFolderの初期設定］ダイアログボックスの［アカウント］タブを開きます。

  2. 削除するiFolderアカウントを選択して、［削除］をクリックします。

  3. 削除処理の範囲を指定するように求めるメッセージが表示されます。次のいずれかの操作を行います。

    • 現在のコンピュータからのみiFolderアカウントを削除する:［サーバからiFolderとファイルを削除する］をオフにして、［はい］をクリックします。iFolderは選択したアカウントをローカルワークステーションからのみ削除します。このアカウントのローカルiFolderは、標準フォルダに戻されます。必要がない場合は、ファイルマネージャを使用してデータのローカルコピーを検索して削除します。

      このアカウントのiFolderへは、iFolderクライアントを持つ他のコンピュータから引き続きアクセスできます。iFolderクライアントをNovell® iFolder® 3.xエンタープライズサーバと一緒に使用する場合、Novell iFolder 3.x Web Accessを使用してどこからでもファイルにアクセスできます。

    • iFolderアカウントおよびiFolderとファイルをサーバから削除する:［サーバからiFolderとファイルを削除する］をオンにして、［はい］をクリックします。iFolderは選択したアカウントをローカルワークステーションから削除します。このアカウントが所有するすべてのiFolderがサーバから削除され、そのiFolderは現在のコンピュータおよびすべてのメンバーのコンピュータで標準フォルダに戻されます。最後に、このアカウントの共有iFolderのメンバーから削除されます。ローカルiFolderはすべて標準フォルダに戻されます。必要がない場合は、ファイルマネージャを使用してデータのローカルコピーを検索して削除します。

      このアカウントを同じまたは別のコンピュータで再び使用する場合、アカウントとそのiFolderを、新規作成のときと同様に設定する必要があります。

    • 削除のキャンセル:削除処理を取り消すには［いいえ］をクリックします。アカウントとそのiFolderは変更されません。
