iFolderの主要機能

iFolderを使用する前に、次の主要機能について理解することが重要です。

  • iFolderクライアント
  • iFolderアカウント
  • 共有iFolder
  • iFolderのアクセス権
  • ファイル同期とデータ管理
  • 同期ログ


iFolderクライアント

iFolderクライアントはオペレーティングシステムに統合され、ネイティブデスクトップ環境でiFolderサービスを提供します。iFolderは次のオペレーティングシステムをサポートしています。

  • Novell Linux Desktop 9
  • Windows 2000/XP
  • Macintosh OS X v10.3以降

iFolderクライアントで、自分が所有するiFolderを管理したり、どの共有iFolderを各コンピュータ上にセットアップするかを選択できます。詳細については、「iFolderの管理」を参照してください。


iFolderアカウント

iFolderを作成する前に、iFolderサービスのアカウントを少なくとも1つ設定する必要があります。iFolderセッションは、iFolderサービスのアカウントへログインするときに開始され、ログアウトするか、またはiFolderクライアントを閉じると終了します。自分のiFolderが他のアクティブなiFolderとファイルを同期するのは、自分のiFolderセッションがアクティブで、オンラインで作業しているときだけです。自分のローカルiFolderへは、アカウントにログインしているかどうかに関係なく、いつでもアクセスできます。詳細については、「iFolderアカウントの設定」を参照してください。


共有iFolder

iFolderは、中央コンピュータを介して、ユーザが指定したユーザグループと、ファイルを選択して共用および同期するローカルディレクトリです。iFolderファイルへはすべてのiFolderメンバーがアクセスでき、権利を付与されたメンバーはファイルの内容を変更できます。自分のiFolderへは、複数のワークステーションからアクセスでき、他のユーザと共用することができます。

iFolderでの作業は、ファイルマネージャや、iFolderクライアントの一部であるiFolderブラウザで直接行います。iFolder内では、ユーザの好みや職場の慣行に合わせて、任意のサブディレクトリ構造を設定できます。このサブディレクトリ構造は、iFolderのメンバー全員に同じものが提供されます。iFolderを共用しているときに、書き込み権を付与されたメンバーユーザがこのディレクトリを変更した場合、変更内容はiFolderのすべてのコピーに適用されます。iFolder配置のガイドラインを満たすコンピュータ上の共有iFolderであれば、どのユーザでも見つけることができます。


iFolderのアクセス権

iFolderクライアントでは、iFolderメンバーに対して所有者および3つのレベルのアクセス権をサポートしています。

  • 所有者:iFolderの所有者としての役割を果たす唯一のユーザです。通常はそのiFolderを作成したユーザになります。所有者ユーザは、iFolderクライアントを使用して、所有権をiFolderの他のメンバーに譲渡することができます。

    iFolderの所有者にはフル制御のアクセス権があります。このユーザはiFolderに対する読み書き可能アクセス権を持ち、メンバーユーザのメンバーシップとアクセス権を管理したり、メンバーのフル制御アクセス権を削除したりできます。

  • フル制御:フル制御のアクセス権を持つ共有iFolderのメンバーです。このメンバーはiFolderへの読み書き可能アクセス権を持ち、所有者を除くすべてのユーザに対するメンバーシップとアクセス権を管理できます。

  • 読書き可能:共有iFolderのメンバーで、iFolder内のディレクトリとファイルに対する読書き可能アクセス権を持ちます。

  • 読み込み専用:共有iFolderのメンバーで、iFolder内のディレクトリとファイルに対する読み込み専用アクセス権を持ちます。このメンバーはファイルの変更はローカルでできますが、変更した内容はiFolderメンバーのコピーへは同期されません。

iFolderユーザはiFolderを所有したり、他のiFolderのメンバーになることができます。所有者ユーザの場合、常にフル制御のアクセス権を持ちます。メンバーユーザの場合、各共有iFolderでのアクセスレベルはさまざまで、フル制御のアクセス権を持つメンバーによってアクセス権が付与されます。


ファイル同期とデータ管理

iFolderアカウントをセットアップするときに［パスワードを保存する］を有効にすると、iFolderの招待やファイルが作業に応じてバックグラウンドで同期されるように設定できます。

iFolderクライアントは、各自のコンピュータのデスクトップ環境へログインするたびにiFolderを自動的に実行するように設定できます。セッションは、ユーザがローカルiFolder内のファイルで作業しているときはバックグラウンドで動作し、変更を追跡および記録します。

ファイルの同期は、指定した間隔またはオンデマンドで実行できます。


同期ログ

同期ログには、他のiFolderに対するファイルのアップロードやダウンロードなどの、iFolderのバックグラウンド状態が表示されます。
