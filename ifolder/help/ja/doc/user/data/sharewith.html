  • ユーザのアクセス権を理解する
  • ユーザをiFolderに追加する
  • ユーザのアクセス権を編集する
  • iFolderからユーザを削除する

ユーザをiFolderに追加する


ファイルマネージャを使ってユーザを追加する

  1. ファイルマネージャでiFolderまで移動し、そのiFolderを選択します。

  2. iFolderを右クリックして、［iFolder］>［プロパティ］の順にクリックします。

  3. ［共用］をクリックします。

  4. ユーザのリストからユーザを1人以上選択します。

  5. 次のアクセス権を選択します。［フル制御］、［読書き可能］、［読み込み専用］。

  6. ［適用］をクリックします。


iFolderブラウザを使用してユーザを追加する

  1. ［通知］エリアでiFolderアイコンをクリックし、［iFolder］をクリックしてiFolderブラウザを開きます。

  2. iFolderのリストでiFolderを選択し、［共用］をクリックして［iFolderのプロパティ］ウィンドウの［共用］タブを開きます。

  3. ［追加］をクリックし、［ユーザの選択］ダイアログボックスを開きます。

  4. ユーザのリストから1人以上のユーザを［選択したユーザ］リストに追加し、［OK］をクリックします。

    招待がサーバと同期されるまで、数秒かかる場合があります。

  5. 1人以上のユーザを選択し、［アクセス］をクリックします。［フル制御］、［読書き可能］、［読み込み専用］のいずれかを選択し、［OK］をクリックします。

  6. iFolderを共用するユーザをすべて招待するまで手順4～5を繰り返します。

  7. ［適用］をクリックし、［OK］をクリックして［iFolderのプロパティ］ウィンドウを閉じます。


ユーザのアクセス権を編集する

iFolderのメンバーの作業関係と状況が変わるときは、ユーザのiFolderへのアクセス権を変更しなければならない場合があります。たとえば、iFolderの［ファイル制御(File Control)］権を信頼するユーザに与える場合などです。


ファイルマネージャを使ってユーザの権限を設定する

  1. ファイルマネージャでiFolderまで移動し、そのiFolderを選択します。

  2. iFolderを右クリックして、［iFolder］>［プロパティ］の順にクリックします。

  3. ［共用］をクリックします。

  4. ユーザのリストからユーザを1人以上選択します。

  5. ［アクセス］を選択し、続いて［フル制御］、［読書き可能］、または［読み込み専用］を指定します。

  6. ［適用］をクリックし、［OK］をクリックして［iFolderのプロパティ］ウィンドウを閉じます。


iFolderブラウザを使用してユーザの権限を設定する

  1. ［通知］エリアでiFolderアイコンをクリックし、［iFolder］をクリックしてiFolderブラウザを開きます。

  2. iFolderのリストでiFolderを選択し、［共用］をクリックします。

  3. ユーザのリストからユーザを1人以上選択します。

  4. ［アクセス］を選択し、続いて［フル制御］、［読書き可能］、または［読み込み専用］を指定します。

  5. ［適用］をクリックし、［OK］をクリックして［iFolderのプロパティ］ウィンドウを閉じます。


iFolderからユーザを削除する

iFolderの共用を解除するには、iFolderのメンバーとしてユーザを削除する必要があります。ユーザのワークステーションのiFolderは標準フォルダになります。データのコピーはこれまでのメンバーのワークステーションには残りますが、共有iFolderと同期されることはありません。


ファイルマネージャを使ってユーザを削除する

  1. ファイルマネージャでiFolderまで移動し、そのiFolderを選択します。

  2. iFolderを右クリックして、［iFolder］>［プロパティ］の順にクリックします。

  3. ［共用］をクリックします。

  4. ユーザのリストからユーザを1人以上選択します。

  5. ［削除］をクリックして、［適用］、［OK］の順にクリックします。


iFolderブラウザを使ってユーザを削除する

  1. ［通知］エリアでiFolderアイコンをクリックし、［iFolder］をクリックしてiFolderブラウザを開きます。

  2. iFolderのリストでiFolderを選択し、［共用］をクリックします。

  3. ユーザのリストからユーザを1人以上選択します。

  4. ［削除］をクリックして、［適用］、［OK］の順にクリックします。
