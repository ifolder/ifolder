iFolderアカウントへのログイン

iFolderアカウントを作成した後、ローカルのiFolderデータ、iFolderユーザのリスト、および共有iFolder通知を同期する場合は常にログインする必要があります。複数のアカウントに同時にログインできます。各アカウントで、次のログイン方法のいずれかを使用します。

  • ［パスワードの保存］を有効にすると、デスクトップにログインするか、iFolderクライアントを開始するたびに、自動的にiFolderアカウントにログインします。

    パスワードが変更されると、次にiFolderアカウントへのログインを試みたときの自動認証は失敗します。新しいパスワードでログインするためのメッセージが表示され、新しいパスワードの指定と保存ができます。

  • iFolderでファイルを同期する準備が整った場合、またはiFolderを管理する準備が整った場合にのみ、［パスワードの保存］を無効にし、iFolderアカウントにログインします。

既存のiFolderアカウントの［パスワードの保存］を有効、または無効にするには、次のことを実行します。

  1. ［通知］エリアでiFolderアイコンを右クリックし、［アカウント］を選択します。

  2. アカウント設定を表示するiFolderアカウントを選択します。

  3. ［パスワードの保存］を選択(有効)、または選択解除(無効)にします。

  4. パスワードを指定し、［OK］または［適用］をクリックします。

  5. ［OK］をクリックしてアカウント設定への変更を確認します。変更を取り消すには［キャンセル］をクリックします。

iFolderアカウントにログインするには、次のことを実行します。

  1. ［通知］スペースでiFolderアイコンを右クリックし、［アカウント］を選択します。

  2. アカウント設定を表示するiFolderアカウントを選択します。

  3. パスワードを指定して［ログイン］をクリックします。

    ログインに成功すると、アカウントの状況が［ログイン］に変わります。

    ログインに失敗すると、パスワードの失敗など、失敗の原因になったエラーメッセージが表示されます。

  4. ［OK］をクリックして［アカウント］ダイアログボックスを閉じます。
