iFolderの移動

iFolderを作成した後は、iFolderへのパスは静的になります。ローカルコンピュータ上で、iFolderを元の場所から別の場所に移動すると、元の場所でiFolderを保持することはできません。iFolderを移動することにより、同期データベースとiFolderフォルダに変換したフォルダとのリンクは壊れます。

iFolderを作成したコンピュータ上でiFolderを移動するには次のことを実行します。

  1. iFolderを標準フォルダに変換します。

  2. フォルダを移動します。

  3. フォルダをiFolderに変換します。

  4. iFolderを共用します(オプション)。

参加中のコンピュータにiFolderを移動するには次のことを実行します。

  1. iFolderで、iFolderの参加を停止するためにiFolderを元に戻しますが、自分には使用可能なままにしておきます。

  2. 使用可能なiFolderを選択し、［セットアップ］をクリックします。

  3. iFolderを配置する先のローカルパスを指定します。

  4. iFolderがファイルをエンタープライズサーバと同期できるようにします。
