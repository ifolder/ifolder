iFolderアカウントからのログアウト

オフラインで作業するときは、iFolderアカウントからログアウトします。

  • Macintosh

  • Linux

  • Windows

Macintosh

  1. ドックでiFolderサービスアイコンをクリックし、 ［アカウント］を選択して［iFolder初期設定］ダイアログを開きます。

  2. ツールバーの［アカウント］アイコンをクリックし、iFolderアカウントのリストを表示します。

  3. 管理するアカウントを見つけ、［ログアウト］をクリックしてiFolderサーバから切断します。

Linux

  1. ［通知］スペースでiFolderサービスアイコンを右クリックし、［アカウント］を選択し、［iFolderの初期設定］ダイアログボックスの［アカウント］タブを開きます。

  2. 管理するアカウントを見つけたら、その隣にある［オンライン］チェックボックスをオフにして、iFolderサーバからの接続を切断します。

  3. ［閉じる］をクリックして、［iFolder初期設定］ダイアログボックスを閉じます。

Windows

  1. ［通知］スペースでiFolderサービスアイコンを右クリックし、［アカウント］を選択し、［iFolderの初期設定］ダイアログボックスの［アカウント］タブを開きます。

  2. 管理するアカウントを見つけたら、その横にある［オンライン］チェックボックスをオフにします。

  3. ［適用］をクリックしてiFolderサーバからの接続を切断してから、［OK］をクリックして［iFolderの初期設定］ダイアログボックスを閉じます。
