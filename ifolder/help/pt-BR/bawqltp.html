Cliente iFolder

O cliente do iFolder integra-se ao seu sistema operacional para fornecer serviços do iFolder em seu ambiente nativo de área de trabalho. O cliente iFolder permite que você gerencie suas próprias pastas iFolder e selecione quais pastas iFolder compartilhadas deseja configurar em cada computador. O iFolder suporta os seguintes sistemas operacionais:

  • SUSE Linux Enterprise Desktop 10 SP1, SP2, e SP3

  • SUSE Linux Enterprise Desktop 11

  • Windows Vista SP1

  • Windows XP SP2 e SP3

  • Windows 7

  • OpenSUSE 11.1

  • OpenSUSE 11.2

  • Apple Macintosh 10.4 e posterior

Para obter mais informações, consulte Informações legais.
