Conta do iFolder

Uma conta do iFolder é criada quando o administrador de serviços do iFolder configura você como usuário em um servidor iFolder. O administrador lhe designa um nome de usuário e uma senha. No caso de empresa, esses dados devem ser seu nome de usuário ou o endereço de e-mail e a senha relacionada. Você deve configurar pelo menos uma conta do iFolder antes de criar pastas do iFolder. Para obter mais informações, consulte a Configurando uma conta do iFolder.

Uma sessão do iFolder começa quando você efetua login em uma conta de serviços do iFolder e termina quando você efetua logout ou quando sai do cliente iFolder. Suas pastas iFolder sincronizam arquivos com outras pastas iFolder ativas quando sua sessão está ativa e você está trabalhando online. É possível acessar dados em suas pastas iFolder locais a qualquer momento, estando você conectado ou não à conta. Para obter informações, consulte a Configurando uma conta do iFolder.

Para obter mais informações, consulte Informações legais.
