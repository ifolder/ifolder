Migrando do iFolder 2.x para o iFolder 3.7 ou posterior

Esta seção aborda o seguinte:

  • Entendendo o processo de migração

  • Procedimento de migração

Entendendo o processo de migração

O Novell iFolder fornece uma migração simples do cliente do iFolder 2.x para o cliente do iFolder 3.7 ou posterior. O Assistente de Migração fornecido no cliente do iFolder ajuda você a migrar os dados existentes do iFolder 2.X de forma rápida e fácil.

Durante a inicialização, o Assistente de Migração detecta a instalação da versão 2.x e solicita que você migre os dados. Durante a migração dos dados, ele solicita novamente que você converta as pastas existentes ou faça uma cópia da pasta do iFolder 2. x. Para obter mais informações sobre isso, consulte a Etapa 6.

O iFolder questiona você sobre a ressincronização dos arquivos com o servidor 3.7 ou posterior e pergunta se é necessário reter o cliente 2.x até o término da sincronização. Durante esse processo, somente os dados são migrados; portanto, o usuário precisa definir a preferência no novo cliente.

Procedimento de migração

  1. Instale o cliente do iFolder para o servidor do iFolder 3.7 ou posterior e, em seguida, configure sua conta para o servidor empresarial do iFolder 3.7 ou posterior.

    Para obter informações sobre a instalação do cliente e a configuração da conta, consulte Gerenciando as contas e preferências do iFolder.

  2. No browser do iFolder, crie uma ou mais pastas do iFolder no seu computador local.

    Você pode usar qualquer nome válido para uma pasta do iFolder. Não crie uma pasta do iFolder no mesmo caminho do diretório atual do iFolder 2.x. Para obter informações, consulte a Criando e fazendo upload de uma pasta do iFolder.

  3. O iFolder exibe uma mensagem sobre a existência de dados da versão anterior e uma solicitação de migração.

    Clique em OK para abrir uma lista de 2 disponíveis.x.

  4. Selecione a pasta do iFolder 2.x Pastas do iFolder que você deseja migrar.

    Clique em Migrar para abrir o Assistente de Migração.

    NOTA:Você também pode acessar o Assistente de Migração comiFolder > Migração > Migrar do iFolder 2.x.

    O Assistente de Migração o direciona durante o processo de migração para que você conclua a migração de dados.

  5. Clique em Avançar para exibir as opções de migração.

  6. Selecione um dos seguintes itens:

    Migrar o iFolder e desconectá-lo do domínio 2.x: Selecione esta opção para desconectar a pasta existente do iFolder 2.x depois de migrá-la para o 3.7 ou posterior.

    Criar uma cópia do iFolder e migrar: Selecione esta opção para criar uma cópia do iFolder 2.x e convertê-los em arquivos do iFolder 3.7 ou posterior. Isso não fará com que os arquivos do domínio iFolder 2.x sejam removidos, mesmo depois que você migrá-los para o iFolder 3.7 ou posterior.

    Se você selecionar essa opção, também poderá selecionar Copiar a pasta pai se quiser copiar não apenas o conteúdo da pasta do 2.x, mas também da pasta pai. Se você deixar essa opção desmarcada, somente o conteúdo da pasta selecionada será migrado para o 3.7 ou posterior.

    Clique em Procurar para encontrar a localização para gravar a pasta migrada.

    Por exemplo, c:/Documents and Settings/<nome_do_usuário>/ifolder2dir é um iFolder 2.x iFolder e dir1 e dir2 são diretórios sob ifolder2dir. Quando você seleciona a opção Copiar o diretório pai, ela cria o ifolder2dir na nova localização e os diretórios dir1 e dir2 são copiados para ela. Se a opção permanecer desmarcada, somente os diretórios dir1 e dir2 serão movidos para o local especificado.

  7. Clique em Avançar para selecionar o domínio e as Opções de Segurança da seguinte maneira:

    1. Selecione o domínio na lista suspensa:

    2. Selecione a opção de segurança desejada:

      • Selecione Criptografado para criptografar a pasta do iFolder selecionada.

        Se você não tiver definido uma frase secreta durante o login, o iFolder solicitará que você a digite. Você pode ignorar essa opção e definir a frase secreta posteriormente.

        Se você não estiver conectado ao domínio iFolder, terá de se conectar ao domínio.

      • Selecione Compartilhado para compartilhar a pasta do iFolder com outros membros do domínio do iFolder

        Você pode migrar um iFolder 2.X para a pasta compartilhada. Quando você tentar migrar uma pasta criptografada do iFolder 2.x para uma pasta compartilhada, será solicitado que você confirme a ação.

        Clique em OK para confirmar sua ação.

  8. Clique em Avançar para migrar a pasta do iFolder selecionada.

  9. Verifique se as informações inseridas estão corretas; se necessário, clique em Voltar para retornar às páginas anteriores e fazer correções.

    Se as informações fornecidas estiverem corretas, clique em Migrar para iniciar a migração.

    Clique em Voltar para voltar e fazer as mudanças necessárias.

  10. Clique em Concluir para concluir a migração.

Para obter mais informações, consulte Informações legais.
