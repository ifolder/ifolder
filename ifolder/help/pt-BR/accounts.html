Configurando uma conta do iFolder

Use o Assistente de Criação de Contas do iFolder para adicionar e configurar uma nova conta. Você deve configurar pelo menos uma conta do iFolder para poder criar pastas iFolder ou compartilhar pastas iFolder no sistema. Você pode se conectar ao servidor iFolder com apenas uma conta a qualquer momento No entanto, para se conectar a mais de um servidor iFolder, você pode configurar várias contas. Você pode efetuar login separadamente em cada conta e ficar conectado simultaneamente em várias contas.

Vários usuários com identidades de login local diferentes terão iFolders no mesmo computador. Na primeira vez que você iniciar o iFolder usando a identidade de login local atual, o iFolder solicitará que você configure uma conta.

IMPORTANTE:O iFolder 3.7 e as versões posteriores apresentam um novo recurso, a criação automática de conta, que ajuda o administrador do iFolder a criar e configurar uma conta para você. Se o administrador do iFolder tiver habilitado a criação automática de conta, você não precisará configurar uma conta para você mesmo. Ao iniciar o iFolder pela primeira vez, você será solicitado a digitar sua senha. Digite a senha e conclua a configuração da conta.

Para configurar uma nova conta:

  1. Para acessar o Assistente de Criação de Contas do iFolder, use um destes métodos:

    • Na primeira vez que você iniciar o iFolder com sua identidade de login local atual, o iFolder solicitará a configuração de uma conta. Clique em Avançar para ir diretamente para o Assistente de Criação de Conta do iFolder.

    • Clique com o botão direito do mouse no nos Serviços do iFolder, na área de notificação, selecione Contas, para abrir a guia Contas da caixa de diálogo Preferências do iFolder, e clique em Adicionar, para abrir o Assistente de Criação de Conta do iFolder.

  2. Siga as instruções na tela para especificar os seguintes valores:

    Parâmetro

    Descrição

    Endereço do Servidor

    O nome DNS ou endereço IP do servidor empresarial onde você tem uma conta. Obtenha informações do administrador do iFolder.

    Por exemplo: svr21.example.com ou 192.168.1.1

    O nome do servidor é sempre exibido junto com seu endereço IP na interface de usuário do iFolder. Por exemplo: iFolder - 192.168.1.1

    IMPORTANTE:Você não pode usar um cliente do iFolder 3.7 ou posterior para se conectar a um servidor do iFolder 3.2.

    Configurar este servidor como o servidor padrão

    Se você tiver vários servidores conectados à sua conta, marque a caixa de seleção Configurar este servidor como o servidor padrão para configurar o servidor selecionado na lista suspensa de servidores como o servidor padrão quando criar ou fizer upload de um iFolder.

    Nome do Usuário

    Especifique sua identidade de usuário para esta conta. O administrador do iFolder decidirá se você usará o formato ID de usuário da rede (nome comum do LDAP) ou endereço de e-mail.

    Por exemplo, se o seu nome é João Silva com um ID de usuário jsilva e um endereço de e-mail joao.silva@exemplo.com, o administrador definirá uma configuração para o servidor que determina qual dos dois formatos será aceito no campo Nome de usuário para fins de autenticação.

    Senha

    Especifique a senha para seu nome de usuário.

    A autenticação ocorre através de canal seguro (SSL). Você é solicitado a validar e aceitar o certificado do servidor.

    Lembrar Senha

    Marque a caixa de seleção Lembrar senha neste computador para efetuar login automaticamente na conta do iFolder sempre que você efetuar login no computador.

    Se a sua senha de rede mudar, a autenticação automática falhará normalmente na próxima vez em que o iFolder tentar efetuar login na conta do iFolder. O iFolder solicitará que você efetue login com a nova senha e permitirá que você especifique e grave a nova senha.

  3. Verifique as informações inseridas. Para fazer mudanças, clique no botão Voltar para retornar às páginas anteriores. Clique em Conectar para continuar e validar sua conexão com o servidor.

  4. Se você for solicitado aAceitar o certificado deste servidor, clique em Detalhes para analisar as informações do certificado e clique em OK para aceitá-lo.

    NOTA:Durante a criação da conta, o login é efetuado pelo SSL. Quando for necessário aceitar o certificado do servidor, você deverá validar o certificado e aceitá-lo para ter uma conexão bem-sucedida.

  5. Se o administrador tiver habilitado a política de criptografia, o iFolder solicitará que você digite a frase secreta.

    Digite a frase secreta no campo Frase secreta.

    Digite-a novamente no campo Digitar novamente a frase secreta.

    Marque a caixa de seleção Lembrar a frase secreta para acessar automaticamente os dados do iFolder criptografado durante a sessão atual. Se você selecionar essa opção, o iFolder poderá lembrar sua frase secreta no próximo processo de login. Essa troca de frase secreta é feita automaticamente.

  6. Selecione o agente de recuperação desejado na lista e clique em OK.

  7. Especifique a localização em que deseja criar a pasta padrão do iFolder.

    Por padrão, o iFolder padrão é armazenado em <diretório pessoal do usuário>/ifolder/nome_do_servidor/nome_do_usuario.

  8. Selecione o tipo da pasta padrão do iFolder que você deseja criar. Se você escolher Criptografia, será solicitado a fornecer a frase secreta.

    IMPORTANTE:Você pode compartilhar apenas as pastas comuns do iFolder. As pastas criptografadas do iFolder não podem ser compartilhadas.

    Se o administrador do iFolder não tiver habilitado a criptografia da conta atual, você só poderá criar iFolders regulares. Se a criptografia estiver habilitada, a opção Regular estará selecionada por padrão.

  9. Selecione Sincronização Segura para permitir a sincronização segura dos arquivos no iFolder que você está criando.

    Você pode habilitar o canal seguro para as pastas comuns e criptografadas do iFolder. Com essa opção selecionada, é possível usar um canal seguro para sincronizar entre o iFolder no servidor e o contido na máquina local.

    Se você remover o iFolder padrão no servidor, na próxima vez que configurar a conta de um thick client, será solicitada a criação de um iFolder padrão novamente.

    No entanto, se você remover o iFolder padrão da máquina local, na próxima vez que configurar a conta, o iFolder solicitará que você faça o download do iFolder padrão.

  10. Caso já tenha configurado um iFolder padrão, você poderá fazer o download do iFolder para a sua máquina local.

    Faça o download do iFolder padrão e clique em Avançar para continuar.

  11. Se uma pasta com o mesmo nome do iFolder padrão já existir no seu sistema, você será solicitado a fundir as duas pastas ou fazer o download do iFolder padrão em um local diferente.

    Clique em OK para fundir as duas pastas ou em Cancelar para fazer o download do iFolder padrão em outro local.

  12. Clique em Concluir para fechar o Assistente de Criação de Conta do iFolder.

    O iFolder sincroniza a lista de usuários referente à conta especificada e identifica as respectivas pastas disponíveis para download. Esse download inicial pode demorar alguns segundos ou minutos, dependendo do tamanho da lista de usuários da conta.

  13. Verifique se a conta configurada aparece na lista de contas.

    1. Clique o botão direito do mouse no ícone Serviços do iFolder na área de notificação e selecione Preferências no menu para abrir a caixa de diálogo Preferências do iFolder, na guia Contas.

      Quando você está conectado ao servidor iFolder, a caixa de seleção Online é marcada.

    2. Feche a caixa de diálogo Preferências do iFolder.

  14. Configure pastas iFolder para a conta utilizando um dos seguintes métodos:

    • Clique o botão direito do mouse no ícone Serviços iFolder na área de notificação e selecione iFolders para abrir o browser do iFolder. Vá para a Criando e fazendo upload de uma pasta do iFolder.

    • Se as pastas do iFolder dessa conta estiverem disponíveis para download a partir do servidor, elas estarão listadas na janela do iFolder em pastas do iFolder em <nome_servidor_iFolder>. O ícone de download indica que o iFolder está disponível para download. Vá para a Fazendo download de uma pasta disponível do iFolder.

Para obter mais informações, consulte Informações legais.
