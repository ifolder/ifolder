Movendo uma pasta iFolder

Depois que uma pasta iFolder é criada, o caminho para o iFolder fica estático. Você não pode mover uma pasta iFolder de sua localização original para outra localização em seu computador local e mantê-la como uma pasta iFolder. A movimentação da pasta iFolder quebra o vínculo entre o banco de dados de sincronização e a pasta que você converteu em uma pasta iFolder.

Para mover uma pasta iFolder no computador onde foi criado:

  1. Em um gerenciador de arquivos ou no browser do iFolder, selecione a pasta do iFolder a ser removida.

  2. Reverta-a para uma pasta normal.

  3. Em um gerenciador de arquivos, mova a pasta para outra localização no computador.

  4. Converta a pasta de volta para uma pasta do iFolder e faça seu upload para o servidor.

  5. (Opcional) Compartilhe o iFolder.

  6. Permita que o iFolder sincronize arquivos com o respectivo servidor.

Para mover uma pasta iFolder em um computador participante:

  1. Reverta as pastas do iFolder para que não participem mais do iFolder no computador atual.

  2. Selecione a pasta disponível do iFolder e clique em Fazer Download de uma Pasta do iFolder.

  3. Especifique um novo caminho local onde deseja colocar a pasta iFolder.

  4. Permita que o iFolder sincronize arquivos com o respectivo servidor.

Para obter mais informações, consulte Informações legais.
