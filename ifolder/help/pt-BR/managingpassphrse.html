Gerenciando frases secretas de iFolders criptografados

Um iFolder criptografado possui uma chave de criptografia de dados exclusiva que é gerada automaticamente durante a criação do iFolder. A chave é criptografada usando a frase secreta fornecida pelo usuário e a chave pública do agente de recuperação.

O agente de recuperação será selecionado durante o processo de criação da conta do iFolder se o administrador do iFolder tiver habilitado a política de criptografia no servidor.

Se o administrador do iFolder tiver habilitado a criptografia, uma caixa de diálogo será exibida durante a criação do iFolder para que você possa inserir a frase secreta. O iFolder usa a frase secreta para gerar uma chave de criptografia exclusiva para criptografar ou descriptografar os dados do iFolder no servidor iFolder. Se você esquecer a frase secreta, não poderá acessar os dados do iFolder. É necessário redefinir a frase secreta para ter acesso aos dados do iFolder.

Recuperando um iFolder criptografado

Use o Assistente de Recuperação de Frase Secreta para redefinir a frase secreta e recuperar um iFolder criptografado. Siga as etapas abaixo para iniciar o Assistente de Recuperação de Frase Secreta:

  1. No browser do iFolder, clique em Segurança > Frase Secreta Esquecida para iniciar o assistente.

  2. Na lista Conta do iFolder, selecione o domínio para o qual a frase secreta deve ser redefinida e clique em Avançar.

    Durante o processo de criação da conta do iFolder, se a opção Server_Default estiver selecionada como o agente de recuperação, a página Configurar nova frase secreta será exibida. Para redefinir a nova frase secreta, consulte Redefinindo a frase secreta quando o agente de recuperação é Server_Default.

    No entanto, se o agente de recuperação selecionado não for Server_Default, será necessário redefinir a frase secreta usando Redefinindo a frase secreta quando o agente de recuperação não é Server_Default.

Redefinindo a frase secreta quando o agente de recuperação é Server_Default

Para configurar a nova frase secreta, siga estas etapas:

  1. Na página Configurar nova frase secreta, especifique a nova frase secreta no campo Nova Frase Secreta.

  2. Digite mais uma vez a nova frase secreta no campo Confirmar Frase Secreta > .

  3. Especifique a senha no campo Senha e clique em Avançar.

  4. Depois que a frase secreta for redefinida, clique em Concluir para fechar o assistente.

Redefinindo a frase secreta quando o agente de recuperação não é Server_Default

Para redefinir a frase secreta quando o agente de recuperação selecionado não for Server_Default, siga estas etapas:

  1. A página de Boas-Vindas do assistente exibe as informações que você precisa para redefinir a frase secreta. Clique em Avançar para exibir a página Mecanismo de Recuperação de Frase Secreta.

  2. Estes são os três métodos que você pode usar no assistente para redefinir sua frase secreta:

    • Usando o arquivo secreto e sua senha: O arquivo secreto e sua senha são usados para redefinir a frase secreta. Caso tenha mantido o arquivo secreto e a senha, você poderá usar o assistente para especificar a localização do arquivo secreto e fornecer sua senha. Para redefinir a frase secreta usando o arquivo secreto, consulte Usando o arquivo secreto para redefinir a frase secreta.

      NOTA:O arquivo secreto e a senha são fornecidos pelo administrador do iFolder.

    • Usando o novo arquivo de dados: O novo arquivo de dados também pode ser usado para redefinir a frase secreta. Você poderá receber o novo arquivo de dados do administrador depois que executar uma operação de exportação do arquivo antigo. Para redefinir a frase secreta usando o novo arquivo de dados, consulte Usando o novo arquivo de dados para redefinir a frase secreta.

    • Exportando o arquivo de dados antigo: Caso não tenha o arquivo secreto ou o novo arquivo de dados, você poderá usar o assistente para exportar o arquivo de dados antigo e enviá-lo por e-mail para o seu administrador. Quando o administrador devolver o novo arquivo de dados, você poderá usar o assistente para recuperar a frase secreta. Para exportar o arquivo de dados antigo, consulte Usando um arquivo exportado para redefinir a frase secreta.

Usando o arquivo secreto para redefinir a frase secreta

Para redefinir a frase secreta usando o arquivo secreto, execute as seguintes etapas:

  1. Selecione a opção Eu tenho o arquivo secreto e a senha e clique em Avançar.

  2. Preencha os campos a seguir:

    Localização do arquivo secreto: Especifique a localização do arquivo secreto ou clique no botão Procurar para localizar e selecionar o arquivo secreto.

    Senha Secreta: Especifique a senha do arquivo secreto.

    Nova frase secreta: Especifique a nova frase secreta.

    Confirmar frase secreta Digite novamente a nova frase secreta.

  3. Clique em Avançar para redefinir a frase secreta.

  4. Clique em Concluir para fechar o assistente.

NOTA:O arquivo secreto e a senha são fornecidos pelo administrador do iFolder.

Usando o novo arquivo de dados para redefinir a frase secreta

Para redefinir a frase secreta usando o arquivo secreto, execute as seguintes etapas:

  1. Selecione a opção Eu tenho o novo arquivo de dados enviado pelo administrador e clique em Avançar.

  2. No campo Localização do novo arquivo de dados, especifique a localização do novo arquivo de dados, ou clique no botão Procurar para localizá-lo e selecioná-lo.

  3. Se o novo arquivo de dados que você recebeu do administrador estiver criptografado, selecione a caixa de seleção O arquivo acima está criptogrado?. Quando você marcar essa caixa de seleção, o campo Senha única será habilitado.

  4. No campo Senha única, especifique a senha usada pelo administrador para criptografar o novo arquivo de dados.

  5. No campo Nova frase secreta, especifique a nova frase secreta.

  6. No campo Confirmar frase secreta, digite novamente a nova frase secreta.

  7. Clique em Avançar para redefinir a frase secreta.

  8. Clique em Concluir para fechar o assistente.

Usando um arquivo exportado para redefinir a frase secreta

Para redefinir a frase secreta quando você não tiver o arquivo secreto ou o novo arquivo de dados:

  1. Selecione a opção Eu não tenho arquivo secreto nem arquivo de dados e clique em Avançar.

  2. No campo Local para gravar o arquivo de dados antigo, especifique o local no qual o arquivo de dados antigo deverá ser gravado e clique no botão Procurar para especificar o local no qual o arquivo de dados antigo deverá ser gravado.

  3. Clique em Avançar para obter o arquivo de dados antigo. A localização do arquivo de dados antigo e o endereço de e-mail do administrador do iFolder são exibidos. Você pode usar essas informações para obter o novo arquivo de dados.

  4. Clique em Concluir para fechar o assistente.

  5. Envie o arquivo de dados antigo por e-mail para o administrador do iFolder com uma solicitação para receber o novo arquivo de dados.

  6. Quando você receber o novo arquivo de dados, consulte Usando o novo arquivo de dados para redefinir a frase secreta para redefinir a frase secreta.

Mudando a frase secreta

Por motivos de segurança, você deve mudar sua frase secreta periodicamente. Você pode redefini-la usando a senha única enviada pelo Agente de Recuperação.

  1. No browser do iFolder, selecione Segurança > Mudar Frase Secreta.

  2. Na janela Mudar Frase Secreta, selecione a conta do iFolder cuja frase secreta você deseja redefinir e especifique os seguintes detalhes:

    Digitar Frase Secreta: Especifique a frase secreta.

    Digitar Nova Frase Secreta: Especifique a nova senha que deseja configurar para a conta do iFolder atual.

    Redigitar Frase Secreta: Especifique a nova frase secreta novamente para confirmação.

    Agente de Recuperação: Selecione o Agente de Recuperação que você deseja definir para a conta do iFolder atual.

  3. Clique em Redefinir.

Para obter mais informações, consulte Informações legais.
