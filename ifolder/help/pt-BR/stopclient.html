Saindo do cliente iFolder

Para sair do cliente do iFolder siga uma das opções abaixo:

  • Clique o botão direito do mouse no ícone Serviços do iFolder na área de notificação e selecione Sair.

  • Clique em iFolder > Sair

    Quando você seleciona Sair, o iFolder exibe uma mensagem de confirmação. Clique em Sim para confirmar sua ação.

Os arquivos são sincronizados com sua conta no servidor iFolder somente quando você está conectado a esse servidor. Você pode parar a sincronização efetuando logout de uma conta. O iFolder continua a ser executado em segundo plano no modo offline. Para obter informações, consulte a Efetuando logout de uma conta do iFolder.

Você pode ter várias contas, sendo que cada uma delas tenha sido criada em um servidor de host diferente (domínio).

Para obter mais informações, consulte Informações legais.
