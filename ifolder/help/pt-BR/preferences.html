Configurando as preferências do iFolder para o cliente

  1. Clique o botão direito do mouse no ícone Serviços do iFolder na área de notificação e selecione Preferências para abrir a caixa de diálogo Preferências do iFolder na guia Geral.

  2. Especifique suas preferências para o seguinte:

    Parâmetro

    Descrição

    Iniciar o iFolder quando efetuar login na área de trabalho (Disponível somente para Windows)

    Selecione essa opção para iniciar o iFolder sempre que efetuar login com o nome de usuário local onde você configurar a conta do iFolder. No caso das contas em que você habilitou a opção Lembrar Minha Senha, a sincronização ocorre em segundo plano durante o intervalo de sincronização especificado.

    Desmarque esta opção para iniciar o iFolder manualmente.

    Exibir caixa de diálogo de confirmação de criação do iFolder

    Marque esta caixa de seleção para receber uma mensagem de confirmação sempre que criar um iFolder, além de indicações visuais normais, como a exibição do iFolder na lista de iFolders.

    Desmarque esta caixa de seleção para não receber mais a mensagem de confirmação.

    Ocultar janela principal do iFolder na inicialização

    Desmarque esta caixa de seleção para exibir a janela do iFolder na inicialização.

    Marque esta caixa de seleção caso deseje que a janela do iFolder permaneça oculta durante a inicialização.

    Exibir registros de sincronização

    Marque esta caixa de seleção caso deseje exibir registros de sincronização na inicialização.

    Desmarque esta caixa de seleção caso deseje ocultar registros de sincronização na inicialização.

    Exibir notificações de violações de políticas

    Marque esta caixa de seleção para exibir uma mensagem pop-up na área de notificação sempre que uma política for violada.

    Notificar quando iFolders forem compartilhados

    Marque esta caixa de seleção para exibir uma mensagem pop-up na área de notificação sempre que receber um convite para compartilhar um iFolder.

    Para ver uma lista dos iFolders que estão disponíveis para download no computador atual, selecione Mostrar iFolders Disponíveis no browser do iFolder sempre que o iFolder estiver sendo executado.

    Notificar quando houver conflitos

    Marque esta caixa de seleção para exibir uma mensagem pop-up na área de notificação sempre que ocorrer um conflito na sincronização de arquivos.

    Desmarque esta caixa de seleção para desabilitar as mensagens pop-up dos conflitos de sincronização.

    Sempre que houver conflitos, o ícone Erro do iFolder será exibido quando o iFolder estiver em execução.

    Notificar quando um usuário ingressar em um domínio do iFolder

    Marque esta caixa de seleção para exibir uma mensagem pop-up na área de notificação sempre que um usuário aceitar um convite para compartilhar um iFolder.

    Desmarque esta caixa de seleção para desabilitar as mensagens pop-up.

    Você pode ver uma lista completa de usuários do iFolder sempre que este estiver execução, selecionando a pasta do iFolder e clicando em Compartilhar com para abrir a caixa de diálogo Propriedades do iFolder na guia Compartilhamento.

    Sincronização

    Sincronizar Pastas iFolder Automaticamente: Marque esta caixa de seleção para habilitar a sincronização de todas as pastas do iFolder neste computador com o login local ou a identidade de logon atual. Desmarque a caixa de seleção para desabilitar a sincronização.

    Sincronizar Pastas do iFolder a Cada: Especifique o intervalo mínimo a ser usado para a sincronização de pastas iFolder neste computador sob a sua identidade de login ou logon local. Especifique o valor e as unidades. O padrão é 5 minutos se essa opção estiver habilitada. O intervalo mínimo pode ser configurado para 5 segundos.

    Se você usar o cliente do iFolder com o servidor do Novell iFolder 3.7 ou posterior, o intervalo de sincronização mínimo efetivo será sempre o maior valor das seguintes configurações:

    • O padrão de zero (0) da política do sistema, a não ser que exista uma política do usuário definida. Se uma política do usuário estiver definida, sendo ela maior ou menor no valor, ela anulará a política do sistema.

    • A política da máquina local ou a configuração na máquina do cliente sincronizando com o servidor.

    • A política do iFolder (coleção).

  3. Clique em Fechar para fechar a caixa de diálogo Preferências do iFolder.

Para obter mais informações, consulte Informações legais.
