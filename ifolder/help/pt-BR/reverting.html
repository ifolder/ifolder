Revertendo um iFolder para uma pasta normal

Você pode reverter uma pasta do iFolder para uma pasta normal no computador local. Após a reversão, o relacionamento de compartilhamento colaborativo permanece para todos os membros do iFolder, mas este não é mais sincronizado com o seu computador. A reversão de uma pasta do iFolder não apaga esse programa e seu conteúdo de estações de trabalho membros ou do servidor.

  1. Em um gerenciador de arquivos ou no browser do iFolder, localize a pasta do iFolder que você deseja reverter para uma pasta normal.

  2. Use um dos seguintes métodos para iniciar a reversão da pasta do iFolder:

    • Em um gerenciador de arquivos, clique o botão direito do mouse na pasta do iFolder e clique em Reverter para uma Pasta Normal.

    • No browser do iFolder, selecione iFolder e clique em iFolder > Reverter para uma pasta normal.

    A caixa de diálogo Reverter este iFolder de volta para uma pasta normal? solicita confirmação e esclarecimento.

  3. Siga um destes procedimentos:

    • Selecione Não para manter a pasta como uma pasta do iFolder no computador local.

    • Selecione Sim para reverter a pasta do iFolder para uma pasta normal.

      A cópia local do iFolder é uma pasta normal e não é mais sincronizada com o servidor. Outros membros do iFolder não são afetados.

    • Se você for o proprietário do iFolder, poderá opcionalmente parar de compartilhar essa pasta como uma pasta do iFolder. Selecione Apagar este iFolder do servidor e clique em Sim.

      A pasta do iFolder é revertida para uma pasta normal no seu computador local e em todos os computadores membros. A cópia de servidor iFolder é permanentemente apagada do servidor.

Para obter mais informações, consulte Informações legais.
