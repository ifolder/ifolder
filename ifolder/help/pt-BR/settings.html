Configurações do iFolder

Você pode usar a guia de Configurações do iFolder para habilitar as notificações.

  1. Selecione a opção Exibir Notificações para habilitar as notificações.

  2. A partir da lista de notificações, selecione os eventos para os quais você deseja habilitar as notificações.

Para obter mais informações, consulte Informações legais.
