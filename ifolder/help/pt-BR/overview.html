Visão geral do iFolder

O Novell iFolder é um aplicativo de compartilhamento de arquivos para clientes do Linux, Windows e Macintosh. É possível compartilhar arquivos em várias pastas do iFolder, cada qual com usuários ou Grupos LDAP distintos. Você controla quem pode fazer parte de um iFolder e o nível de acesso de cada membro. Além disso, você também pode fazer parte de pastas iFolder que terceiros compartilham com você.

O iFolder apresenta uma interface de usuário simples e fácil de usar para migração do lado do cliente do iFolder 2.X para o iFolder 3.6 e posterior, e para um upgrade do lado do cliente do iFolder 3.X para o iFolder 3.6 e posterior.
