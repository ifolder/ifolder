Configurando as definições do programa de varredura de vírus local para o tráfego do iFolder

Se você usar o software de varredura de vírus no seu computador, exclua o diretório ..\simias\WorkArea\ da verredura. Por exemplo, a localização do diretório ..\simias\WorkArea\ no Windows é c:\documents and settings\nomedeusuario\local settings\application data\simias\WorkArea\.simias.wf.guid.

O diretório ..\simias\WorkArea\ é o local onde o iFolder disponibiliza os arquivos para download no servidor. Se um arquivo do iFolder tiver um vírus anexado e o software de varredura de vírus detectar esse vírus enquanto o arquivo estiver no diretório ..\simias\WorkArea, esse software exibirá uma mensagem indicando que um vírus foi encontrado e moverá o arquivo para quarentena. Em seguida, o iFolder tentará sincronizar o arquivo novamente.

Se a área de trabalho não for explorada, a sincronização poderá ser concluída. O arquivo infectado será transferido por download e movido para a pasta local do iFolder. O programa de varredura de vírus detecta o arquivo infectado no iFolder e o move desta pasta para a quarentena, protegendo o computador da infecção. O iFolder detecta que o arquivo foi apagado e quando a próxima sincronização começar, o iFolder removerá o arquivo do servidor e dos computadores dos membros.

Porém, se o iFolder para onde é realizado o download de um arquivo for Apenas leitura, a exclusão local não será forçada para o servidor e para os computadores dos outros membros. O vírus deve ser detectado nas outras partes onde as permissões possibilitam que as mudanças sejam sincronizadas. O programa de varredura de vírus provavelmente alertará o usuário sobre o vírus para que ele possa alertar outros membros do iFolder sobre isso.

Para obter mais informações, consulte Informações legais.
