Browser do iFolder

O browser do iFolder permite que você gerencie seus iFolders. A tabela abaixo resume os elementos do browser do iFolder:

Tabela 2 Elementos do browser do iFolder

Nome do Elemento

Descrição

Criar iFolder

Permite que faça a conversão de uma pasta na máquina local em um iFolder e faça upload desse iFolder no servidor iFolder ao qual está conectado.

Para obter mais informações, consulte a Criando e fazendo upload de uma pasta do iFolder.

Sincronizar agora

Permite iniciar imediatamente uma sincronização dos iFolders na máquina local e no servidor.

Compartilhar com

Permite compartilhar o iFolder com outros usuários.

Para obter mais informações, consulte a Compartilhando um iFolder.

Reverter para uma pasta normal

Permite reverter um iFolder para uma pasta normal. Você também pode apagar o iFolder do servidor. Para obter mais informações, consulte a Revertendo um iFolder para uma pasta normal.

Servidor

Exibe o endereço IP do servidor iFolder ao qual você está conectado.

Nome

Exibe o nome do iFolder selecionado.

Proprietário

Exibe o nome do proprietário do iFolder selecionado.

Direitos

Exibe os direitos do proprietário do iFolder.

Arquivos/pastas a serem sincronizados

Indica o número de arquivos ou pastas que estão aguardando a sincronização.

Hora da última sincronização bem-sucedida

Indica a hora e a data da última sincronização bem-sucedida.

Fazer download de um iFolder

Permite fazer o download de um iFolder do servidor iFolder para a máquina local.

Para obter mais informações, consulte a Fazendo download de uma pasta disponível do iFolder.

Tipo de iFolder

Indica se o iFolder selecionado é criptografado ou regular.

Tamanho do iFolder

Exibe o tamanho do iFolder selecionado.

Fundir

A ação de fundir ajuda você a consolidar os dados das pastas do iFolder e minimizar a transferência de dados entre o cliente e o servidor iFolder. Você pode fundir o conteúdo em uma pasta do iFolder no servidor com o conteúdo em um conteúdo da pasta existente na máquina local. Para obter mais informações, consulte a Fundindo pastas do iFolder.

Apagar do servidor

Permite apagar o iFolder selecionado do servidor.

Lista Conta

Exibe a lista de servidores nos quais você configurou uma conta.

Conectar/Desconectar

Permite que você se conecte ou cancele a conexão com um servidor iFolder. Se você estiver conectado ao servidor iFolder, o botão Desconectarserá exibido. Se você não estiver conectado a um servidor iFolder, o botão Conectar será exibido.

Resolver

Permite resolver conflitos de arquivo. Para resolver conflitos entre versões de arquivos, selecione a versão correta na caixa de diálogo Resolução do Conflito. Para obter mais informações, consulte a Resolvendo conflitos de arquivos.

NOTA:No cliente Macintosh, para resolver os conflitos de nome, é necessário atualizar manualmente o nome do arquivo ou da pasta para garantir que caracteres não suportados não sejam usados ( \ / : * ? " < > | ; ) tÃ©cnico.

Usuário

Exibe o nome do usuário que efetuou login no servidor iFolder selecionado.

Número de iFolders

Exibe o número total de iFolders no servidor iFolder e a máquina local do servidor iFolder selecionado.

Quota de Disco

Exibe os valores dos campos Usado e Disponível.

Usado

Exibe o espaço no disco do servidor que está sendo ocupado por todos os arquivos e diretórios no servidor iFolder selecionado.

Disponível

Exibe o espaço no disco do servidor que está disponível para os arquivos ou diretórios no servidor iFolder selecionado.

Sincronização automática

Exibe o intervalo de tempo entre dois ciclos consecutivos de sincronização automática

Para obter mais informações, consulte Informações legais.
