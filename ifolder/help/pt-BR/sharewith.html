Compartilhando um iFolder

Você pode compartilhar uma pasta iFolder com vários usuários. Esta seção discute as seguintes tarefas:

  • Compreendendo os direitos de acesso do usuário

  • Acessando a guia Compartilhamento

  • Adicionando um usuário a uma pasta iFolder

  • Modificando os direitos de acesso do usuário

  • Removendo um usuário de uma pasta iFolder

  • Transferindo a propriedade para um usuário do iFolder

Acessando a guia Compartilhamento

Você pode gerenciar a participação do iFolder na guia Compartilhamento na caixa de diálogo Propriedades do iFolder.

IMPORTANTE:Você poderá compartilhar pastas do iFolder se o administrador do iFolder tiver habilitado a política de compartilhamento para sua conta.

  1. Se o iFolder não estiver em execução, inicie-o.

  2. Execute um dos procedimentos a seguir para abrir a caixa de diálogo Propriedades do iFolder na guia Compartilhamento.

    • Em um gerenciador de arquivos, clique o botão direito do mouse na pasta do iFolder que deseja compartilhar e selecioneCompartilhar com.

    • No browser do iFolder, selecione a pasta do iFolder e, em seguida, Compartilhar com.

Adicionando um usuário a uma pasta iFolder

  1. Na guia Compartilhamento, clique em Adicionar para abrir a caixa de diálogo Selecionar Usuário.

  2. Na lista de usuários, selecione um ou mais usuários ou Grupos LDAP adicionando-os à lista Usuários a serem adicionados e clique em OK.

    Pode levar alguns segundos até que os usuários se sincronizem com o servidor. Logo após a sincronização, aparecerá uma notificação na parte inferior direita da página.

  3. Na guia Compartilhamento, selecione um ou mais usuários aos quais você deseja designar o mesmo direito de acesso.

  4. Clique em Direitos, selecione o direito de acesso Controle Total, Leitura/Gravação ou Somente Leitura e clique em OK.

  5. Repita as etapas anteriores até que todos os usuários ou Grupos LDAP desejados tenham sido convidados a compartilhar a pasta do iFolder e tiverem definido seus direitos de acesso.

  6. Clique em Fechar para fechar a janela Propriedades do iFolder.

Modificando os direitos de acesso do usuário

Conforme as mudanças nos relacionamentos de trabalho e no status dos membros de uma pasta do iFolder, talvez seja necessário mudar o direito de acesso de um usuário a essa pasta. Por exemplo, convém conceder a um usuário confiável o direito Controle Total à pasta do iFolder.

  1. Na guia Compartilhamento, selecione um ou mais usuários que devem ter o mesmo direito de acesso.

  2. Selecione Direitos e especifique Controle Total, Leitura/Gravação ou Somente Leitura.

    Quando você convida pela primeira vez um usuário ou Grupo LDAP a participar, a opção Tornar este usuário o proprietário do iFolder está desabilitada (esmaecida) na caixa de diálogo Acesso. Se você pretende tornar o novo usuário o proprietário, deverá esperar até que o estado dele mude de Usuário Convidado paraUsuário do iFolder; depois poderá transferir a propriedade. Quando você transfere a propriedade para um Grupo LDAP, todos os membros desse grupo recebem os direitos de propriedade.

  3. Clique em OK e em OK novamente para fechar a janela Propriedades do iFolder.

Removendo um usuário de uma pasta iFolder

Para cancelar o compartilhamento de uma pasta do iFolder, você precisa remover um usuário ou Grupo LDAP como membro de uma pasta do iFolder. A pasta do iFolder na estação de trabalho do membro do Grupo LDAP ou do usuário se torna uma pasta normal. Uma cópia dos dados permanecem na estação de trabalho do membro anterior, mas os arquivos não são mais sincronizados com a pasta compartilhada do iFolder.

  1. Na guia Compartilhamento, selecione um ou mais usuários na lista.

  2. Clique em Remover e em Sim.

Transferindo a propriedade para um usuário do iFolder

O proprietário de uma pasta do iFolder pode transferir a propriedade dessa pasta para outro membro individual ou Grupo LDAP cujo estado seja Usuário do iFolder. Após a transferência, o proprietário original torna-se membro com o direito Leitura/Gravação, e o espaço consumido pelo iFolder é contabilizado em relação ao novo proprietário. Quando um Grupo LDAP se torna proprietário de uma pasta do iFolder, todos os membros do grupo recebem os direitos de propriedade.

  1. Em um gerenciador ou no browser do iFolder, clique o botão direito do mouse na pasta do iFolder que deseja transferir e selecione Compartilhar com para abrir a caixa de diálogo Propriedades do iFolder na guia Compartilhamento.

  2. Na lista de usuários, selecione o Usuário do iFolder que deverá ser o novo proprietário e clique em Direitos.

  3. Selecione Tornar este usuário o proprietário do iFolder.

    A opção Tornar este usuário o proprietário do iFolder estará desabilitada (esmaecida) na caixa de diálogo Acesso se o estado do usuário for Usuário Convidado. Nesse caso, saia da caixa de diálogo, espere até que o estado do usuário mude deUsuário Convidado para Usuário do iFolder e repita as etapas anteriores para transferir a propriedade.

  4. Clique em OK e em OK novamente para fechar a janela Propriedades do iFolder.
