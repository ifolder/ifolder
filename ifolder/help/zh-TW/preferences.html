設定用戶端的 iFolder 優先設定

  1. 以滑鼠右鍵按一下通知區域中的「iFolder 服務」圖示，然後選取「優先設定」開啟「優先設定」對話方塊至「一般」索引標籤。

  2. 針對下列各項指定您的優先設定：

    參數

    描述

    登入桌面時啟動 iFolder (僅可於 Windows 用戶端中使用)

    選取此核取方塊，這樣每當您以設定 iFolder 帳戶所使用的本地使用者名稱登入時，便會啟動 iFolder。針對您已啟用「記住密碼」的帳戶，在您登入時就不會提示指定密碼。

    取消選取此核取方塊，可手動啟動 iFolder。

    成功建立 iFolder 時顯示確認對話方塊

    選取此核取方塊，這樣每當建立 iFolder 時，除了一般視覺線索 (如在 iFolder 清單中出現 iFolder) 之外，還會接收到確認訊息。

    取消選取此核取方塊，可停止接收確認訊息。

    隱藏 iFolder 系統匣圖示 (只在 Windows 用戶端中可用)

    選取此核取方塊以隱藏顯示在系統匣中的 iFolder 圖示。

    取消選取此核取方塊以在系統匣中顯示 iFolder 圖示。

    警告：如果您選取此核取方塊，請確保已取消選取「啟動時隱藏 iFolder 主視窗」核取方塊。否則，您將不會看到顯示 iFolder 應用程式正在執行的視覺提示。

    啟動時隱藏 iFolder 主視窗

    若要在啟動時隱藏 iFolder 視窗，請選取此核取方塊。

    取消選取此核取方塊可在啟動時顯示 iFolder 視窗。

    顯示同步記錄

    若要在啟動時顯示同步記錄，請選取此核取方塊。

    若要在啟動時隱藏同步記錄，請取消選取此核取方塊。

    違反配額規則時

    選取此核取方塊可在違反磁碟配額規則時，在通知區域顯示快顯訊息。

    取消選取此核取方塊以停用此通知。

    違反檔案大小規則時

    選取此核取方塊可在違反檔案大小規則時，在通知區域顯示快顯訊息。

    取消選取此核取方塊以停用此通知。

    違反檔案排除規則時

    選取此核取方塊可在違反檔案排除規則時，在通知區域顯示快顯訊息。

    取消選取此核取方塊以停用此通知。

    磁碟已滿時

    選取此核取方塊可在本地系統中的磁碟已滿時，在通知區域顯示快顯訊息。

    取消選取此核取方塊以停用此通知。

    所需許可無法使用時

    選取此核取方塊，可在無法使用對檔案或資料夾執行作業所需的許可時，在通知區域顯示快顯訊息。例如，如果您只有讀取權限，將無法下載檔案。

    取消選取此核取方塊以停用此通知。

    檔案路徑超過最佳限制時

    選取此核取方塊可在檔案路徑超過系統定義的最大長度時，在通知區域顯示快顯訊息。例如，在基於 Windows 的平台上，路徑中的字元數必須少於 248 個字元。

    取消選取此核取方塊以停用此通知。

    共享 iFolders 時

    選取此核取方塊，這樣每當接收到新的共享 iFolder 邀請時，就會在通知區域中顯示快顯訊息。

    取消選取此核取方塊以停用此通知。

    發生衝突時

    選取此核取方塊，這樣每當同步化檔案發生衝突時，便會在通知區域中啟用快顯訊息。

    取消選取此核取方塊，可停用同步化衝突的快顯訊息。

    每當衝突發生時，會在 iFolder 上顯示「iFolder 錯誤」 圖示。

    同步化

    自動同步化 iFolder: 選取此核取方塊，可讓此電腦上屬於目前本地登入或登入身分下的所有 iFolder 同步化。取消選取此核取方塊，可提用同步化。

    同步化 iFolder 間隔: 指定用於同步化此電腦上屬於目前本地登入或登入身分下之 iFolder 的最小間隔。指定值及單位。如果啟用這個選項，預設值是 5 分鐘。間隔最小可設為 5 秒鐘。

    如果您搭配使用 iFolder 用戶端與 Novell iFolder 3.7 或更新版本，則有效的最小同步化間隔一律為下列設定中最大的值：

    • 系統規則預設為零 (0)，除非存在使用者規則集。 如果設定使用者規則，則使用者規則會覆寫系統規則，而不論使用者規則的值較之是大還是小。

    • 本地機器規則，或與伺服器同步化之用戶端機器的設定。

    • iFolder (集合) 規則。

  3. 按一下「關閉」以關閉「iFolder 優先設定」對話方塊。

有關更詳細的資訊，請參閱 法律聲明.
