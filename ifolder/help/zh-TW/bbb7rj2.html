從共用 iFolder 移除成員資格

從下載的共用 iFolder 移除成員資格：

重要：如果與 LDAPGroup 共用 iFolder，則不允許 LDAPGroup 的個別成員從該 iFolder 移除他們的成員資格。

  1. 在檔案管理員或 iFolder 瀏覽器中，找到要從中移除成員資格的 iFolder。

  2. 以滑鼠右鍵按一下 iFolder 然後選取「回復為一般資料夾」以顯示確認對話方塊。

  3. 在確認對話方塊中，選取「同時從 iFolder 移除我的成員資格」。

  4. 按一下「是」以確定。

    iFolder 的本地副本便成為一般資料夾，並不再與伺服器同步。

從伺服器上的共用 iFolder 移除成員資格：

  1. 在 iFolder 瀏覽器中，找到要從中刪除成員資格的 iFolder。

  2. 以滑鼠右鍵按一下 iFolder 然後選取「移除我的成員資格」。

  3. 按一下「確定」，確認您的動作。

有關更詳細的資訊，請參閱 法律聲明.
