啟動 iFolder 用戶端

當 iFolder 執行時，工作列的通知區域會顯示「iFolder 服務」圖示。iFolder 會在桌面環境中整合。在檔案管理員中、桌面上或 iFolder 瀏覽器中檢視 iFolder 時，該 iFolder 上會出現 iFolder 徽章 (綠色的 “i”)。加密的 iFolder 由 徽章 (鎖定的資料夾) 指示。

重要：您必須開啟工作列的通知區域，iFolder 執行時才能顯示「iFolder 服務」圖示。

您隨時可以從本地使用本地 iFolder 中的檔案。當 iFolder 用戶端正在執行時，它會在您工作時記錄您對本地 iFolder 資料所做的變更。如果您在 iFolder 執行時變更本地 iFolder 資料，iFolder 會識別且在下次您啟動 iFolder 時記錄其差異。

當 iFolder 沒有執行時，iFolder 會出現在檔案管理員中或桌面上。您可以存取 iFolder 本地副本中的檔案，但是無法管理 iFolder。此外，您必須登入帳戶，才能建立 iFolder 或同步化其檔案。

附註：建立帳戶期間，使用者登入會透過 SSL 進行。系統提示您接受伺服器證書時，您必須驗證並接受該證書，才能成功連接。

如需有關停止 iFolder 用戶端的詳細資訊，請參閱 離開 iFolder 用戶端。

Linux

  • 登入時自動啟動 iFolder

  • 手動啟動 iFolder

登入時自動啟動 iFolder

若要啟用登入桌面時自動啟動 iFolder，請執行以下動作之一：

  • 開啟應用程式功能表，按一下「更多應用程式」，找出並以滑鼠右鍵按一下「iFolder 3」，然後從功能表選取「新增至啟動程式」。

  • 確定您登出時 iFolder 正在執行，然後儲存工作階段設定。登入桌面時，請選取此工作階段選項。

  • 開啟終端機視窗，以根部使用者身份登入，然後執行 /opt/gnome/bin/gnome-session-properties 以新增 iFolder 為您登入電腦時要啟動的程式。

若要停用登入時自動啟動 iFolder，請執行以下動作之一：

  • 開啟應用程式功能表，找出並以滑鼠右鍵按一下 iFolder 3，然後選取「從啟動程式中移除」。

  • 確定您登出時 iFolder 沒有在執行，然後儲存工作階段設定。登入桌面時，請選取此工作階段選項。

手動啟動 iFolder

  1. 使用您建立 iFolder 時要所使用的本地 Linux 使用者識別登入您的電腦。

  2. 請使用下列方法之一來啟動 iFolder：

    • 在工作列中，開啟應用程式功能表，按一下「更多應用程式」，在可用的選項中搜尋或捲動以找出 iFolder 3，以滑鼠右鍵按一下 iFolder 3，然後選取「啟動 iFolder 3」。

    • 開啟終端機外圍程序，然後輸入

      /usr/bin/ifolder

Windows

  • 登入時自動啟動 iFolder

  • 手動啟動 iFolder

登入時自動啟動 iFolder

當您登入桌上型電腦時，若要自動啟動 iFolder：

  1. 以滑鼠右鍵按一下通知區域中的「iFolder 服務」圖示，然後選取「優先設定」開啟「iFolder 優先設定」對話方塊至「一般」索引標籤。

  2. 選取「登入桌面時啟動 iFolder」，然後按一下「套用」。

  3. 按一下「確定」關閉「iFolder 優先設定」對話方塊。

若要停用登入桌面時自動啟動 iFolder：

  1. 以滑鼠右鍵按一下「iFolder 服務」圖示，然後選取「優先設定」開啟「iFolder 優先設定」對話方塊至「一般」索引標籤。

  2. 取消選取「登入桌面時啟動 iFolder」，然後按一下「套用」。

  3. 按一下「確定」關閉「iFolder 優先設定」對話方塊。

手動啟動 iFolder

  1. 使用建立 iFolder 時想使用的本地 Windows 使用者身份登入電腦。

  2. 如果 iFolder 在成功登入時沒有自動啟動，請按一下「啟動 > 程式」 > 「iFolder 3」，然後選取「iFolder 3 用戶端」應用程式將它啟動。

跨平台用戶端

  • 登入時自動啟動 iFolder

  • 手動啟動 iFolder

登入時自動啟動 iFolder

若要在登入 Macintosh 電腦時自動開啟 iFolder，則將 iFolder 加入「啟動項目」中。

  1. 啟動 iFolder。

  2. 從固定點，以滑鼠右鍵按一下 iFolder 圖示，並選取「登入時開啟」選項。

    或

    從固定點，按住 CTRL 鍵，按一下 iFolder 圖示，然後選取「登入時開啟」選項。

若要關閉自動啟動 iFolder，則從「啟動項目」移除 iFolder。

  1. 從固定點，以滑鼠右鍵按一下 iFolder 圖示，並取消選取「登入時開啟」選項。

    或

    從固定點，以滑鼠右鍵按一下 iFolder 圖示，或按住 CTRL 鍵再按一下 iFolder 圖示，然後取消選取「登入時開啟」選項。

手動啟動 iFolder

  1. 以建立 iFolder 時想使用的本地 Macintosh 使用者身份登入電腦。

  2. 請使用下列方法之一來啟動 iFolder：

    • 開啟 Applications 目錄，(此目錄位於 Macintosh HD)，選取 Applications，然後按兩下「iFolder 3」。

    • 在高亮顯示處 (Apple 按鈕 + 空白鍵，或按一下右上方的放大鏡) 輸入 iFolder，然後按一下 Applications 下的 「iFolder 3」。

    • 開啟終端機外圍程序，以 root 使用者身份登入，然後輸入 

      /Applications/iFolder 3.app/Contents/MacOS/iFolder 3

有關更詳細的資訊，請參閱 法律聲明.
