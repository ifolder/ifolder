移動 iFolder

建立 iFolder 之後，iFolder 的路徑是靜態的。您無法將 iFolder 從其原始位置移動至本機電腦的其他位置並使其仍為 iFolder。移動 iFolder 將導致同步資料庫與轉換為 iFolder 之資料夾間的連結被中斷。

若要移動建立 iFolder 所在電腦上的 iFolder：

  1. 將 iFolder 轉換為一般資料夾。

  2. 移動該資料夾。

  3. 將該資料夾轉換為 iFolder。

  4. (選擇性的) 共用該 iFolder。

若要移動某個參與電腦上的 iFolder：

  1. 在 iFolder 中，恢復 iFolder 以停止參與 iFolder，但仍使其可用。

  2. 選取可用的 iFolder，然後按一下「設定」。

  3. 指定您要放置 iFolder 的新本地路徑。

  4. 允許 iFolder 與企業伺服器同步化檔案。
