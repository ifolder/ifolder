登入 iFolder 帳戶

設定 iFolder 帳戶之後，每當您要同步化本地 iFolder 資料、iFolder 使用者清單及共用的 iFolder 通知時，都必須登入。您可以同時登入多個帳戶。針對每個帳戶，請使用下列其中一種登入方法：

  • 開啟「記住密碼」，以便每次您登入桌面或啟動 iFolder 用戶端時，都可以自動登入 iFolder 帳戶。

    每當您變更密碼，則下次 iFolder 嘗試登入 iFolder 帳戶時，自動驗證將正常失敗。iFolder 會提示您使用新密碼登入，並允許您指定並儲存新的密碼。

  • 關閉「記住密碼」，並只在您準備好要同步化其 iFolder 中的檔案或要管理 iFolder 時才登入 iFolder 帳戶。

若要開啟或關閉現有 iFolder 帳戶的「記住密碼」：

  1. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 圖示 ，然後選取「帳戶」。

  2. 選取 iFolder 帳戶，以檢視其帳戶設定。

  3. 選取 (開啟) 或取消選取 (關閉)「記住密碼」。

  4. 指定密碼，然後按一下「確定」或「套用」。

  5. 按一下「確定」確認對「帳戶」設定的變更，或按一下「取消」撤消此變更。

若要登入 iFolder 帳戶：

  1. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 圖示 ，然後選取「帳戶」。

  2. 選取 iFolder 帳戶，以檢視其帳戶設定。

  3. 指定密碼，然後按一下「登入」。

    登入成功後，帳戶的狀態會變更為「登入」。

    如果登入失敗，則會出現一則錯誤訊息報告原因，如密碼失敗。

  4. 按一下「確定」，關閉「帳戶」對話方塊。
