檢視和設定 iFolder 的內容

使用「iFolder 內容」頁以檢視和設定選定 iFolder 的下列資訊。


瞭解 iFolder 內容

下列參數將報告伺服器上已使用的磁碟空間：

參數 描述

可用的

對選定 iFolder 中檔案或目錄目前可用的伺服器磁碟空間總數：該值等於容量減去已用空間。

已使用

目前被選定 iFolder 中所有檔案及目錄佔用的伺服器磁碟空間總數：

以下之一：

  • 設定容量
  • 容量

  • 設定容量：如果您是 iFolder 擁有者，則可為 iFolder 指定容量。

  • 容量：如果您是成員，此值將報告由 iFolder 擁有者配置給選定 iFolder 的伺服器磁碟空間總數。如果沒有設定容量，該值就為空或 N/A (不適用)。

下列參數將報告 iFolder 之本地副本的同步化狀態：

參數 描述

上次成功同步化

最近一次成功同步化 iFolder 本地副本與 iFolder 伺服器的完成時間。

待同步的檔案/資料夾

iFolder 中需要同步化之儲存項目 (檔案和資料夾) 的總數。

自動同步化間隔

iFolder 的同步化間隔。指定以分鐘、小時或日為時間單位的值。

當 iFolder 客戶端搭配使用 Novell iFolder 3.x 企業伺服器時，此間隔無法改寫由 iFolder 管理員為伺服器或伺服器上之帳戶指定的最小間隔，但它可改寫全客戶端的同步化間隔。

立即同步化

按一下按鈕，視需要啟動同步化。


從檔案管理員存取 iFolder 內容

  1. 在檔案管理員中，導覽至「iFolder」，然後選取它。

  2. 以滑鼠右鍵按一下「iFolder」，然後按一下「iFolder」>「內容」。


從 iFolders 瀏覽器存取 iFolder 內容

  1. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 應用程式圖示 ，然後按一下「iFolder」，以開啟 iFolder 瀏覽器。

  2. 在 iFolder 清單中，選取「iFolder」，然後按一下「內容」。
