iFolder 說明

iFolderTM 是 Linux*、Windows* 及 Macintosh* 用戶端的開放式來源檔案共用應用程式。使用 iFolder，無論您是在線上、還是離線，任何時候您的本地檔案都可以跨越電腦的界限自動跟隨您。您可以共用多個 iFolder 中的檔案，每個都針對不同的使用者群組。您可以控制誰可以參與 iFolder，並控制參與者對其中檔案的存取權限。您還可以參與其他使用者與您共用的 iFolder。

本說明系統說明如何使用與管理 iFolder 用戶端。主要對象為產品的使用者，內容分成以下小節：

  • iFolder 概觀
  • 管理 iFolder 帳戶及優先設定
  • 管理 iFolder


文件慣例

在 Novell 文件中，大於符號 (>) 用於分隔步驟中的動作以及前後參照路徑中的項目。

商標符號 (®、TM 等) 代表 Novell 商標。星號 (*) 代表協力廠商商標。

雖然部份平台的單一路徑名稱以反斜線寫入，其他平台則用斜線，不過路徑名稱以反斜線表示。要求使用正斜線的平台使用者，例如 Linux 或 UNIX*，應依據軟體的要求使用正斜線。
