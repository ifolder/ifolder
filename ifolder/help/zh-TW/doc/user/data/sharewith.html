共用 iFolder

您可與多個使用者共用 iFolder。本節討論下列任務：

  • 瞭解使用者存取權限
  • 新增使用者至 iFolder
  • 修改使用者存取權限
  • 從 iFolder 移除使用者

新增使用者至 iFolder


利用檔案管理員新增使用者

  1. 在檔案管理員中，導覽至「iFolder」，然後選取它。

  2. 以滑鼠右鍵按一下「iFolder」，然後按一下「iFolder」>「內容」。

  3. 按一下「共用」。

  4. 從使用者清單中選取一個或多個使用者。

  5. 選取存取權限：「完全控制」、「讀／寫」或「唯讀」。

  6. 按一下「套用」。


利用 iFolder 瀏覽器新增使用者

  1. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 圖示 ，然後按一下「iFolder」，以開啟 iFolder 瀏覽器。

  2. 在 iFolder 清單中，選取「iFolder」，然後按一下「共用」 以開啟「共用」索引標籤的「iFolder 內容」視窗。

  3. 按一下「新增」，以開啟「選取使用者」對話方塊。

  4. 從使用者清單中，將一個或多個使用者加入到「選取使用者」清單以選取他們，然後按一下「確定」。

    邀請可能需要數秒鐘的時間才能同步至伺服器。

  5. 選取一個或多個使用者，按一下「存取」，選取「完全控制」、「讀／寫」或「唯讀」等存取權限，然後按一下「確定」。

  6. 重複步驟 4 和 5，直至您已邀請到所有想要的使用者來共用 iFolder。

  7. 按一下「套用」，然後按一下「確定」，以關閉「iFolder 內容」視窗。


修改使用者存取權限

如果 iFolder 成員的工作關係和狀態發生變更，您可能需要變更 iFolder 的使用者存取權限。例如，您可能想給予受信任的使用者 iFolder 的「檔案控制」權限。


利用檔案管理員設定使用者權限

  1. 在檔案管理員中，導覽至「iFolder」，然後選取它。

  2. 以滑鼠右鍵按一下「iFolder」，然後按一下「iFolder」>「內容」。

  3. 按一下「共用」。

  4. 從使用者清單中選取一個或多個使用者。

  5. 選取「存取」，然後指定「完全控制」、「讀／寫」或「唯讀」等權限。

  6. 按一下「套用」，然後按一下「確定」，以關閉「iFolder 內容」視窗。


利用 iFolder 瀏覽器設定使用者權限

  1. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 圖示 ，然後按一下「iFolder」，以開啟 iFolder 瀏覽器。

  2. 在 iFolder 清單中，選取「iFolder」，然後按一下「共用」。

  3. 從使用者清單中選取一個或多個使用者。

  4. 選取「存取」，然後指定「完全控制」、「讀／寫」或「唯讀」等權限。

  5. 按一下「套用」，然後按一下「確定」，以關閉「iFolder 內容」視窗。


從 iFolder 移除使用者

若要取消共用 iFolder，您必須移除作為 iFolder 成員的使用者。使用者工作站上的 iFolder 成為一般資料夾。資料的副本保留在先前成員的工作站上，但檔案將不再與共用的 iFolder 保持同步。


利用檔案管理員移除使用者

  1. 在檔案管理員中，導覽至「iFolder」，然後選取它。

  2. 以滑鼠右鍵按一下「iFolder」，然後按一下「iFolder」>「內容」。

  3. 按一下「共用」。

  4. 從使用者清單中選取一個或多個使用者。

  5. 依次按一下「移除」、「套用」和「確認」。


利用 iFolder 瀏覽器移除使用者

  1. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 圖示 ，然後按一下「iFolder」，以開啟 iFolder 瀏覽器。

  2. 在 iFolder 清單中，選取「iFolder」，然後按一下「共用」。

  3. 從使用者清單中選取一個或多個使用者。

  4. 依次按一下「移除」、「套用」和「確認」。
