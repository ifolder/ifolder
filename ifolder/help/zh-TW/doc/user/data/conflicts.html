解決檔案衝突

當同時儲存的檔案版本間發生衝突，或檔案名稱僅大小寫不同時，該檔案就會設定「名稱衝突」旗標並儲存在衝突區域中，同時會傳送警告以透過 iFolder 用戶端設定檔案的旗標。


版本衝突

版本衝突時，應先與其他成員協調，再解決衝突。然後在「衝突解決」對話方塊中選擇正確的版本。

  1. 當發生衝突時，在「通知」區域中會出現立現式訊息。按一下 X，關閉該訊息。

    您可以停用立現式行為，方法是在「iFolder 優先設定」 > 「一般」索引標籤中設定優先設定。

  2. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 圖示 ，然後按一下「iFolder」，以開啟 iFolder 瀏覽器。

  3. 選取衝突的檔案，然後按一下「解決」，以檢視「解決衝突」對話方塊。

  4. 檢視本地機器上及伺服器上檔案的資訊，以決定要保留的檔案版本，然後按一下「儲存此版本」來保留正確的檔案版本。

  5. 解決衝突之後，按一下「確定」。


檔案名稱衝突

當本地檔案版本與伺服器檔案版本名稱的大小寫不同時，會發生檔案名稱衝突。其內容可能相同也可能不同。這些類型的衝突可藉由重新命名檔案來解決，但重新命名檔案時需謹慎。

  • 如果內容相同，則可以在「解決衝突」對話方塊中重新命名其中一個檔案。
  • 如果內容不同，請務必建立本地檔案的副本並為其提供不同名稱，然後返回「解決衝突」對話方塊並儲存該檔案的伺服器版本。這可讓兩個版本的檔案以不同名稱同時存在於 iFolder 中。如果在「解決衝突」對話方塊內重新命名一個檔案版本，該檔案版本便會改寫另一個版本，使另一個檔案的內容遭到損毀。

  1. 當發生衝突時，在「通知」區域中會出現立現式訊息。按一下 X，關閉該訊息。

  2. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 圖示 ，然後按一下「iFolder」，以開啟 iFolder 瀏覽器。

  3. 選取衝突的檔案，然後按一下「解決」，以檢視「解決衝突」對話方塊。

  4. 確認衝突是屬於檔案名稱衝突，然後判斷內容是否相同。

  5. 執行下列動作之一：

    • 如果內容相同，則重新命名其中一個檔案，並按一下「儲存此版本」，然後按一下「確定」以關閉「解決衝突」對話方塊。您已解決此問題。
    • 如果內容不同，請繼續步驟 6。

  6. 關閉「解決衝突」對話方塊。

  7. 開啟檔案管理員，並導覽至衝突檔案的本地副本。

  8. 複製該檔案，並使用唯一名稱將其儲存在 iFolder 中。

    iFolder 會將該檔案作為新的檔案在伺服器及成員電腦間進行同步。

  9. 返回「解決衝突」對話方塊，選取該檔案的伺服器版本，按一下「儲存此版本」，然後按一下「確定」。

    該檔案的伺服器版本即會下載至您的本地機器，並改寫具有該名稱之檔案的本地副本。
