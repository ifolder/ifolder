刪除 iFolder 帳戶

您可以指定是否只從目前電腦上移除帳戶還是同時從伺服器將您擁有的 iFolder 也一併移除，基本上會停止您為該帳戶所設定的共用 iFolder。

如果您從電腦上移除 iFolder 帳戶，則 iFolder 的相關本地副本將被恢復為一般資料夾。您可以從其他電腦或 Web 繼續存取這些 iFolder。

如果您從伺服器上移除 iFolder 帳戶：

  • 您擁有的所有 iFolder 均為未共用。
  • 成員電腦上的 iFolder 本地副本將被恢復為一般資料夾。
  • 將從伺服器移除 iFolder 及其內容。
  • 您作為與他人共用的 iFolder 成員，也將被移除。

  1. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 應用程式圖示 ，然後選取「帳戶」以開啟「帳戶」索引標籤的「iFolder 優先設定」對話方塊。

  2. 選取您要刪除的 iFolder 帳戶，然後按一下「刪除」。

  3. 會出現訊息提示您判斷刪除動作的範圍。請執行下列動作之一：

    • 只從目前電腦上移除 iFolder 帳戶：從伺服器取消選取「移除我的 iFolder 和檔案」，然後按一下「是」。iFolder 只從本地工作站移除選取的帳戶。此帳戶的本地 iFolder 將恢復為一般資料夾。使用檔案管理員找出並刪除不再需要之資料的本地複本。

      您可以從使用 iFolder 客戶端的其他電腦繼續存取此帳戶的 iFolder。當 iFolder 客戶端搭配使用 Novell® iFolder® 3.x 企業伺服器時，您可以從使用 Novell iFolder 3.x 網路存取的任何位置存取檔案。

    • 從伺服器移除 iFolder 帳戶以及 iFolder 和檔案。從伺服器選取「移除我的 iFolder 和檔案」，然後按一下「是」。iFolder 會從本地工作站移除選取的帳戶。同時會從伺服器移除此帳戶下您所擁有的所有 iFolder，並將 iFolder 恢復為目前電腦和所有成員電腦上的一般資料夾。最後，您作為此帳戶上共用 iFolder 的成員也會被移除。本地 iFolder 都將恢復為一般資料夾。使用檔案管理員找出並刪除不再需要之資料的本地複本。

      如果您決定在此部電腦或其他電腦上再次使用此帳戶，則必須以新建的方式設定帳戶及其 iFolder。

    • 取消刪除：按一下「否」撤銷刪除動作。您的帳戶及其 iFolder 維持不變。
