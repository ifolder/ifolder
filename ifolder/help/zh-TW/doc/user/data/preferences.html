設定用戶端的 iFolder 優先設定

  1. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 應用程式圖示 ，然後選取「優先設定」以開啟「一般」索引標籤的「iFolder 優先設定」對話方塊。


  2. 請指定下列值：

    參數 描述

    登入桌面時啟動 iFolder

    選取此選項，這樣每當您以設定 iFolder 帳戶所使用的本地使用者名稱登入時，便會啟動 iFolder。針對您有開啟「記住密碼」的那些帳戶，會在指定的「同步化間隔」於背景中進行同步化。

    取消選取此選項，可手動啟動 iFolder。

    建立 iFolder 時顯示確認對話方塊

    選取此選項，這樣每當建立 iFolder 時，除了一般視覺線索 (如在 iFolder 清單中出現 iFolder) 之外，還會接收到確認訊息。

    取消選取此選項，可停止接收確認訊息。

    共用 iFolder 的通知

    選取此選項，這樣每當接收到新的共用 iFolder 邀請時，就會在「通知」區域中開啟立現式訊息。

    會根據您在「iFolder 優先設定」中指定的「同步化間隔」來擷取邀請。該間隔可控制檔案、iFolder 共用邀請及使用者清單的同步化。您還可以按一下「重新整理」，視需要更新邀請。

    取消選取此選項，可關閉邀請的立現式訊息行為。

    衝突通知

    選取此選項，這樣每當同步化檔案發生衝突時，便會在「通知」區域中開啟立現式訊息。

    取消選取此選項，可關閉同步化衝突的立現式訊息行為。

    使用者加入時通知

    選取此選項，這樣每當使用者接受共用 iFolder 邀請時，就會在「通知」區域中開啟立現式訊息。只有當使用者在其自己的電腦上實際設定 iFolder 時，才會被視為接受。

    取消選取此選項，可關閉使用者共用確認的立現式訊息行為。

    同步化

    自動同步化 iFolder：選取此核取方塊，便會開啟此電腦上以目前本地登入識別登入之所有 iFolder 的同步化。取消選取此核取方塊，可關閉同步化。

    同步化 iFolder 間隔：指定用於同步化此電腦上以目前本地登入識別登入之 iFolder 的最小間隔。指定值及單位。

    如果您搭配使用 iFolder 用戶端與 Novell iFolder 3.x，則有效的最小同步化間隔一律為下列設定中最大的值：

    • 系統規則 (預設為零)，除非存在使用者規則集。如果設定使用者規則，則使用者規則會覆寫系統規則，而不論使用者規則的值較之是大還是小。
    • 本地機器規則，或與伺服器同步化之用戶端機器的設定。
    • iFolder (集合) 規則。

  3. 完成時，按一下「套用」以使變更生效。

  4. 按一下「確定」，關閉「iFolder 優先設定」對話方塊。
