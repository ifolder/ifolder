管理 iFolder

iFolderTM 是一個本地目錄，用於選擇性地與使用者指定的成員共用並同步化檔案。所有成員都可以存取 iFolder 檔案，那些具有適當權限的成員還可以對其進行修改及讀取。

本節將討論 iFolder 的下列任務：

  • 放置 iFolder 的指示
  • 檔案類型和大小不同步的指示
  • iFolder 及其資料夾和檔案的命名慣例
  • 建立 iFolder
  • 共用 iFolder
  • 設定可用的 iFolder
  • 檢視和設定 iFolder 的內容
  • 同步化檔案
  • 解決檔案衝突
  • 將 iFolder 恢復為一般資料夾
  • 移動 iFolder
  • 刪除 iFolder
