管理已加密 iFolder 的密碼片語

加密的 iFolder 有唯一的資料加密金鑰，這是在 iFolder 建立期間自動產生。該金鑰是使用密碼片語 (由使用者提供) 以及復原代理程式的公用金鑰所加密。

若 iFolder 管理員已在伺服器上啟用加密規則，則建立 iFolder 帳戶期間會選取復原代理程式。

若 iFolder 管理員已啟用加密，則建立 iFolder 時會顯示一個對話方塊，讓您輸入您的密碼片語。iFolder 使用該密碼片語來產生唯一的加密金鑰，以加密或解密 iFolder 伺服器上的 iFolder 資料。若忘記該密碼片語，您將無法存取 iFolder 資料。您必須重設密碼片語，才能取得 iFolder 資料的存取權。

復原加密的 iFolder

使用「密碼片語復原精靈」來重設密碼片語及復原加密的 iFolder。依照下列步驟啟動「密碼片語復原精靈」：

  1. 在 iFolder 瀏覽器中，按一下「安全性」>「忘記密碼片語」以啟動精靈。

  2. 從「iFolder 帳戶」清單中，選取要重設密碼片語的帳戶，然後按一下「下一步」。

    在建立 iFolder 帳戶期間，若已選取 Server_Default 選項做為復原代理程式，則會顯示「設定新密碼片語」頁面。若要重設新密碼片語，請參閱在復原代理程式是 Server_Default 時重設密碼片語。

    換句話說，若選取 Server_Default 以外的復原代理程式，則您必須使用在復原代理程式不是 Server_Default 時重設密碼片語來重設密碼片語。

在復原代理程式是 Server_Default 時重設密碼片語

若要設定新密碼片語，請執行下列步驟：

  1. 在「設定新密碼片語」頁面中的「新密碼片語」欄位指定新密碼片語。

  2. 在「確認密碼片語」欄位重新輸入新密碼片語。

  3. 在「密碼」欄位指定密碼，然後按一下「下一步」。

  4. 成功重設密碼片語之後，請按一下「完成」關閉精靈。

在復原代理程式不是 Server_Default 時重設密碼片語

若要在選取的復原代理程式不是 Server_Default 的情況下重設密碼片語，請依照下列步驟執行：

  1. 精靈的「歡迎使用」頁面會顯示重設密碼片語所需的資訊。按一下「下一步」以顯示「密碼片語復原方式」頁面。

  2. 在該精靈中，有三種方式可用來重設密碼片語：

    • 使用祕密檔案與其密碼： 使用祕密檔案與其密碼來重設密碼片語。若您有維護祕密檔案與其密碼，則可以使用精靈來指定祕密檔案的位置並提供其密碼。若要使用祕密檔案來重設密碼片語，請參閱使用祕密檔案來重設密碼片語。

      附註：您可以向 iFolder 管理員索取祕密檔案與其密碼。

    • 使用新資料檔： 您也可以使用新資料檔來重設密碼片語。執行完舊資料檔的輸出操作之後，您可以向管理員索取新資料檔。若要使用新資料檔來重設密碼片語，請參閱使用新資料檔來重設密碼片語。

    • 輸出舊資料檔： 若沒有祕密檔案或新資料檔，您可以使用精靈來輸出舊資料檔，然後使用電子郵件將該檔案寄給您的管理員。當管理員送回新資料檔時，您就可以使用精靈來重設密碼片語。若要輸出舊資料檔，請參閱使用輸出的檔案來重設密碼片語。

使用祕密檔案來重設密碼片語

若要使用祕密檔案來重設密碼片語，請執行下列步驟：

  1. 選取「我有祕密檔案與其密碼」選項，然後按一下「下一步」。

  2. 填寫下列欄位：

    祕密檔案位置： 指定祕密檔案位置，或按一下「瀏覽」按鈕尋找並選取祕密檔案。

    祕密密碼： 指定祕密檔案的密碼。

    新密碼片語： 指定新密碼片語。

    確認密碼片語 再次輸入新密碼片語。

  3. 按一下「下一步」以重設密碼片語。

  4. 按一下「完成」，便可關閉精靈。

附註：您可以向 iFolder 管理員索取祕密檔案與其密碼。

使用新資料檔來重設密碼片語

若要使用新資料檔來重設密碼片語，請執行下列步驟：

  1. 選取「我有管理員傳送給我的新資料檔」選項，然後按一下「下一步」。

  2. 在「新資料檔位置」欄位中，指定新資料檔的位置，或按一下「瀏覽」按鈕尋找並選取新資料檔。

  3. 若管理員送給您的新資料檔已加密，請選取「上列檔案是否已加密？」核取方塊。當您選取此核取方塊時，會啟用「一次性密碼」欄位。

  4. 在「一次性密碼」欄位中，指定管理員用來加密新資料檔的密碼。

  5. 在「新密碼片語」欄位中，指定新密碼片語。

  6. 在「確認密碼片語」欄位中，再次輸入新密碼片語。

  7. 按一下「下一步」以重設密碼片語。

  8. 按一下「完成」，便可關閉精靈。

使用輸出的檔案來重設密碼片語

若要在沒有祕密檔案或新資料檔的情況下重設密碼片語：

  1. 選取「我沒有祕密檔案或資料檔」選項，然後按一下「下一步」。

  2. 在「要用來儲存舊資料檔的位置」欄位中，指定要用來儲存舊資料檔的位置，或按一下「瀏覽」按鈕以指定要用來儲存舊資料檔的位置。

  3. 按一下「下一步」以取得舊資料檔。會顯示舊資料檔位置與 iFolder 管理員的電子郵件地址。您可以使用此資訊來取得新資料檔。

  4. 按一下「完成」，便可關閉精靈。

  5. 使用電子郵件將舊資料檔寄給您的 iFolder 管理員，並要求取得新資料檔。

  6. 當您收到新資料檔時，請參閱使用新資料檔來重設密碼片語以重設您的密碼片語。

變更密碼片語

基於安全性理由，您必須定期變更密碼片語。您可以使用您的「復原」代理程式傳送給您的一次性密碼片語來重設密碼片語。

  1. 在 iFolder 瀏覽器中，選取「安全性」>「變更密碼片語」。

  2. 在「變更密碼片語」視窗中，選取要重設密碼片語的 iFolder 帳戶，並指定下列詳細資料：

    輸入通關密語： 指定目前的密碼片語。

    輸入新的通關密語： 指定要為目前 iFolder 帳戶設定的新密碼片語。

    重新輸入通關密語： 再次指定新密碼片語進行確認。

    復原代理程式： 選取您為目前 iFolder 帳戶設定的復原代理程式。

  3. 按一下「重設」。

有關更詳細的資訊，請參閱 法律聲明.
