刪除 iFolder 帳戶

您可以指定是否要僅從目前電腦移除帳戶，或同時從伺服器移除您擁有的 iFolder。

如果僅從電腦移除 iFolder 帳戶：

  • 此帳戶的本地 iFolder 將恢復為一般資料夾。 使用檔案管理員找出並刪除不再需要之資料的本地複本。

  • 您可以使用 iFolder 用戶端從其他電腦繼續存取這些 iFolder。

  • 當搭配 Novell iFolder 3.7 或更新版本的企業伺服器使用 iFolder 用戶端時，您也可以從具有 Novell iFolder 3.7 或更新版本之 Web 存取的任意處存取檔案。

如果您從伺服器上移除 iFolder 帳戶：

  • 您擁有的所有 iFolder 均為未共用。

  • 成員電腦上的 iFolder 本地副本將被恢復為一般資料夾。 使用檔案管理員找出並刪除不再需要之資料的本地複本。

  • 將從伺服器移除 iFolder 及其內容。

  • 您作為與他人共用的 iFolder 成員，也將被移除。

  • 帳戶會從本地電腦移除。

如果您決定在此部電腦或其他電腦上再次使用此帳戶，則必須以新建的方式設定帳戶及其 iFolder。

刪除 iFolder 帳戶：

  1. 以滑鼠右鍵按一下通知區域中的「iFolder 服務」圖示，然後選取「帳戶設定」開啟「iFolder 優先設定」對話方塊至「帳戶」索引標籤。

  2. 選取您要刪除的 iFolder 帳戶，然後按一下「移除」。

  3. 會出現訊息提示您決定刪除動作的範圍。

    請執行下列其中一個步驟：

    • 只從目前電腦上移除 iFolder 帳戶： 確定已取消選取「從伺服器移除我的 iFolder 及檔案」，然後按一下「是」。

    • 從伺服器移除 iFolder 帳戶以及 iFolder 和檔案。 選取「從伺服器移除我的 iFolder 及檔案」，然後按一下「是」。

    • 取消刪除： 按一下「否」退出刪除動作。您的帳戶及其 iFolder 維持不變。

有關更詳細的資訊，請參閱 法律聲明.
