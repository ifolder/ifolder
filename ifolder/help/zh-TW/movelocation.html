移動 iFolder

在您建立 iFolder 後，至 iFolder 的路徑會是靜態。您無法在您的本機電腦上由其原來了位置移動 iFolder 至另一個位置，並且還保有 iFolder。移動 iFolder 將導致同步資料庫與轉換為 iFolder 之資料夾間的連結被中斷。

若要移動建立 iFolder 所在電腦上的 iFolder：

  1. 在檔案管理員或 iFolder 瀏覽器中，選取所要移動的 iFolder。

  2. 恢復 iFolder 至一般資料夾。

  3. 在檔案管理員中，將資料夾移動到電腦上的另一個位置。

  4. 將資料夾轉換回 iFolder，並將 iFolder 上載至伺服器。

  5. (選擇性的) 共用該 iFolder。

  6. 允許 iFolder 同步化檔案和 iFolder 伺服器。

若要移動某個參與電腦上的 iFolder：

  1. 在 iFolder 中，恢復 iFolder 以停止參與目前電腦上的 iFolder。

  2. 選取可用的 iFolder，然後按一下「下載 iFolder」。

  3. 指定您要放置 iFolder 的新本地路徑。

  4. 允許 iFolder 同步化檔案和 iFolder 伺服器。

有關更詳細的資訊，請參閱 法律聲明.
