iFolder 用戶端

iFolder 用戶端整合了作業系統，在您的原生桌上型環境中提供 iFolder 服務。 iFolder 用戶端可讓您管理自己的 iFolder，以及選取要在每個電腦上設定的共用 iFolder。iFolder 支援下列作業系統：

  • SUSE Linux Enterprise Desktop (SLED) 10 、SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP1 64 位元

  • openSUSE 11.3

    iFolder Linux 用戶端必須使用 Mono 架構的 Linux 系統，以及支援 iFolder Nautilus 外掛程式的 GNOME 桌面。

  • Windows XP SP3 32 位元

  • Windows Vista SP1

  • Windows 7

  • Macintosh OS X 32 位元 (Intel 架構) 10.5 或以上版本 (需使用 Mono 2.4.2.3)。不支援 PowerPc 架構。

有關更詳細的資訊，請參閱 法律聲明.
