登出 iFolder 帳戶

當您想要離線工作時，可以登出 iFolder 帳戶。

  • 跨平台用戶端

  • Linux

  • Windows

跨平台用戶端

  1. 按一下固定點中的「iFolder 服務」圖示，然後選取「帳戶」開啟「iFolder 優先設定」對話方塊。

  2. 從工具列按一下「帳戶」圖示，檢視目前 iFolder 領域中的帳戶清單。

  3. 找出您要管理的帳戶，然後按一下「登出」中斷與 iFolder 伺服器的連接。

Linux

  1. 以滑鼠右鍵按一下通知區域中的「iFolder 服務」圖示，然後選取「帳戶」開啟「iFolder 優先設定」對話方塊至「帳戶」索引標籤。

  2. 找出您要管理帳戶，然後取消選取旁邊的「線上」核取方塊，以解除連接 iFolder 伺服器。

  3. 按一下「關閉」以關閉「iFolder 優先設定」對話方塊。

Windows

  1. 以滑鼠右鍵按一下通知區域中的「iFolder 服務」圖示，然後選取「帳戶」開啟「iFolder 優先設定」對話方塊至「帳戶」索引標籤。

  2. 找出您要管理帳戶，然後取消選取旁邊的「線上」核取方塊。

  3. 按一下「套用」以解除連接 iFolder 伺服器，然後按一下「關閉」以關閉「iFolder 優先設定」對話方塊。

有關更詳細的資訊，請參閱 法律聲明.
