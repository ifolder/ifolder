Konfigurace místní brány firewall pro provoz aplikace iFolder

Pokud používáte místní bránu firewall, musíte ji nastavit tak, aby povolila aplikaci iFolder zvolit místní dynamický port, nebo musíte nakonfigurovat místní statický port pro klienta aplikace iFolder, který bude komunikovat lokálně (na stejném počítači) se serverem Mono XSP. Aplikace iFolder používá webové služby Mono XSP ke komunikaci přes protokoly HTTP nebo HTTPS (SSL jako výchozí) s podnikovými servery Novell iFolder, na nichž máte založeny účty iFolder. Správce služeb iFolder konfiguruje služby, které určují, zda se ve vašem prostředí použije protokol HTTP nebo HTTPS.

Vaše brána firewall neblokuje provoz mezi serverem Mono XSPa podnikovým serverem Novell iFolder, protože Mono XSP používá pro provoz standardní porty protokolu HTTP nebo HTTPS. Pokud však blokujete provoz mezi klientem a serverem Mono XSP, klient nemá možnost se serverem komunikovat. Nebude se moci přihlásit ke svému účtu iFolder a synchronizovat data ve složkách iFolder.

Na začátku není v konfiguraci aplikace iFolder uveden konkrétní port, který má být použit pro komunikaci klienta a serveru Mono XSP. Při prvním spuštění aplikace iFolder se otevře dostupný místní port, který bude přijímat žádosti ze serveru Mono XSP a odesílat data na server přes Mono XSP. Zapíše číslo portu do konfiguračního souboru a poté daný port používá, ledaže by port při spuštění aplikace iFolder nebyl k dispozici. Pokud daný port používá jiná aplikace, aplikace iFolder vybere jiný dostupný port, znovu nakonfiguruje port v konfiguračním souboru a tento nový port poté používá. Pokud např. aplikace iFolder není spuštěná a daný port zabere jiná aplikace, při dalším spuštění si aplikace iFolder musí vybrat a nakonfigurovat pro použití jiný port.


Použití místního dynamického portu

Pokud na vašem počítači používáte místní bránu firewall, měla by tato brána detekovat místní provoz mezi aplikací iFolder a serverem Mono XSP pokaždé, když aplikace iFolder kontaktuje server, aby proběhla synchronizace seznamu uživatelů a dat ve složkách iFolder. Pokud povolíte provoz, nebude se při každém spuštění klienta zobrazovat místní upozornění. Pokud povolíte provoz trvale, brána firewall by vás již nadále neměla upozorňovat.


Použití místního statického portu

V části Správce služby souboru Simias.config můžete volitelně přidat parametr Rozsah portů webové služby a povolit pro server Mono XSP používání statického portu nebo rozsahu portů pro tento místní provoz. Tento port nebo rozsah portů lze poté otevřít v bráně firewall a povolit tak komunikaci pro server Mono XSP.

Soubor Simias.config lze nalézt v těchto umístěních:

Operační systém Umístění souboru Simias.config

Linux

/home/username/.local/share/simias/Simias.config

Windows

C:\Documents and Settings\username\Local Settings\Application Data\simias\Simias.config

Macintosh

/Users/username/.local/share/simias/Simias.config

Přidejte nastavení rozsahu portů webové služby do části Správce služby v souboru Simias.config. Příklad:

<konfigurace>
  <název části="SpravceSluzby">
    <název nastavení="RozsahPortuWeboveSluzby" hodnota="monoxsp_port" />   
  </sekce>
</konfigurace>

Nahraďte parametr monoxsp_port číslem portu nebo rozsahem čísel portů:

  • Chcete-li použít jeden statický port, zadejte hodnotu čtyřmístného čísla portu (jako např. 1234). Příklad: 
    <název nastavení="RozsahPortuWeboveSluzby" hodnota="1234" />
  • Chcete-li použít jakýkoli dostupný port v rámci uvedeného rozsahu portů, zadejte rozsah portů (jako např. 1234-1236). Příklad: 
    <název nastavení="RozsahPortuWeboveSluzby" hodnota="1234-1236" />

Konfigurace systému pro použití specifického portu nebo rozsahu portů:

  1. Nastavte ve své bráně firewall, aby povolovala provoz na portu nebo rozsahu portů, které chcete používat pro provoz aplikace iFolder.

    Informace o tom, jak otevřít port v bráně firewall, naleznete v příslušné dokumentaci k této bráně firewall.

  2. Pokud chcete ukončit klienta aplikace iFolder, klepněte pravým tlačítkem myši na ikonu aplikace iFolder a vyberte položku Konec.

  3. Přidáním parametru Rozsah portů webové služby v části Správce služby změňte soubor Simias.config.

  4. Spusťte klienta aplikace iFolder.

    Více informací naleznete v části Spuštění klienta aplikace iFolder.
