Spuštění klienta aplikace iFolder


Spuštění aplikace iFolder pro Novell Linux Desktop 9

  1. Přihlaste se k počítači pod identitou uživatele, kterou chcete použít při tvorbě složek iFolder.

  2. Ke spuštění aplikace iFolder použijte jeden z následujících postupů:

    • Otevřete prostředí terminálu a zadejte

      /opt/novell/ifolder3/bin/ifolder

    • Pokud chcete, aby se aplikace iFolder spouštěla při přihlášení automaticky, přesvědčte se, že aplikace iFolder během odhlášení od relace GNOME běží, a relaci uložte. Můžete také otevřít okno terminálu, přihlásit se jako uživatel root, a poté spustit příkaz /opt/gnome/bin/gnome-session-properties, chcete-li přidat aplikaci iFolder jako program určený ke spuštění při přihlášení ke svému počítači.

  3. Pokud jste u svého účtu iFolder zakázali možnost Pamatovat si heslo, musíte se přihlásit k účtu, chcete-li synchronizovat seznam uživatelů, synchronizovat složky iFolder a pracovat se svými složkami iFolder.

    Více informací naleznete v části Přihlášení k účtu aplikace iFolder.


Spuštění aplikace iFolder pro systém Windows 2000/XP

  1. Přihlaste se k počítači pod identitou uživatele Windows, kterou chcete použít při tvorbě složek iFolder.

  2. Pokud se aplikace iFolder nespustí automaticky po úspěšném přihlášení, spustíte ji klepnutím na tlačítko Start > Programy > iFolder, a výběrem aplikace iFolder.

    Aplikace iFolder se nebude automaticky spouštět, pokud zakážete nastavení Spouštět aplikaci iFolder při přihlášení do pracovního prostředí v Předvolbách aplikace iFolder.

  3. Pokud jste u svého účtu iFolder zakázali možnost Pamatovat si heslo, musíte se přihlásit k účtu, chcete-li synchronizovat seznam uživatelů, synchronizovat složky iFolder a pracovat se svými složkami iFolder.

    Více informací naleznete v části Přihlášení k účtu aplikace iFolder.


Spuštění aplikace iFolder pro systém Macintosh OS X v10.3 nebo novější

  1. Přihlaste se k počítači pod identitou uživatele, kterou chcete použít při tvorbě složek iFolder.

  2. Ke spuštění aplikace iFolder použijte jeden z následujících postupů:

    • Otevřete nabídku Aplikace, vyberte podnabídku Internet a vyberte iFolder.
    • Chcete-li spustit aplikaci iFolder z příkazového řádku, otevřete prostředí terminálu a zadejte

      /opt/novell/ifolder3/bin/ifolder

    • Chcete-li, aby se aplikace iFolder spouštěla automaticky při přihlášení k počítači Macintosh, přidejte ji do Položek ke spuštění.

  3. Pokud jste u svého účtu iFolder zakázali možnost Pamatovat si heslo, musíte se přihlásit k účtu, chcete-li synchronizovat seznam uživatelů, synchronizovat složky iFolder a pracovat se svými složkami iFolder.

    Více informací naleznete v části Přihlášení k účtu aplikace iFolder.
