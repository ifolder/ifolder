Lösen von Dateikonflikten

Konflikte können zwischen Dateiversionen auftreten, die zur selben Zeit auf zwei verschiedenen Computern gespeichert wurden, oder wenn sich Dateinamen nur bezüglich der Groß-/Kleinschreibung unterscheiden. Wenn Konflikte auftreten, wird die Datei im Konfliktbereich gespeichert und mit dem Status Namenskonflikt versehen. Das iFolder-Fehlersymbol iFolder-Anwendungssymbol wird für den iFolder-Ordner angezeigt. Wenn die Option Über Konflikte benachrichtigen in den iFolder-Einstellungen aktiviert ist, wird außerdem eine Warnung an den Benachrichtigungsbereich gesendet. Sie können das Anzeigeverhalten von Benachrichtigungen steuern, indem Sie die entsprechende Einstellung auf der Registerkarte iFolder-Einstellungen > Allgemein festlegen.

Informationen zum Auflösen von Dateikonflikten finden Sie unter:

  • Auflösen von Dateiversionskonflikten

  • Auflösen von Dateinamenkonflikten

Auflösen von Dateiversionskonflikten

Um Dateiversionskonflikte aufzulösen, müssen Sie im Dialogfeld zur Konfliktauflösung die richtige Version auswählen.

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Dienstesymbol und klicken Sie dann auf iFolder-Ordner, um den iFolder-Browser zu öffnen.

  2. Wählen Sie den iFolder-Ordner aus, bei dem der Konflikt besteht, und klicken Sie dann auf Auflösen, um das Dialogfeld Konflikte lösen anzuzeigen.

    Es werden zwei Versionen der gleichen Datei angezeigt:

    • Lokale Version: Die auf Ihrem lokalen Computer gespeicherte Datei.

    • Serverversion: Die auf dem Server gespeicherte Datei.

    Sie können die Informationen zur Datei auf Ihrem lokalen Computer und zur Datei auf dem Server anzeigen, um zu bestimmen, welche Dateiversion Sie behalten möchten.

  3. Klicken Sie für jede Datei auf Öffnen, um die Dateien anzuzeigen.

  4. Wählen Sie für In iFolder speichern entsprechend Ihren Anforderungen eine der folgenden Optionen aus.

    Lokale Version: Wählen Sie diese Option aus, wenn die lokale Version in dem iFolder-Ordner gespeichert werden soll.

    Serverversion: Wählen Sie diese Option aus, wenn die Serverversion in dem iFolder-Ordner gespeichert werden soll.

  5. Wählen Sie Aktivieren aus, um die Option In Konfliktübersicht speichern zu aktivieren. Wenn Sie diese Option aktivieren, wird die konfliktverursachende Datei automatisch in der Konfliktübersicht gespeichert. Ihre Auswahl legt fest, welche Version zur späteren Verwendung in der Konfliktübersicht gespeichert wird.

  6. Wenn Sie den Konflikt aufgelöst haben, klicken Sie auf Speichern.

Auflösen von Dateinamenkonflikten

Dateinamenskonflikte treten auf, wenn die Namen der lokalen Dateiversion und der Dateiversion auf dem Server sich durch die Groß- und Kleinschreibung unterscheiden. Der Inhalt kann unterschiedlich oder identisch sein. Diese Arten von Konflikten können durch Umbenennen der Dateien gelöst werden, Sie sollten beim Umbenennen von Dateien allerdings Vorsicht walten lassen.

  • Wenn der Inhalt identisch ist, können Sie eine der Dateien mithilfe des Dialogfelds "Konflikte lösen" umbenennen.

  • Wenn der Inhalt unterschiedlich ist, müssen Sie eine Kopie der lokalen Datei erstellen und ihr einen anderen Namen geben, bevor Sie zum Dialogfeld "Konflikte lösen" zurückkehren und die Serverversion der Datei speichern. Dadurch können beide Versionen der Datei mit verschiedenen Dateinamen im iFolder-Ordner gespeichert werden. Wenn Sie eine Datei im Dialogfeld "Konflikte lösen" umbenennen, überschreibt diese Dateiversion die andere und der Inhalt der anderen Datei wird gelöscht.

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Dienstesymbol und klicken Sie dann auf iFolder-Ordner, um den iFolder-Browser zu öffnen.

  2. Wählen Sie die Datei aus, bei der der Konflikt besteht, und klicken Sie dann auf Auflösen, um das Dialogfeld "Konflikte lösen" anzuzeigen.

  3. Bestätigen Sie, dass es sich um einen Dateinamenskonflikt handelt, und ermitteln Sie, ob der Inhalt identisch ist oder nicht.

  4. Führen Sie einen der folgenden Vorgänge aus:

    • Wenn der Inhalt identisch ist, benennen Sie eine der Dateien um, klicken Sie auf Diese Version speichern und klicken Sie dann auf OK, um das Dialogfeld "Konflikte lösen" zu schließen. Der Vorgang ist abgeschlossen.

    • Wenn der Inhalt unterschiedlich ist, fahren Sie fort mit Schritt 5.

  5. Schließen Sie das Dialogfeld "Konflikte lösen":

  6. Öffnen Sie einen Datei-Manager und wechseln Sie zur lokalen Kopie der Konfliktdatei.

  7. Kopieren Sie die Datei und speichern Sie sie mit einem eindeutigen Namen im iFolder-Ordner.

    iFolder synchronisiert die Datei als neue Datei auf dem Server und den Mitgliedercomputern.

  8. Kehren Sie zum Dialogfeld Konflikte lösen zurück, wählen Sie die Serverversion der Datei aus, klicken Sie auf Diese Version speichern und klicken Sie dann auf OK.

    Die Serverversion der Datei wird auf Ihren lokalen Computer heruntergeladen und überschreibt die lokale Kopie der Datei mit diesem Namen.

    HINWEIS:Für Macintosh-Client: Um die Namenskonflikte zu lösen, müssen Sie den Datei- oder Ordnernamen manuell aktualisieren, um sicher zu gehen, dass keine ungültigen Zeichen ( \ / : * ? " < > | ; ) enthalten sind.
