Konfigurieren von iFolder-Konten

Verwenden Sie den Assistenten zum Erstellen von iFolder-Konten, um ein neues Konto hinzuzufügen und zu konfigurieren. Vor der Erstellung von iFolder-Ordnern bzw. der Freigabe von iFolder-Ordnern auf Ihrem System muss mindestens ein iFolder-Konto konfiguriert werden. Sie können immer nur für jeweils ein Konto gleichzeitig eine Verbindung zum iFolder-Server herstellen. Sie können jedoch mehrere Konten konfigurieren, wenn Sie eine Verbindung zu mehreren iFolder-Servern herstellen möchten. Sie können sich an jedem Konto einzeln anmelden und können gleichzeitig an mehrere Konten angemeldet sein.

Mehrere Benutzer mit verschiedenen lokalen Anmeldenamen verfügen über iFolder-Ordner auf demselben Computer. Wenn Sie iFolder zum ersten Mal unter Ihrer aktuellen lokalen Anmeldeidentität starten, fordert iFolder Sie zum Einrichten eines iFolder-Kontos auf.

WICHTIG:iFolder 3.7 und höhere Versionen enthalten eine neue Funktion zur automatischen Kontoerstellung, die dem iFolder-Administrator die Erstellung und Konfiguration von Konten erleichtert. Wenn Ihr iFolder-Administrator die automatische Kontoerstellung aktiviert hat, brauchen Sie nicht selbst ein Konto zu konfigurieren. Wenn Sie erstmals einen iFolder starten, werden Sie aufgefordert, ein Passwort einzugeben. Geben Sie Ihr Passwort ein, um Ihr Konto vollständig zu konfigurieren.

So konfigurieren Sie ein neues Konto:

  1. Greifen Sie mithilfe einer der folgenden Methoden auf den Assistenten zum Erstellen von iFolder-Konten zu.

    • Wenn Sie iFolder zum ersten Mal unter Ihrer aktuellen lokalen Anmeldeidentität starten, fordert iFolder Sie zum Einrichten eines iFolder-Kontos auf. Klicken Sie auf Weiter, um direkt zum Assistenten zum Erstellen von iFolder-Konten zu wechseln.

    • Klicken Sie mit der rechten Maustaste auf das Symbol für iFolder-Services im Benachrichtigungsbereich und wählen Sie Konten aus, um die Registerkarte Konten im Dialogfeld "iFolder-Einstellungen" zu öffnen. Klicken Sie anschließend auf Hinzufügen, um den Assistenten zum Erstellen von iFolder-Konten zu öffnen.

  2. Folgen Sie den Bildschirmanweisungen, um die folgenden Werte anzugeben:

    Parameter

    Beschreibung

    Serveradresse

    Der DNS-Name bzw. die IP-Adresse des iFolder-Enterprise-Servers, auf dem sich Ihr Konto befindet. Fragen Sie Ihren iFolder-Administrator nach diesen Informationen.

    Beispiel: svr21.beispiel.com oder 192.168.1.1

    Der Servername wird in der iFolder-Benutzeroberfläche immer zusammen mit seiner IP-Adresse angezeigt. Beispiel: iFolder - 192.168.1.1

    WICHTIG:Es ist nicht möglich, einen iFolder-Client der Version 3.7 oder höher zum Verbinden mit einem iFolder 3.2-Server zu verwenden.

    Diesen Server als Standardserver festlegen

    Wenn mit Ihrem Konto mehrere Server verbunden sind, aktivieren Sie das Kontrollkästchen für Diesen Server als Standardserver festlegen, um den in der Dropdown-Liste der Server ausgewählten Server als Standardserver für das Erstellen und Heraufladen von iFolder-Ordnern auszuwählen.

    Benutzername

    Geben Sie Ihre Benutzeridentität für dieses Konto an. Der für Sie zuständige iFolder-Administrator entscheidet, ob es sich bei dem von Ihnen verwendeten Format um Ihre Netzwerk-Benutzer-ID (allgemeiner LDAP-Name) oder Ihre Email-Adresse handeln soll.

    Wenn Ihr Name beispielsweise Richard Bauer, Ihre Benutzer-ID rbauer und Ihre Email-Adresse richard.bauer@beispiel.com lautet, konfiguriert Ihr Administrator eine Einstellung für den Server, durch die bestimmt wird, welches der beiden Formate zu Authentifizierungszwecken im Feld Benutzername akzeptiert wird.

    Passwort

    Geben Sie das Passwort zu Ihrem Benutzernamen an.

    Die Authentifizierung erfolgt über einen gesicherten Kanal (SSL). Sie werden aufgefordert, das Serverzertifikat zu bestätigen und zu akzeptieren.

    Passwort speichern

    Aktivieren Sie die Option Passwort auf diesem Computer speichern, um bei jeder Anmeldung an Ihrem Computer automatisch bei diesem iFolder-Konto angemeldet zu werden.

    Wenn sich Ihr Netzwerkpasswort ändert, tritt ein Fehler auf, wenn iFolder das nächste Mal versucht, sich bei dem iFolder-Konto anzumelden. iFolder fordert Sie zur Anmeldung mit Ihrem neuen Passwort auf und ermöglicht Ihnen die Angabe und Speicherung des neuen Passworts.

  3. Überprüfen Sie die eingegebenen Informationen. Klicken Sie auf die Schaltfläche Zurück, wenn Sie Änderungen vornehmen möchten, und sehen Sie sich die vorigen Seiten an. Klicken Sie auf Verbinden, um fortzufahren und Ihre Verbindung zum Server zu validieren.

  4. Wenn Sie zum Akzeptieren des Serverzertifikats aufgefordert werden, klicken Sie auf Details, um die Zertifikatsinformationen zu überprüfen, und klicken Sie dann auf OK, um das Zertifikat zu akzeptieren.

    HINWEIS:Während der Kontenerstellung erfolgt die Anmeldung über SSL. Wenn Sie aufgefordert werden, das Serverzertifikat zu akzeptieren, müssen Sie das Zertifikat bestätigen und akzeptieren, damit eine erfolgreiche Verbindung möglich ist.

  5. Wenn der Administrator die Verschlüsselungsrichtlinie aktiviert hat, fordert iFolder Sie zur Eingabe Ihres Passwortsatzes auf.

    Geben Sie Ihren Passwortsatz im Feld Passwortsatz ein.

    Geben Sie den Passwortsatz im Feld Passwortsatz wiederholen erneut ein.

    Aktivieren Sie das Kontrollkästchen für Passwortsatz speichern, um während der aktuellen Sitzung automatisch auf die verschlüsselten iFolder-Daten zuzugreifen. Wenn Sie diese Option auswählen, kann iFolder Ihren Passwortsatz für den nächsten Anmeldevorgang speichern. Dieser Passwortsatzaustausch erfolgt automatisch.

  6. Wählen Sie den gewünschten Wiederherstellungsagenten aus der Liste aus und klicken Sie auf OK.

  7. Geben Sie den Speicherort an, an dem Sie den standardmäßigen iFolder-Ordner erstellen möchten.

    Standardmäßig wird dieser iFolder-Ordner unter <Basisverzeichnis_des_Benutzers>/ifolder/servername/benutzername gespeichert.

  8. Wählen Sie den Typ des standardmäßigen iFolder-Ordners aus, den Sie erstellen möchten. Wenn Sie die Option Verschlüsselung auswählen, werden Sie zur Eingabe des Passwortsatzes aufgefordert.

    WICHTIG:Sie können nur die regulären iFolder-Ordner freigeben. Verschlüsselte iFolder-Ordner können nicht freigegeben werden.

    Wenn Ihr iFolder-Administrator die Verschlüsselung für Ihr aktuelles Konto nicht aktiviert hat, können Sie nur reguläre iFolder-Ordner erstellen. Dagegen ist bei aktivierter Verschlüsselung die Option Regulär standardmäßig ausgewählt.

  9. Wählen Sie SecureSync, um die sichere Synchronisierung der Dateien im von Ihnen erstellten iFolder-Ordner zu aktivieren.

    Sie können sowohl für reguläre als auch für verschlüsselte iFolder-Ordner den sicheren Kanal aktivieren. Wenn diese Option ausgewählt ist, ermöglicht iFolder die Verwendung eines sicheren Kanals zur Synchronisierung zwischen den iFolder-Ordnern auf dem Server und dem lokalen Computer.

    Wenn Sie den standardmäßigen iFolder-Ordner vom Server entfernen, fordert Sie iFolder beim nächsten Konfigurieren des Kontos von einem Thick-Client aus erneut zum Erstellen eines standardmäßigen iFolder-Ordners auf.

    Wenn Sie den Standard-iFolder-Ordner jedoch auf dem lokalen Computer entfernen, werden Sie bei der nächsten Konfiguration des Kontos zum Herunterladen des Standard-iFolder-Ordners aufgefordert.

  10. Wenn Sie bereits einen Standard-iFolder-Ordner eingerichtet haben, können Sie ihn auf den lokalen Computer herunterladen.

    Laden Sie den Standard-iFolder-Ordner herunter und klicken Sie zum Fortfahren auf Weiter.

  11. Wenn auf Ihrem System bereits ein Ordner vorhanden ist, dessen Name mit dem Namen des Standard-iFolder-Ordners identisch ist, werden Sie aufgefordert, entweder die beiden Ordner zusammenzuführen, oder den Standard-iFolder-Ordner an einen anderen Standort herunterzuladen.

    Klicken Sie auf OK, um die beiden Ordner zusammenzuführen, oder klicken Sie auf Abbrechen, um den Standard-iFolder-Ordner an einen anderen Standort herunterzuladen.

  12. Klicken Sie auf Fertig stellen, um den Assistenten zum Erstellen von iFolder-Konten zu schließen.

    iFolder synchronisiert die Benutzerliste für das angegebene Konto und identifiziert iFolder-Ordner, die zum Herunterladen zur Verfügung stehen. Dieser erste Download kann ein paar Sekunden oder ein paar Minuten dauern, je nach Größe der Benutzerliste für das Konto.

  13. Überprüfen Sie, ob das eingerichtete Konto in der Liste der Konten angezeigt wird.

    1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Dienstesymbol und wählen Sie dann im Menü Einstellungen aus, um das Dialogfeld iFolder-Einstellungen auf der Registerkarte Konten zu öffnen.

      Wenn Sie mit dem iFolder-Server verbunden sind, ist das Kontrollkästchen Online aktiviert.

    2. Schließen Sie das Dialogfeld "iFolder-Einstellungen".

  14. Gehen Sie zur Einrichtung von iFolder-Konten für das Konto gemäß einer der nachfolgenden Verfahrensweisen vor:

    • Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Dienstesymbol und wählen Sie dann iFolder-Ordner aus, um den iFolder-Browser zu öffnen. Fahren Sie mit Erstellen und Hochladen eines iFolder-Ordners fort.

    • Wenn für dieses Konto iFolder-Ordner zum Herunterladen vom Server verfügbar sind, werden sie im iFolder-Fenster unter den iFolder-Ordnern und <iFolder-Servername> aufgelistet. Das Download-Symbol zeigt an, dass der iFolder-Ordner heruntergeladen werden kann. Fahren Sie mit Herunterladen verfügbarer iFolder-Ordner fort.
