Verschieben von iFolder-Ordnern

Nach dem Erstellen eines iFolder-Ordners ist der Pfad zu diesem iFolder-Ordner statisch. Sie können einen iFolder-Ordner nicht von seinem ursprünglichen Standort an einen anderen Standort auf Ihrem lokalen Computer verschieben und ihn gleichzeitig als iFolder-Ordner beibehalten. Durch das Verschieben eines iFolder-Ordners wird die Verbindung zwischen der Synchronisierungsdatenbank und dem in einen iFolder-Ordner konvertierten Ordner unterbrochen.

So verschieben Sie einen iFolder-Ordner auf dem Computer, auf dem er erstellt wurde:

  1. Wählen Sie in einem Dateimanager oder im iFolder-Browser den zu verschiebenden iFolder-Ordner aus.

  2. Wandeln Sie den iFolder-Ordner in einen normalen Ordner um.

  3. Verschieben Sie den Ordner in einem Dateimanager zu einem anderen Speicherort auf dem Computer.

  4. Wandeln Sie den Ordner in einen iFolder-Ordner um und laden Sie den iFolder-Ordner auf den Server hoch.

  5. (Optional) Geben Sie den iFolder-Ordner frei.

  6. Lassen Sie zu, dass der iFolder-Ordner Dateien mit dem iFolder-Server synchronisiert.

So verschieben Sie einen iFolder-Ordner auf einen teilnehmenden Computer:

  1. Wandeln Sie den iFolder-Ordner um, damit er nicht mehr im iFolder-Ordner auf dem aktuellen Computer vorhanden ist.

  2. Wählen Sie den verfügbaren Ordner aus und klicken Sie dann auf Einen iFolder herunterladen.

  3. Geben Sie einen neuen lokalen Pfad an, unter dem der iFolder-Ordner gespeichert werden soll.

  4. Lassen Sie zu, dass der iFolder-Ordner Dateien mit dem iFolder-Server synchronisiert.
