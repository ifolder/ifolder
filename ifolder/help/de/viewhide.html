Anzeigen und Ausblenden verfügbarer iFolder-Ordner

Sie können sich entscheiden, welche iFolder-Ordner auf Ihre Computer heruntergeladen werden sollen. Wenn iFolder-Ordner, die Sie nicht heruntergeladen haben, auf dem Server verfügbar sind, werden sie im iFolder-Browser unter dem Servernamen des entsprechenden Kontos aufgelistet. Der Servername ist ein vom Administrator angegebener beschreibender Name des Servers, auf dem Sie ein iFolder-Konto besitzen.

Zum Anzeigen der verfügbaren iFolder-Ordner im iFolder-Browser klicken Sie auf Verfügbare iFolder-Ordner anzeigen.

Zum Ausblenden der verfügbaren iFolder-Ordner im iFolder-Browser klicken Sie auf Verfügbare iFolder-Ordner ausblenden.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
