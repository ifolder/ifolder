Konfigurieren von iFolder-Einstellungen für den Client

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Dienstesymbol und wählen Sie dann Einstellungen aus, um das Dialogfeld mit den iFolder-Einstellungen auf der Registerkarte Allgemein zu öffnen.

  2. Geben Sie die Einstellung für Folgendes an:

    Parameter

    Beschreibung

    iFolder bei der Anmeldung am Desktop starten (nur im Windows-Client verfügbar)

    Wenn Sie diese Option auswählen, wird iFolder jedes Mal gestartet, wenn Sie sich bei dem Computer mit dem lokalen Benutzernamen anmelden, auf dem Sie das iFolder-Konto eingerichtet haben. Für die Konten, für die Sie die Option Passwort nicht vergessen aktiviert haben, wird die Synchronisierung in den angegebenen Synchronisierungsintervallen im Hintergrund ausgeführt.

    Deaktivieren Sie diese Option, um iFolder manuell zu starten.

    Bei erfolgreicher Erstellung von iFolder das Bestätigungsdialogfeld anzeigen

    Aktivieren Sie dieses Kontrollkästchen, um bei jeder Erstellung eines iFolder-Ordners zusätzlich zu den normalen optischen Hinweisen (beispielsweise das Einblenden des jeweiligen iFolder-Ordners in der Liste der iFolder-Ordner) eine Bestätigungsmeldung zu erhalten.

    Deaktivieren Sie dieses Kontrollkästchen, wenn Sie die Bestätigungsmeldung nicht mehr erhalten möchten.

    iFolder-Hauptfenster beim Starten ausblenden

    Deaktivieren Sie dieses Kontrollkästchen, um das iFolder-Fenster beim Starten anzuzeigen.

    Aktivieren Sie dieses Kontrollkästchen, wenn das iFolder-Fenster beim Starten abgeblendet bleiben soll.

    Synchronisierungsprotokolle anzeigen

    Aktivieren Sie dieses Kontrollkästchen, wenn die Synchronisierungsprotokolle beim Start angezeigt werden sollen.

    Deaktivieren Sie dieses Kontrollkästchen, wenn die Synchronisierungsprotokolle beim Start ausgeblendet werden sollen.

    Benachrichtigung für Richtlinienverletzungen anzeigen

    Aktivieren Sie dieses Kontrollkästchen, um eine Popup-Meldung im Benachrichtigungsbereich anzuzeigen, wenn eine Richtlinie verletzt wurde.

    Benachrichtigung über Freigabe von iFolder-Ordnern

    Aktivieren Sie dieses Kontrollkästchen, um eine Popup-Meldung im Benachrichtigungsbereich anzuzeigen, wenn Sie eine neue Einladung zur Nutzung eines freigegebenen iFolder-Ordners erhalten.

    Um eine Liste der iFolder-Ordner anzuzeigen, die zum Herunterladen auf den aktuellen Computer zur Verfügung stehen, wählen Sie im iFolder-Browser Verfügbare iFolder-Ordner anzeigen aus, wenn iFolder ausgeführt wird.

    Benachrichtigung über Auftreten von Konflikten

    Aktivieren Sie dieses Kontrollkästchen, um eine Popup-Meldung im Benachrichtigungsbereich anzuzeigen, wenn bei der Synchronisierung von Dateien ein Konflikt auftritt.

    Deaktivieren Sie dieses Kontrollkästchen, um die Popup-Meldungen für Synchronisierungskonflikte zu deaktivieren.

    Wenn Konflikte auftreten, wird das iFolder-Fehlersymbol auf dem iFolder-Ordner angezeigt, wenn iFolder ausgeführt wird.

    Benachrichtigung über Eintritt eines Benutzers in die iFolder-Domäne

    Aktivieren Sie dieses Kontrollkästchen, um eine Popup-Meldung im Benachrichtigungsbereich anzuzeigen, wenn ein Benutzer eine Einladung zur Nutzung eines freigegebenen iFolder-Ordners annimmt.

    Deaktivieren Sie dieses Kontrollkästchen, um die Popup-Meldungen zu deaktivieren.

    Wenn iFolder ausgeführt wird, können Sie jederzeit eine vollständige Liste aller iFolder-Benutzer anzeigen, indem Sie den iFolder-Ordner auswählen und dann auf Gemeinsam verwenden mit klicken, um das Dialogfeld "iFolder-Eigenschaften" aufzurufen und die Registerkarte Freigabe zu öffnen.

    Synchronisierung

    iFolder automatisch synchronisieren: Aktivieren Sie dieses Kontrollkästchen, um die Synchronisierung für alle iFolder-Ordner auf diesem Computer unter Ihrem aktuellen lokalen Anmeldenamen oder Ihrer Anmeldeidentität zu aktivieren. Deaktivieren Sie das Kontrollkästchen, um die Synchronisierung zu deaktivieren.

    Synchronisiert automatisch alle: Geben Sie das Mindestintervall an, das für die Synchronisierung von iFolder-Ordnern auf diesem Computer unter Ihrem aktuellen lokalen Anmeldenamen oder Ihrer Anmeldeidentität verwendet werden soll. Geben Sie den Wert und die Einheiten an. Der Standardwert ist 5 Minuten, wenn diese Option aktiviert ist. Der zulässige Mindestwert für das intervall beträgt 5 Sekunden.

    Wenn Sie den iFolder-Client mit einem Novell iFolder-Server der Version 3.7 oder höher verwenden, entspricht das effektive minimale Synchronisierungsintervall stets dem größten Wert der folgenden Einstellungen:

    • Die Systemrichtlinie ist standardmäßig auf null (0) gesetzt, es sei denn, es wurde eine Benutzerrichtlinie festgelegt. Wenn eine Benutzerrichtlinie festgelegt wurde, wird die Systemrichtlinie von der Benutzerrichtlinie überschrieben. Dabei spielt es keine Rolle, ob die Benutzerrichtlinie einen größeren oder einen kleineren Wert aufweist.

    • Die lokale Computerrichtlinie oder die Einstellung auf dem Client-Computer, über den die Synchronisierung mit dem Server erfolgt.

    • Die iFolder-(Sammlungs-)Richtlinie

  3. Klicken Sie auf Schließen, um das Dialogfeld "iFolder-Einstellungen" schließen.
