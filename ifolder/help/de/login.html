Anmelden bei iFolder-Konten

Sie können jederzeit lokal mit Dateien im iFolder-Verzeichnis arbeiten. Sie müssen an einem iFolder-Konto angemeldet sein, um die Benutzerliste und die iFolder-Daten zu synchronisieren, neue iFolder-Ordner hochzuladen, freigegebene iFolder-Ordner herunterzuladen, iFolder-Ordner freizugeben und iFolder-Ordner zu verwalten. Sie können sich einzeln anmelden und gleichzeitig bei mehreren Konten angemeldet sein.

Mit einer der folgenden Methoden können Sie sich bei den einzelnen Konten anmelden:

  • Automatisch anmelden: Aktivieren Sie Passwort speichern für Ihr iFolder-Konto, wenn Sie automatisch angemeldet werden wollen, sobald der iFolder-Client gestartet wird. Sie können Passwort auf diesem Computer speichern gleich bei der Konfiguration des Kontos aktivieren oder dies später durch Ändern der entsprechenden Kontoeinstellungen nachholen.

    Wenn Sie Ihr Passwort ändern, erfolgt keine automatische Authentifizierung, wenn Sie das nächste Mal versuchen, sich am iFolder-Konto anzumelden. Wenn Sie von iFolder zur Anmeldung aufgefordert werden, können Sie das neue Passwort eingeben und das Kontrollkästchen für Passwort speichern erneut aktivieren, falls gewünscht.

  • Bei Bedarf anmelden: Deaktivieren Sie das Kontrollkästchen für Passwort speichern, wenn Sie sich am iFolder-Konto nur bei Bedarf anmelden möchten.

So aktivieren oder deaktivieren Sie die Funktion Passwort speichern für ein vorhandenes iFolder-Konto:

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Dienstesymbol und wählen Sie dann Konten aus, um das Dialogfeld mit den iFolder-Einstellungen auf der Registerkarte Konten zu öffnen.

  2. Wählen Sie das iFolder-Konto aus und klicken Sie dann auf Eigenschaften, um die Einstellungen im Dialogfeld Eigenschaften anzuzeigen.

  3. Klicken Sie auf die Registerkarte Identität.

  4. Wählen Sie Passwort speichern oder heben Sie die Auswahl dafür auf (bzw. aktivieren/deaktivieren Sie das entsprechende Kontrollkästchen).

  5. Geben Sie das Passwort im Feld Passwort ein.

  6. Klicken Sie auf Schließen, um den Vorgang abzuschließen.

  7. Klicken Sie auf Schließen im Dialogfeld iFolder-Eigenschaften, um Ihre Änderungen zu speichern und sich am Server anzumelden.

So melden Sie sich bei einem iFolder-Konto an:

  1. Klicken Sie mit der rechten Maustaste im Benachrichtigungsbereich auf das iFolder-Dienstesymbol und wählen Sie dann Konten aus.

  2. Wählen Sie das iFolder-Konto aus und aktivieren Sie dann daneben das Kontrollkästchen Online.

  3. Wenn Sie nach einem Passwort gefragt werden, geben Sie das Passwort an und klicken Sie dann auf Verbinden.

    Wenn Passwort speichern für das Konto aktiviert ist, werden Sie nur zur Eingabe eines Passworts aufgefordert, wenn Ihr Passwort abgelaufen ist.

    Wenn bei der Anmeldung ein Fehler auftritt, wird der Grund hierfür in einer Fehlermeldung angegeben; es kann sich beispielsweise um einen Fehler hinsichtlich des Passworts handeln.

  4. Schließen Sie nach erfolgreicher Verbindungsherstellung das Dialogfeld iFolder-Einstellungen.
