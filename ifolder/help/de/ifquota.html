Festlegen einer iFolder-Quote

Sie können keine Quote für Ihre eigenen iFolder-Ordner festlegen. Der iFolder-Administrator legt die Quote für Ihr Benutzerkonto fest. Der Administrator kann die Quote auf eine von vier Stufen festlegen: System, LDAPGroup, Benutzer und iFolder.

Es folgen Einzelheiten über das Anzeigen der für Ihr Konto festgelegten Quoteneigenschaften:

  • Quote

  • Speicherparameter für auf dem iFolder-Server gespeicherte iFolder-Ordner

Wenn der Administrator eine iFolder-Quote aktiviert hat, die geringer als Ihr aktueller Speicherverbrauch für iFolder-Daten ist, werden die Daten so lange nicht mehr synchronisiert, bis sie die Grenze unterschreiten oder die Quote auf einen Wert erhöht wird, der mindestens so groß wie der verbrauchte Speicherplatz ist.
