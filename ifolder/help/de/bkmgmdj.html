Ändern des Passworts

Sie können das Passwort für Ihr iFolder-Konto ändern.

  1. Wählen Sie im iFolder-Browser Sicherheit > Passwort ändern aus.

  2. Wählen Sie im Fenster Passwort ändern das iFolder-Konto aus, für das Sie das Passwort ändern möchten, und geben Sie die folgenden Details an:

    Aktuelles Passwort: Geben Sie Ihr aktuelles Passwort an.

    Neues Passwort: Geben Sie das neue Passwort an, das Sie für das aktuelle iFolder-Konto festlegen möchten.

    Neues Passwort bestätigen: Geben Sie das neue Passwort zur Bestätigung erneut an.

    Passwort speichern: Aktivieren Sie das Kontrollkästchen für Passwort speichern, um bei jeder Anmeldung an Ihrem Computer automatisch beim iFolder-Konto angemeldet zu werden.

  3. Klicken Sie auf Zurücksetzen.

    HINWEIS:iFolder ändert das Passwort direkt am LDAP-Server. Wenn derselbe LDAP-Server als Benutzerverwaltungsquelle für andere Anwendungen verwendet wird, steht das neue Passwort auch für diese Anwendungen zur Verfügung. Nach der Änderung Ihres Passworts läuft die iFolder-Sitzung aus und Sie werden abgemeldet. Sie müssen sich anschließend neu anmelden, um eine Verbindung zu den iFolder-Services herzustellen.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
