Beenden des iFolder-Clients

Um den iFolder-Client zu beenden, führen Sie eines der folgenden Verfahren aus:

  • Klicken Sie mit der rechten Maustaste im Benachrichtigungsbereich auf das iFolder-Dienstesymbol und wählen Sie dann Beenden aus.

  • Klicken Sie auf iFolder > Beenden.

    Wenn Sie Beenden auswählen, zeigt iFolder eine Bestätigungsmeldung an. Klicken Sie auf Ja, um die Aktion zu bestätigen.

Die Dateien werden nur mit Ihrem iFolder-Serverkonto synchronisiert, wenn Sie mit dem iFolder-Server verbunden sind. Sie können die Synchronisierung stoppen, indem Sie sich bei einem Konto abmelden. Im Offlinemodus wird iFolder weiterhin im Hintergrund ausgeführt. Informationen hierzu erhalten Sie unterAbmelden von einem iFolder-Konto.

Sie können über mehrere Konten verfügen, von denen jedes auf einem anderen Hostserver (Domäne) erstellt wurde.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
