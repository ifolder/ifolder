Freigeben von iFolder-Ordnern

Sie können einen iFolder-Ordner gemeinsam mit mehreren Benutzern verwenden. In diesem Abschnitt werden die folgenden Aufgaben behandelt:

  • Benutzerzugriffsrechte

  • Zugriff auf die Registerkarte "Freigabe"

  • Hinzufügen von Benutzern zu iFolder-Ordnern

  • Bearbeiten von Benutzerzugriffsrechten

  • Entfernen von Benutzern aus iFolder-Ordnern

  • Übertragen der Eigentümerschaft auf einen iFolder-Benutzer

Zugriff auf die Registerkarte "Freigabe"

Sie können die Mitgliedschaft des iFolder-Ordners auf der Registerkarte Freigabe im Dialogfeld "iFolder-Eigenschaften" verwalten.

WICHTIG:Sie können iFolder-Ordner nur freigeben, wenn der iFolder-Administrator die Freigaberichtlinie für Ihr Konto aktiviert hat.

  1. Starten Sie ggf. iFolder.

  2. Führen Sie eine der folgenden Aktionen aus, um das Dialogfeld "iFolder-Eigenschaften" aufzurufen und die Registerkarte Freigabe zu öffnen.

    • Klicken Sie in einem Dateimanager mit der rechten Maustaste auf den freizugebenden iFolder-Ordner und wählen Sie dann Gemeinsam verwenden mit aus.

    • Wählen Sie im iFolder-Browser den iFolder-Ordner und wählen Sie dann Gemeinsam verwenden mit aus.

Hinzufügen von Benutzern zu iFolder-Ordnern

  1. Klicken Sie auf der Registerkarte Freigabe auf Hinzufügen, um das Dialogfeld "Benutzer auswählen" zu öffnen.

  2. Wählen Sie aus der Liste der Benutzer einen oder mehrere Benutzer oder LDAP-Gruppen aus, indem Sie sie der Liste Hinzuzufügende Benutzer hinzufügen, und klicken Sie anschließend auf OK.

    Es kann einige Sekunden dauern, bis die Benutzer auf dem Server synchronisiert werden. Kurz nach der Synchronisierung wird rechts unten auf der Seite eine Benachrichtigung angezeigt.

  3. Wählen Sie auf der Registerkarte Freigabe einen oder mehrere Benutzer aus, denen Sie dieselben Zugriffsrechte zuweisen möchten.

  4. Klicken Sie auf Rechte, wählen Sie aus den Zugriffsrechten Vollständige Kontrolle, Lesen/Schreiben oder Nur Lesen aus und klicken Sie dann auf OK.

  5. Wiederholen Sie die vorherigen Schritte, bis Sie alle gewünschten Benutzer und LDAP-Gruppen zur gemeinsamen Nutzung des iFolder-Ordners eingeladen und ihre Zugriffsrechte festgelegt haben.

  6. Klicken Sie auf Schließen, um das Fenster "iFolder-Eigenschaften" zu schließen.

Bearbeiten von Benutzerzugriffsrechten

Da sich die Arbeitsbeziehungen und der Status der Mitglieder eines iFolder-Ordners ändern, müssen Sie möglicherweise das Zugriffsrecht eines Benutzers für den iFolder-Ordner ändern. Beispielsweise können Sie einem vertrauenswürdigen Benutzer das Recht der vollständigen Kontrolle für den iFolder-Ordner gewähren.

  1. Wählen Sie auf der Registerkarte Freigabe einen oder mehrere Benutzer aus, die dieselben Zugriffsrechte haben sollen.

  2. Wählen Sie Rechte aus und geben Sie dann das Recht Vollständige Kontrolle, Lesen/Schreiben oder Nur Lesen an.

    Wenn Sie zum ersten Mal einen Benutzer oder eine LDAP-Gruppe zur Teilnahme einladen, ist die Option Diesen Benutzer zum iFolder-Eigentümer machen im Dialogfeld "Zugriff" abgeblendet. Wenn Sie den neuen Benutzer zum Eigentümer machen möchten, müssen Sie warten, bis sich der Status des Benutzers von Eingeladener Benutzer in iFolder-Benutzer ändert. Dann können Sie die Eigentümerschaft übertragen. Wenn Sie die Eigentümerschaft auf eine LDAP-Gruppe übertragen, erhalten alle Mitglieder der LDAP-Gruppe Eigentümerrechte.

  3. Klicken Sie auf OK und klicken Sie danach auf OK, um das Fenster "iFolder-Eigenschaften" zu schließen.

Entfernen von Benutzern aus iFolder-Ordnern

Wenn Sie die Freigabe eines iFolder-Ordners aufheben möchten, müssen Sie einen Benutzer oder eine LDAP-Gruppe als Mitglied eines iFolder-Ordners entfernen. Der iFolder-Ordner auf der Arbeitsstation des Benutzers oder der LDAP-Gruppe wird zum normalen Ordner. Eine Kopie der Daten verbleibt auf der Arbeitsstation des ehemaligen Mitglieds, die Dateien werden jedoch nicht mehr mit dem freigegebenen iFolder-Ordner synchronisiert.

  1. Wählen Sie auf der Registerkarte Freigabe mindestens einen Benutzer aus der Liste aus.

  2. Klicken Sie auf Entfernen und anschließend auf Ja.

Übertragen der Eigentümerschaft auf einen iFolder-Benutzer

Der Eigentümer eines iFolder-Ordners kann die Eigentümerschaft des iFolder-Ordners auf ein anderes Mitglied oder auf eine LDAP-Gruppe übertragen, das oder die den Status eines iFolder-Benutzers besitzt. Nach der Übertragung wird der ursprüngliche Eigentümer zum Mitglied mit Lese-/Schreibberechtigung und der vom iFolder-Ordner belegte Speicherplatz wird dem neuen Besitzer angerechnet. Wenn eine LDAP-Gruppe zum Eigentümer eines iFolder-Ordners wird, erhalten alle Mitglieder der Gruppe Eigentümerrechte.

  1. Klicken Sie in einem Dateimanager oder im iFolder-Browser mit der rechten Maustaste auf den zu übertragenden iFolder-Ordner und wählen Sie dann Gemeinsam verwenden mit aus, um das Dialogfeld iFolder-Eigenschaften mit der Registerkarte Freigabe zu öffnen.

  2. Wählen Sie aus der Liste der Benutzer den iFolder-Benutzer aus, der der neue Eigentümer sein soll, und klicken Sie anschließend auf Rechte.

  3. Wählen Sie Diesen Benutzer zum iFolder-Eigentümer machen.

    Die Option Diesen Benutzer zum iFolder-Eigentümer machen wird im Dialogfeld "Zugriff" abgeblendet dargestellt, wenn der Benutzer den Status Eingeladener Benutzer innehat. Verlassen Sie in diesem Fall das Dialogfeld, warten Sie, bis sich der Benutzerstatus von Eingeladener Benutzer in iFolder-Benutzer geändert hat, und wiederholen Sie die vorherigen Schritte, um die Eigentümerschaft zu übertragen.

  4. Klicken Sie auf OK und klicken Sie danach auf OK, um das Fenster "iFolder-Eigenschaften" zu schließen.
