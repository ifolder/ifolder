Übersicht über iFolder

Novell iFolder ist eine Dateifreigabe-Anwendung für Linux-, Windows- und Macintosh-Clients. Sie können Dateien in mehreren iFolder-Ordnern freigeben, zu denen jeweils andere Benutzer oder LDAP-Gruppen gehören. Sie können steuern, welche Benutzer einen iFolder-Ordner nutzen dürfen und welche Zugriffsstufe den einzelnen Mitgliedern zugewiesen wird. Sie können zudem iFolder-Ordner nutzen, die andere Benutzer für Sie freigeben.

iFolder stellt Ihnen eine einfache, problemlose Benutzerschnittstelle für die clientseitige Migration von iFolder 2.x auf iFolder 3.6 und höher bereit sowie für ein clientseitiges Upgrade von iFolder 3.x auf iFolder 3.6 und höher.
