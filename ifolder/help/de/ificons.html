Einzelheiten zu den iFolder-Symbolen

In der folgenden Tabelle werden iFolder-Symbole und die zugehörigen Aufgaben beschrieben.

Tabelle 1 Beschreibung der iFolder-Symbole

iFolder-Symbol

Beschreibung

Das iFolder-Dienstesymbol wird für den iFolder 3-Client im Desktop-Anwendungsmenü angezeigt.

Wenn iFolder ausgeführt wird, wird das iFolder-Dienstesymbol auch im Benachrichtigungsbereich der Desktop-Taskleiste angezeigt. Klicken Sie mit der rechten Maustaste darauf, um ein Menü mit möglichen Aufgaben einzublenden, darunter iFolder-Browser, Kontoeinstellungen, Synchronisierungsprotokoll, Einstellungen und Hilfe.

Das Symbol iFolder-Ordner erstellen wird im iFolder-Browser angezeigt.

Sie können einen neuen iFolder-Ordner erstellen oder einen vorhandenen Ordner in einen iFolder-Ordner umwandeln. Wenn Sie über mehrere Konten verfügen, müssen Sie das zu verwendende Konto angeben. Ein iFolder-Ordner kann nur zu einem Konto gehören. Informationen hierzu erhalten Sie unterErstellen und Hochladen eines iFolder-Ordners.

Das Symbol iFolder-Download wird angezeigt, wenn ein iFolder-Ordner auf dem angegebenen Server für Sie verfügbar ist, Sie ihn jedoch noch nicht auf den lokalen Computer heruntergeladen haben.

Sie können beliebige Kombinationen von iFolder-Ordnern von verschiedenen Konten auf den lokalen Computer herunterladen. Informationen hierzu erhalten Sie unterHerunterladen verfügbarer iFolder-Ordner.

Das Symbol Verschlüsselter iFolder-Ordner (gesperrter iFolder-Ordner) gibt an, dass der iFolder-Ordner verschlüsselt ist und im iFolder-Browser angezeigt wird.

Mit dem bei der Anmeldung angegebenen Passwortsatz können Sie die Dateien im verschlüsselten iFolder-Ordner abrufen. Weitere Informationen zur Verschlüsselung finden Sie unter Verwalten von Passwortsätzen für verschlüsselte iFolder-Ordner.

Das iFolder-Symbol (ein grünes "i") wird für iFolder-Ordner angezeigt, wenn die iFolder-Diensteanwendung für iFolder-Ordner ausgeführt wird, die in einem Dateimanager, auf dem Desktop oder im iFolder-Browser angezeigt werden.

Wenn iFolder nicht ausgeführt wird, werden iFolder-Ordner als normale Ordner angezeigt, wenn sie im Dateimanager oder auf dem Desktop angezeigt werden. Wenn iFolder nicht ausgeführt wird, steht auch der iFolder-Browser nicht zur Verfügung.

Sie sollten Ihre lokale Sammlung von iFolder-Ordnern in einem separaten Verzeichnis organisieren, das nur iFolder-Ordner enthält. Auf diese Weise können Sie sich leicht merken, welche Ordner iFolder-Ordner sind, auch wenn iFolder nicht ausgeführt wird und keine Symbole für die Markierung als iFolder-Ordner zur Verfügung stehen.

Das iFolder-Warnungssymbol zeigt an, dass der Server nicht verfügbar ist. iFolder wird ausgeführt, aber Sie sind nicht mit dem iFolder-Server verbunden. Mögliche Ursachen sind:

  • Sie sind nicht am iFolder-Konto angemeldet. Informationen hierzu erhalten Sie unterAnmelden bei iFolder-Konten.

  • Das Konto wird auf Clientseite deaktiviert. Das bedeutet, dass der Parameter Automatisch verbinden deaktiviert wird. Informationen finden Sie in Automatisch verbinden in Anzeigen und Bearbeiten von iFolder-Konto-Einstellungen.

  • Der Server ist inaktiv oder Ihr Konto wurde auf Serverseite deaktiviert. Ihr iFolder-Administrator kann Ihnen in diesem Fall weiterhelfen.

  • Ihre Firewall blockiert den iFolder-Datenverkehr. Informationen zur Lösung dieses Problems finden Sie in Konfigurieren von lokalen Firewall-Einstellungen für den iFolder-Verkehr.

  • Ihre Netzwerkverbindung ist ausgefallen. Überprüfen Sie Ihre lokalen Netzwerkeinstellungen.

Das Symbol iFolder-Konflikt zeigt an, dass ein Konflikt zwischen der lokalen Kopie und der Serverkopie vorliegt. Lösen Sie den Konflikt, indem Sie angeben, welche Dateiversion behalten werden soll. Weitere Informationen zum Dateikonflikt finden Sie in Lösen von Dateikonflikten.

Das iFolder-Symbol für das Warten auf Synchronisierung zeigt an, dass der iFolder-Ordner Dateien enthält, die mit dem Server synchronisiert werden müssen und dass der Synchronisierungsvorgang noch nicht begonnen hat.

iFolder synchronisiert einen iFolder-Ordner nach dem anderen und stellt die Synchronisierungsaufgaben in eine Warteschlange. Um sicherzustellen, dass Dateien in diesem lokalen iFolder-Ordner mit den Dateien auf dem Server synchronisiert werden, warten Sie mit dem Beenden von iFolder, bis die Synchronisierung beginnt und vollständig ausgeführt wurde, was Sie an den Änderungen am Symbol erkennen können.

Das iFolder-Synchronisierungssymbol zeigt an, dass Dateien im iFolder-Ordner momentan mit dem Server synchronisiert werden.

Das Symbol iFolder-Fehler zeigt an, dass eine oder mehrere Dateien in dem iFolder-Ordner nicht synchronisiert wurden, weil es einen Richtlinienkonflikt aufgrund einer Dateigrößenbeschränkung oder einer Beschränkung bei der Anzahl der iFolder-Ordner gibt.

Im Dialogfeld "Freigabe" zeigt das iFolder-Symbol für den Eigentümerbenutzer an, dass der Benutzer den freigegebenen iFolder-Ordner besitzt.

Im Dialogfeld "Freigabe" zeigt das iFolder-Symbol für den aktuellen Benutzer an, dass der Benutzer den iFolder-Ordner aktuell verwaltet. Wenn der aktuelle Benutzer der Eigentümer ist, wird stattdessen das Eigentümerbenutzer-Symbol angezeigt.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
