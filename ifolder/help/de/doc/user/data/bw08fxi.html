Starten des iFolder-Client


Starten von iFolder unter Novell Linux Desktop 9

  1. Melden Sie sich unter der Benutzeridentität bei Ihrem Computer an, mit der Sie iFolder-Ordner erstellen möchten.

  2. Wählen Sie zum Starten von iFolder eine der folgenden Vorgehensweisen:

    • Öffnen Sie eine Terminal-Shell und geben Sie Folgendes ein:

      /opt/novell/ifolder3/bin/ifolder

    • Wenn iFolder automatisch gestartet werden soll, wenn Sie sich anmelden, stellen Sie sicher, dass iFolder ausgeführt wird, wenn Sie sich bei der GNOME-Sitzung abmelden, und speichern Sie die Sitzung dann. Sie können auch ein Terminal-Fenster öffnen, sich als Benutzer "root" anmelden und dann /opt/gnome/bin/gnome-session-properties ausführen, um iFolder als Programm hinzuzufügen, das bei Ihrer Anmeldung gestartet werden soll.

  3. Wenn Sie die Option "Passwort speichern" für Ihr iFolder-Konto deaktiviert haben, müssen Sie sich bei diesem Konto anmelden, um die Benutzerliste zu synchronisieren, iFolder-Ordner zu synchronisieren und Ihre iFolder-Ordner zu verwalten.

    Weitere Informationen hierzu erhalten Sie unter Anmelden bei iFolder-Konten.


Starten von iFolder unter Windows 2000/XP

  1. Melden Sie sich unter der Windows-Benutzeridentität bei Ihrem Computer an, mit der Sie iFolder-Ordner erstellen möchten.

  2. Wenn iFolder nach der erfolgreichen Anmeldung nicht automatisch gestartet wird, wählen Sie die Optionsfolge "Start" > "Programme" > "iFolder" und wählen Sie dann die iFolder-Anwendung aus.

    iFolder wird nicht automatisch gestartet, wenn Sie in den iFolder-Einstellungen die Option "iFolder starten, wenn Anmeldung am Desktop erfolgt" deaktivieren.

  3. Wenn Sie die Option "Passwort speichern" für Ihr iFolder-Konto deaktiviert haben, müssen Sie sich bei diesem Konto anmelden, um die Benutzerliste zu synchronisieren, iFolder-Ordner zu synchronisieren und Ihre iFolder-Ordner zu verwalten.

    Weitere Informationen hierzu erhalten Sie unter Anmelden bei iFolder-Konten.


Starten von iFolder unter Macintosh OS X ab Version 10.3

  1. Melden Sie sich unter der Benutzeridentität bei Ihrem Computer an, mit der Sie iFolder-Ordner erstellen möchten.

  2. Wählen Sie zum Starten von iFolder eine der folgenden Vorgehensweisen:

    • Rufen Sie das Menü "Programme" auf, wählen Sie das Untermenü "Internet" aus und klicken Sie dann auf "iFolder".
    • Wenn Sie iFolder über die Befehlszeile starten möchten, öffnen Sie eine Terminal-Shell und geben Sie Folgendes ein:

      /opt/novell/ifolder3/bin/ifolder

    • Wenn iFolder automatisch gestartet werden soll, wenn Sie sich bei Ihrem Macintosh-Computer anmelden, fügen Sie iFolder den Startelementen hinzu.

  3. Wenn Sie die Option "Passwort speichern" für Ihr iFolder-Konto deaktiviert haben, müssen Sie sich bei diesem Konto anmelden, um die Benutzerliste zu synchronisieren, iFolder-Ordner zu synchronisieren und Ihre iFolder-Ordner zu verwalten.

    Weitere Informationen hierzu erhalten Sie unter Anmelden bei iFolder-Konten.
