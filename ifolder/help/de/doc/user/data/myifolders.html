Verwalten von iFolder-Ordnern

Ein iFolderTM-Ordner ist ein lokales Verzeichnis, in dem ausgewählte Dateien mit Mitgliedern freigegeben und synchronisiert werden, die der Benutzer angibt. Sämtliche Mitglieder können auf die iFolder-Dateien zugreifen; die Dateien können von denjenigen Mitgliedern geändert und gelesen werden, die hierzu berechtigt sind.

In diesem Abschnitt werden die folgenden Aufgaben für iFolder behandelt:

  • Richtlinien für das Suchen nach iFolder-Ordnern
  • Richtlinien für nicht zu synchronisierende Dateitypen und -größen
  • Benennungskonventionen für iFolder-Ordner und zugehörige Ordner und Dateien
  • Erstellen von iFolder-Ordnern
  • Freigeben von iFolder-Ordnern
  • Einrichten verfügbarer iFolder-Ordner
  • Anzeigen und Konfigurieren der Eigenschaften von iFolder-Ordnern
  • Synchronisieren von Dateien
  • Auflösen von Dateikonflikten
  • Zurücksetzen von iFolder-Ordnern auf normale Ordner
  • Verschieben von iFolder-Ordnern
  • Löschen von iFolder-Ordnern
