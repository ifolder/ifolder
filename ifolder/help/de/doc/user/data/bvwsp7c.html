Löschen von iFolder-Konten

Sie können angeben, ob das Konto nur vom aktuellen Computer entfernt werden soll oder ob die iFolder-Ordner, deren Eigentümer Sie sind, vom Server entfernt werden sollen. Auf diese Weise werden im Wesentlichen die freigegebenen iFolder-Ordner inaktiv, die Sie für dieses Konto eingerichtet haben.

Wenn Sie ein iFolder-Konto von Ihrem Computer entfernen, werden die zugehörigen lokalen Kopien von iFolder-Ordnern auf normale Ordner zurückgesetzt. Von anderen Computern aus bzw. über das Web können Sie weiterhin auf die iFolder-Ordner zugreifen.

Wenn Sie Ihr iFolder-Konto vom Server entfernen, geschieht Folgendes:

  • Die Freigabe sämtlicher iFolder-Ordner, deren Eigentümer Sie sind, wird aufgehoben.
  • Die lokalen Kopien von iFolder-Ordnern auf Mitgliedercomputern werden auf normale Ordner zurückgesetzt.
  • Der iFolder-Ordner und sein Inhalt werden vom Server entfernt.
  • Sie werden als Mitglied von iFolder-Ordnern entfernt, die andere Benutzer zuvor für Sie freigegeben haben.

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das Symbol für die iFolder-Anwendung () und wählen Sie dann "Konten" aus, um das Dialogfeld "iFolder-Einstellungen" mit der Registerkarte "Konten" aufzurufen.

  2. Wählen Sie das zu löschende iFolder-Konto aus und klicken Sie dann auf "Löschen".

  3. In einer Meldung werden Sie aufgefordert, den Umfang der Löschaktion zu bestimmen. Führen Sie einen der folgenden Schritte aus:

    • iFolder-Konto nur vom aktuellen Computer entfernen: Heben Sie die Auswahl von "Eigene iFolder-Ordner und -Dateien vom Server entfernen" auf und klicken Sie dann auf "Ja". Daraufhin entfernt iFolder das ausgewählte Konto lediglich von der lokalen Arbeitsstation. Die lokalen iFolder-Ordner für dieses Konto werden auf normale Ordner zurückgesetzt. Wenn die lokale Kopie der Daten nicht benötigt wird, machen Sie sie mithilfe eines Datei-Managers ausfindig und löschen Sie sie.

      Von anderen Computern aus können Sie mit dem iFolder-Client weiterhin auf Ihre iFolder-Ordner für dieses Konto zugreifen. Wenn der iFolder-Client mit einem Novell® iFolder® 3.x-Enterprise-Server verwendet wird, ist der Zugriff mit Novell iFolder 3.x Web Access von allen Standorten aus möglich.

    • iFolder-Konto und Ihre iFolder-Ordner und -Dateien vom Server entfernen: Wählen Sie "Meine iFolder-Ordner und Dateien vom Server entfernen" aus und klicken Sie dann auf "Ja". Daraufhin entfernt iFolder das ausgewählte Konto von der lokalen Arbeitsstation. Zudem werden sämtliche iFolder-Ordner, die Sie für dieses Konto besitzen, vom Server entfernt und die iFolder-Ordner werden auf dem aktuellen Computer und allen Mitgliedercomputern auf normale Ordner zurückgesetzt. Abschließend werden Sie als Mitglied für freigegebene iFolder-Ordner in diesem Konto entfernt. Die lokalen iFolder-Ordner werden alle auf normale Ordner zurückgesetzt. Wenn die lokale Kopie der Daten nicht benötigt wird, machen Sie sie mithilfe eines Datei-Managers ausfindig und löschen Sie sie.

      Wenn Sie dieses Konto auf diesem oder einem anderen Computer erneut verwenden möchten, müssen Sie hinsichtlich der Einrichtung des Kontos und der zugehörigen iFolder-Ordner genauso vorgehen wie bei der Neuerstellung.

    • Löschvorgang abbrechen: Klicken Sie auf "Nein", um die Löschaktion zu verwerfen. Ihr Konto und die zugehörigen iFolder-Ordner bleiben unverändert.
