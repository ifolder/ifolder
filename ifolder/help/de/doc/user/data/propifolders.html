Anzeigen und Konfigurieren der Eigenschaften von iFolder-Ordnern

Auf der Seite "iFolder-Eigenschaften" können Sie die folgenden Informationen zu einem ausgewählten iFolder-Ordner anzeigen und konfigurieren.


iFolder-Eigenschaften

Mit den folgenden Parametern wird der auf dem Server verwendete Speicherplatz angegeben:

Parameter Beschreibung

Verfügbar

Der zurzeit im ausgewählten iFolder-Ordner insgesamt für Dateien und Verzeichnisse verfügbare Speicherplatz des Servers. Der Wert entspricht der Speicherquote abzüglich des belegten Speichers.

Belegt

Der zurzeit im ausgewählten iFolder-Ordner insgesamt von allen Dateien und Verzeichnissen belegte Speicherplatz des Servers.

Eine der folgenden Optionen:

  • Quote festlegen
  • Quote

  • Quote festlegen: Wenn Sie der Eigentümer des iFolder-Ordners sind, können Sie eine Speicherquote für den iFolder-Ordner angeben.

  • Quote: Wenn Sie ein Mitglied sind, wird durch den Wert der Serverspeicherplatz insgesamt angegeben, der dem ausgewählten iFolder-Ordner durch den Eigentümer des iFolder-Ordners zugewiesen wurde. Wenn keine Quote festgelegt wurde, ist kein Wert angegeben oder lautet "N/Z" (nicht zutreffend).

Mit den folgenden Parametern wird der Synchronisierungsstatus für die lokale Kopie des iFolder-Ordners angegeben:

Parameter Beschreibung

Letzte erfolgreiche Synchronisierung

Die Zeit zum Abschließen der letzten erfolgreichen Synchronisierung Ihrer lokalen Kopie des iFolder-Ordners mit dem iFolder-Server.

Zu synchronisierende Dateien/Ordner

Die Gesamtanzahl der Speicherobjekte (Dateien und Ordner) im iFolder-Ordner, die synchronisiert werden müssen.

Automatisch synchronisieren alle

Das Synchronisierungsintervall für den iFolder-Ordner. Der Wert kann in Minuten, Stunden oder Tagen angegeben werden.

Wenn der iFolder-Client mit einem Novell iFolder-Enterprise-Server der Version 3.x verwendet wird, kann mit dem Intervall nicht das vom iFolder-Administrator angegebene Mindestintervall für den Server oder Ihr Konto auf dem Server überschrieben werden. Es kann jedoch das Synchronisierungsintervall für den gesamten Client überschrieben werden.

Jetzt synchronisieren

Klicken Sie auf die Schaltfläche, um bei Bedarf eine Synchronisierung zu initiieren.


Zugreifen auf iFolder-Eigenschaften über einen Datei-Manager

  1. Navigieren Sie im Datei-Manager zum iFolder-Ordner und wählen Sie ihn dann aus.

  2. Klicken Sie mit der rechten Maustaste auf den iFolder-Ordner und klicken Sie dann auf "iFolder" > "Eigenschaften".


Zugreifen auf iFolder-Eigenschaften über einen Browser für iFolder-Ordner

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Symbol () und klicken Sie dann auf "iFolder-Ordner", um den Browser für iFolder-Ordner zu öffnen.

  2. Wählen Sie in der Liste der iFolder-Ordner den iFolder-Ordner aus und klicken Sie anschließend auf "Eigenschaften".
