Konfigurieren von iFolder-Einstellungen für den Client

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das Symbol für die iFolder-Anwendung () und wählen Sie dann "Einstellungen" aus, um das Dialogfeld "iFolder-Einstellungen" mit der Registerkarte "Allgemein" aufzurufen.


  2. Geben Sie folgende Werte an:

    Parameter Beschreibung

    iFolder starten, wenn Anmeldung am Desktop erfolgt

    Wenn Sie diese Option auswählen, wird iFolder jedes Mal gestartet, wenn Sie sich bei dem Computer mit dem lokalen Benutzernamen anmelden, auf dem Sie das iFolder-Konto eingerichtet haben. Für die Konten, für die Sie die Option "Passwort speichern" aktiviert haben, wird die Synchronisierung in den angegebenen Synchronisierungsintervallen im Hintergrund ausgeführt.

    Deaktivieren Sie diese Option, um iFolder manuell zu starten.

    Beim Erstellen von iFolder-Ordnern Bestätigungsdialogfeld einblenden

    Wählen Sie diese Option aus, um bei jeder Erstellung eines iFolder-Ordners zusätzlich zu den normalen optischen Hinweisen (beispielsweise das Einblenden des jeweiligen iFolder-Ordners in der Liste der iFolder-Ordner) eine Bestätigungsmeldung zu erhalten.

    Wenn Sie die Bestätigungsmeldung nicht mehr erhalten möchten, heben Sie die Auswahl dieser Option aus.

    Benachrichtigen bei Freigabe von iFolder-Ordnern

    Wenn Sie diese Option auswählen, wird jedes Mal, wenn Sie eine neue Einladung zur Freigabe eines iFolder-Ordners erhalten, eine Popup-Meldung eingeblendet.

    Einladungen werden basierend auf dem Synchronisierungsintervall abgerufen, das Sie unter "iFolder-Einstellungen" angeben. Durch dieses Intervall werden die Synchronisierung von Dateien, Einladungen zur Freigabe von iFolder-Ordnern sowie die Liste der Benutzer gesteuert. Bei Bedarf können die Einladungen auch über "Aktualisieren" auf den neuesten Stand gebracht werden.

    Wenn keine Popup-Meldungen hinsichtlich Einladungen erwünscht sind, heben Sie die Auswahl dieser Option auf.

    Bei Konflikten benachrichtigen

    Wenn Sie diese Option auswählen, wird jedes Mal, wenn bei der Synchronisierung von Dateien ein Konflikt auftritt, im Benachrichtigungsbereich eine Popup-Meldung eingeblendet.

    Wenn keine Popup-Meldungen zu Synchronisierungskonflikten erwünscht sind, heben Sie die Auswahl dieser Option auf.

    Benachrichtigen, wenn ein Benutzer beitritt

    Wenn Sie diese Option auswählen, wird jedes Mal, wenn ein Benutzer eine Einladung zur Freigabe eines iFolder-Ordners annimmt, eine Popup-Meldung eingeblendet. Die Annahme ist bis zu dem Zeitpunkt verschlüsselt, wenn der Benutzer den iFolder-Ordner auf dem Computer einrichtet.

    Wenn keine Popup-Meldungen hinsichtlich der Bestätigung der Benutzerfreigabe erwünscht sind, heben Sie die Auswahl dieser Option auf.

    Synchronisierung

    iFolder automatisch synchronisieren: Aktivieren Sie dieses Kontrollkästchen, um die Synchronisierung für alle iFolder-Ordner auf diesem Computer unter Ihrer aktuellen lokalen Anmelde-Identität zu ermöglichen. Deaktivieren Sie das Kontrollkästchen, um die Synchronisierung zu deaktivieren.

    iFolder synchronisieren alle: Geben Sie das Mindestintervall an, das für die Synchronisierung von iFolder-Ordnern auf diesem Computer unter Ihrer aktuellen lokalen Anmelde-Identität verwendet werden soll. Geben Sie den Wert und die Einheiten an.

    Wenn Sie den iFolder-Client mit Novell iFolder 3.x verwenden, entspricht das effektive minimale Synchronisierungsintervall stets dem größten Wert der folgenden Einstellungen:

    • Die Systemrichtlinie (Standardwert: null), es sei denn, es wurde eine Benutzerrichtlinie festgelegt. Wenn eine Benutzerrichtlinie festgelegt wurde, wird die Systemrichtlinie von der Benutzerrichtlinie überschrieben. Dabei spielt es keine Rolle, ob die Benutzerrichtlinie einen größeren oder einen kleineren Wert aufweist.
    • Die Richtlinie des lokalen Computers oder die Einstellung auf dem Client-Computer, über den die Synchronisierung mit dem Server erfolgt.
    • Die iFolder-(Sammlungs-)Richtlinie

  3. Wenn Sie fertig sind, klicken Sie auf "Anwenden", damit Ihre Änderungen wirksam werden.

  4. Klicken Sie auf "OK", um das Dialogfeld "iFolder-Einstellungen" zu schließen.
