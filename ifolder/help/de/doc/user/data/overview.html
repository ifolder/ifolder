Übersicht über iFolder

iFolderTM ist eine Anwendung für die Dateifreigabe, die für Linux-, Windows*-, und Macintosh*-Clients entwickelt wurde. Sie können Dateien in mehreren iFolder-Ordnern freigeben, von denen jeder einer anderen Benutzergruppe zugehörig ist. Sie können steuern, welche Benutzer einen iFolder-Ordner nutzen dürfen und welche Zugriffsstufe den einzelnen Mitgliedern zugewiesen wird. Sie können zudem iFolder-Ordner nutzen, die andere Benutzer für Sie freigeben.
