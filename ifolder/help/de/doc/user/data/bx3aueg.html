iFolder-Freigabe

Wenn Sie an unterschiedlichen Orten tätig sind oder mit anderen zusammenarbeiten, ist die gewissenhafte Verwaltung von Dateiversionen unerlässlich. Mit iFolder steht Ihnen die jeweils aktuelle Version Ihrer Dateien auf sämtlichen Computern zur Verfügung, auf denen der iFolder-Client installiert und ein freigegebener iFolder-Ordner eingerichtet wurde. iFolder ermöglicht Ihnen zudem die Freigabe mehrerer Ordner und ihrer jeweiligen Inhalte für andere berechtigte Benutzer. Sie steuern, welche Benutzer den jeweiligen iFolder-Ordner nutzen dürfen und welche Zugriffsstufe den einzelnen Mitgliedern zugewiesen wird. Ebenso können Sie freigegebene iFolder-Ordner nutzen, die sich im Besitz anderer Benutzer in Ihrer für die Zusammenarbeit konfigurierten Umgebung befinden.

Der iFolder-Client unterstützt die Freigabe, indem von einem zentralen Computer aus Dateien auf mehreren Computern synchronisiert werden. Außerdem können Sie:

  • Dateien für mehrere Computer freigeben
  • Dateien für andere Benutzer freigeben
  • Im Besitz mehrerer iFolder-Ordner sein
  • Mehrere iFolder-Ordner nutzen, die andere Benutzer für Sie freigeben
  • Als Eigentümer oder Mitglied iFolder-Ordner mehrerer Konten nutzen
