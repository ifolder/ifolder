Anmelden bei iFolder-Konten

Nach der Einrichtung eines iFolder-Kontos müssen Sie sich immer anmelden, wenn Sie Ihre lokalen iFolder-Daten, die Liste der iFolder-Benutzer sowie Ihre freigegebenen iFolder-Benachrichtigungen synchronisieren möchten. Sie können bei mehreren Konten gleichzeitig angemeldet sein. Mit einer der folgenden Methoden können Sie sich bei den einzelnen Konten anmelden:

  • Aktivieren Sie die Option "Passwort speichern", damit die Anmeldung beim iFolder-Konto immer automatisch erfolgt, wenn Sie sich bei Ihrem Desktop anmelden oder den iFolder-Client starten.

    Immer, wenn sich Ihr Passwort ändert, tritt ein Fehler auf, wenn iFolder das nächste Mal versucht, sich bei dem iFolder-Konto anzumelden. iFolder fordert Sie zur Anmeldung mit Ihrem neuen Passwort auf und ermöglicht Ihnen die Angabe und Speicherung des neuen Passworts.

  • Sie sollten die Option "Passwort speichern" nur deaktivieren und sich bei einem iFolder-Konto anmelden, wenn Sie Dateien in den zugehörigen iFolder-Ordnern synchronisieren oder die iFolder-Ordner verwalten möchten.

So aktivieren bzw. deaktivieren Sie "Passwort speichern" für ein bestehendes iFolder-Konto:

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Symbol () und wählen Sie dann "Konten" aus.

  2. Wählen Sie das iFolder-Konto aus, um die zugehörigen Konto-Einstellungen anzuzeigen.

  3. Wählen Sie "Passwort speichern" aus bzw. heben Sie die Auswahl auf (Option aktivieren oder deaktivieren).

  4. Geben Sie das Passwort an und klicken Sie dann auf "OK" oder "Anwenden".

  5. Klicken Sie auf "OK", um die an den Konto-Einstellungen vorgenommene Änderung zu bestätigen, oder klicken Sie auf "Abbrechen", um die Änderung zu verwerfen.

So melden Sie sich bei einem iFolder-Konto an:

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Symbol () und wählen Sie dann "Konten" aus.

  2. Wählen Sie das iFolder-Konto aus, um die zugehörigen Konto-Einstellungen anzuzeigen.

  3. Geben Sie das Passwort an und klicken Sie dann auf "Anmelden".

    Bei erfolgreicher Anmeldung ändert sich der Kontostatus in "Angemeldet".

    Wenn bei der Anmeldung ein Fehler auftritt, wird der Grund hierfür in einer Fehlermeldung angegeben; es kann sich beispielsweise um einen Fehler hinsichtlich des Passworts handeln.

  4. Klicken Sie auf "OK", um das Dialogfeld "Konten" zu schließen.
