Affichage et modification des paramètres du compte iFolder

Les paramètres suivants peuvent être affichés dans les boîtes de dialogue Paramètres du compte et Propriétés du compte :

Tableau 1 Paramètres et propriétés du compte

Paramètre

Description

Emplacement

Modifiable par l'utilisateur (oui/non)

En ligne

Indique si vous êtes actuellement connecté au serveur iFolder pour un compte.

Cochez la case En ligne en regard du compte à connecter au serveur iFolder.

Désélectionnez la case En ligne en regard du compte à déconnecter du serveur iFolder.

Préférences iFolder > onglet Comptes

Oui

Nom du serveur

Affiche le nom descriptif du serveur iFolder où vos dossiers iFolder sont stockés pour le compte. Ce nom est fourni par le serveur et contrôlé par l'administrateur iFolder.

Ce paramètre n'est pas modifiable par l'utilisateur.

Préférences iFolder > onglet Comptes

Propriétés du compte > onglet Serveur

Non

Nom d'utilisateur

Affiche l'identité de l'utilisateur pour ce compte, par exemple votre ID utilisateur réseau (nom commun LDAP) ou votre adresse de messagerie. Ce paramètre n'est pas modifiable.

Le nom d'utilisateur est spécifié lorsque vous configurez le compte dans l'Assistant Compte iFolder. Il ne peut pas être modifié pour un compte existant.

Pour employer un nom d'utilisateur différent sur le même serveur, vous devez supprimer le compte existant, puis en créer un autre.

Préférences iFolder > onglet Comptes

Propriétés du compte > onglet Identité

Non

Adresse

Affiche le nom DNS ou l'adresse IP du serveur d'entreprise iFolder sur lequel se trouve votre compte. Demandez ces informations à votre administrateur iFolder.

Par exemple : svr21.exemple.com ou 192.168.1.1

Vous pouvez modifier l'adresse après vous être déconnecté du serveur iFolder.

Propriétés du compte > onglet Serveur

Oui

Mot de passe

Permet de spécifier le mot de passe pour votre nom d'utilisateur.

Propriétés du compte > onglet Identité

Oui

Mémoriser le mot de passe

Indique si iFolder se logue automatiquement pour vous au démarrage de iFolder. Si le paramètre Connexion automatique du compte est activé (par défaut), iFolder tente de se connecter avec le mot de passe enregistré.

Sélectionnez Mémoriser le mot de passe pour vous loguer automatiquement à ce compte iFolder lorsque vous vous loguez à votre ordinateur.

Si votre mot de passe de réseau change, l'authentification automatique échoue la prochaine fois qu'iFolder essaie de vous loguer au compte iFolder. iFolder vous invite à vous loguer avec votre nouveau mot de passe et vous permet de le spécifier et de l'enregistrer.

Propriétés du compte > onglet Identité

Oui

Quota

Indique l'espace total alloué à votre compte iFolder sur le serveur d'entreprise. Cette valeur peut correspondre au quota par défaut défini pour tous les utilisateurs ou à un quota propre à l'utilisateur. Si aucun quota n'est défini, la valeur est vide ou N/A(non applicable). Vous pouvez également afficher l'espace disponible dans la fenêtre principale de iFolder pour chaque serveur auquel vous vous êtes connecté.

Propriétés du compte > onglet Espace disque

Non

Utilisé

Indique l'espace total actuellement utilisé par tous les dossiers iFolder que vous possédez pour ce compte. Ce total n'inclut pas l'espace utilisé par les dossiers iFolder que vous partagez avec d'autres, mais dont vous n'êtes pas propriétaire.

Propriétés du compte > onglet Espace disque

Non

Disponible

Indique l'espace total actuellement disponible pour vos dossiers iFolder sur ce compte iFolder. La valeur est égale au quota moins l'espace utilisé.

Propriétés du compte > onglet Espace disque

Non

Connexion automatique

Si cette valeur est sélectionnée (activée), iFolder se connecte automatiquement à ce compte au démarrage. Pour désactiver la connexion automatique, désélectionnez cette option.

Ce paramètre ne régit que la capacité côté client à activer ou désactiver le compte.

Lorsque vous utilisez iFolder avec un serveur Novell iFolder 3.7 ou version ultérieure, l'administrateur système de iFolder peut désactiver votre compte iFolder (généralement de façon temporaire), côté serveur. Dans ce cas, la situation suivante se produit :

  • Si vous êtes logué lorsque le compte est désactivé du côté serveur, vous pouvez gérer vos dossiers iFolder et vos données iFolder continuent à être synchronisées avec le serveur jusqu'à ce que vous clôturiez votre session en vous déloguant ou en désactivant la synchronisation, ou - si cela arrive en premier - jusqu'à ce que le serveur déconnecte votre session.

  • Si le compte est désactivé sur le serveur, vous recevez un message contextuel vous en informant chaque fois que vous essayez de vous loguer. Contactez votre administrateur système iFolder pour obtenir de l'aide.

Propriétés du compte, tous les onglets

Oui

Compte par défaut

Si vous possédez plusieurs comptes, cochez la case Compte par défaut pour que ce compte soit celui sélectionné par défaut dans la liste déroulante qui s'affiche lorsque vous créez et téléchargez des dossiers iFolder.

Cette valeur ne peut pas être modifiée si vous ne possédez qu'un seul compte.

Propriétés du compte, tous les onglets

Oui, si vous possédez plusieurs comptes

Paramètres de codage

Vous pouvez définir et redéfinir vos paramètres de codage qui sont affichés dans le menu Sécurité. Pour plus d'informations sur les paramètres de codage, reportez-vous à la section Gestion des phrases secrètes pour les dossiers iFolder codés.

Menu Sécurité

L'utilisateur peut modifier la phrase secrète

  1. Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, puis sélectionnez Paramètres du compte pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Comptes.

    Sous l'onglet Comptes, vous pouvez gérer les paramètres suivants pour chaque compte :

    • En ligne (sélectionnez l'option pour vous connecter et désélectionnez-la pour vous déconnecter)

    • Nom du serveur (affichage uniquement)

    • Nom d'utilisateur (affichage uniquement)

  2. Sélectionnez le compte iFolder à gérer, puis cliquez sur Propriétés pour ouvrir la boîte de dialogue Propriétés du compte dans l'onglet Serveur.

    Sous l'onglet Serveur, vous pouvez gérer les paramètres suivants pour le compte sélectionné :

    • Nom (affichage uniquement)

    • Adresse (affichage uniquement si vous êtes connecté ; modifiable si vous êtes déconnecté)

    • Description du serveur (affichage uniquement ; faites défiler pour afficher toutes les informations)

    • Connexion automatique (sélectionnez cette option pour activer la connexion automatique ; désélectionnez-la pour désactiver la connexion automatique)

    • Compte par défaut (sélectionnez cette option pour que le compte soit celui par défaut ; uniquement disponible si vous possédez plusieurs comptes)

  3. Sélectionnez l'onglet Identité pour gérer les paramètres suivants pour le compte sélectionné :

    • Nom d'utilisateur (affichage uniquement)

    • Mot de passe (précisez le mot de passe)

    • Mémoriser le mot de passe (sélectionnez cette option pour activer la mémorisation du mot de passe ; désélectionnez-la pour désactiver la mémorisation))

    • Connexion automatique (sélectionnez cette option pour activer la connexion automatique ; désélectionnez-la pour désactiver la connexion automatique)

    • Compte par défaut (sélectionnez cette option pour que le compte soit celui par défaut ; uniquement disponible si vous possédez plusieurs comptes)

  4. Sélectionnez l'onglet Espace disque pour afficher les paramètres suivants pour le compte sélectionné :

    • Quota (affichage uniquement)

    • Utilisé (affichage uniquement)

    • Disponible (affichage uniquement)

    • Connexion automatique (sélectionnez cette option pour activer la connexion automatique ; désélectionnez-la pour désactiver la connexion automatique)

    • Compte par défaut (sélectionnez cette option pour que le compte soit celui par défaut ; uniquement disponible si vous possédez plusieurs comptes)

  5. Une fois terminé, fermez la boîte de dialogue Propriétés du compte, puis la boîte de dialogue Préférences iFolder .

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
