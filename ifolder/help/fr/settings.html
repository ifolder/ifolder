Paramètres iFolder

L'onglet Paramètres iFolder permet d'activer les notifications.

  1. Cochez la case Afficher des notifications pour activer les notifications.

  2. Dans la liste des notifications, sélectionnez les événements pour lesquels vous souhaitez activer les notifications.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
