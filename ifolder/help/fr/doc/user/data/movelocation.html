Déplacement d'un dossier iFolder

Une fois que vous avez créé un dossier iFolder, le chemin d'accès à celui-ci est statique. Vous ne pouvez pas déplacer un dossier iFolder de son emplacement d'origine vers un autre emplacement sur votre ordinateur et le conserver en tant que dossier iFolder. Si vous déplacez un dossier iFolder, le lien entre la base de données de synchronisation et le dossier que vous avez converti en dossier iFolder se rompt.

Pour déplacer un dossier iFolder sur l'ordinateur où vous l'avez créé :

  1. Convertissez le dossier iFolder en dossier normal.

  2. Déplacez le dossier.

  3. Convertissez le dossier en dossier iFolder.

  4. (Facultatif) Partagez le dossier iFolder.

Pour déplacer un dossier iFolder sur un ordinateur participant :

  1. Convertissez le dossier iFolder de sorte qu'il ne soit plus un dossier participant à iFolder mais conservez-le pour qu'il soit toujours disponible.

  2. Sélectionnez ce dossier, puis cliquez sur Configurer.

  3. Spécifiez le chemin du nouvel emplacement du dossier iFolder.

  4. Autorisez le dossier iFolder à synchroniser les fichiers avec le serveur d'entreprise.
