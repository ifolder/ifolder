Conversion d'un dossier iFolder en dossier normal

Tout membre peut convertir un dossier iFolder en dossier normal. Une fois que le dossier est converti, la relation de partage permettant à tous les membres du dossier iFolder de collaborer demeure, mais le dossier iFolder n'est plus synchronisé sur cet ordinateur. Le dossier iFolder et son contenu restent disponibles sur les postes de travail des membres.

  1. Dans un gestionnaire de fichiers ou sur le bureau, recherchez le dossier iFolder que vous souhaitez convertir en dossier normal.

  2. Utilisez l'une des méthodes suivantes :

    • Cliquez avec le bouton droit sur le dossier iFolder, puis sélectionnez iFolder > Revenir à un dossier normal.
    • Sélectionnez le dossier iFolder, puis dans la barre d'outils du gestionnaire de fichiers, cliquez sur Fichier > iFolder > Revenir à un dossier normal.

    L'icône iFolder est remplacée par une icône de dossier normal. Les données sont conservées intégralement sur les ordinateurs des membres.
