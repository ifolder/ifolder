Synchronisation des fichiers


Synchronisation des fichiers sur demande

  1. Dans la zone de notification, cliquez avec le bouton droit sur l'icône iFolder , puis sélectionnez Dossiers iFolder pour ouvrir le navigateur de dossiers iFolder.

  2. Dans la liste des dossiers iFolder, sélectionnez le dossier iFolder voulu.

  3. Effectuez l'une des opérations suivantes :

    • Cliquez sur Synchroniser .
    • Sélectionnez iFolder > Synchroniser maintenant.


Configuration de l'intervalle de synchronisation

  1. Dans la zone de notification, cliquez avec le bouton droit de la souris sur l'icône iFolder , puis sélectionnez Préférences pour ouvrir la boîte de dialogue Préférences iFolder à l'onglet Général.

  2. Effectuez l'une des opérations suivantes :

    • Sélectionnez Synchroniser pour activer la synchronisation en arrière-plan.
    • Désélectionnez Synchroniser pour désactiver la synchronisation.

  3. Si vous activez la synchronisation, spécifiez l'intervalle de synchronisation des fichiers, par exemple toutes les heures ou tous les deux jours.

    Lorsque le client iFolder est utilisé avec Novell iFolder 3.x, l'intervalle de synchronisation système minimum, en général toutes les 5 minutes, est configuré par l'administrateur iFolder. Vous ne pouvez pas remplacer l'intervalle de synchronisation minimum pour le système. L'intervalle de synchronisation minimum absolu pour le système est de 1 minute.
