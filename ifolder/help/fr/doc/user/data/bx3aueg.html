Partage iFolder

En principe, lorsque vous travaillez à divers emplacements ou avec d'autres personnes, vous devez gérer les différentes versions des fichiers avec beaucoup de précaution. Avec iFolder, la toute dernière version de vos fichiers peut vous suivre sur tous les ordinateurs sur lesquels le client iFolder est installé avec la fonction de dossiers partagés activée. iFolder vous permet également de partager plusieurs dossiers iFolder et leur contenu avec d'autres utilisateurs autorisés. Vous choisissez qui participe à chaque iFolder et vous déterminez le niveau d'accès de chaque membre. De même, vous pouvez participer aux dossiers iFolder partagés qui appartiennent à d'autres utilisateurs de votre environnement de collaboration.

Le client iFolder prend en charge le partage en synchronisant des fichiers sur plusieurs ordinateurs via un ordinateur central. Il vous permet d'effectuer les opérations suivantes :

  • partager des fichiers entre plusieurs ordinateurs ;
  • partager des fichiers avec d'autres utilisateurs ;
  • posséder plusieurs dossiers iFolder ;
  • participer à plusieurs dossiers iFolder que d'autres utilisateurs partagent avec vous ;
  • participer à plusieurs comptes en tant que propriétaire ou membre de dossiers iFolder.
