Affichage et configuration des propriétés d'un dossier iFolder

La page Propriétés des dossiers iFolder vous permet d'afficher et de configurer les informations suivantes concernant un dossier iFolder sélectionné.


Présentation des propriétés iFolder

Les paramètres suivants indiquent l'espace disque utilisé sur le serveur :

Paramètre Description

Disponible

Espace disque total disponible sur le serveur pour les fichiers ou les répertoires dans le dossier iFolder sélectionné. La valeur est égale au quota moins l'espace utilisé.

Utilisé

Espace disque total occupé sur le serveur par l'ensemble des fichiers et des répertoires dans le dossier iFolder sélectionné.

Reportez-vous aux paramètres suivants :

  • Définir un quota
  • Quota

  • Définir un quota : Si vous êtes le propriétaire du dossier iFolder, vous pouvez spécifier un quota pour celui-ci.

  • Quota : Si vous êtes un membre, la valeur indique l'espace disque total alloué sur le serveur au dossier iFolder sélectionné par son propriétaire. Si aucun quota n'est défini, la valeur est vide ou égale à S/O (sans objet).

Les paramètres suivants indiquent l'état de la synchronisation de la copie locale du dossier iFolder :

Paramètre Description

Dernière synchronisation réussie

Heure de fin de la dernière synchronisation de la copie locale du dossier iFolder avec le serveur iFolder.

Fichiers/dossiers à synchroniser

Nombre total d'éléments de stockage (fichiers et dossiers) dans le dossier iFolder devant être synchronisés.

Synchronisation automatique tous les :

Intervalle de synchronisation du dossier iFolder. Spécifiez la valeur en minutes, heures ou jours.

Lorsque le client iFolder est utilisé avec un serveur d'entreprise Novell iFolder 3.x , l'intervalle ne peut pas remplacer l'intervalle minimum spécifié par l'administrateur iFolder pour le serveur ou votre compte sur le serveur, mais il peut remplacer l'intervalle de synchronisation pour le client.

Synchroniser maintenant

Cliquez sur ce bouton pour lancer une synchronisation sur demande.


Accès aux propriétés iFolder à partir d'un gestionnaire de fichiers

  1. Dans le gestionnaire de fichiers, accédez au dossier iFolder et sélectionnez-le.

  2. Cliquez avec le bouton droit de la souris sur le dossier iFolder, puis sur iFolder > Propriétés.


Accès aux propriétés iFolder à partir du navigateur de dossiers iFolder

  1. Dans la zone de notification, cliquez avec le bouton droit sur l'icône iFolder , puis cliquez sur Dossiers iFolder pour ouvrir le navigateur de dossiers iFolder.

  2. Dans la liste des dossiers iFolder, sélectionnez le dossier voulu, puis cliquez sur Propriétés.
