Démarrage du Client iFolder


Démarrage de iFolder pour Novell Linux Desktop 9

  1. Loguez-vous à votre ordinateur avec l'identité utilisateur que vous avez employée pour créer des dossiers iFolder.

  2. Démarrez iFolder en faisant appel à l'une des méthodes suivantes :

    • Ouvrez un shell terminal et entrez :

      /opt/novell/ifolder3/bin/ifolder

    • Si vous voulez démarrer iFolder automatiquement lorsque vous vous loguez, vérifiez qu'il est exécuté lorsque vous vous déloguez de votre session GNOME, puis enregistrez la session. Vous pouvez également ouvrir une fenêtre de terminal, vous loguer en tant qu'utilisateur racine, puis exécuter /opt/gnome/bin/propriétés-session-gnome pour ajouter iFolder comme programme à démarrer lorsque vous vous loguez à votre ordinateur.

  3. Si vous avez décoché la case Mémoriser le mot de passe pour votre compte iFolder, vous devez vous loguer au compte afin de synchroniser la liste des utilisateurs et les dossiers iFolder et de gérer ces dossiers.

    Pour plus d'informations, reportez-vous à Connexion à un compte iFolder.


Démarrage de iFolder pour Windows 2000/XP

  1. Loguez-vous à votre ordinateur avec l'identité utilisateur Windows que vous avez employée pour créer des dossiers iFolder.

  2. Si iFolder ne démarre pas automatiquement après que vous vous êtes connecté, lancez-le en cliquant sur Démarrer > Programmes > iFolder, puis sélectionnez l'application iFolder.

    iFolder ne démarre pas automatiquement si vous décochez la case Démarrer iFolder lors de la connexion au bureau dans les Préférences iFolder.

  3. Si vous avez décoché la case Mémoriser le mot de passe pour votre compte iFolder, vous devez vous loguer au compte afin de synchroniser la liste des utilisateurs et les dossiers iFolder et de gérer ces dossiers.

    Pour plus d'informations, reportez-vous à Connexion à un compte iFolder.


Démarrage de iFolder pour Macintosh OS X v10.3 et version ultérieure

  1. Loguez-vous à votre ordinateur avec l'identité utilisateur que vous avez employée pour créer des dossiers iFolder.

  2. Démarrez iFolder en faisant appel à l'une des méthodes suivantes :

    • Ouvrez le menu Applications et sélectionnez le sous-menu Internet, puis iFolder.
    • Pour démarrer iFolder à partir de la ligne de commande, ouvrez un shell terminal et entrez :

      /opt/novell/ifolder3/bin/ifolder

    • Si vous voulez démarrer iFolder automatiquement lorsque vous vous loguez à votre ordinateur Macintosh, ajoutez iFolder aux éléments de démarrage.

  3. Si vous avez décoché la case Mémoriser le mot de passe pour votre compte iFolder, vous devez vous loguer au compte afin de synchroniser la liste des utilisateurs et les dossiers iFolder et de gérer ces dossiers.

    Pour plus d'informations, reportez-vous à Connexion à un compte iFolder.
