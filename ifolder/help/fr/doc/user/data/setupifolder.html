Configuration d'un dossier iFolder disponible

Lorsque vous partagez des dossiers iFolder avec d'autres utilisateurs, ils apparaissent avec une icône indiquant que le dossier iFolder est disponible dans le navigateur des dossiers iFolder. Pour accéder au dossier iFolder partagé sur l'ordinateur local, configurez ce dossier pour qu'il soit actif sur l'ordinateur.

L'emplacement du dossier iFolder local doit respecter les directives suivantes :

  • Directives pour la localisation des dossiers iFolder
  • Conventions de dénomination des fichiers et des dossiers d'un compte iFolder

Pour configurer un dossier iFolder :

  1. Dans la zone de notification, cliquez avec le bouton droit sur l'icône iFolder , puis sélectionnez Dossiers iFolder pour ouvrir le navigateur de dossiers iFolder.

  2. Dans la liste des dossiers iFolder, sélectionnez le dossier voulu, puis cliquez sur Configurer .

  3. Si vous n'êtes pas logué au compte, loguez-vous avec votre nom d'utilisateur et votre mot de passe, puis cliquez sur OK.

  4. Naviguez jusqu'à l'emplacement où vous souhaitez créer un dossier iFolder local.

  5. Cliquez deux fois sur OK.
