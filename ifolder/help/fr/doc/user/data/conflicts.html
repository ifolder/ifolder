Résolution des conflits de fichiers

En cas de conflits entre des versions de fichiers enregistrés au même moment ou lorsque des fichiers ne présentent que des différences au niveau des lettres (majuscules ou minuscules), le fichier est signalé en tant que Conflit de noms et stocké dans la zone conflictuelle, puis une alerte est transmise pour signaler le fichier au client iFolder.


Conflits de versions

Avant de résoudre un conflit de versions, vous devez d'abord avertir les autres membres. Choisissez ensuite la version qui convient dans la boîte de dialogue Résoudre les conflits.

  1. En cas de conflit, un message contextuel s'affiche dans la zone de notification. Cliquez sur X pour fermer la fenêtre.

    Si vous ne souhaitez pas que ce message s'affiche, vous pouvez le spécifier dans l'onglet Général des Préférences iFolder.

  2. Dans la zone de notification, cliquez avec le bouton droit de la souris sur l'icône iFolder , puis cliquez sur Dossiers iFolder pour ouvrir le navigateur de dossiers iFolder.

  3. Sélectionnez le fichier à l'origine du conflit, puis cliquez sur Résoudre pour afficher la boîte de dialogue Résoudre les conflits.

  4. Consultez les informations relatives au fichier sur l'ordinateur local. Regardez sur le serveur quelle version du fichier vous souhaitez conserver , puis cliquez sur Enregistrer.

  5. Une fois que le conflit est résolu, cliquez sur OK.


Conflits de noms de fichier

Les conflits de noms de fichier se produisent lorsque la version locale du fichier et celle du serveur présentent des différences au niveau des majuscules/minuscules. Ces fichiers peuvent avoir le même contenu ou non. Vous pouvez résoudre ce type de conflit en renommant les fichiers tout en veillant à l'endroit où vous renommez le fichier.

  • Si le contenu est identique, vous pouvez renommer l'un des fichiers dans la boîte de dialogue Résoudre les conflits.
  • S'il est différent, veillez à créer un copie du fichier local en lui attribuant un nom différent, puis revenez à la boîte de dialogue Résoudre les conflits et enregistrez la version serveur du fichier. Les deux fichiers peuvent alors exister dans iFolder sous des noms différents. Si vous renommez une version du fichier dans la boîte de dialogue Résoudre les conflits, cette version remplacera l'autre (son contenu sera détruit).

  1. En cas de conflit, un message contextuel s'affiche dans la zone de notification. Cliquez sur X pour ignorer le message.

  2. Dans la zone de notification, cliquez avec le bouton droit de la souris sur l'icône iFolder , puis cliquez sur Dossiers iFolder pour ouvrir le navigateur de dossiers iFolder.

  3. Sélectionnez le fichier à l'origine du conflit, puis cliquez sur Résoudre pour afficher la boîte de dialogue Résoudre les conflits.

  4. Confirmez qu'un conflit de noms de fichier existe et déterminez si le contenu est identique.

  5. Effectuez l'une des opérations suivantes :

    • Si le contenu est identique, renommez l'un des fichiers et cliquez sur Enregistrer, puis sur OK pour fermer la boîte de dialogue Résoudre les conflits. Vous avez terminé.
    • Si le contenu est différent, passez à l'Etape 6.

  6. Fermez la boîte de dialogue Résoudre les conflits.

  7. Ouvrez un gestionnaire de fichiers, puis accédez à la copie locale du fichier à l'origine du conflit.

  8. Copiez le fichier et enregistrez-le sous un nom distinct dans iFolder.

    iFolder effectue la synchronisation du nouveau fichier avec le serveur et les ordinateurs membres.

  9. Retournez à la boîte de dialogue Résoudre les conflits, sélectionnez la version serveur du fichier. Cliquez sur Enregistrer, puis sur OK.

    La version serveur du fichier est téléchargée sur l'ordinateur local et remplace la copie locale par le fichier du même nom.
