Modification du mot de passe

Vous pouvez modifier le mot de passe de votre compte iFolder.

  1. Dans le navigateur iFolder, sélectionnez Sécurité > Changer le mot de passe.

  2. Dans la fenêtre Changer le mot de passe, sélectionnez le compte iFolder pour lequel vous souhaitez changer le mot de passe, puis spécifiez les détails suivants :

    Mot de passe actuel : spécifiez votre mot de passe actuel.

    Nouveau mot de passe : spécifiez le nouveau mot de passe à définir pour le compte iFolder actuel.

    Confirmer le nouveau mot de passe : retapez le nouveau mot de passe à des fins de confirmation.

    Mémoriser le mot de passe: Cochez la case Mémoriser le mot de passe pour vous loguer automatiquement au compte iFolder lorsque vous vous loguez à votre ordinateur.

  3. Cliquez sur Réinitialiser.

    REMARQUE :iFolder change le mot de passe directement sur le serveur LDAP. Si ce même serveur LDAP est une source de gestion des utilisateurs pour d'autres applications, le nouveau mot de passe s'applique également à ces applications. Une fois votre mot de passe modifié, la session iFolder expire et vous êtes délogué. Vous devez ensuite vous reloguer pour vous connecter aux services iFolder.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
