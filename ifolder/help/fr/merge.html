Fusion de dossiers iFolder

La fusion contribue à consolider les données iFolder et à minimiser le transfert de données entre serveur et client iFolder. Vous pouvez fusionner le contenu d'un dossier iFolder du serveur avec celui d'un dossier existant de la machine locale.

  • Fusion iFolder-dossier

  • Déplacement de dossiers iFolder existants

  • Fusion de dossiers iFolder 2.x migrés avec des dossiers iFolder 3.7 ou version ultérieure

Fusion iFolder-dossier

Vous pouvez fusionner un dossier iFolder et un dossier normal de même nom. Par exemple, vous pouvez fusionner le contenu d'un dossier iFolder nommé entreprise avec celui d'un dossier normal également nommé entreprise.

Supposons que le dossier iFolder entreprise a trois sous-dossiers : rh, finance et marketing. Vous souhaitez télécharger le contenu du dossier iFolder entreprise dans le dossier nommé entreprise qui ne contient que les dossiers rh et finance. Dans ce cas, vous ne devez pas télécharger les trois dossiers dans le dossier entreprise ni augmenter la vitesse de transfert. Au lieu de cela, vous pouvez fusionner le dossier iFolder entreprise dans le dossier entreprise de sorte que la fonction Fusionner dans le dossier évite intelligemment le téléchargement des sous-dossiers rh et finance et ne télécharge que le sous-dossier marketing. Si les données du dossier iFolder entreprise et du dossier entreprise diffèrent, un conflit de version s'affiche. Vous pouvez résoudre les conflits manuellement. De même, si un conflit de nom de fichier s'affiche, vous pouvez le résoudre manuellement.

Pour fusionner un dossier iFolder sur le serveur avec le contenu d'un dossier existant sur la machine locale :

  1. Dans le navigateur iFolder, cliquez avec le bouton droit sur le dossier iFolder désiré dans la liste de dossiers iFolder sur le serveur sélectionné, puis sélectionnez Fusionner dans le dossier.

  2. Recherchez et sélectionnez le dossier et cliquez sur OK.

Le contenu de ces deux dossiers est alors comparé et les fichiers qui ne sont pas présents dans le dossier sont téléchargés à partir du dossier iFolder. Les fichiers qui ne sont pas présents dans le dossier iFolder, mais disponible dans le dossier sont téléchargés dans le dossier iFolder.

Déplacement de dossiers iFolder existants

Pour déplacer un dossier iFolder existant nommé entreprise à un emplacement différent sur la machine cliente :

  1. Convertissez le dossier iFolder entreprise en un dossier normal.

    IMPORTANT :Vous ne devez pas supprimer le dossier iFolder du serveur.

    Pour plus d'informations, reportez-vous au Conversion d'un dossier iFolder en dossier normal.

  2. Déplacez le dossier entreprise.

    IMPORTANT :Vous ne devez pas changer le nom du dossier pendant son déplacement.

  3. Dans la fenêtre de navigation iFolder, sélectionnez le dossier iFolder entreprise sur le serveur, puis cliquez sur Fusionner dans le dossier.

  4. Sélectionnez le dossier entreprise sur la machine locale, puis cliquez sur OK pour fusionner le contenu.

    REMARQUE :Si vous utilisez la fonction Synchroniser maintenant (plutôt que Fusionner dans le dossier), comme indiqué à la Déplacement d'un dossier iFolder, toutes les données du dossier iFolder du serveur seront vérifiées et l'ensemble des données synchronisées, augmentant ainsi la durée de transfert de données. À l'inverse, la fonction Fusionner dans le dossier minimise le transfert de données dans la mesure où elle se limite à fusionner le contenu du dossier iFolder et du dossier normal.

    Pour plus d'informations sur la fusion, reportez-vous à la section Fusion iFolder-dossier.

Fusion de dossiers iFolder 2.x migrés avec des dossiers iFolder 3.7 ou version ultérieure

Vous pouvez fusionner un dossier iFolder 2.x migré avec un dossier iFolder créé avec un serveur iFolder 3.7 ou version ultérieure. Pour plus d'informations sur la migration, reportez-vous à la section Migration et mise à niveau de Novell iFolder.

Après migration, les dossiers iFolder 2.x sont déplacés sur le serveur iFolder 3.7 ou version ultérieure. Vous pouvez décider de télécharger sur la machine client les dossiers iFolder migrés sur la version 3.7 ou une version ultérieure du serveur. Cependant, il est possible que ces dossiers iFolder existent déjà sur la machine cliente. Si c'est le cas, utilisez la fonction de fusion pour éviter un transfert de données inutile du serveur au client.

Pour fusionner des dossiers iFolder 2.x migrés avec des dossiers iFolder 3.7 ou version ultérieure,

  1. migrez les dossiers iFolder 2.x dans le domaine iFolder 3.7 ou version ultérieure.

  2. Convertissez les dossiers iFolder 2.x de la machine locale en dossiers normaux.

  3. Sélectionnez la copie serveur de chaque dossier iFolder et cliquez sur Fusionner dans le dossier.

  4. Pour fusionner le contenu, sélectionnez le dossier normal et cliquez sur OK.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
