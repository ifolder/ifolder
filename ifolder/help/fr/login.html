Connexion à un compte iFolder

Vous pouvez à tout moment travailler en local sur les fichiers se trouvant dans les répertoires iFolder. Vous devez être logué à un compte iFolder pour synchroniser la liste des utilisateurs ainsi que les données iFolder, pour télécharger des dossiers iFolder nouveaux ou partagés, pour partager des dossiers iFolder et pour les gérer. Vous pouvez vous loguer séparément et être logué simultanément à plusieurs comptes.

Pour vous loguer à un compte, procédez de l'une des façons suivantes :

  • Se loguer automatiquement : activez l'option Mémoriser mon mot de passe pour votre compte iFolder si vous souhaitez vous loguer automatiquement au démarrage du client iFolder. Vous pouvez activer l'option Mémoriser le mot de passe sur cet ordinateur lors de la configuration du compte ou lors de la modification ultérieure des paramètres du compte.

    Lorsque votre mot de passe a été modifié, l'authentification automatique échoue tout simplement lors de votre prochaine tentative de login au compte iFolder. Lorsque iFolder vous invite à vous loguer, vous pouvez entrer le nouveau mot de passe et cocher de nouveau la case Mémoriser le mot de passe le cas échéant.

  • Se loguer comme souhaité : désactivez la case à cocher Mémoriser le mot de passe si vous souhaitez choisir la façon de vous loguer au compte iFolder.

Pour activer ou désactiver l'option Mémoriser le mot de passe pour un compte iFolder existant :

  1. Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, puis sélectionnez Comptes pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Comptes.

  2. Sélectionnez le compte iFolder, puis cliquez sur Propriétés pour afficher ses paramètres dans la boîte de dialogue Propriétés.

  3. Cliquez sur l'onglet Identité.

  4. Cochez (pour activer) ou désélectionnez (pour désactiver) la case Mémoriser le mot de passe.

  5. Entrez votre mot de passe dans le champ Mot de passe.

  6. Cliquez sur Fermer pour valider votre choix.

  7. Cliquez sur OK dans la boîte de dialogue Propriétés iFolder pour enregistrer les modifications et vous loguer au serveur.

Pour se loguer à un compte iFolder :

  1. Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, puis sélectionnez Comptes.

  2. Sélectionnez le compte iFolder, puis cochez la case En ligne en regard de celui-ci.

  3. Si le système vous demande un mot de passe, entrez-le, puis cliquez sur Connecter.

    Si l'option Mémoriser le mot de passe est activée pour le compte, vous ne devez pas entrer votre mot de passe sauf s'il a expiré.

    Si elle échoue, un message d'erreur indique la cause, par exemple, la saisie d'un mot de passe erroné.

  4. Une fois la connexion établie, fermez la boîte de dialogue Préférences iFolder.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
