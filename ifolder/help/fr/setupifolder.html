Téléchargement d'un dossier iFolder disponible

Lorsque vous partagez des dossiers iFolder avec d'autres membres, ces dossiers s'affichent avec l'icône iFolder Téléchargementsous Dossiers iFolder dans Nom du serveur iFolder dans votre navigateur iFolder, le Nom du serveur iFolder correspondant au nom du serveur pour ce compte. Pour participer au dossier iFolder partagé sur votre ordinateur, vous pouvez télécharger le dossier iFolder.

Lorsqu'un nouvel utilisateur s'ajoute à la liste de vos dossiers iFolder partagés, iFolder annonce l'arrivée d'un nouvel utilisateur partagé avec un message contextuel dans la zone Notification.

L'utilisateur partagé recevra également un message contextuel dans la zone Notification immédiatement après avoir terminé de télécharger le dossier iFolder partagé. Le message indique que <Nom de l'utilisateur qui a partagé le dossier iFolder> a rejoint les membres du dossier <Nom du dossier iFolder>. Dès qu'un nouvel utilisateur rejoint les membres partagés, tous les utilisateurs membres existants sont avertis de l'arrivée du nouveau membre partagé.

  1. Cliquez avec le bouton droit sur l'icône Services iFolder , puis cliquez sur Dossiers iFolder pour ouvrir le navigateur iFolder.

  2. Dans la liste des dossiers iFolder, sélectionnez le dossier iFolder disponible, puis cliquez sur Télécharger un dossier iFolder.

  3. Si vous n'êtes pas logué au compte, faites-le en entrant votre nom d'utilisateur et mot de passe, puis cliquez sur Connecter.

  4. Recherchez l'emplacement dans lequel vous souhaitez créer un dossier iFolder local, puis spécifiez le nom à utiliser localement pour ce dossier iFolder.

    L'emplacement et le nom du dossier doivent respecter les directives suivantes :

    • Directives sur l'emplacement et l'utilisation des dossiers iFolder

    • Conventions de dénomination des fichiers et des dossiers d'un compte iFolder

  5. Cliquez deux fois sur OK.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
