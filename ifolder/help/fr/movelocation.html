Déplacement d'un dossier iFolder

Une fois que vous avez créé un dossier iFolder, le chemin d'accès à celui-ci est statique. Vous ne pouvez pas déplacer un dossier iFolder de son emplacement d'origine vers un autre emplacement sur votre ordinateur et le conserver en tant que dossier iFolder. Si vous déplacez un dossier iFolder, le lien entre la base de données de synchronisation et le dossier que vous avez converti en dossier iFolder se rompt.

Pour déplacer un dossier iFolder sur l'ordinateur où vous l'avez créé :

  1. Dans un gestionnaire de fichiers ou dans le navigateur iFolder, sélectionnez le dossier iFolder à déplacer.

  2. Convertissez le dossier iFolder en dossier normal.

  3. Dans le gestionnaire de fichiers, déplacez le dossier vers un autre emplacement sur l'ordinateur.

  4. Rétablissez le dossier en dossier iFolder et téléchargez-le sur le serveur.

  5. (Facultatif) Partagez le dossier iFolder.

  6. Autorisez le dossier iFolder à synchroniser des fichiers avec le serveur iFolder.

Pour déplacer un dossier iFolder sur un ordinateur participant :

  1. Dans les dossiers iFolder, convertissez le dossier iFolder pour cesser de participer au dossier iFolder sur l'ordinateur actuel.

  2. Sélectionnez le dossier iFolder disponible, puis cliquez sur Télécharger un dossier iFolder.

  3. Spécifiez le chemin du nouvel emplacement du dossier iFolder.

  4. Autorisez le dossier iFolder à synchroniser des fichiers avec le serveur iFolder.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
