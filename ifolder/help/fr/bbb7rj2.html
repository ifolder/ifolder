Suppression d'une adhésion d'un dossier iFolder partagé

Pour supprimer une adhésion d'un dossier iFolder partagé téléchargé :

IMPORTANT :Les membres individuels d'un groupe LDAP ne sont pas autorisés à supprimer leur adhésion d'un dossier iFolder si ce dernier est partagé avec ce groupe LDAP.

  1. Dans le gestionnaire de fichiers ou dans le navigateur iFolder, localisez le dossier iFolder duquel vous souhaitez supprimer votre adhésion.

  2. Cliquez avec le bouton droit sur le dossier iFolder et sélectionnez Revenir à un dossier normal pour afficher la boîte de dialogue de confirmation.

  3. Dans la boîte de dialogue de confirmation, sélectionnez Supprimer également mon adhésion du dossier iFolder.

  4. Cliquez sur Oui pour confirmer.

    La copie locale du dossier iFolder est un dossier normal et n'est plus synchronisée avec le serveur.

Pour supprimer une adhésion d'un dossier iFolder partagé sur le serveur :

  1. Dans le navigateur iFolder, localisez le dossier iFolder duquel vous souhaitez supprimer votre adhésion.

  2. Cliquez avec le bouton droit sur le dossier iFolder et sélectionnez Supprimer mon adhésion.

  3. Cliquez sur OK pour confirmer votre opération.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
