Création et téléchargement d'un dossier iFolder

Vous pouvez créer un dossier iFolder en convertissant un dossier existant qui répond aux critères suivants :

  • Directives sur l'emplacement et l'utilisation des dossiers iFolder

  • Conventions de dénomination des fichiers et des dossiers d'un compte iFolder

Si le dossier n'existe pas, vous pouvez accéder à un emplacement et y créer un dossier dans le cadre du processus.

IMPORTANT :Vous ne pouvez partager que les dossiers iFolder ordinaires. Les dossiers iFolder codés ne peuvent pas être partagés.

Au sein de iFolder, vous pouvez configurer une structure de sous-répertoires adaptée à vos habitudes professionnelles ou personnelles. La structure de sous-répertoires est constante pour tous les dossiers iFolders membres. Chaque poste de travail peut spécifier un répertoire parent différent dans le cas d'un dossier iFolder partagé.

Lorsque vous dépassez la limite du nombre de dossiers iFolder qu'un utilisateur peut créer, iFolder affiche un message d'erreur. Par exemple, si votre administrateur a défini la stratégie Nombre de dossiers iFolder par utilisateur sur 10, lorsque vous tentez de créer le onzième dossier iFolder, une notification vous signale la violation de la stratégie.

Linux

Vous pouvez créer des dossiers iFolder en utilisant un gestionnaire de fichiers ou le navigateur iFolder, qui est disponible dans l'application cliente. Pour créer un dossier iFolder, vous pouvez procéder de l'une des façons suivantes :

  • Création de dossiers iFolder dans le navigateur de dossiers iFolder

  • Création de dossiers iFolder dans un gestionnaire de fichiers

Création de dossiers iFolder dans le navigateur de dossiers iFolder

  1. Dans la zone de notification, utilisez l'une des méthodes suivantes pour ouvrir le navigateur iFolder :

    • Cliquez sur l'icône Services iFolder .

    • Cliquez avec le bouton droit sur l'icône Services iFolder , puis cliquez sur Dossiers iFolder.

  2. Cliquez sur Créer un dossier iFolder.

  3. Sous Autres options, indiquez le compte iFolder à utiliser pour ce dossier iFolder.

    Si vous disposez de plusieurs comptes iFolder, le compte par défaut est affiché en premier. Pour sélectionner un autre compte, utilisez la liste déroulante.

  4. Dans le navigateur, accédez au répertoire dans lequel créer un dossier iFolder.

  5. Sélectionnez le dossier normal à convertir en dossier iFolder, puis effectuez l'une des opérations suivantes :

    • Si le dossier normal existe, sélectionnez-le, puis cliquez sur Ajouter.

    • Dans le cas contraire, cliquez sur Créer un dossier pour le créer, sélectionnez le nouveau dossier, puis cliquez sur Ajouter.

  6. Sélectionnez le Type de dossier iFolder parmi les options proposées. Vous pouvez décider de créer un dossier iFolder codé ou ordinaire.

    IMPORTANT :Vous n'êtes autorisé à créer des dossiers iFolder codés que si votre administrateur iFolder a défini votre stratégie de codage sur On (activé). Si ce n'est pas le cas, vous ne pouvez créer que des dossiers iFolder ordinaires et cette option est sélectionnée par défaut. Dans ce cas, les deux options de Type sont désactivées.

    • Sélectionnez Codage basé sur phrase secrète pour activer le codage du dossier iFolder sélectionné.

      Si vous n'avez pas défini la phrase secrète lors de la session de login et que vous sélectionnez Codage basé sur phrase secrète, un message vous invite à entrer la phrase secrète. Indiquez la phrase secrète dans boîte de dialogue Phrase secrète.

    • Sélectionnez Normal pour activer le partage du dossier iFolder sélectionné.

      iFolder indique la disponibilité de dossiers iFolder partagés au moyen d'un message contextuel dans la zone Notification. Les utilisateurs partagés peuvent cliquer sur le message pour installer le dossier iFolder tout de suite ou fermer le message pour télécharger le fichier ultérieurement.

  7. Sélectionnez Synchronisation sécurisée pour activer la synchronisation sécurisée des fichiers dans le dossier iFolder que vous créez.

    Vous pouvez activer un canal sécurisé pour les dossiers iFolder ordinaires et codés. Lorsque cette option est sélectionnée, iFolder vous permet d'utiliser un canal sécurisé pour effectuer la synchronisation entre iFolder sur le serveur et sur la machine locale.

    REMARQUE :Vous pouvez également activer la synchronisation sécurisée pour un dossier iFolder sélectionné sur votre machine en cliquant avec le bouton droit sur l'option iFolder > Propriétés dans la fenêtre principale de iFolder. Pour ce faire, cliquez avec le bouton droit sur iFolder et sélectionnez Propriétés, puis sous l'onglet Général, sélectionnez la case Synchronisation sécurisée.

  8. Cliquez sur OK.

    Le dossier iFolder apparaît dans la liste Dossiers iFolder sur cet ordinateur.

  9. Si le message Dossier iFolder créé apparaît, effectuez l'une des opérations suivantes :

    • Cliquez sur Fermer pour fermer le message.

    • Sélectionnez Ne plus afficher ce message pour désactiver les notifications utlérieures, puis cliquez sur Fermer pour fermer le message.

      Si, par la suite, vous souhaitez recevoir des messages de confirmation, allez dans Préférences iFolder, sélectionnez Afficher la boîte de dialogue de confirmation lors de la création de dossiers iFolder, puis cliquez sur OK.

Création de dossiers iFolder dans un gestionnaire de fichiers

  1. Accédez au dossier à convertir en dossier iFolder à l'aide du gestionnaire de fichiers, puis sélectionnez ce dossier.

  2. Pour cela, utilisez l'une des méthodes suivantes :

    • Cliquez avec le bouton droit sur l'icône de dossier, puis cliquez sur Convertir en dossier iFolder.

    • Dans la barre d'outils, cliquez sur Fichier > Convertir en dossier iFolder.

    L'icône de dossier se transforme en dossier marqué du symbole iFolder (« i » vert) .

  3. Si le message Dossier iFolder créé apparaît, effectuez l'une des opérations suivantes :

    • Cliquez sur Fermer pour fermer le message.

    • Sélectionnez Ne plus afficher ce message pour désactiver les notifications ultérieures, puis cliquez sur Fermer pour fermer le message.

      Si, par la suite, vous souhaitez recevoir des messages de confirmation, allez dans Préférences iFolder, sélectionnez Afficher la boîte de dialogue de confirmation lors de la création de dossiers iFolder, puis cliquez sur OK.

Création de dossiers iFolder sur un point de montage FAT32 (Linux)

Sous Linux, iFolder prend en charge la création et l'utilisation de dossiers iFolder sur un point de montage des systèmes de fichiers FAT32. Avant de pouvoir créer de nouveaux dossiers iFolder ou installer des dossiers iFolder partagés sur un volume FAT32 sur votre ordinateur fonctionnant sous Linux, l'utilisateur iFolder doit posséder le point de montage FAT32 dans lequel vous souhaitez stocker les dossiers iFolder. En outre, le fichier /etc/fstab doit explicitement identifier l'ID utilisateur local de l'utilisateur iFolder pour le point de montage.

IMPORTANT :Cette opération est obligatoire, car les systèmes de fichiers FAT32 ne prennent pas en charge les autorisations de fichiers. Tous les autres systèmes de fichiers ne prenant pas en charge les autorisations rencontrent le même problème.

Si vous disposez d'un système à démarrage double, vous pouvez accéder aux données contenues dans les dossiers iFolder comme s'il s'agissait de dossiers normaux à partir du système dans lequel le dossier n'est pas configuré en tant que dossier iFolder. Si vous avez configuré le dossier en tant que dossier iFolder sur les deux plates-formes, les données sont stockées dans deux dossiers iFolder distincts sur le serveur. Ceci utilise le double des ressources qui vous sont allouées sur le serveur.

Comme les systèmes de fichiers FAT32 ne prennent pas en charge les autorisations de systèmes de fichiers POSIX, vérifiez que l'utilisateur iFolder est le propriétaire local du point de montage. Modifiez le fichier /etc/fstab afin de définir le paramètre ID utilisateur (UID) de l'identité du login local de l'utilisateur pour l'utilisateur iFolder qui crée des dossiers iFolder sur ce point de montage.

  1. Loguez-vous à votre ordinateur en tant qu'utilisateur racine.

  2. Créez le répertoire où monter le système de fichiers FAT32. A l'invite de commande, tapez

    mkdir /home/username/fsmount

    Remplacez nom_utilisateur par le nom d'utilisateur de l'utilisateur souhaitant utiliser l'unité FAT32 pour des dossiers iFolder. Remplacez montagefs par le nom de répertoire que vous souhaitez utiliser comme point de montage. Ainsi, si le nom d'utilisateur est jdurand et le nom de répertoire fat32mntpt pour une unité ou une partition FAT32, tapez

    mkdir /home/jsmith/fat32mntpt

  3. Montez l'unité FAT32 au point de montage voulu. A l'invite de commande, tapez

    mount -t vfat /dev/hda1 /home/username/fsmount

    Remplacez /dev/hdal par le nom de l'unité ou de la partition du périphérique non-Linux. Remplacez nom d'utilisateur par le nom d'utilisateur de l'utilisateur iFolder. Remplacez fsmount par le nom de répertoire que vous souhaitez utiliser comme point de montage.

    Pour poursuivre l'exemple, l'emplacement du système de fichiers FAT32 (vfat) se situe sur le périphérique hda4 (/dev/hda4), le nom d'utilisateur est jdurand et le répertoire de point de montage est /home/jdurand/fat32mntpt, tapez

    mount -t vfat /dev/hda4 /home/jsmith/fat32mntpt

  4. Éditez le fichier /etc/fstab en ajoutant le paramètre de l'utilisateur iFolderuid=nom_utilisateur sur la ligne qui définit le point de montage FAT32.

    /dev/hda1  /home/username/fsmount  vfat  uid=username

    Toujours selon le même exemple, modifiez le fichier/etc/fstab en ajoutant uid=jdurand au point de montage :

    /dev/hda4  /home/jsmith/fat32mntpt  vfat  uid=jsmith

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
