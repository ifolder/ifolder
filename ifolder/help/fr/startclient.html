Démarrage du client iFolder

Lorsque iFolder s'exécute, l'icône Services iFolder apparaît dans la zone Notification de la barre des tâches. iFolder est intégré dans l'environnement de bureau. Le symbole de iFolder (un « i » vert) s'affiche sur les dossiers iFolder lorsque ceux-ci s'affichent dans un gestionnaire de fichiers, sur le Bureau ou dans le navigateur iFolder. Le dossier codé iFolder est indiqué par le symbole (dossier verrouillé).

IMPORTANT :La zone Notification de la barre des tâches doit être activée pour que l'icône Services iFolder apparaisse lorsque iFolder est en cours d'exécution.

Vous pouvez à tout moment travailler en local sur les fichiers se trouvant dans vos dossiers iFolder locaux. Lorsque le client iFolder est en cours d'exécution, il enregistre les changements que vous apportez aux données iFolder locales au fur et à mesure de votre travail. Si vous modifiez les données iFolder locales alors que iFolder n'est pas en cours d'exécution, iFolder identifie et consigne les différences au prochain démarrage.

Lorsque iFolder ne s'exécute pas, les dossiers iFolder apparaissent dans le gestionnaire de fichiers ou sur le Bureau. Il vous est possible d'accéder aux fichiers dans votre copie locale d'un dossier iFolder, mais pas de gérer les dossiers iFolder. En outre, vous devez vous loguer à un compte pour créer un dossier iFolder ou synchroniser ses fichiers.

REMARQUE :Lors de la création du compte, le login de l'utilisateur s'effectue via SSL. Lorsque vous êtes invité à accepter le certificat de serveur, vous devez le valider et l'accepter pour pouvoir établir une connexion.

Pour plus d'informations sur l'arrêt du client iFolder, reportez-vous à la section Fermeture du client iFolder.

Démarrage automatique de iFolder lors du login

Pour activer le démarrage automatique de iFolder lors du login, effectuez l'une des opérations suivantes :

  • Ouvrez le menu Applications, cliquez sur Plus d'applications, localisez et cliquez avec le bouton droit sur iFolder 3, puis sélectionnez Ajouter aux programmes de démarrage dans le menu.

  • Assurez-vous que iFolder est en cours d'exécution lorsque vous vous déloguez, puis enregistrez les paramètres de la session. Sélectionnez cette option de session lorsque vous vous loguez sur l'ordinateur.

  • Ouvrez une fenêtre de terminal, loguez-vous en tant qu'utilisateur root, puis exécutez /opt/gnome/bin/gnome-session-properties pour ajouter iFolder aux programmes qui démarrent lorsque vous vous loguez.

Pour désactiver le démarrage automatique de iFolder lors du login, effectuez l'une des opérations suivantes :

  • Ouvrez le menu Applications, localisez et cliquez avec le bouton droit sur iFolder 3, puis sélectionnez Supprimer des programmes de démarrage.

  • Assurez-vous que iFolder n'est pas en cours d'exécution lorsque vous vous déloguez, puis enregistrez les paramètres de la session. Sélectionnez cette option de session lorsque vous vous loguez sur l'ordinateur.

Démarrage manuel de iFolder

  1. Loguez-vous avec l'identité d'utilisateur Linux local que vous souhaitez utiliser lorsque vous créez des dossiers iFolder.

  2. Démarrez iFolder en faisant appel à l'une des méthodes suivantes :

    • Dans la barre des tâches, ouvrez le menu Applications, cliquez sur Plus d'applications, localisez iFolder 3 en lançant une recherche ou en faisant défiler les options disponibles, cliquez avec le bouton droit sur iFolder 3, puis sélectionnez Démarrer iFolder 3.

    • Ouvrez un shell terminal et entrez :

      /usr/bin/ifolder

Démarrage automatique de iFolder lors du login

Pour démarrer automatiquement iFolder lors du login sur l'ordinateur :

  1. Cliquez avec le bouton droit sur l'icône Services iFolder , puis sélectionnez Préférences pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Général.

  2. Sélectionnez Démarrer iFolder lors du login sur l'ordinateur, puis cliquez sur Appliquer.

  3. Cliquez sur OK pour fermer la boîte de dialogue Préférences iFolder.

Pour désactiver le démarrage automatique de iFolder lors du login sur l'ordinateur :

  1. Cliquez avec le bouton droit sur l'icône Services iFolder, puis sélectionnez Préférences pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Général.

  2. Désélectionnez Démarrer iFolder lors du login sur l'ordinateur, puis cliquez sur Appliquer.

  3. Cliquez sur OK pour fermer la boîte de dialogue Préférences iFolder.

Démarrage manuel de iFolder

  1. Loguez-vous avec l'identité d'utilisateur Windows local que vous souhaitez utiliser lorsque vous créez des dossiers iFolder.

  2. Si l'application iFolder ne démarre pas automatiquement après le login, démarrez-la en cliquant sur Démarrer > Programmes > iFolder 3, puis sélectionnez l'application Client iFolder 3.

Démarrage automatique de iFolder lors du login

Pour activer le démarrage automatique de iFolder lorsque vous vous loguez à votre ordinateur Macintosh, ajoutez iFolder aux Éléments de démarrage.

  1. Démarrez iFolder.

  2. À partir du dock, cliquez avec le bouton droit sur l'icône iFolder et sélectionnez l'option Ouvrir au login.

    ou

    À partir du dock, appuyez sur la touche Ctrl, cliquez sur l'icône iFolder, puis sélectionnez Ouvrir au login.

Pour désactiver le démarrage automatique de iFolder, supprimez iFolder des Éléments de démarrage.

  1. À partir du dock, cliquez avec le bouton droit sur l'icône iFolder et désélectionnez Ouvrir au login.

    ou

    À partir du dock, cliquez avec le bouton droit sur l'icône iFolder ou cliquez sur la touche Ctrl, puis désélectionnez Ouvrir au login.

Démarrage manuel de iFolder

  1. Loguez-vous avec l'identité d'utilisateur Macintosh local que vous souhaitez utiliser lorsque vous créez des dossiers iFolder.

  2. Démarrez iFolder en faisant appel à l'une des méthodes suivantes :

    • Ouvrez le répertoire Applications sur l'unité Macintosh HD, sélectionnez Applications, puis double-cliquez sur iFolder 3.

    • Tapez iFolder dans Spotlight (bouton Apple + barre d'espace ou cliquez sur l'icône de la loupe en haut à droite), puis cliquez sur le dossier iFolder 3 sous Applications.

    • Ouvrez un shell de terminal, loguez-vous en tant qu'utilisateur racine, puis entrez 

      /Applications/iFolder 3.app/Contents/MacOS/iFolder 3

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
