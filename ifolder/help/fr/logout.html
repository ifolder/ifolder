Déconnexion d'un compte iFolder

Déloguez-vous d'un compte iFolder lorsque vous souhaitez travailler hors ligne.

  • Macintosh

  • Linux

  • Windows

Macintosh

  1. Cliquez sur l'icône Services iFolder dans le dock, puis sélectionnez Comptes pour ouvrir la boîte de dialogue Préférences iFolder.

  2. Cliquez sur l'icône Comptes dans la barre d'outils pour afficher une liste des comptes iFolder.

  3. Localisez le compte que vous souhaitez gérer, puis cliquez sur Logout pour vous déconnecter du serveur iFolder.

Linux

  1. Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, puis sélectionnez Comptes pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Comptes.

  2. Localisez le compte à gérer, puis désélectionnez la case En ligne en regard de celui-ci pour le déconnecter du serveur iFolder.

  3. Cliquez sur Fermer pour fermer la boîte de dialogue Préférences iFolder.

Windows

  1. Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, puis sélectionnez Comptes pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Comptes.

  2. Localisez le compte à gérer, puis désélectionnez la case En ligne en regard de celui-ci.

  3. Cliquez sur Appliquer pour le déconnecter du serveur iFolder, puis cliquez sur OK pour fermer la boîte de dialogue Préférences iFolder.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
