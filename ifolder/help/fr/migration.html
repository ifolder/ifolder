Migration de iFolder 2.x vers iFolder 3.7 ou version ultérieure.

Cette section fournit les informations suivantes :

  • Présentation du processus de migration

  • Procédure de migration

Présentation du processus de migration

Novell iFolder permet une migration simple à partir du client iFolder 2.x vers le client iFolder 3.7 ou version ultérieure. L'assistant de migration fourni dans le client iFolder vous aide à migrer rapidement et aisément des données iFolder 2.x.

Au démarrage, l'Assistant de migration détecte l'installation de la version 2.x et vous invite à migrer vos données. Lors de la migration des données, il vous invite de nouveau à convertir les dossiers existants ou à faire une copie du dossier iFolder 2.x. Pour plus d'informations à ce sujet, reportez-vous à l'Étape 6.

iFolder vous invite à procéder à la resynchronisation des fichiers avec le serveur 3.7 ou version ultérieure et vous demande si vous devez conserver le client 2.x jusqu'à la fin de la synchronisation. Au cours de ce processus, seules les données sont migrées. Par conséquent, l'utilisateur doit définir la préférence dans le nouveau client.

Procédure de migration

  1. Installez le client iFolder pour le serveur iFolder 3.7 ou version ultérieure, puis configurez votre compte pour le serveur d'entreprise iFolder 3.7 ou version ultérieure.

    Pour plus d'informations sur l'installation du client et la configuration du compte, reportez-vous à la rubrique Gestion des comptes et des préférences iFolder.

  2. Dans le navigateur iFolder, créez un ou plusieurs dossiers iFolder sur votre ordinateur local.

    Vous pouvez utiliser n'importe quel nom valide pour un dossier iFolder. Ne créez pas de dossier iFolder dans le même chemin que votre répertoire iFolder 2.x actuel. Pour plus d'informations, reportez-vous à Création et téléchargement d'un dossier iFolder.

  3. iFolder affiche un message signalant l'existence de données d'une version précédente et vous invite à procéder à leur migration.

    Cliquez sur OK pour ouvrir une liste des dossiersiFolder 2.x disponibles.

  4. Sélectionnez le dossier iFolder 2.x à migrer.

    Cliquez sur Migrer pour afficher l'Assistant de migration.

    REMARQUE :Vous pouvez également accéder à l'assistant de migration via iFolder > Migration > Migrer de iFolder 2.x.

    L'Assistant de migration vous guide dans le processus de migration de vos données.

  5. Cliquez sur Suivant pour afficher les options de migration.

  6. Sélectionnez l'une des options suivantes :

    Migrer le dossier iFolder et le déconnecter du domaine 2.x : sélectionnez cette option pour déconnecter le dossier existant du dossier iFolder 2.x après l'avoir migré vers la version 3.7 ou une ultérieure.

    Créez une copie du dossier iFolder et effectuez la migration : sélectionnez cette option pour créer une copie du dossier iFolder 2.x et les convertir au format 3.7 ou format ultérieur. Les fichiers ne sont pas supprimés du domaine iFolder 2.x même après leur migration vers iFolder 3.7 ou version ultérieure.

    Si vous sélectionnez cette option, vous pouvez également cocher la case Copier le dossier parent pour copier non seulement le contenu du dossier 2.x, mais aussi le dossier parent. Si vous ne sélectionnez pas cette option, seul le contenu du dossier sélectionné sera migré vers iFolder 3.7 ou version ultérieure.

    Cliquez sur Parcourir pour trouver l'emplacement de destination du dossier migré.

    Par exemple, c:/Documents and Settings/<nom d'utilisateur>/ifolder2dir est un dossier iFolder 2.x, et dir1 et dir2 correspondent aux répertoires sous ifolder2dir. Si vous sélectionnez l'option Copier le répertoire parent, le dossier ifolder2dir est créé dans le nouvel emplacement et les répertoires dir1 et dir2 y sont copiés. Si cette option n'est pas sélectionnée, seuls les répertoiresdir1 et dir2 sont déplacés vers l'emplacement spécifié.

  7. Cliquez sur Suivant pour sélectionner le domaine et les options de sécurité comme suit :

    1. Sélectionnez le domaine dans la liste déroulante.

    2. Sélectionnez l'option de sécurité souhaitée :

      • Sélectionnez Codé pour coder le dossier iFolder sélectionné.

        Si vous n'avez pas défini votre phrase secrète lors du login, iFolder vous invite à l'entrer. Vous pouvez ignorer cette option et définir la phrase secrète ultérieurement.

        Si vous n'êtes pas actuellement connecté au domaine iFolder, vous êtes invité à le faire.

      • Sélectionnez Partagé pour partager le dossier iFolder avec d'autres membres dans le domaine iFolder

        Vous pouvez migrer un dossier iFolder 2.x codé vers le dossier partagé. Lorsque vous tentez de migrer un dossier iFolder 2.x codé vers un dossier partagé, vous êtes invité à vérifier l'opération.

        Cliquez sur OK pour confirmer votre opération.

  8. Cliquez sur Suivant pour migrer le dossier iFolder sélectionné.

  9. Vérifiez si les informations entrées sont correctes. Au besoin, cliquez sur Précédent pour retourner aux pages précédentes et apporter les corrections.

    Si les informations sont correctes, cliquez surMigrer pour lancer le processus de migration.

    Cliquez sur Précédent pour retourner aux pages précédentes et apporter les modifications nécessaires.

  10. Cliquez sur Terminer pour terminer le processus de migration.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
