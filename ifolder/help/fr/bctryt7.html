Mise à jour des clients iFolder 3.x

Cette section fournit les informations suivantes :

  • Mise à niveau automatique vers iFolder 3.7 ou versions ultérieures

  • Mise à niveau manuelle vers iFolder 3.7 ou versions ultérieures

IMPORTANT :Avant de mettre à niveau votre client 3.4.1, vous devez sauvegarder votre zone de stockage Simias locale. Si vous n'effectuez pas la sauvegarde, vous perdez toutes les informations de configuration de l'ordinateur car la banque Simias est effacée lorsque la mise à niveau échoue.

Mise à niveau automatique vers iFolder 3.7 ou versions ultérieures

La mise à niveau automatique n'est possible que si vous disposez des droits d'accès à la racine car le processus nécessite l'installation des progiciels RPM. Le mot de passe racine est exigé pendant la mise à niveau. Si vous ne disposez pas des droits d'accès à la racine, le progiciel n'est pas installé automatiquement. Les progiciels du client sont téléchargés dans un répertoire temporaire. Le client Linux est téléchargé dans /tmp/<32chardir>/ifolder3-linux.tar.gz. Cela vous permet d'effectuer la mise à niveau plus tard puisque l'emplacement contient deux scripts d'installation exécutables manuellement. Tant que vous n'avez pas mis à niveau le client, vous ne pouvez pas vous connecter au serveur iFolder 3.7 ou version ultérieure avec un client iFolder 3.x.

  1. À partir d'un client iFolder 3.x antérieur à la version 3.7, essayez de vous connecter au serveur iFolder. Vous êtes invité à effectuer une mise à niveau.

  2. Cliquez sur Oui pour mettre automatiquement à niveau votre client iFolder 3.x.

Dans le cas de iFolder 3.4.1, si vous cliquez sur Non et tentez de connecter le client iFolder 3.4.1 au serveur iFolder 3.7 ou version ultérieure, un message d'erreur s'affiche. Cliquez sur OK pour le faire disparaître.

Mise à niveau manuelle vers iFolder 3.7 ou versions ultérieures

  1. Cliquez sur Mettre à niveau dans le menu Aide. Vous êtes invité à effectuer une mise à niveau. Cliquez sur Accepter pour mettre à niveau votre client iFolder 3.x.

    REMARQUE :Lorsque vous cliquez sur l'option de mise à niveau, iFolder vérifie les mises à niveau pour tous les serveurs connectés.

    ou

  1. Ouvrez un terminal. À l’invite de commande, exécutez install-ifolder.sh.

  2. Entrez le mot de passe racine lorsque vous y êtes invité.

    Cette opération ferme le client iFolder s'il est déjà en cours d'exécution, puis met à jour le serveur Web XSP et le client iFolder avec le plug-in Nautilus-ifolder3.

  3. Démarrez iFolder à partir du menu Application.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
