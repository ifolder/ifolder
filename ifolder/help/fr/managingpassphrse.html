Gestion des phrases secrètes pour les dossiers iFolder codés

Chaque dossier iFolder dispose d'une clé de codage de données unique qui est générée automatiquement lors de la création du dossier iFolder. Cette clé est codée à l'aide d'une phrase secrète spécifiée par l'utilisateur et également à l'aide de la clé publique de l'agent de récupération.

L'agent de récupération est sélectionné au cours du processus de création du compte iFolder si l'administrateur iFolder a activé la stratégie de codage sur le serveur.

Si l'administrateur iFolder a activé le codage, une boîte de dialogue s'affiche pendant la création du compte iFolder afin que vous puissiez entrer votre phrase secrète. iFolder utilise la phrase secrète pour générer une clé de codage unique afin de coder ou de décoder les données iFolder sur le serveur iFolder. Si vous oubliez la phrase secrète, vous ne pouvez pas accéder aux données iFolder. Vous devez redéfinir votre phrase secrète pour accéder aux données iFolder.

Récupération d'un dossier iFolder codé

À l'aide de l'assistant de récupération de la phrase secrète, redéfinissez cette dernière et récupérez un dossier iFolder codé. Suivez la procédure ci-dessous pour lancer l'assistant de récupération de la phrase secrète :

  1. Dans le navigateur iFolder, cliquez sur Sécurité > Phrase secrète oubliée pour lancer l'assistant.

  2. Dans la liste Compte iFolder, sélectionnez le domaine pour lequel la phrase secrète doit être redéfinie et cliquez sur Suivant.

    Au cours du processus de création du compte iFolder, si l'option Server_Default a été sélectionnée en tant qu'agent de récupération, la page Définir une nouvelle phrase secrète s'affiche. Pour redéfinir la nouvelle phrase secrète, reportez-vous à la section Redéfinition de la phrase secrète lorsque l'agent de récupération est Server_Default.

    Par contre, si vous avez sélectionné un agent de récupération autre que Server_Default, vous devez redéfinir la phrase secrète comme décrit à la section Redéfinition de la phrase secrète lorsque l'agent de récupération n'est pas Server_Default.

Redéfinition de la phrase secrète lorsque l'agent de récupération est Server_Default

Pour définir la nouvelle phrase secrète, suivez la procédure ci-dessous :

  1. Sur la page Définir une nouvelle phrase secrète, spécifiez cette dernière dans le champ Nouvelle phrase secrète.

  2. Retapez la phrase secrète dans le champ Confirmer la phrase secrète > .

  3. Entrez votre mot de passe dans le champ Mot de passe, puis cliquez sur Suivant.

  4. Une fois la phrase secrète redéfinie, cliquez sur Terminer pour refermer l'assistant.

Redéfinition de la phrase secrète lorsque l'agent de récupération n'est pas Server_Default

Pour redéfinir la phrase secrète lorsque l'agent de récupération sélectionné n'est pas Server_Default, suivez la procédure ci-dessous :

  1. La page de bienvenue de l'assistant affiche des informations nécessaires à la redéfinition de la phrase secrète. Cliquez sur Suivant pour afficher la page du mécanisme de récupération de la phrase secrète.

  2. L'assistant propose trois méthodes permettant de redéfinir votre phrase secrète :

    • Utilisation du fichier secret et de son mot de passe : le fichier secret et son mot de passe sont utilisés pour redéfinir la phrase secrète. Si vous avez gardé le fichier secret et son mot de passe, vous pouvez utiliser l'assistant pour spécifier l'emplacement du fichier secret ainsi que son mot de passe. Pour redéfinir la phrase secrète à l'aide du fichier secret, voir la section Utilisation du fichier secret pour redéfinir la phrase secrète.

      REMARQUE :vous pouvez obtenir le fichier secret et son mot de passe auprès de l'administrateur iFolder.

    • Utilisation du nouveau fichier de données : le nouveau fichier de données peut également être utilisé pour redéfinir la phrase secrète. Vous pouvez obtenir le nouveau fichier de données auprès de l'administrateur après avoir exporté l'ancien fichier de données. Pour redéfinir la phrase secrète à l'aide du nouveau fichier de données, reportez-vous à la section Utilisation du nouveau fichier de données pour redéfinir la phrase secrète.

    • Exportation de l'ancien fichier de données : si vous ne disposez ni du fichier secret ni du nouveau fichier de données, vous pouvez utiliser l'assistant pour exporter l'ancien fichier de données et l'envoyer par message électronique à votre administrateur. Lorsque l'administrateur renvoie le nouveau fichier de données, vous pouvez utiliser l'assistant pour redéfinir votre phrase secrète. Pour exporter l'ancien fichier de données, reportez-vous à la section Utilisation d'un fichier exporté pour redéfinir la phrase secrète.

Utilisation du fichier secret pour redéfinir la phrase secrète

Pour redéfinir la phrase secrète à l'aide du fichier secret, suivez la procédure ci-dessous :

  1. Sélectionnez l'option Je dispose du fichier secret et de son mot de passe, puis cliquez sur Suivant.

  2. Renseignez les champs suivants :

    Emplacement du fichier secret : spécifiez l'emplacement du fichier secret ou cliquez sur le bouton Parcourir pour rechercher et sélectionner le fichier secret.

    Mot de passe du fichier secret : spécifiez le mot de passe du fichier secret.

    Nouvelle phrase secrète : spécifiez la nouvelle phrase secrète.

    Confirmer la phrase secrète retapez la nouvelle phrase secrète.

  3. Cliquez sur Suivant pour redéfinir la phrase secrète.

  4. Cliquez sur Terminer pour fermer l'assistant.

REMARQUE :vous pouvez obtenir le fichier secret et son mot de passe auprès de l'administrateur iFolder.

Utilisation du nouveau fichier de données pour redéfinir la phrase secrète

Pour redéfinir la phrase secrète à l'aide du nouveau fichier de données, suivez la procédure ci-dessous :

  1. Sélectionnez l'option Je dispose du nouveau fichier de données envoyé par l'administrateur, puis cliquez sur Suivant.

  2. Dans le champ Emplacement du nouveau fichier de données, spécifiez l'emplacement approprié ou cliquez sur le bouton Parcourir pour le rechercher et le sélectionner.

  3. Si le nouveau fichier de données que vous recevez de l'administrateur est codé, cochez la case Le fichier ci-dessus est-il codé ?. Lorsque vous cochez cette case, le champ Mot de passe à usage unique est activé.

  4. Dans le champ Mot de passe à usage unique, spécifiez le mot de passe utilisé par l'administrateur pour coder le nouveau fichier de données.

  5. Dans le champ Nouvelle phrase secrète, spécifiez la nouvelle phrase secrète.

  6. Dans le champ Confirmer la phrase secrète, retapez la nouvelle phrase secrète.

  7. Cliquez sur Suivant pour redéfinir la phrase secrète.

  8. Cliquez sur Terminer pour fermer l'assistant.

Utilisation d'un fichier exporté pour redéfinir la phrase secrète

Pour redéfinir la phrase secrète lorsque vous ne disposez pas du fichier secret ni du nouveau fichier de données :

  1. Sélectionnez l'option Je ne dispose d'aucun fichier secret ni de données, puis cliquez sur Suivant.

  2. Dans le champ Emplacement d'enregistrement de l'ancien fichier de données, spécifiez l'emplacement approprié ou cliquez sur le bouton Parcourir pour le définir.

  3. Cliquez sur Suivant pour obtenir l'ancien fichier de données. L'emplacement de l'ancien fichier de données et l'adresse électronique de l'administrateur iFolder s'affichent. Vous pouvez utiliser ces informations pour obtenir le nouveau fichier de données.

  4. Cliquez sur Terminer pour fermer l'assistant.

  5. Envoyez par message électronique l'ancien fichier de données à votre administrateur iFolder en lui demandant de vous renvoyer le nouveau fichier de données.

  6. Lorsque vous recevez le nouveau fichier de données, consultez la section Utilisation du nouveau fichier de données pour redéfinir la phrase secrète pour redéfinir votre phrase secrète.

Modification de la phrase secrète

Pour des raisons de sécurité, vous devez changer régulièrement votre phrase secrète. Vous pouvez redéfinir la phrase secrète en utilisant la phrase secrète à usage unique envoyée par votre agent de récupération.

  1. Dans le navigateur iFolder, sélectionnez Sécurité > Changer la phrase secrète.

  2. Dans la fenêtre Changer la phrase secrète, sélectionnez le compte iFolder pour lequel vous souhaitez redéfinir la phrase secrète, puis spécifiez les détails suivants :

    Entrer la phrase secrète : spécifiez votre phrase secrète actuelle.

    Entrer la nouvelle phrase secrète : spécifiez la nouvelle phrase secrète à définir pour le compte iFolder actuel.

    Retaper la phrase secrète : retapez la nouvelle phrase secrète à des fins de confirmation.

    Agent de récupération : Sélectionnez l'agent de récupération que vous avez défini pour le compte iFolder actuel.

  3. Cliquez sur Réinitialiser.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
