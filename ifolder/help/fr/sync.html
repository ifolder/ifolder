Synchronisation des fichiers sur demande

  1. Dans la zone Notification, cliquez avec le bouton droit sur l'icône Services iFolder , puis cliquez sur les dossiers iFolder pour ouvrir le navigateur iFolder.

  2. Dans la liste des dossiers iFolder, sélectionnez le dossier iFolder, puis cliquez sur Synchroniser maintenant.

Configuration de l'intervalle de synchronisation

  1. Dans la zone Notification, cliquez avec le bouton droit sur l'icône Services iFolder , puis sélectionnez Préférences pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Général.

  2. Effectuez l'une des opérations suivantes :

    • Sélectionnez Synchroniser pour activer la synchronisation en arrière-plan.

    • Désélectionnez Synchroniser pour désactiver la synchronisation.

  3. Si vous activez la synchronisation, spécifiez l'intervalle de synchronisation des fichiers, par exemple toutes les heures ou tous les deux jours.

    Lorsque le client iFolder est utilisé avec Novell iFolder, l'intervalle de synchronisation minimum du système (en général, toutes les 5 minutes) est configuré par l'administrateur iFolder. Vous ne pouvez pas remplacer l'intervalle de synchronisation minimum pour le système. L'intervalle de synchronisation minimum absolu pour le système est de 1 minute.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
