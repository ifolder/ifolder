Coexistence de clients Novell iFolder 2.x et iFolder

Le client iFolder pour Novell iFolder 3.7 ou versions ultérieures et le client Novell iFolder 2.x peuvent coexister sur le même ordinateur, mais ils ne sont pas intégrés et leurs logiciels ne doivent pas être installés dans le même dossier. Si vous envisagez d'utiliser le client Novell iFolder 2.x sur votre ordinateur, vous devez respecter les restrictions suivantes :

  • Les répertoires locaux dans lesquels vous stockez vos données Novell iFolder 2.x ne doivent pas être convertis en dossiers iFolder.

  • Les répertoires contenant un répertoire local dans lequel vous stockez vos données Novell iFolder 2.x ne doivent pas être convertis en dossiers iFolder.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
