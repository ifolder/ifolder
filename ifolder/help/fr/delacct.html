Suppression d'un compte iFolder

Vous pouvez décider de supprimer un compte uniquement de l'ordinateur utilisé ou de supprimer aussi vos dossiers iFolder du serveur.

Si vous supprimez un compte iFolder uniquement de votre ordinateur :

  • Les dossiers iFolder locaux de ce compte sont transformés en dossiers normaux. Faites appel à un gestionnaire de fichiers pour localiser et supprimer la copie locale des données si elle n'est pas requise.

  • Vous pouvez continuer à accéder aux dossiers iFolder à partir d'autres ordinateurs avec le client iFolder.

  • Lorsque le client iFolder est utilisé avec un serveur d'entreprise Novell iFolder 3.7 ou version ultérieure, vous pouvez également accéder aux fichiers à partir de n'importe quel endroit avec Novell iFolder 3.7 Web Access ou une version ultérieure.

Si vous supprimez votre compte iFolder du serveur :

  • Tous les dossiers iFolder que vous possédez ne sont pas partagés.

  • Les copies locales des dossiers iFolder enregistrées sur les ordinateurs membres sont transformées en dossiers normaux. Faites appel à un gestionnaire de fichiers pour localiser et supprimer la copie locale des données si elle n'est pas requise.

  • Le dossier iFolder et son contenu sont supprimés du serveur.

  • Vous êtes supprimé du système comme membre des dossiers iFolder que d'autres utilisateurs partagent avec vous.

  • Le compte est supprimé de l'ordinateur local.

Si vous décidez d'utiliser à nouveau ce compte sur cet ordinateur ou sur un autre ordinateur, vous devez configurer ce compte et ses dossiers iFolder comme vous l'avez fait lorsque vous l'avez créé.

Pour supprimer un compte iFolder :

  1. Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, puis sélectionnez Paramètres du compte pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Comptes.

  2. Sélectionnez le compte iFolder à supprimer, puis cliquez sur Supprimer.

  3. Un message vous invite à déterminer la portée de la suppression.

    Effectuez l'une des opérations suivantes :

    • Supprimez le compte iFolder uniquement de l'ordinateur sur lequel vous travaillez : Assurez-vous que la case Supprimer mes dossiers iFolder et mes fichiers du serveur est déselectionnée, puis cliquez sur Oui.

    • Supprimez le compte iFolder et vos dossiers et fichiers iFolder du serveur : Cochez la case Supprimer mes dossiers iFolder et mes fichiers du serveur, puis cliquez sur Oui.

    • Annulez la suppression : Cliquez sur Non pour annuler la suppression. Votre compte et ses dossiers iFolder restent intacts.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
