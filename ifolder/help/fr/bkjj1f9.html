Navigateur iFolder

Le navigateur iFolder vous permet de gérer vos dossiers iFolder. Le tableau ci-dessous synthétise les éléments du navigateur iFolder :

Tableau 2 Éléments du navigateur iFolder

Nom de l'élément

Description

Créer un dossier iFolder

Vous permet de convertir un dossier se trouvant sur la machine locale en un dossier iFolder et de le télécharger sur le serveur iFolder auquel vous êtes connecté.

Pour plus d'informations, reportez-vous à la Création et téléchargement d'un dossier iFolder

Synchroniser maintenant

Vous permet d'initier immédiatement une synchronisation des dossiers iFolder entre la machine locale et le serveur.

Partager avec

Vous permet de partager votre dossier iFolder avec d'autres utilisateurs.

Pour plus d'informations, reportez-vous à la Partage d'un iFolder

Rétablir en dossier normal

Vous permet de rétablir le dossier iFolder en dossier normal. Vous pouvez également supprimer le dossier iFolder du serveur. Pour plus d'informations, reportez-vous à la Conversion d'un dossier iFolder en dossier normal

Serveur

Affiche l'adresse IP du serveur iFolder auquel vous êtes connecté.

Nom

Affiche le nom du dossier iFolder sélectionné.

Propriétaire

Affiche le nom du propriétaire du dossier iFolder sélectionné.

Droits

Affiche les droits dont dispose le propriétaire du dossier iFolder.

Fichiers/dossiers à synchroniser

Indique le nombre de fichiers ou de dossiers en attente de synchronisation.

Heure de la dernière synchronisation réussie

Indique l'heure et la date de la dernière synchronisation réussie.

Télécharger un dossier iFolder

Vous permet de télécharger un dossier iFolder du serveur iFolder sur votre machine locale.

Pour plus d'informations, reportez-vous à la Téléchargement d'un dossier iFolder disponible

Type de dossier iFolder

Indique si le dossier iFolder sélectionné est codé ou ordinaire.

Taille du dossier iFolder

Affiche la taille du dossier iFolder sélectionné.

Fusionner

La fusion contribue à consolider les données iFolder et à minimiser le transfert de données entre serveur et client iFolder. Vous pouvez fusionner le contenu d'un dossier iFolder du serveur avec celui d'un dossier existant de la machine locale. Pour plus d'informations, reportez-vous à la Fusion de dossiers iFolder

Supprimer du serveur

Vous permet de supprimer le dossier iFolder sélectionné du serveur.

Liste des comptes

Affiche la liste des serveurs sur lesquels un compte est configuré.

Se connecter/Se déconnecter

Vous permet de vous connecter ou de vous déconnecter d'un serveur iFolder. Si vous êtes connecté au serveur iFolder, le bouton Se déconnecter s'affiche et si vous n'êtes pas connecté à un serveur iFolder, le bouton Se connecter apparaît.

Résoudre

Vous permet de résoudre les conflits de fichiers. Pour résoudre les conflits de version de fichier, vous devez sélectionner la version correcte dans la boîte de dialogue Résolution des conflits. Pour plus d'informations, reportez-vous à la Résolution des conflits de fichiers

REMARQUE :En ce qui concerne les clients Macintosh, vous devez mettre à jour manuellement le nom du fichier ou du dossier afin de résoudre les conflits de noms. Veillez à ce qu'il ne comporte aucun caractère non pris en charge ( \ / : * ? " < > | ; ).

Utilisateur

Affiche le nom de l'utilisateur logué au serveur iFolder sélectionné.

Nombre de dossiers iFolder

Affiche le nombre total de dossiers iFolder sur le serveur iFolder et la machine locale pour le serveur iFolder sélectionné.

Quota de disque

Affiche les valeurs des champs Utilisé et Disponible.

Utilisé

Affiche l'espace disque total occupé sur le serveur par l'ensemble des fichiers et des répertoires sur le serveur iFolder sélectionné.

Disponible

Affiche l'espace disque total disponible sur le serveur pour les fichiers ou les répertoires sur le serveur iFolder sélectionné.

Synchronisation automatique

Affiche l'intervalle de temps entre deux cycles de synchronisation automatique consécutifs.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
