Chiusura di una sessione del client iFolder

Per chiudere il client iFolder, eseguire una delle azioni indicate di seguito:

  • Fare clic con il pulsante destro del mouse sull'icona Servizi iFolder nell'area di notifica, quindi scegliere Esci.

  • Fare clic su iFolder > Esci

    Quando si seleziona Esci, viene visualizzato un messaggio di conferma. Fare clic su Sì per confermare l'azione.

La sincronizzazione dei file con il conto del server iFolder viene eseguita solo se si è connessi al server. Per interrompere la sincronizzazione, eseguire il logout dal conto. iFolder continua l'esecuzione in background in modalità non in linea. Per ulteriori informazioni, vedere il Logout da un conto iFolder.

È possibile disporre di più conti, ciascuno creato in un server host (dominio) diverso.

Per ulteriori informazioni, vedere Note legali.
