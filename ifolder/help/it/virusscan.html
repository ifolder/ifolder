Configurazione delle impostazioni del software antivirus locale per il traffico iFolder

Se si utilizza software di scansione antivirus nel computer, escludere la directory ..\simias\WorkArea\ dalla scansione. Ad esempio, l’ubicazione della directory ..\simias\WorkArea\ in Windows è c:\Documents and Settings\nomeutente\Impostazioni locali\Dati applicazioni\simias\WorkArea\.simias.wf.guid.

La directory ..\simias\WorkArea\ è l'ubicazione in cui iFolder memorizza provvisoriamente i file per lo scaricamento dal server. Se un file iFolder contiene un virus e il software di scansione antivirus lo rileva mentre il file è memorizzato nella directory ..\simias\WorkArea, verrà visualizzato un messaggio popup del software antivirus che indica la presenza del virus e il file verrà posto in quarantena, forzando un nuovo tentativo di sincronizzazione da parte di iFolder.

Per consentire il completamento della sincronizzazione, è necessario pertanto evitare di eseguire la scansione della cartella WorkArea. Il file infetto verrà così scaricato e spostato nella cartella iFolder locale. Il software antivirus rileverà quindi la presenza del file infetto nella cartella iFolder e lo metterà in quarantena, proteggendo così il computer dall’infezione. iFolder rileverà la cancellazione del file e alla successiva sincronizzazione rimuoverà il file dal server e dai computer membri.

Tuttavia, se la cartella iFolder in cui il file viene scaricato è di sola lettura, la cancellazione locale non verrà eseguita sul server e sugli altri computer membri. Il virus dovrà essere rilevato in un’altra postazione in cui le autorizzazioni consentono la sincronizzazione delle modifiche. Poiché in genere il software antivirus dell'utente visualizza un avviso di presenza di virus, è compito dell'utente informare gli altri membri.

Per ulteriori informazioni, vedere Note legali.
