Annullamento della conversione di una cartella normale in cartella iFolder

Qualsiasi membro può annullare la conversione in cartella iFolder. Una volta annullata la conversione, la relazione di condivisione di collaborazione rimane per tutti i membri iFolder, ma la cartella non verrà più sincronizzata su questo computer. La cartella iFolder e i suoi contenuti non vengono cancellati dalle workstation partecipanti.

  1. In un manager di file o sul desktop individuare la cartella iFolder che si desidera ripristinare in cartella normale.

  2. Effettuare una delle seguenti operazioni:

    • Fare clic con il pulsante destro del mouse sulla cartella iFolder, quindi selezionare iFolder > Ripristina cartella normale.
    • Selezionare la cartella iFolder, quindi sulla barra degli strumenti del manager di file fare clic su File > iFolder > Ripristina cartella normale.

    L’icona della cartella iFolder viene modificata in icona di cartella normale. I dati rimangono invariati su tutti i computer membri.
