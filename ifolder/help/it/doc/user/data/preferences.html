Configurazione delle Preferenze di iFolder per il client

  1. Fare clic con il pulsante destro del mouse sull'icona dell'applicazione iFolder nell'area di notifica e scegliere Preferenze per aprire la scheda Generale della finestra di dialogo Preferenze di iFolder.


  2. Specificare i valori seguenti:

    Parametro Descrizione

    Avvia iFolder al login al desktop

    Selezionare questa opzione per avviare iFolder ogni volta che si esegue il login con il nome utente locale utilizzato per configurare il conto iFolder. Per gli account per i quali è stata abilitata l'opzione Memorizza parola d'ordine, la sincronizzazione viene eseguita in background in base all'intervallo di sincronizzazione impostato.

    Deselezionare l'opzione per avviare iFolder manualmente.

    Mostra finestra di conferma per la creazione di cartelle iFolder

    Selezionare questa opzione per consentire la visualizzazione di un messaggio di conferma quando viene creata una cartella iFolder, in aggiunta alle consuete indicazioni, ad esempio l'inserimento della cartella nell'elenco delle cartelle iFolder.

    Deselezionare l'opzione per impedire la visualizzazione del messaggio di conferma.

    Notifica cartelle iFolder condivise

    Selezionare questa opzione per abilitare la visualizzazione di un messaggio popup nell'area di notifica quando si riceve un nuovo invito a condividere una cartella iFolder.

    La frequenza di recupero degli inviti dipende dall'intervallo di sincronizzazione specificato nelle Preferenze di iFolder. L'intervallo consente di controllare la sincronizzazione dei file, degli inviti alla condivisione di cartelle iFolder e dell'elenco degli utenti. È inoltre possibile fare clic su Aggiorna per recuperare gli inviti su richiesta.

    Deselezionare l'opzione per disabilitare la visualizzazione del messaggio popup relativo agli inviti.

    Notifica di conflitti

    Selezionare questa opzione per abilitare la visualizzazione di un messaggio popup nell'area di notifica se si verifica un conflitto durante la sincronizzazione dei file.

    Deselezionare l'opzione per disabilitare la visualizzazione del messaggio popup relativo ai conflitti di sincronizzazione.

    Notifica adesione utente

    Selezionare questa opzione per abilitare la visualizzazione di un messaggio popup nell'area di notifica quando un utente accetta un invito alla condivisione di una cartella iFolder. L'accettazione è vincolata all'effettiva configurazione di una cartella iFolder nel computer dell'utente.

    Deselezionare l'opzione per disabilitare la visualizzazione del messaggio popup relativo alle conferme di condivisione degli utenti.

    Sincronizzazione

    Sincronizza cartelle iFolder automaticamente: selezionare questa casella di controllo per abilitare la sincronizzazione di tutte le cartelle iFolder presenti nel computer con l'identità di login/accesso locale attuale. Deselezionare la casella di controllo per disabilitare la sincronizzazione.

    Sincronizza cartelle iFolder ogni: specificare l'intervallo minimo per la sincronizzazione delle cartelle iFolder disponibili nel computer con l'identità di login/accesso locale attuale. Specificare il valore e le unità.

    Se si utilizza il client iFolder con Novell iFolder 3.x, l'intervallo minimo di sincronizzazione effettivo corrisponde sempre al valore maggiore tra quelli delle impostazioni seguenti:

    • Le norme del sistema (valore di default uguale a zero), a meno che non siano impostate norme utente. In questo caso, le norme utente hanno priorità su quelle di sistema indipendentemente dal fatto che il relativo valore sia maggiore o minore.
    • Le norme del computer locale oppure l'impostazione nel computer client che esegue la sincronizzazione con il server.
    • Le norme di iFolder (raccolta).

  3. Al termine, fare clic su Applica per rendere effettive le modifiche.

  4. Fare clic su OK per chiudere la finestra di dialogo Preferenze di iFolder.
