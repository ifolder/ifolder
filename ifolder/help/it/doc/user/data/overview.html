Panoramica su iFolder

iFolderTM è un'applicazione per la condivisione dei file mediante client Linux*, Windows* e Macintosh*. È possibile condividere i file in più cartelle iFolder a ognuna delle quali è associato un diverso gruppo di utenti e controllare la partecipazione alla cartella iFolder da parte di altri utenti e i livelli di accesso per ciascun membro. È inoltre possibile partecipare alle cartelle iFolder condivise di altri utenti.
