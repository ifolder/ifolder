Configurazione delle impostazioni del software antivirus locale per il traffico iFolder

Se nel computer viene utilizzato un software antivirus, escludere la directory ..\simias\WorkArea\ dal controllo antivirus. Ad esempio, l’ubicazione della directory ..\simias\WorkArea\ in Windows è c:\Documents and Settings\nomeutente\Impostazioni locali\Dati applicazioni\simias\WorkArea\.simias.wf.guid.

La directory ..\simias\WorkArea\ è l’ubicazione in cui iFolder memorizza provvisoriamente i file per lo scaricamento dal server. Se un file iFolder contiene un virus e il software antivirus effettua la scansione del virus mentre il file si trova nella directory ..\simias\WorkArea, verrà visualizzato un messaggio di rilevamento virus, il file verrà messo in quarantena, quindi iFolder tenterà nuovamente di sincronizzare il file.

Se la directory WorkArea non viene sottoposta a scansione, sarà possibile completare la sincronizzazione e il file infetto verrà scaricato e spostato nella cartella iFolder locale. Il software antivirus rileverà quindi la presenza del file infetto nella cartella iFolder e lo metterà in quarantena, proteggendo così il computer dall’infezione. iFolder rileverà la cancellazione del file e alla successiva sincronizzazione rimuoverà il file dal server e dai computer membri.

Tuttavia, se la cartella iFolder in cui il file viene scaricato è di sola lettura, la cancellazione locale non verrà eseguita sul server e sugli altri computer membri. Il virus dovrà essere rilevato in un’altra postazione in cui le autorizzazioni consentono la sincronizzazione delle modifiche. Probabilmente il software antivirus dell’utente visualizzerà un avviso di presenza di virus e quindi l’utente ne potrà informare gli altri membri.
