Indicazioni per l'ubicazione delle cartelle iFolder

In genere, è possibile collocare le cartelle iFolder in qualsiasi ubicazione del disco rigido locale ogni volta che si crea una cartella iFolder o che se ne configura una condivisa da altri con l’utente. Tuttavia esistono alcune limitazioni pratiche e specifiche. Per l’ubicazione delle cartelle iFolder sul computer in uso tenere presente le seguenti indicazioni:

  • Il nome di una cartella iFolder deve essere univoco nella directory in cui viene collocata la cartella.
  • Il nome di una cartella iFolder non può contenere i seguenti caratteri non validi: 
    \/:*?"<>|;

    Per altre convenzioni di denominazione, vedere Convenzioni di denominazione per le cartelle iFolder e i relativi file e cartelle.

  • La cartella iFolder deve risiedere su un disco rigido locale; l’ubicazione non può trovarsi su un’unità di rete o non fisica.
  • Una cartella iFolder non può trovarsi alla radice dell’unità.
  • Una cartella iFolder può essere creata solo da una cartella esistente.
  • Una cartella iFolder può essere creata solo da una cartella per la quale si dispone dei diritti di accesso in lettura e scrittura dei file. Quando si configura una cartella iFolder condivisa su un computer, è necessario disporre dei diritti di accesso alla cartella in cui si colloca la cartella iFolder.
  • Una cartella iFolder non può contenere o trovarsi in una cartella di sistema, ad esempio la cartella di Windows o un’unità di sistema Linux o Mac.
  • Una cartella iFolder non può contenere o trovarsi in una cartella di applicazioni, ad esempio la cartella Programmi.
  • La cartella iFolder non può contenere o trovarsi nella cartella dei metadati iFolder. L’ubicazione di default della cartella di dati è la seguente:

    • Linux: nella cartella home nella sottodirectory .local\share

    • Windows: nella directory C:\Documents and Settings\nomeutente_accesso_Windows\Impostazioni locali\Dati applicazioni\simias\

    • Macintosh: nella cartella home nella sottodirectory .local\share

  • Una cartella iFolder non può contenere o trovarsi in altre cartelle iFolder, compresa la cartella iFolder di Novell® iFolder® 2.1x.
