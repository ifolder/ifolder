Visualizzare e nascondere le cartelle iFolder disponibili

È possibile scegliere quali cartelle iFolder scaricare in ciascun computer. Se esistono cartelle iFolder disponibili nel server ma non ancora scaricate, queste sono elencate in Nome server del relativo conto nel browser delle cartelle iFolder. Nome server è il nome descrittivo del server specificato dall'amministratore in cui si dispone di un conto iFolder.

Per visualizzare le cartelle iFolder disponibili nel browser delle cartelle iFolder, fare clic su Visualizza cartelle iFolder disponibili.

Per nascondere le cartelle iFolder disponibili nel browser delle cartelle iFolder, fare clic su Nascondi cartelle iFolder disponibili.

Per ulteriori informazioni, vedere Note legali.
