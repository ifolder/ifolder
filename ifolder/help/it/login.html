Login a un conto iFolder

È possibile utilizzare i file memorizzati nelle directory iFolder locali in qualunque momento. È necessario avere eseguito il login a un conto iFolder per potere sincronizzare l'elenco utenti e i dati iFolder, caricare nuove cartelle iFolder, scaricare le cartelle iFolder condivise, condividere le cartelle iFolder e gestirle. È possibile eseguire il login separatamente ed essere collegati simultaneamente a più conti.

Per ogni conto, eseguire una delle operazioni seguenti:

  • Esecuzione automatica del login: abilitare Memorizza password per il conto iFolder se si desidera eseguire automaticamente il login ad ogni avvio del client iFolder. È possibile abilitare Memorizza la password in questo computer quando si configura il conto o modificando le relative impostazioni in un secondo momento.

    Ogni volta che viene modificata la password, l'autenticazione automatica non viene eseguita correttamente al successivo tentativo di eseguire il login al conto iFolder. Se iFolder richiede di eseguire il login, è possibile immettere la nuova password e selezionare di nuovo la casella di controllo Memorizza password, se lo si desidera.

  • Login in base alle necessità: Deselezionare la casella di controllo Memorizza password se si desidera eseguire il login a un conto iFolder solo in base alle necessità.

Per abilitare o disabilitare la funzione Memorizza password per un conto iFolder esistente:

  1. Fare clic con il pulsante destro del mouse sull'icona Servizi iFolder nell'area di notifica e scegliere Conti per aprire la scheda Conti della finestra di dialogo Preferenze di iFolder.

  2. Selezionare il conto iFolder, quindi fare clic su Proprietà per visualizzarne le impostazioni nella finestra di dialogo Proprietà.

  3. Fare clic sulla scheda Identità.

  4. Selezionare (abilitare) o deselezionare (disabilitare) Memorizza password.

  5. Fornire la password nel campo Password.

  6. Fare clic su Chiudi per completare le operazioni.

  7. Fare clic su Chiudi nella finestra di dialogo Proprietà iFolder per salvare le modifiche ed eseguire il login al server.

Per eseguire il login a un conto iFolder:

  1. Fare clic con il pulsante destro del mouse sull'icona Servizi iFolder nell'area di notifica e scegliere Conti.

  2. Selezionare il conto iFolder, quindi selezionare la casella di controllo In linea accanto al conto.

  3. Se viene chiesto di immettere una password, specificarla e fare clic su Connetti.

    Se l'opzione Memorizza password è abilitata per il conto, non viene richiesto di immettere una password a meno che questa non sia scaduta.

    In caso contrario, viene visualizzato un messaggio di errore che ne indica la causa, ad esempio un errore nella password.

  4. Dopo avere eseguito la connessione, chiudere la finestra di dialogo Preferenze di iFolder.
