Impostazione di una quota iFolder

Non è possibile impostare una quota per le cartelle iFolder di cui si è proprietari. L'amministratore di iFolder imposta la quota per l'account utente. L'amministratore può impostare la quota a quattro livelli differenti: a livello di sistema, di gruppo LDAP, di utente e di cartella iFolder.

Per ulteriori dettagli sulla visualizzazione delle proprietà delle quote impostate per il conto utente, vedere quanto segue:

  • Quota

  • Parametri di memorizzazione per le cartelle iFolder memorizzate nel server iFolder

Se l'amministratore abilita una quota di sistema con valore inferiore rispetto allo spazio totale dell'utente attuale per i dati della cartella iFolder, la sincronizzazione dei dati delle quote dell'utente verrà interrotta finché i dati non rientreranno nel limite o la quota non raggiungerà un valore maggiore dello spazio totale utilizzato.

Per ulteriori informazioni, vedere Note legali.
