Cancellazione di un conto iFolder

È possibile specificare se si desidera rimuovere un conto solo dal computer attuale o se rimuovere dal server anche le cartelle iFolder di cui si è proprietari.

Se si rimuove un conto iFolder solo dal computer:

  • Le cartelle iFolder locali associate a tale conto verranno convertite in cartelle normali. Se la copia locale dei dati non è più necessaria, utilizzare un manager di file per individuarne l’ubicazione e cancellarla.

  • È ancora possibile accedere alle cartelle iFolder da altri computer con il client iFolder.

  • Quando il client iFolder client viene utilizzato con un server aziendale Novell iFolder 3.7 o versione successiva, l'accesso ai file può anche essere eseguito da qualsiasi postazione provvista di accesso Web Novell iFolder 3.7 o versione successiva.

Se si rimuove un conto iFolder dal server:

  • Verrà annullata la condivisione di tutte le cartelle iFolder di cui si è proprietari.

  • Le copie locali delle cartelle iFolder presenti sui computer membri verranno convertite in cartelle normali. Se la copia locale dei dati non è più necessaria, utilizzare un manager di file per individuarne l’ubicazione e cancellarla.

  • La cartella iFolder e i relativi contenuti verranno rimossi dal server.

  • Lo stato di membro delle cartelle iFolder condivise di altri utenti verrà revocato.

  • Il conto viene rimosso dal computer locale.

Se si decide di utilizzare questo conto su questo o su un altro computer, è necessario configurare il conto e le relative cartelle iFolder come all’inizio.

Per cancellare un conto iFolder:

  1. Fare clic con il pulsante destro del mouse sull'icona Servizi iFolder nell'area di notifica e scegliere Impostazioni conto per aprire la scheda Conti della finestra di dialogo Preferenze di iFolder.

  2. Selezionare il conto iFolder che si desidera cancellare, quindi fare clic su Cancella.

  3. Verrà visualizzato un messaggio in cui viene chiesto di definire il limite dell'azione di rimozione.

    Effettuare una delle seguenti operazioni:

    • Rimozione del conto iFolder solo dal computer in uso: Assicurarsi che Rimuovi cartelle iFolder e file personali dal server sia deselezionato, quindi fare clic su Sì.

    • Rimozione del conto iFolder, delle cartelle iFolder e dei rispettivi file personali dal sever: Selezionare Rimuovi cartelle iFolder e file personali dal server, quindi fare clic su Sì.

    • Annullamento della cancellazione: Fare clic su No per annullare l'operazione di cancellazione. Il conto e le relative cartelle iFolder rimarranno invariati.

Per ulteriori informazioni, vedere Note legali.
