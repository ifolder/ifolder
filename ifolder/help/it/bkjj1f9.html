Browser delle cartelle iFolder

Il browser consente di gestire le cartelle iFolder. Nella tabella indicata di seguito vengono riepilogati gli elementi del browser:

Tabella 2 Elementi del browser delle cartelle iFolder

Nome elemento

Descrizione

Crea cartella iFolder

Consente di convertire una cartella sul computer locale in una cartella iFolder e caricarla sul server iFolder a cui si è connessi.

Per ulteriori informazioni, vedere Creazione e caricamento di una cartella iFolder.

Sincronizza ora

Consente di iniziare immediatamente una sincronizzazione delle cartelle iFolder sul computer locale e sul server.

Condividi con

Consente di condividere la cartella iFolder con altri utenti.

Per ulteriori informazioni, vedere Condivisione di una cartella iFolder.

Ripristina cartella normale

Consente di ripristinare una cartella normale da un cartella iFolder. È possibile anche eliminare la cartella iFolder dal server. Per ulteriori informazioni, vedere Ripristino di una cartella normale da una cartella iFolder.

Server

Visualizza l'indirizzo IP del server iFolder a cui si è connessi.

Nome

Visualizza il nome della cartella iFolder selezionata.

Proprietario

Visualizza il nome del proprietario della cartella iFolder selezionata.

Diritti

Visualizza i diritti di cui dispone il proprietario della cartella iFolder.

File/cartelle da sincronizzare

Indica il numero di file o cartelle in attesa di sincronizzazione.

Ultima sincronizzazione completata

Indica l'ora e la data dell'ultima sincronizzazione.

Scarica una iFolder

Consente di scaricare una cartella iFolder dal server iFolder sul computer locale.

Per ulteriori informazioni, vedere Scaricamento di una cartella iFolder disponibile.

Tipo di cartella iFolder

Indica se la cartella iFolder selezionata è cifrata o normale.

Dimensioni cartella iFolder

Visualizza le dimensioni della cartella iFolder selezionata.

Fondi

Con la funzionalità di fusione è possibile consolidare i dati delle cartelle iFolder e ridurre in questo modo i tempi necessari per il loro trasferimento tra il server e il client iFolder. È possibile fondere il contenuto in una cartella iFolder sul server con quello di una cartella esistente nel computer locale. Per ulteriori informazioni, vedere Fusione di cartelle iFolder.

Cancella dal server

Consente di cancellare la cartella iFolder selezionata dal server.

Elenco domini

Visualizza l'elenco di server in cui è stato configurato un conto.

Connetti/Disconnetti

Consente di eseguire la connessione o disconnessione da un server iFolder. Se si è connessi al server iFolder, viene visualizzato il pulsante Disconnetti e, se non si è connessi a tale server, viene visualizzato il pulsante Connetti.

Risolvi

Consente di risolvere i conflitti dei file. Per risolvere i conflitti tra le versioni dei file, è necessario selezionare la versione corretta nella finestra di dialogo per la risoluzione dei conflitti. Per ulteriori informazioni, vedere Risoluzione dei conflitti tra file.

Utente

Visualizza il nome dell'utente che ha eseguito il login al server iFolder selezionato.

Numero della cartella iFolder

Visualizza il numero totale di cartelle iFolder sul server iFolder e il computer locale per il server iFolder selezionata.

Quota disco

Visualizza i valori dei campi In uso e Disponibile.

In uso

Visualizza la quantità totale di spazio su disco del server attualmente utilizzato da tutti i file e le directory sul server iFolder selezionato.

Disponibile

Visualizza la quantità totale di spazio su disco del server attualmente disponibile per i file e le directory nel server iFolder selezionato.

Sincronizzazione automatica

Visualizza l'intervallo di tempo tra due cicli di sincronizzazione automatici consecutivi.

Per ulteriori informazioni, vedere Note legali.
