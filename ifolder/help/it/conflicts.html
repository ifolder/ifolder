Risoluzione dei conflitti tra file

È possibile che si verifichino conflitti tra versioni di file salvate contemporaneamente in computer diversi o nel caso in cui i nomi di file differiscono solo per l'utilizzo di maiuscole e minuscole. Quando si verifica un conflitto, il file viene memorizzato nell'area dei conflitti ed è contrassegnato con lo stato Conflitto nome. L'icona di errore di iFolder viene visualizzata sulla cartella iFolder. Se l'opzione Notifica conflitti è abilitata nella finestra di dialogo Preferenze di iFolder, viene inoltre inviato un avviso all'area di notifica. È possibile controllare il comportamento delle notifiche popup impostando le preferenze nella scheda Generale > della finestra di dialogo Preferenze di iFolder.

Per risolvere i conflitti di file, vedere gli argomenti seguenti:

  • Risoluzione dei conflitti di versione di file

  • Risoluzione dei conflitti di nome di file

Risoluzione dei conflitti di versione di file

Per risolvere i conflitti tra le versioni di file, è necessario selezionare la versione corretta nella finestra di dialogo relativa alla risoluzione dei conflitti.

  1. Nell'area di notifica fare clic con il pulsante destro del mouse sull'icona Servizi iFolder e scegliere Cartelle iFolder per aprire il browser delle cartelle iFolder.

  2. Selezionare la cartella iFolder in conflitto e fare clic su Risolvi per aprire la finestra di dialogo Risolvi conflitti.

    È possibile visualizzare due versioni dello stesso file:

    • Versione locale: File salvato nel computer locale.

    • Versione del server: File salvato nel server.

    È possibile confrontare le informazioni relative al file nel computer locale con quelle relative al file nel server per individuare la versione del file che si desidera mantenere.

  3. Fare clic su Apri per ciascuno dei file in modo da visualizzarli.

  4. Selezionare una o più delle opzioni seguenti per il salvataggio nella cartella iFolder a seconda delle preferenze.

    Versione locale: Selezionare questa opzione se si desidera salvare la versione locale nella cartella iFolder.

    Versione del server: Selezionare questa opzione se si desidera salvare la versione del server nella cartella iFolder.

  5. selezionare Abilita per abilitare l'opzione per il salvataggio nella cartella dei conflitti. Quando si abilita questa opzione, il file in conflitto viene salvato automaticamente nella cartella dei conflitti. L'utente seleziona quale versione deve essere salvata nella cartella dei conflitti locale per poter essere utilizzato successivamente.

  6. Dopo avere risolto il conflitto, fare clic su Salva.

Risoluzione dei conflitti di nome di file

Questo tipo di conflitto si verifica quando il nome della versione locale del file e quello della corrispondente versione sul server differiscono per l'utilizzo di maiuscole e minuscole. Il contenuto dei file può quindi essere identico o meno. È possibile risolvere tali conflitti rinominando i file, tuttavia è necessario eseguire tale operazione con la massima cautela.

  • Se il contenuto è identico, è possibile rinominare uno dei file nella finestra di dialogo Risolvi conflitti.

  • In caso contrario, eseguire una copia del file locale e assegnargli un nome diverso. Tornare quindi alla finestra di dialogo Risolvi conflitti e salvare la versione del file disponibile sul server. Ciò consente di mantenere entrambe le versioni dei file nella cartella iFolder con nomi differenti. Se si rinomina una versione di file nella finestra di dialogo Risolvi conflitti, tale versione sovrascriverà l'altra, il che determina l'eliminazione definitiva del contenuto del file sostituito.

  1. Nell'area di notifica fare clic con il pulsante destro del mouse sull'icona Servizi iFolder e scegliere Cartelle iFolder per aprire il browser delle cartelle iFolder.

  2. Selezionare il file in conflitto e fare clic su Risolvi per aprire la finestra di dialogo Risolvi conflitti.

  3. Accertarsi che si tratti di un conflitto relativo ai nomi dei file e verificare se il contenuto dei file è identico o meno.

  4. Effettuare una delle seguenti operazioni:

    • Se il contenuto è identico, rinominare uno dei file, fare clic su Salva questa versione, quindi su OK per chiudere la finestra di dialogo Risolvi conflitti. Ciò consente di concludere il task.

    • Se il contenuto è diverso, passare a Passaggio 5.

  5. Chiudere la finestra di dialogo Risolvi conflitti.

  6. Aprire un manager di file per individuare la copia locale del file in conflitto.

  7. Copiare il file e salvarlo con un nome univoco nella cartella iFolder.

    Verrà eseguita la sincronizzazione del file come nuovo elemento nel server e nei computer membri.

  8. Tornare alla finestra di dialogo Risolvi conflitti, selezionare la versione del file memorizzata nel server, fare clic su Salva questa versione e quindi su OK.

    La versione del file sul server viene scaricata nel computer locale e sostituita alla copia locale del file con lo stesso nome.
