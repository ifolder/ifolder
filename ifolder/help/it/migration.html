Migrazione da iFolder 2.x a iFolder 3.7 o versione successiva

In questa sezione verranno illustrati i seguenti argomenti:

Informazioni sul processo di migrazione

Novell iFolder consente di eseguire la migrazione dei dati da iFolder 2.x al client iFolder 3.7 o versione successiva. La Migrazione guidata fornita nel client iFolder consente di eseguire la migrazione dei dati esistenti di iFolder 2.x in modo semplice e rapido.

All'avvio, la Migrazione guidata rileva l'installazione iFolder 2.x e chiede di eseguire la migrazione dei dati. Durante la migrazione dei dati, viene chiesto nuovamente di convertire le cartelle esistenti o di eseguire una copia della cartella iFolder 2.x. Per ulteriori informazioni a tale proposito, vedere Passaggio 6.

Viene visualizzato un messaggio relativo alla nuova sincronizzazione dei file nel server iFolder 3.7 o versione successiva e viene chiesto se è necessario mantenere il client 2.x fino al termine della sincronizzazione. Poiché durante il processo viene eseguita solo la migrazione dei dati, l'utente deve impostare le preferenze nel nuovo client.

Procedura di migrazione

  1. Installare il client iFolder per il server iFolder 3.7 o versione successiva, quindi configurare il conto per il server aziendale iFolder 3.7 o versione successiva.

    Per informazioni sull'installazione del client e la configurazione del conto, vedere Gestione dei conti e delle preferenze di iFolder.

  2. Nel browser delle cartelle iFolder, creare una o più cartelle iFolder nel computer locale.

    È possibile utilizzare qualsiasi nome valido per una cartella iFolder. Non creare una cartella iFolder nella stessa directory utilizzata per le cartelle iFolder 2.x. Per ulteriori informazioni, vedere il Creazione e caricamento di una cartella iFolder.

  3. Verrà visualizzato un messaggio relativo all'esistenza di dati della versione precedente e verrà richiesto di eseguire la migrazione.

    Fare clic su OK per aprire un elenco delle cartelle iFolder 2 disponibili.x disponibili.

  4. Selezionare le cartelle iFolder 2.x di cui si desidera eseguire la migrazione.

    Fare clic su Esegui migrazione per visualizzare la Migrazione guidata.

    NOTA:È inoltre possibile avviare la Migrazione guidata scegliendo iFolder > Migrazione > Esegui migrazione da 2.x.

    La Migrazione guidata consente di eseguire le varie fasi della procedura di migrazione dei dati.

  5. Scegliere Avanti per visualizzare le opzioni di migrazione.

  6. Selezionare una delle opzioni seguenti:

    Eseguire la migrazione della cartella iFolder e disconnetterla dal server 2.x: Selezionare questa opzione per disconnettere la cartella esistente da iFolder 2.x server al termine della migrazione alla versione 3.7 o successiva.

    Creare una copia della cartella iFolder ed eseguire la migrazione: Selezionare questa opzione per creare una copia di iFolder 2.x e convertirli in iFolder 3.7 o versione successiva. Questa operazione non rimuove i file dal dominio di iFolder 2.x server anche al termine della migrazione dei file a iFolder 3.7 o versione successiva.

    Se si seleziona questa opzione, è anche possibile selezionare Copia la cartella superiore se si desidera copiare non solo il contenuto della cartella iFolder 2.x, ma anche quello della cartella superiore. Se non si seleziona questa opzione, verrà eseguita la migrazione alla versione 3.7 o successiva solo dei contenuti della cartella selezionata.

    Fare clic su Sfoglia per scegliere l'ubicazione di salvataggio della cartella di cui si è eseguita la migrazione.

    Ad esempio, C:/Documents and Settings/<nome utente>/dirifolder2 è una cartella iFolder 2.x e dir1 e dir2 sono le sottodirectory di dirifolder2. Se si seleziona l'opzione Copia la cartella superiore, viene creata la cartella dirifolder2 nella nuova ubicazione e le directory dir1 e dir2 vengono copiate all'interno della nuova cartella. Se l'opzione rimane deselezionata, verranno spostate nell'ubicazione specificata solo dir1 e dir2.

  7. Scegliere Avanti per selezionare il server e le opzioni di sicurezza come segue:

    1. Selezionare il server dall'elenco a discesa.

    2. Selezionare l'opzione di sicurezza desiderata:

      • Selezionare Cifrata per cifrare la cartella iFolder selezionata.

        Se non si è impostata la stringa di cifratura durante il login, viene chiesto di immetterla. È possibile ignorare l'immissione e impostare la stringa di cifratura in un secondo momento.

        Se non si è attualmente connessi al server iFolder, viene richiesto di eseguire la connessione.

      • Selezionare Condivisa per condividere la cartella iFolder con altri membri del server iFolder.

        È possibile eseguire la migrazione di una cartella iFolder 2.x in una cartella condivisa. Se si prova a eseguire la migrazione da una cartella iFolder 2.x in una cartella condivisa, verrà richiesto di confermare l'operazione.

        Fare clic su OK per confermare l'azione.

  8. Fare clic su Avanti per eseguire la migrazione della cartella iFolder selezionata.

  9. Verificare la correttezza delle informazioni immesse e fare clic su Indietro per tornare alle pagine precedenti e apportare eventuali correzioni.

    Se le informazioni immesse sono corrette, fare clic su Esegui migrazione per iniziare la migrazione.

    Fare clic su Indietro per tornare alle pagine precedenti e apportare eventuali modifiche necessarie.

  10. Fare clic su Fine per completare la migrazione.

Per ulteriori informazioni, vedere Note legali.
