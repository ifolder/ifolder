Gestione delle cartelle iFolder

Una cartella iFolder è una directory locale utilizzata per condividere e sincronizzare in modo selettivo file con membri specificati dall'utente. Gli utenti che dispongono dei privilegi necessari possono modificare e leggere i file iFolder.
