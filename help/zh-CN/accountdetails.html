查看和修改 iFolder 帐户设置

可在“帐户设置”和“帐户属性”对话框中查看以下参数：

参数

说明

位置

可由用户修改（是/否）

联机

指示当前您是否连接到某个帐户的 iFolder 服务器。

选中该帐户旁边的联机复选框以连接到该帐户的 iFolder 服务器。

取消选中该帐户旁边的“联机”复选框以断开与 iFolder 服务器的连接。

“iFolder 首选设置” > “帐户”选项卡

是

服务器名

显示为帐户储存 iFolder 所在的 iFolder 服务器的描述名称。此名称由该服务器提供并由 iFolder 管理员进行控制。

用户不可修改此设置。

“iFolder 首选设置” > “帐户”选项卡

“帐户属性”>“服务器”选项卡

否

用户名

显示此帐户的用户身份，如网络用户 ID（LDAP 常用名）或电子邮件地址。不可修改此设置。

在“iFolder 帐户助理”中设置帐户时指定用户名。不可为现有帐户修改用户名。

要在同一服务器上使用不同的用户名，必须删除现有帐户，然后为该用户名创建一个不同的帐户。

“iFolder 首选设置” > “帐户”选项卡

“帐户属性”>“身份”选项卡

否

地址

显示您的帐户所在的 iFolder Enterprise Server 的 DNS 名称或 IP 地址。从 iFolder 管理员处获取此信息。

例如： svr21.example.com 或 192.168.1.1

您可以修改地址，但进行更改时，必须中断与 iFolder 服务器的连接。

“帐户属性”>“服务器”选项卡

是

口令

允许为用户名指定口令。

“帐户属性”>“身份”选项卡

是

保存口令

指示 iFolder 启动时 iFolder 是否自动登录。如果帐户的“自动连接”参数已启用（默认），则帐户会尝试使用保存的口令进行连接。

选中保存口令可以在登录计算机时自动登录到此 iFolder 帐户。

如果更改了网络口令，则在下一次 iFolder 尝试登录 iFolder 帐户时，自动鉴定将失败。iFolder 将提示您用新口令登录并且允许您指定和保存新口令。

“帐户属性”>“身份”选项卡

是

定额

报告 Enterprise Server 上分配给您的 iFolder 帐户的空间总量。此值可能是为所有用户设置的默认定额，也可能是用户特有的定额。如果未设置定额，则该值为空或为 N/A（不适用）。对于已连接的每个服务器，还可以在 iFolder 主窗口中查看可用空间。

“帐户属性”>“磁盘空间”选项卡

否

已用空间

报告此帐户上您拥有的所有 iFolder 当前正在使用的空间总量。不包括与您共享但为其他人拥有的 iFolder 所使用的空间。

“帐户属性”>“磁盘空间”选项卡

否

可用空间

报告此 iFolder 帐户上您当前可用于 iFolder 的空间总量。此值等于定额减去已使用空间。

“帐户属性”>“磁盘空间”选项卡

否

自动连接

如果选择（启用）了此值，则 iFolder 启动时将自动连接到此帐户。取消选中该设置可禁用连接到帐户。

该设置仅管理客户端启用或禁用帐户的能力。

当您同时使用 iFolder 和 Novell® iFolder® 3.7 或更高版本服务器时，iFolder 系统管理员可禁用您在服务器端的 iFolder 帐户（通常是临时的）。禁用发生时可能出现以下情况：

  • 如果从服务器端禁用帐户时您已登录，则您可以管理 iFolder，并且您的 iFolder 数据继续与服务器保持同步，直到您通过注销或禁用同步结束会话，或者服务器断开您的会话（以先发生的为准）为止。

  • 如果帐户在服务器端被禁用，则您在尝试登录时将收到一个弹出讯息，通知您该帐户已被禁用。请与您的 iFolder 系统管理员联系以获取帮助。

所有选项卡上的“帐户属性”

是

帐户为默认

当您有多个帐户时，请选中“帐户为默认”复选框以在创建和上载 iFolder 时使此帐户成为帐户下拉列表中默认选择的帐户。

如果只有一个帐户，则不能修改此值。

所有选项卡上的“帐户属性”

是，如果有多个帐户

加密设置

您可以设置和重设置可从“安全性”菜单查看的加密设置。有关加密设置的更多信息，请参见管理加密 iFolder 的通行口令

“安全性”菜单

用户可以重设置通行口令

  1. 在通知区域中右击“iFolder 服务” 图标，然后选择“帐户设置”以打开“iFolder 首选设置”并进入“帐户”选项卡。

    在“帐户”报告中，可以管理每个帐户的以下参数：

    • 联机（选中可进行连接；取消选中可中断连接）

    • 服务器名（仅供查看）

    • 用户名（仅供查看）

  2. 选择要管理的 iFolder 帐户，然后单击“属性”打开“帐户属性”对话框，显示“服务器”选项卡。

    在“服务器”选项卡上，可以管理所选帐户的以下参数：

    • 名称（仅供查看）

    • 地址（如果已连接则仅可查看；如果已中断连接，则可修改）

    • 服务器说明（仅查看；滚动以查看所有信息）

    • 自动连接（选中可启用；取消选中可禁用）

    • 帐户为默认（选中可启用；只有存在多个帐户时才可用）

  3. 选择“身份”选项卡可管理所选帐户的以下参数：

    • 用户名（仅供查看）

    • 口令（指定口令）

    • 记住口令（选中可启用；取消选中可禁用）

    • 自动连接（选中可启用；取消选中可禁用）

    • 帐户为默认（选中可启用；只有存在多个帐户时才可用）

  4. 选择“磁盘空间”选项卡可查看所选帐户的以下参数：

    • 定额（仅查看）

    • 已用（仅查看）

    • 可用（仅查看）

    • 自动连接（选中可启用；取消选中可禁用）

    • 帐户为默认（选中可启用；只有存在多个帐户时才可用）

  5. 完成后，请关闭“帐户属性”对话框，然后关闭“iFolder 首选设置”对话框。

有关更多信息，请参见 法律声明.
