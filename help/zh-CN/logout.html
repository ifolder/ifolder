注销 iFolder 帐户

如果要脱机工作，则注销 iFolder 帐户。

  • Macintosh

  • Linux

  • Windows

Macintosh

  1. 在对接台中单击 iFolder 服务图标，然后选择“帐户”以打开“iFolder 首选设置”对话框。

  2. 从工具栏单击“帐户”图标以查看当前 iFolder 域中的帐户列表。

  3. 找到要管理的帐户，然后单击“注销”以从 iFolder 服务器断开。

Linux

  1. 在通知区域中右击“iFolder 服务” 图标，然后选择“帐户”以打开“iFolder 首选设置”对话框并进入“帐户”选项卡。

  2. 找到您希望管理的帐户，然后取消选择它旁边的“联机”复选框以断开与 iFolder 服务器的连接。

  3. 单击“关闭”可关闭“iFolder 首选设置”对话框。

Windows

  1. 在通知区域中右击“iFolder 服务” 图标，然后选择“帐户”以打开“iFolder 首选设置”对话框并进入“帐户”选项卡。

  2. 找到您希望管理的帐户，然后取消选择它旁边的“联机”复选框。

  3. 单击“应用”断开与 iFolder 服务器的连接，然后单击“确定”关闭“iFolder 首选设置”对话框。

有关更多信息，请参见 法律声明.
