iFolder 客户程序

iFolder 客户程序可与您的操作系统集成以在本机桌面环境中提供 iFolder 服务。通过使用 iFolder 客户程序，您可以管理自己的 iFolder，并且可以选择要在每台计算机上设置的共享 iFolder。iFolder 支持以下操作系统：

  • SUSE® Linux Enterprise Desktop 10 SP1和 SP2

  • SUSE Linux Enterprise Desktop 11

  • Windows Vista SP1

  • Windows XP SP2

  • OpenSUSE 11.1

  • Apple Macintosh® 10.4 和更高版本

有关更多信息，请参见 法律声明.
