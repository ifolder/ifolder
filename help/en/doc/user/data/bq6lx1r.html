Modifying Contact Information

  1. In the Address Book, find the contact that you want to edit.

    For information, see Finding a Contact.

  2. To open the Contact Editor, do one of the following:

    • Select the contact, then click Edit > Contact.
    • Double-click the contact.

  3. Modify the contact information, as needed.

    Make sure to complete the required User ID, First Name, Last Name, and E-mail address fields.

  4. Click OK.

  5. Close the Address Book.
