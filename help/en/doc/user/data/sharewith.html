Adding a User to an iFolder


Add a User with a File Manager

  1. In the file manager, navigate to the iFolder, then select it.

  2. Right-click the iFolder, then click iFolder > Properties.

  3. Click Share With.

  4. From the list of users, select one or more users.

  5. Select the access right: Full Control, Read/Write, or Read Only.

  6. Click Apply.


Add a User with My iFolders

  1. In the status bar, right-click the iFolder icon , then click My iFolders to open the My iFolders browser.

  2. In the list of iFolders, select the iFolder, then click Share With.

  3. From the list of users, select one or more users.

  4. Select the access right: Full Control, Read/Write, or Read Only.

  5. Click Apply.


Modifying User Access Rights

As the working relationships and status of members of an iFolder workgroup change, you might need to change a user's access right for the iFolder. For example, you might want to give a trusted user the File Control right for the iFolder.


Setting User Rights with a File Manager

  1. In the file manager, navigate to the iFolder, then select it.

  2. Right-click the iFolder, then click iFolder > Properties.

  3. Click Share With.

  4. From the list of users, select one or more users.

  5. Select the access right: Full Control, Read/Write, or Read Only.

  6. Click Apply.


Setting User Rights with My iFolders

  1. In the status bar, right-click the iFolder icon , then click My iFolders to open the My iFolders browser.

  2. In the list of iFolders, select the iFolder, then click Share With.

  3. From the list of users, select one or more users.

  4. Select the access right: Full Control, Read/Write, or Read Only.

  5. Click Apply.


Removing a User from an iFolder

To unshare an iFolder, you must remove a user as a member of an iFolder workgroup. The iFolder on the user's workstation becomes a normal folder. A copy of the data remains on the former member's workstation, but the files are no longer synchronized with the workgroup.


Removing Users with a File Manager

  1. In the file manager, navigate to the iFolder, then select it.

  2. Right-click the iFolder, then click iFolder > Properties.

  3. Click Share With.

  4. From the list of users, select one or more users.

  5. Click Remove.


Removing Users with My iFolders

  1. In the status bar, right-click the iFolder icon , then click My iFolders to open the My iFolders browser.

  2. In the list of iFolders, select the iFolder, then click Share With.

  3. From the list of users, select one or more users.

  4. Click Remove.
