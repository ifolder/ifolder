Adding an Address Book

You can maintain multiple books with the iFolder Address Book.

  1. In the Address Book, do one of the following:

    • Click New Book under the Books list.
    • Click File > New > Address Book.
    • Right-click in the Books list area, then click Create.

    The Create Address Book dialog opens.

  2. Type a name for the address book, then click OK.

    The address book appears in the Books list.

  3. Close Address Book.
