Configuring Preferences for the Client

  1. Right-click the iFolder application icon in the Notification area, then select Preferences to open the iFolder Preferences dialog box to the General tab.

  2. Specify the following values:

    Parameter Description

    Start iFolder When Logging In to the Desktop

    Select this option to lauch iFolder whenever you log in with the local username where you set up the iFolder account. For those accounts where you have enabled Remember Password, synchronization occurs in the background at the specified Synchronization Interval.

    Deselect this option to start iFolder manually.

    Show Confirmation Dialog When Creating iFolders

    Select this option to receive a confirmation message each time you create an iFolder in addition to the normal visual cues, such as the iFolder appearing in the list of iFolders.

    Deselect this option stop receiving the confirmation message.

    Notify of Shared iFolders

    Select this option to enable a bubble-pop-up message in the Notification area whenever you receive a new invitation to share an iFolder.

    NOTE:  In this release, invitations are synchronized from the server every 1 minute and the frequency is not configurable. In the final release, invitations will be retrieved based on the Synchronization Interval you specify in iFolder Preferences. The interval will control synchronization of files, iFolder share invitations, and the roster of users. You can also click Refresh to update invitations on demand.

    Deselect this option to disable the pop-up message behavior for invitations.

    Notify of Collisions

    Select this option to enable a bubble-pop-up message in the Notification area whenever a conflict occurs when synchronizing files.

    Deselect this option to disable the pop-up message behavior for synchronization conflicts.

    Notify When a User Joins

    Select this option to enable a bubble-pop-up message in the Notification area whenever a user accepts an invitation to share an iFolder. Acceptance is keyed to when the user actually sets up the iFolder on his or her own computer.

    Deselect this option to disable the pop-up message behavior for user share confirmation.

    Synchronization

    Select Sync to enable background synchronization; deselect Sync to disable it.

    If you enable synchronization, specify the interval for synchronizing files, such as every 1 hour or every 2 days. The minimum Sync Interval, typically every 5 minutes, is configured by the iFolder administrator. You cannot override the system minimum.

  3. When you are done, click Apply to effect your changes.

  4. Click OK to close the iFolder Preferences dialog box.
