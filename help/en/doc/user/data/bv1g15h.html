Workgroup and Enterprise Server Sharing

The iFolder client supports sharing in two modes: Workgroup and Enterprise Server. Workgroup mode supports synchronization across multiple computers through a peer-based host. Enterprise Server mode supports synchronization across multiple computers through a central server. In an enterprise environment, your administrator determines whether Workgroup mode is enabled for use in addition to the Enterprise Server mode.
