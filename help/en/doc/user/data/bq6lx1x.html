Using the Contact Picker

Use the Contact Picker to specify which contacts in the Address Book you want to participate in the shared iFolder. It supports the following tasks:

  • Accessing the Contact Picker
  • Adding Members to the Shared iFolder
  • Adding New Contacts to the Address Book from the Contact Picker
  • Viewing or Modifying Information about a Contact


Accessing the Contact Picker

  1. In a file manager or on the desktop, locate the iFolder that you want to manage.

  2. Right-click the iFolder, then click iFolder > Properties.

  3. Click Advanced to manage the iFolder workgroup membership.

  4. Click Add.

    The Contact Picker opens.


Adding Members to the Shared iFolder

  1. In the Contact Picker, from the Contacts list, select one or more usernames you want to add to the Share With list.

    1. (Optional) Use the search function to find the contacts you want to add as a member.

      For information, see Finding a Contact.

    2. (Conditional) If the user is not in the Contacts list, add the user as a contact.

      For information, see Adding a Contact.

  2. Click Add.

    The selected usernames appear in the Share With list (right column).

  3. (Conditional) If you need to remove a contact from the Share With list, select the contact, then click Remove.

    This action only removes the entry from the Share With list; it does not delete the contact from your address book.

  4. Click OK to accept the list of contacts you want to add.

    The contacts are added to the Share With list in iFolder Advanced Properties.


Adding New Contacts to the Address Book from the Contact Picker

  1. In the Contact Picker, to open the Contact Editor, click New Contact under the Contact list.

  2. Type the user contact information.

    At a minimum, you must complete the User ID, First Name, Last Name, and E-mail fields.

  3. Click OK.

    The new contact appears in the Contact list.


Viewing or Modifying Information about a Contact

  1. Use one of these methods to open the Contact Editor:

    • Select the contact from the Contacts list, then double-click the contact.
    • Select the contact from the Contacts list, then click Edit.

  2. View or modify information about the contact.

    At a minimum, the contact information must include the user ID, first name, last name, and e-mail address.

  3. When you are done, click OK to close the window.
