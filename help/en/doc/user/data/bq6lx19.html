Managing iFolder Contacts

The iFolder address book is a collection of information about contacts in your workgroup. Use the address book to manage information about members of your iFolders. At a minimum, the address book contains the user ID, first and last names, and e-mail addresses of each contact that are members of your iFolders. You can add an unlimited number of contacts in the address books.
