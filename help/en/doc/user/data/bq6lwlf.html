Converting a Folder to an iFolder

Any local directory can be converted to an iFolder, if the directory is not the root directory and does not contain an iFolder.

  1. In a file manager or on the desktop, locate the folder that you want to convert to an iFolder.

  2. Use one of the following methods:

    • Right-click the folder icon, then click Convert to an iFolder.
    • In the toolbar, click File > Convert to an iFolder.

    The folder icon changes to an iFolder icon and iFolder displays a confirmation message.

  3. (Optional) Check Do Not Display This Message Again to avoid this message in the future.

  4. To dismiss the confirmation message, click OK.
