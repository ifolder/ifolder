Setting Up an Available iFolder

When others share iFolders with you, the iFolders appear with an Available iFolder icon in your iFolders browser. To participate in the shared iFolder on your current local computer, set up the iFolder to make it active on the computer.

The local iFolder location must satisfy the following guidelines:

  • Guidelines for Locating iFolders
  • Naming Conventions for an iFolder and Its Folders and Files

To set up an iFolder:

  1. In the Notification area, right-click the iFolder icon , then click iFolders to open the iFolders browser.

  2. In the list of iFolders, select the available iFolder, then click Set Up .

  3. If you are not logged in to the account, log in with your username and password, then click OK.

  4. Browse to the location where you want to create a local iFolder.

  5. Click OK twice.
