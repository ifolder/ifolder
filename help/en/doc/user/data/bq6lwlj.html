Viewing Properties of an iFolder

To view information about an iFolder, right-click the iFolder, then click iFolder > Properties.

Click Advanced to manage membership in the selected iFolder workgroup. For information, see Sharing an iFolder.

The following table describes the iFolder properties.

Field Name Description

Owner

A unique identifier for the iFolder Bill owner.

CreationTime

The time stamp in Greenwich Mean Time when the iFolder was created.

ModifyTime

The time stamp in Greenwich Mean Time when the iFolder was last modified.

CollectionId

A unique identifier for the data store.

IDPath

The path for the data collection on the current workstation.

DomainName

The hostname of the workstation that hosts the selected iFolder.

DocumentRoot

The path for the selected iFolder on the local workstation.

MasterIncarnation

Indicates that the selected iFolder is a host to other iFolder members.

LocalIncarnation

If On, indicates that the selected iFolder originated locally and the current workstation is the primary iFolder host.
