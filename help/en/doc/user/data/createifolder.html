Creating an iFolder

Whenever you create an iFolder, you specify an existing normal folder that you want to be converted to an iFolder. If the normal folder does not exist, one is created as part of the process.

You can create iFolders directly in your file manager and in the My iFolders browser available in the client application. Within the iFolder, you can set up any subdirectory structure that suits your personal or corporate work habits. The subdirectory structure is constant across all member iFolders. Each workstation can specify a different parent directory for the shared iFolder.


Guidelines for Folders to be Converted

You can make any local directory an iFolder if it meets certain criteria:

  • The root directory cannot be an iFolder.
  • An iFolder cannot contain another iFolder.
  • The iFolder open source client and the Novell iFolder® 2.1x Client can coexist on the same computer, but they are not integrated and their software should not be installed in the same folder. If you also use Novell iFolder 2.1x on your workstation, you must comply with the following restrictions:
    • Any local directory where you store your Novell iFolder 2.1x data should not be converted to an iFolder.
    • Any directory that contains a local directory where you store your Novell iFolder 2.1x data should not be converted to an iFolder.


Creating iFolders in a File Manager

  1. Use the file manager to navigate to the folder you want to convert to an iFolder, then select the folder.

  2. Use one of the following methods:

    • Right-click the folder icon, then click Convert to an iFolder.
    • In the toolbar, click File > Convert to an iFolder.

    The folder icon changes to an iFolder icon and iFolder displays a confirmation message. It is okay to dismiss the confirmation message.


Creating iFolders in My iFolders

  1. In the status bar, right-click the iFolder icon , then click My iFolders to open the My iFolders browser.

  2. Click Create.

  3. Specify the IP address or DNS name of the enterprise server for the account you want to use for this iFolder.

  4. In the browser, navigate to the directory where you want to create an iFolder.

  5. Select the normal folder you want to make an iFolder, using one of these methods:

    • If the normal folder exists, select the folder, then click OK
    • If the normal folder does not exist, click Create New Folder to create it, select the newly created folder, then click OK.

    The iFolder appears in the My iFolders list.
