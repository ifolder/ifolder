Deleting an iFolder Account

If you delete an iFolder account from your computer, the related local iFolders are reverted to normal folders.

  1. Right-click the iFolder application icon in the Notification area, then select Accounts to open the iFolder Preferences dialog box to the Accounts tab.

  2. Select the iFolder account you want to delete, then click Delete.

  3. A message prompts you to determine if the extent of the delete action. Click one of the following:

    • Yes: Click Yes to remove this account from all workstations, to remove all iFolders owned by this account from the server, and to remove the user from all memberships in shared iFolders on this account. The local iFolders are reverted to normal folders. Use a file manager to locate and delete the local copy of the data if it is not needed.

    • No: Click No to remove this account only from the local workstation. The local iFolders are reverted to normal folders. Use a file manager to locate and delete the local copy of the data if it is not needed.

    • Cancel: Click Cancel to back out of the delete action. Your account and its iFolders remain untouched.
