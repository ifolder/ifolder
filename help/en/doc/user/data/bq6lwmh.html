Accepting an Invitation to Share an iFolder

  1. When an invitation to share an iFolder arrives via e-mail, save the attached IFI file to a local directory.

  2. (Conditional) If the iFolder client is not currently installed on your computer, click the link to download iFolder, then install the iFolder client.

    For information, see Getting Started in the iFolder User Guide.

    The iFolder client begins running automatically when you log in to the desktop. The iFolder Services icon appears in the desktop status bar.

  3. To open the Invitation Wizard, right-click the iFolder icon in the status bar, then click Invitation Wizard.

  4. Read the invitation introduction, then click Next.

  5. Specify the path, including the IFI filename, to the iFolder invitation you want to accept. You can also browse to the location.

  6. Click Next.

  7. Specify the path where you want to place the local iFolder directory for this iFolder workgroup.

    The path cannot reside in an existing iFolder and it cannot be the root directory.

  8. Click Next > Close.
