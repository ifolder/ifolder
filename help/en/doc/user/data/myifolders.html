Managing iFolders

An iFolder is a local directory that selectively shares and synchronizes files with user-specified members. The iFolder files are accessible to all members and can be modified and read by those with privileges to do so.

This section discusses the following tasks for iFolder:

  • Sharing an iFolder
  • Setting Up an Available iFolder
  • Viewing Properties of an iFolder
  • Synchronizing Files
  • Resolving File Conflicts
  • Reverting an iFolder to a Normal Folder
  • Deleting an iFolder
