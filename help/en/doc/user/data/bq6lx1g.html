Finding a Contact

The Address Book provides a search function to enable you to easily locate contacts in your address book. You can search by typing in the search field, using the constraints to limit the search.

  1. From the left drop-down menu in the Search area, specify whether to search by First Name, Last Name, User Name, or E-mail.

  2. From the right drop-down menu in the Search area, specify the constraints to limit the search.

    Constrain the search to contacts that Begin With, End With, or Contain the letters you plan to type in the Search field.

  3. In the Search field, type information to identify the contact or contacts you seek.

    As you begin to type letters in the Search field, the entries that match the search criteria appear in the contact list. The search might return multiple contacts in the Contacts list.

  4. From the Contacts List, select one or more usernames, as desired.

    To select multiple contacts, press the Control key while selecting them.

  5. Continue with your management task for the selected contacts.
