Criando e fazendo upload de uma pasta do iFolder

Você pode criar uma pasta iFolder convertendo uma pasta existente que satisfaça as seguintes diretrizes:

  • Diretrizes para a localização e o uso de pastas do iFolder

  • Convenções de nomeação para um iFolder e suas pastas e arquivos

Se não existir uma pasta, você poderá procurar uma localização e criar a nova pasta como parte do processo.

IMPORTANTE:Você pode compartilhar apenas as pastas comuns do iFolder. As pastas criptografadas do iFolder não podem ser compartilhadas.

Dentro da pasta iFolder, é possível configurar qualquer estrutura de subdiretório que se adapte aos seus hábitos de trabalho pessoais ou profissionais. A estrutura de subdiretórios é constante em todos os membros das pastas iFolder. Cada estação de trabalho pode especificar um diretório pai diferente para a pasta iFolder compartilhada.

Quando o limite de número de pastas do iFolder for excedido, um usuário poderá criá-las e o iFolder exibirá uma mensagem de erro. Por exemplo, se o administrador tiver definido a política de Número de pastas do iFolder por usuário para 10 e você tentar criar mais uma, o iFolder 3.7 o notificará sobre a violação da política.

Linux

Você pode criar pastas do iFolder com um gerenciador de arquivos ou com o browser do iFolder, que está disponível no aplicativo cliente. Use os seguintes métodos para criar uma pasta iFolder:

  • Criando pastas iFolder no browser de pastas iFolder

  • Criando pastas iFolder em um gerenciador de arquivos

Criando pastas iFolder no browser de pastas iFolder

  1. Na área de notificação, use um dos seguintes métodos para abrir o browser do iFolder:

    • Clique no ícone de Serviços do iFolder .

    • Clique o botão direito do mouse no ícone de Serviços do iFolder e clique em Pastas do iFolder.

  2. Clique em Criar iFolder.

  3. Em Mais Opções, especifique a Conta do iFolder que você deseja usar para essa pasta do iFolder.

    Se você tiver várias contas do iFolder, a conta padrão será listada primeiro. Use a lista suspensa para selecionar uma conta diferente.

  4. No browser, navegue até o diretório onde deseja criar uma pasta iFolder.

  5. Selecione a pasta normal que deseja transformar em uma pasta iFolder, utilizando um dos seguintes métodos:

    • Se a pasta normal existir, selecione-a e clique em Adicionar.

    • Se a pasta normal não existir, clique em Criar Pasta para criá-la, selecione a pasta criada recentemente e clique em Adicionar

  6. Selecione o Tipo de pasta do iFolder entre as opções. Você pode decidir se deseja criar um iFolder criptografado ou comum.

    IMPORTANTE:Você terá permissão para criar pastas do iFolder criptografadas somente se o administrador do iFolder tiver definido uma política de criptografia como Ativada. Caso contrário, você poderá criar somente pastas comuns do iFolder e essa opção estará selecionada por padrão. Nesse caso, ambas as opções para Tipo estão desabilitadas.

    • Selecione Criptografia da Frase Secreta para habilitar a criptografia do iFolder selecionado.

      Se você não tiver definido a frase secreta durante a sessão de login e selecionar Criptografia da Frase Secreta, será solicitada a frase secreta. Especifique a frase secreta na caixa de diálogo Frase Secreta.

    • Selecione Comum para permitir o compartilhamento do iFolder selecionado.

      O iFolder anuncia a disponibilidade de pastas compartilhadas do iFolder com uma mensagem de pop-up na área de notificação. Os usuários compartilhados podem clicar na mensagem para configurar a pasta do iFolder nesse momento, ou fechá-la para fazer download da pasta posteriormente.

  7. Selecione Sincronização Segura para permitir a sincronização segura dos arquivos no iFolder que você está criando.

    Você pode habilitar o canal seguro para as pastas comuns e criptografadas do iFolder. Com essa opção selecionada, é possível usar um canal seguro para sincronizar entre o iFolder no servidor e o contido na máquina local.

    NOTA:Você também pode habilitar a sincronização segura para um iFolder selecionado na sua máquina usando a opção do botão direito do mouse iFolder > Propriedades, na janela principal do iFolder. Para isso, clique com o botão direito no iFolder e selecione Propriedades e, na guia Geral, marque a caixa de seleção Sincronização Segura.

  8. Clique em OK.

    A pasta do iFolder aparece na lista iFolders Neste Computador.

  9. Se a mensagem iFolder Criado for mostrada, execute um dos seguintes procedimentos:

    • Clique em Fechar para descartar a mensagem.

    • Selecione Não Mostrar Esta Mensagem Novamente para desabilitar futuras notificações e clique em Fechar para descartar a mensagem.

      Se você decidir posteriormente que deseja obter mensagens de confirmação, vá para Preferências do iFolder, selecione Mostrar Caixa de Diálogo de Confirmação ao Criar Pastas do iFolder e clique emOK.

Criando pastas iFolder em um gerenciador de arquivos

  1. Use o gerenciador de arquivos para navegar até a pasta que deseja converter em uma pasta iFolder e, em seguida, selecione a pasta.

  2. Para isso, use um destes dois métodos:

    • Clique o botão direito do mouse no ícone de pasta e clique em Converter em um iFolder.

    • Na barra de ferramentas, clique em Arquivo > Converter em um iFolder.

    O ícone da pasta muda para uma pasta com uma sobreposição do emblema do iFolder ("i" verde).

  3. Se a mensagem iFolder Criado for mostrada, execute um dos seguintes procedimentos:

    • Clique em Fechar para descartar a mensagem.

    • Selecione Não mostrar esta mensagem novamentepara desabilitar futuras notificações e clique em Fechar para ignorá-la.

      Se você decidir depois que deseja obter mensagens de confirmação, vá para Preferências do iFolder, selecione Mostrar Caixa de Diálogo de Confirmação ao Criar Pastas do iFolder e clique em OK.

Criando pastas iFolder em um Ponto de Montagem FAT32 (Linux)

No Linux, o iFolder suporta a criação e a utilização de pastas iFolder em um ponto de montagem para um sistema de arquivos FAT32. Para criar novas pastas do iFolder ou configurar pastas compartilhadas do iFolder em um volume FAT32 no seu computador Linux, o usuário do iFolder deve possuir o ponto de montagem FAT32 em que essas pastas devem ser armazenadas, e o arquivo /etc/fstab deve identificar explicitamente o ID do usuário local do iFolder referente ao ponto de montagem.

IMPORTANTE:Isso é necessário porque a partição FAT32 não suporta permissões de arquivo. Qualquer outro sistema de arquivos que não suporte permissões terá o mesmo problema.

Para sistemas com inicialização dupla, você pode acessar os dados em pastas iFolder como pastas normais do sistema onde elas não estiverem configuradas como pastas iFolder. Se você configurar a pasta como uma pasta iFolder em ambas as plataformas, o conjunto de dados será armazenado em duas pastas iFolder diferentes no servidor, o que consumirá o dobro dos recursos de disco do servidor alocado.

Como o sistema de arquivos FAT32 não suporta permissões do sistema de arquivos POSIX, verifique se o usuário do iFolder é o proprietário local do ponto de montagem. Modifique o arquivo /etc/fstab para definir o parâmetro ID de usuário (UID) da identidade de login local do usuário do iFolder que cria pastas nesse ponto de montagem.

  1. Efetue login no computador como o usuário raiz.

  2. Crie o diretório onde deseja montar o sistema de arquivos FAT32. No prompt de comando, digite

    mkdir /home/username/fsmount

    Substitua nomedeusuario pelo nome do usuário que planeja usar a unidade FAT32 para pastas do iFolder. Substitua fsmount pelo nome do diretório que você deseja usar como ponto de montagem. Por exemplo, se o nome do usuário for jsmith e o nome do diretório for fat32mntpt referente a uma unidade ou partição FAT32, digite

    mkdir /home/jsmith/fat32mntpt

  3. Monte a unidade FAT32 no ponto de montagem desejado. No prompt de comando, digite

    mount -t vfat /dev/hda1 /home/username/fsmount

    Substitua /dev/hda1 pelo nome do dispositivo ou da partição do dispositivo não-Linux. Substitua nomedeusuario pelo nome do usuário do iFolder. Substitua fsmount pelo nome do diretório que você deseja usar como ponto de montagem.

    Continuando o exemplo, onde o sistema de arquivos FAT32 (vfat) está no dispositivo hda4 (/dev/hda4), o nome do usuário é jsmith, e o diretório do ponto de montagem é /home/jsmith/fat32mntpt, digite

    mount -t vfat /dev/hda4 /home/jsmith/fat32mntpt

  4. Edite o arquivo /etc/fstab adicionando o parâmetrouid=usernamedo usuário do iFolder na linha que define o ponto de montagem FAT32.

    /dev/hda1  /home/username/fsmount  vfat  uid=username

    Continuando o exemplo, modifique o arquivo /etc/fstab adicionandouid=jsmithao ponto de montagem:

    /dev/hda4  /home/jsmith/fat32mntpt  vfat  uid=jsmith
