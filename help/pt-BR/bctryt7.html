Fazendo upgrade do cliente do iFolder 3.x Clientes

Esta seção aborda o seguinte:

  • Fazendo upgrade automático para o iFolder 3.7

  • Upgrade manual para iFolder 3.7

IMPORTANTE:Antes de fazer upgrade do cliente 3.4.1, faça backup do seu armazenamento local do Simias. Se você não fizer o backup, perderá todas as informações de configuração do computador, pois o armazenamento do Simias será apagado se o upgrade falhar.

Fazendo upgrade automático para o iFolder 3.7

O upgrade automático só será possível se você tiver direitos de acesso raiz, já que esse processo precisa instalar os pacotes RPM. A senha raiz é solicitada durante o upgrade. Se você não tiver direitos de acesso à raiz, o pacote não será instalado automaticamente. No entanto, os pacotes clientes serão descarregados em um local temporário. O cliente Linux é descarregado em /tmp/<32chardir>/ifolder3-linux.tar.gz Isso permite que você faça upgrade posteriormente, pois a localização contém dois scripts de instalação que você pode executar manualmente. Somente após fazer upgrade do cliente, você poderá conectar o domínio do iFolder 3.7 com um cliente do iFolder 3.x.

  1. De um cliente do iFolder 3.x cliente anterior ao iFolder 3.7, tente se conectar ao servidor iFolder. Você será perguntado se deseja fazer upgrade.

  2. Clique em Sim para fazer upgrade automático do cliente do iFolder 3.x.

No caso do iFolder 3.4.1, se você clicar em Não e tentar conectar o cliente do iFolder 3.4.1 ao servidor iFolder 3.7, ele exibirá uma mensagem de erro. Clique em OK para descartar a mensagem.

NOTA:Um cliente do iFolder 3.X (que não sejam clientes do iFolder 3.4.) se conecta ao servidor do iFolder 3.7 mesmo que você opte por não fazer upgrade. Apesar de o cliente 3.X permanecer conectado ao servidor do 3.7, você não pode executar operações do iFolder com esse cliente. Quando você tenta converter uma pasta existente em uma pasta do iFolder, o servidor não considera essas mudanças e exibe uma mensagem de erro informando que o servidor não está disponível. A única operação que você pode fazer é o download de novos RPMs do servidor para instalar o iFolder 3.7.

Upgrade manual para iFolder 3.7

  1. Abra um terminal. No prompt de comando, execute install-ifolder.sh.

  2. Digite a senha raiz quando for solicitado.

    Essa ação fecha o cliente do iFolder se ele já estiver em execução e, em seguida, atualiza o servidor da Web XSP e o cliente do iFolder junto com o plug-in do Nautilus-ifolder3.

  3. Inicie o iFolder a partir do menu do Aplicativo.

Para obter mais informações, consulte Informações legais.
