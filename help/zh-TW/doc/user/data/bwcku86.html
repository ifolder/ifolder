跨平台考量

Linux、Windows 和 Macintosh 版 iFolder 用戶端的使用者介面符合其各自平台的介面慣例。在每個平台上執行一般任務的方式是並行的，但您可能會觀察到在以下方面的細微差異，包括按鈕、顯示區、功能標籤、桌面通知區域的位置和啟動 iFolder 優先設定等等。除程序明顯不同的部份之外，本指南僅提供針對三種平台之一的指示。
