iFolder 的優點

iFolder 具有下列優點：

  • 與本地桌面環境整合，因而便於建立和管理多個 iFolder
  • 具有高度延伸性和彈性，對每位使用者的 iFolder 數量和每個 iFolder 的成員數量都沒有實際限制
  • 可讓您輕鬆並選擇性地共用個人和企業檔案
  • 可讓您控制您所擁有的 iFolder 成員使用者的存取層級，或擁有必要的權限以控制成員之存取權限的位置
  • 將您的檔案更新到多個工作站上的成員 iFolder，而使用者不會察覺此過程
  • 追蹤並記錄您在離線時所做的變更，並於上線時同步那些變更
  • 提供成員 iFolder 的保全驗證，以同步資料
  • 提供另一種通過電子郵件交換檔案的方法，這個方法可以避免電子郵件附件中的檔案大小限制，並減少電子郵件的儲存需求
