刪除 iFolder

如果您從集合中刪除 iFolder，則會刪除 iFolder 及其內容的本地副本。這也可能影響 iFolder 的共用關係，要視您所持有的「存取」權限而定。下表以刪除 iFolder 之使用者的「存取」權限為依據，概述了對共用關係及成員 iFolder 的影響。

刪除 iFolder 不會從您的工作站解除安裝 iFolder 用戶端。

  1. 在檔案管理員或 iFolder 瀏覽器中，找到您要刪除的 iFolder。

  2. 以滑鼠右鍵按一下該 iFolder，然後按一下「刪除」。
