iFolder 共用

通常，當您在多個位置工作或與其他人合作時，必須認真負責地管理檔案版本。使用 iFolder，您便可以在已安裝 iFolder 用戶端並設定共用 iFolder 的任何電腦上，始終保持檔案的最新版本。iFolder 還可讓您與已授權的使用者共用多個 iFolder 及其各自的內容。您可以決定每個 iFolder 的參與者以及每個成員的存取層級。同樣地，您也可以參與合作環境中其他人所擁有的共用 iFolder。

iFolder 用戶端支援共用的方式是透過中央電腦同步多台電腦間的檔案。它可讓您執行下列動作：

  • 在電腦間共用檔案
  • 與其他人共用檔案
  • 擁有多個 iFolder
  • 參與其他使用者與您共用的多個 iFolder
  • 作為 iFolder 的擁有者或成員參與多個帳戶
