啟動 iFolder 用戶端


啟動 iFolder for Novell Linux Desktop 9

  1. 使用您想在建立 iFolder 時使用的使用者識別登入電腦。

  2. 請使用下列方法之一來啟動 iFolder：

    • 開啟終端機外圍程序，然後輸入

      /opt/novell/ifolder3/bin/ifolder

    • 如果您需要在登入時自動啟動 iFolder，請確定當您登出 GNOME 會期時 iFolder 正在執行，然後儲存此會期。您也可以開啟終端機視窗、以根部使用者身份登入，然後執行 /opt/gnome/bin/gnome-session-properties，以將 iFolder 作為要在登入電腦時啟動的程式加入。

  3. 如果關閉 iFolder 帳戶的「記住密碼」，則必須登入帳戶以同步「使用者」清單及 iFolder，並管理 iFolder。

    若需要更多資訊，請參閱登入 iFolder 帳戶。


啟動 iFolder for Windows 2000/XP

  1. 使用您想在建立 iFolder 時使用的 Windows 使用者識別登入電腦。

  2. 如果在成功登入時沒有自動啟動 iFolder，則請依次按一下「開始」>「程式集」>「iFolder」以啟動 iFolder，然後選取 iFolder 應用程式。

    如果您關閉於「iFolder 優先設定」中設定的「登入桌面時啟動 iFolder」，則 iFolder 將不會自動啟動。

  3. 如果關閉 iFolder 帳戶的「記住密碼」，則必須登入帳戶以同步「使用者」清單及 iFolder，並管理 iFolder。

    若需要更多資訊，請參閱登入 iFolder 帳戶。


啟動 iFolder for Macintosh OS X v10.3 與更新版本

  1. 使用您想在建立 iFolder 時使用的使用者識別登入電腦。

  2. 請使用下列方法之一來啟動 iFolder：

    • 開啟「應用程式」功能表，選取「網際網路」子功能表，然後選取「iFolder」。
    • 若要從指令行啟動 iFolder，請開啟終端機外圍程序，然後輸入

      /opt/novell/ifolder3/bin/ifolder

    • 如果您想要在登入 Macintosh 電腦時自動啟動 iFolder，請將 iFolder 加入「啟動項目」。

  3. 如果關閉 iFolder 帳戶的「記住密碼」，則必須登入帳戶以同步「使用者」清單及 iFolder，並管理 iFolder。

    若需要更多資訊，請參閱登入 iFolder 帳戶。
