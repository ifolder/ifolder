放置 iFolder 的指示

通常，一旦建立 iFolder 或設定與您共用的 iFolder 後，您便可以將 iFolder 放置在本地硬碟的任何位置。但是，實際上有一些特定的限制。請於電腦中放置 iFolder 時考慮下列指示：

  • iFolder 的名稱必須是您放置 iFolder 之目錄中唯一的資料夾名稱。
  • iFolder 的名稱不能包含下列無效字元： 
    \/:*?"<>|;

    對於其他命名慣例，請參閱 iFolder 及其資料夾和檔案的命名慣例。

  • iFolder 必須位於本地硬碟中，該位置不能是網路磁碟機或非實體磁碟機。
  • iFolder 不能位於磁碟機的根目錄。
  • iFolder 只能從現有的資料夾建立。
  • iFolder 只能從您具有讀取和寫入檔案之檔案系統存取權限的資料夾中建立。當在電腦上設定共用的 iFolder 時，對要放置 iFolder 的資料夾您必須具有檔案系統存取權限。
  • iFolder 不能包含或位於系統資料夾內，例如 Windows 資料夾或 Linux 或 Mac 系統磁碟機。
  • iFolder 不能包含或位於應用程式資料夾中，例如「程式集」資料夾。
  • iFolder 不能包含或位於 iFolder 中繼資料資料夾。資料夾的預設位置如下：

    • Linux：在 .local\share 次目錄的主資料夾中

    • Windows：在 C:\Documents and Settings\WindowsLogonUsername\Local Settings\Application Data\simias 目錄中

    • Macintosh：在 .local\share 次目錄的主資料夾中

  • iFolder 不能包含或位於其他 iFolder 中，包括您的 Novell® iFolder® 2.1x iFolder。
