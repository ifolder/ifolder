設定 iFolder 帳戶

使用「iFolder 帳戶建立精靈」來新增和設定新帳戶。您必須至少設定一個 iFolder 帳戶，才能在系統上建立 iFolder 或共用 iFolder。 您一次只能使用一個帳戶連接到 iFolder 伺服器。然而，若要連接到多部 iFolder 伺服器，您可以設定多個帳戶。您可以分別登入各個帳戶，也可以同時登入多個帳戶。

使用不同本地登入身分的多位使用者將可在相同電腦上擁有 iFolder。第一次使用您目前的本地登入身分開啟 iFolder 時，iFolder 會提示您設定 iFolder 帳戶。

重要：iFolder 3.7 推出了一項可自動建立帳戶的新功能，此功能可協助 iFolder 管理員為您建立和設定帳戶。如果您的 iFolder 管理員啟用自動建立帳戶，您就不需要自行設定帳戶。當您首次啟動 iFolder 時，會提示您輸入密碼。輸入您的密碼並完成帳戶的設定。

設定新帳戶：

  1. 使用下列其中一種方法存取「iFolder 帳戶建立精靈」：

    • 第一次使用您目前的本地登入身分開啟 iFolder 時，iFolder 會提示您設定 iFolder 帳戶。按一下「下一步」直接進入「iFolder 帳戶建立精靈」。

    • 以滑鼠右鍵按一下通知區域中的「iFolder 服務」圖示，選取「帳戶」開啟「iFolder 優先設定」對話方塊至「帳戶」索引標籤，然後按一下「新增」開啟「iFolder 帳戶建立精靈」。

  2. 依照螢幕上的指示指定下列值：

    參數

    描述

    伺服器位址

    帳戶所在之 iFolder 企業伺服器的 DNS 名稱或 IP 位址。 從您的 iFolder 管理員取得此資訊。

    例如： svr21.example.com 或 192.168.1.1

    在 iFolder 使用者介面中，伺服器名稱一定會與其 IP 位址一起顯示。例如：iFolder - 192.168.1.1

    重要：您無法使用 iFolder 3.7 用戶端連接到 iFolder 3.2 伺服器。

    將此伺服器設為預設伺服器

    若有多部伺服器連接到您的帳戶，請選取「將此伺服器設為預設伺服器」核取方塊，將您在伺服器下拉式清單中選取的伺服器設定為建立或上載 ifolder 時的預設伺服器。

    使用者名稱

    指定此帳戶的使用者識別。 您的 iFolder 管理員會決定您應該使用的格式是您的網路使用者 ID (LDAP 通用名稱) 或電子郵件地址。

    例如，如果您的姓名是 John Smith 並且使用者 ID 是 jsmith 而電子郵件地址是 john.smith@example.com，您的管理員就會為伺服器進行設定，該設定會決定要在「使用者名稱」欄位中使用這兩種格式中的哪一種，來進行驗證。

    密碼

    指定使用者名稱的密碼。

    將透過安全通道 (SSL) 進行驗証。系統會提示您驗證及接受伺服器證書。

    記住密碼

    選取「在這台電腦上記住密碼」核取方塊可以在每次登入電腦時都自動登入到此 iFolder 帳戶。

    如果您的網路密碼發生了變更，則下一次 iFolder 嘗試登入到 iFolder 帳戶時自動驗證理所當然就會失敗。iFolder 會提示您使用新密碼登入並且允許您指定並儲存這個新密碼。

  3. 驗證您輸入的資訊。若要進行任何變更，請按一下「上一步」按鈕以重新瀏覽先前的頁面。按一下「連接」以繼續，並驗證您與伺服器的連接。

  4. 如果系統提示您「接受此伺服器的證書」，請按一下「詳細資料」檢視證書資訊，然後按一下「確定」接受。

    附註：建立帳戶期間，會透過 SSL 進行登入。系統提示您接受伺服器證書時，您必須驗證並接受該證書，才能成功連接。

  5. 若管理員已啟用加密規則，iFolder 會提示您輸入您的密碼片語。

    在「密碼片語」欄位中輸入您的密碼片語。

    再次於「重新輸入密碼片語」欄位中輸入密碼片語。

    選取「記住密碼片語」核取方塊以在目前工作階段自動存取加密的 iFolder 資料。若選取此選項，iFolder 可記住您的密碼片語以便在下次登入時使用。此密碼片語交換程序會自動完成。

  6. 從清單中選取想要的「復原」代理程式然後按一下「確定」。

  7. 請指定要建立預設 iFolder 的位置。

    根據預設，預設 iFolder 會儲存在 <users home directory>/ifolder/server_name/user_name。

  8. 請選取要建立的預設 iFolder 類型。若選擇「加密」，系統將提示您提供密碼片語。

    重要：您只能共用一般 iFolder。加密的 iFolders 無法共用。

    如果您的 iFolder 管理員尚未為您目前的帳戶啟用加密，您只能建立一般 iFolder，而若已啟用加密，預設會選取「一般」選項。

  9. 選取「安全同步」使您建立的 iFolder 中的檔案能夠進行安全同步。

    您可以為一般和加密的 iFolder 啟用安全通道。選取這個選項後，iFolder 可讓您使用安全通道進行伺服器和本地機器上 iFolder 間的同步。

    如果移除伺服器上的預設 iFolder，下次從任何複雜用戶端設定帳戶時，iFolder 都會提示您重新建立預設 iFolder。

    然而，若從本地機器移除預設 iFolder，則下次您設定該帳戶時，iFolder 會提示您下載預設 iFolder。

  10. 如果您已經設定預設 iFolder，您可以將它下載至本地機器。

    下載預設 iFolder，並按一下「下一步」繼續。

  11. 若您的系統上已經有和預設 ifolder 同名的資料夾，則系統會提示您合併兩個資料夾或將預設 ifolder 下載至不同位置。

    按一下「確定」合併兩個資料夾，或按一下「取消」將預設 iFolder 下載至不同位置

  12. 按一下「完成」關閉「iFolder 帳戶建立精靈」。

    iFolder 會同步指定帳戶的使用者清單並找出可下載的 iFolder。起初的下載可能需要數秒至數分鐘，視帳戶的使用者清單大小而定。

  13. 請確認設定的帳戶出現在帳戶清單中。

    1. 以滑鼠右鍵按一下通知區域中的「iFolder 服務」圖示，然後從功能表中選取「優先設定」開啟「iFolder 優先設定」對話方塊至「帳戶」索引標籤。

      連接至 iFolder 伺服器時，便會選取「線上」核取方塊。

    2. 關閉「iFolder 優先設定」對話方塊。

  14. 使用下列其中一個方法為帳戶設定 iFolder：

    • 以滑鼠右鍵按一下通知區域中的「iFolder 服務」圖示，然後選取「iFolder」開啟 iFolder 瀏覽器。 請繼續進行 建立和上傳 iFolder。

    • 如果此帳戶的 iFolder 可從伺服器下載，便會列在 <iFolder_server_name> 上 iFolder 底下的 iFolder 視窗。下載圖示代表有可供下載的 iFolder。請繼續進行 下載可用的 iFolder。
