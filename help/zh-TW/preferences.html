設定用戶端的 iFolder 優先設定

  1. 以滑鼠右鍵按一下通知區域中的「iFolder 服務」圖示，然後選取「優先設定」開啟「優先設定」對話方塊至「一般」索引標籤。

  2. 針對下列各項指定您的優先設定：

    參數

    描述

    登入桌面時啟動 iFolder (僅可於 Windows 用戶端中使用)

    選取此選項，這樣每當您以設定 iFolder 帳戶所使用的本地使用者名稱登入時，便會啟動 iFolder。對於已經開啟「記住密碼」的帳戶，會在指定的同步化間隔於背景發生同步化。

    取消選取此選項，可手動啟動 iFolder。

    成功建立 iFolder 時顯示確認對話方塊

    選取此核取方塊，這樣每當建立 iFolder 時，除了一般視覺線索 (如在 iFolder 清單中出現 iFolder) 之外，還會接收到確認訊息。

    取消選取此核取方塊，可停止接收確認訊息。

    啟動時隱藏 iFolder 主視窗

    取消選取此核取方塊可在啟動時顯示 iFolder 視窗。

    若要在啟動時隱藏 iFolder 視窗，請選取此核取方塊。

    顯示同步記錄

    若要在啟動時顯示同步記錄，請選取此核取方塊。

    若要在啟動時隱藏同步記錄，請取消選取此核取方塊。

    顯示規則違規通知

    選取此核取方塊可在違反規則時在通知區域顯示快顯訊息。

    共享 iFolders 時通知我

    選取此核取方塊，這樣每當接收到新的共享 iFolder 邀請時，就會在通知區域中顯示快顯訊息。

    若要檢視目前電腦上可下載的 iFolder 清單，則每當 iFolder 執行時，在 iFolder 瀏覽器中選取「顯示可用的 iFolder」。

    發生衝突時通知我

    選取此核取方塊，這樣每當同步化檔案發生衝突時，便會在通知區域中啟用快顯訊息。

    取消選取此核取方塊，可停用同步化衝突的快顯訊息。

    每當衝突發生時，當 iFolder 在執行中，會在 iFolder 顯示「iFolder 錯誤」 圖示。

    使用者加入 iFolder 網域時通知我

    選取此核取方塊，這樣每當使用者接受共享 iFolder 邀請時，就會在通知區域中顯示快顯訊息。

    取消選取此核取方塊可停用快顯訊息。

    藉由選取 iFolder，然後按一下「共享者」開啟「iFolder 內容」對話方塊到「共用」索引標籤，每當 iFolder 執行時，您可以檢視 iFolder 使用者的完整清單。

    同步化

    自動同步化 iFolder： 選取此核取方塊，便會啟用此電腦上屬於您目前本地登入或登入識別之所有 iFolder 的同步化。取消選取此核取方塊，可提用同步化。

    同步化 iFolder 間隔： 指定用於同步化此電腦上屬於您目前本地登入或登入識別之 iFolder 的最小間隔。指定值及單位。如果啟用這個選項，預設值是 5 分鐘。間隔最小可設為 5 秒鐘。

    如果您搭配使用 iFolder 用戶端與 Novell iFolder 3.7，則有效的最小同步化間隔一律為下列設定中最大的值：

    • 系統政策預設為零 (0)，除非存在使用者規則集。如果設定使用者規則，則使用者規則會覆寫系統規則，而不論使用者規則的值較之是大還是小。

    • 本地機器規則，或與伺服器同步化之用戶端機器的設定。

    • iFolder (集合) 規則。

  3. 按一下「關閉」以關閉「iFolder 優先設定」對話方塊。

有關更詳細的資訊，請參閱 法律聲明.
