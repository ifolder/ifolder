瞭解 iFolder 圖示

下列表格描述了 iFolder 圖示及相關的工作。

iFolder 圖示

描述

桌面「應用程式」功能表的「iFolder 3 用戶端」會以「iFolder 服務」圖示顯示。

當 iFolder 執行時，「iFolder 服務」圖示也會出現在桌面工作列的「通知」區域。以滑鼠右鍵按一下可能工作的功能表，包括「iFolder 瀏覽器」、「帳戶設定」、「同步記錄」、「優先設定」以及「說明」。

iFolder 瀏覽器中會顯示「建立 iFolder」圖示。

您可以建立新的 iFolder 或將現有資料夾轉換為 iFolder。如果擁有多個帳戶，您必須指定要使用的帳戶；iFolder 僅能屬於一個帳戶所有。 如需更多資訊，請參閱建立和上傳 iFolder。

當指定之伺服器上有可供您使用的 iFolder 但您尚未將它下載到您的本地機器時，會顯示「iFolder 下載」圖示。

您可以將 iFolder 的組合從不同的帳戶下載至本地電腦。如需更多資訊，請參閱下載可用的 iFolder。

「已加密 iFolder」圖示 (鎖定的 iFolder) 表示 iFolder 已加密，並且顯示在 iFolder 瀏覽器中。

登入時提供的密碼片語可讓您取回已加密之 iFolder 中的檔案。如需有關加密的詳細資訊，請參閱 管理已加密 iFolder 的密碼片語。

iFolder 圖示 (綠色的「i」) 會在「iFolder 服務」應用程式為檔案管理員、桌面上或 iFolder 瀏覽器中的 iFolder 執行時，顯示在 iFolder 上。

當 iFolder 未執行時，則在檔案管理員或桌面上會顯示為一般資料夾。iFolder 未執行時無法使用 iFolder 瀏覽器。

建議您在僅含有 iFolder 的個別目錄中，組織管理您的 iFolder 本地集合。 如此一來，當 iFolder 未執行且將資料夾標示為 iFolder 時，也能輕鬆記住哪些資料夾為 iFolder。

「iFolder 警告」圖示表示伺服器無法使用。iFolder 正在執行但您未連接到 iFolder 伺服器。可能的原因有：

  • 您並未登入 iFolder 帳戶。如需更多資訊，請參閱登入 iFolder 帳戶。

  • 帳戶已在用戶端停用，代表「自動連接」參數已停用。 如需詳細資訊，請參閱 檢視和修改 iFolder 帳戶設定 中的「自動連接」。

  • 伺服器關閉或您的帳戶已在伺服器端停用。發生這種情況時，請向您的 iFolder 管理員尋求協助。

  • 您的防火牆封鎖了 iFolder 流量。 如需有關解決這個問題的詳細資訊，請參閱 設定 iFolder 流量的本地防火牆設定。

  • 網路連接故障。請檢查您的本地網路設定。

「iFolder 衝突」圖示表示本機副本和伺服器副本之間有檔案衝突。指出要保留的檔案版本即可解決衝突。如需有關檔案衝突的詳細資訊，請參閱 解決檔案衝突。

「iFolder 等待同步化」圖示表示 iFolder 包含要與伺服器同步的檔案，而且同步化程序尚未開始。

iFolder 會一次同步一個 iFolder，並將同步工作排於佇列中。 為了確保本地 iFolder 中的檔案與伺服器上的檔案同步，請依照圖示變更的指示，等待同步開始且順利完成再關閉 iFolder。

「iFolder 同步化」圖示表示 iFolder 中的檔案目前正在與伺服器同步。

「iFolder 錯誤」圖示表示因為檔案大小限制或 iFolder 數目限制造成規則衝突，導致 iFolder 中有一個或多個檔案尚未同步。

在「共用」對話方塊中，「iFolder 擁有者使用者」圖示表示使用者擁有共用 iFolder。

在「共用」對話方塊中，「iFolder 目前使用者」圖示表示使用者目前正在管理 iFolder。如果目前使用者是擁有者，則會改為顯示「擁有者使用者」圖示。

有關更詳細的資訊，請參閱 法律聲明.
