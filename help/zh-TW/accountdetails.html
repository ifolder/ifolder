檢視和修改 iFolder 帳戶設定

下列參數可在「帳戶設定」和「帳戶內容」對話方塊中檢視：

參數

描述

位置

使用者可修改 (是/否)

線上

代表您目前是否連接至 iFolder 伺服器中的帳戶。

選取帳戶旁的線上核取方塊以連接至 iFolder 伺服器的此帳戶。

請不選帳戶旁的線上核取方塊以中斷 iFolder 伺服器的連接。

iFolder 優先設定 > 帳戶索引標籤

是

伺服器名稱

顯示 iFolder 伺服器中您的帳戶儲存 iFolder 位置的描述性名稱。此名稱是由伺服器提供並由 iFolder 管理員控制。

使用者不能修該此項設定。

iFolder 優先設定 > 帳戶索引標籤

帳戶內容 > 伺服器索引標籤

否

使用者名稱

顯示此帳戶的使用者識別，例如您的網路使用者 ID (LDAP 一般名稱) 或您的電子郵件地址。 此設定無法修改。

當您在「iFolder 帳戶助理」中設定帳戶時會指定使用者名稱。若為現有帳戶便無法修改。

若要在相同伺服器上使用不同的使用者名稱，您必須刪除現有帳戶，然後為該名稱建立不同的帳戶。

iFolder 優先設定 > 帳戶索引標籤

帳戶內容 > 識別 索引標籤

否

位址

顯示帳戶所在之 iFolder 企業伺服器的 DNS 名稱或 IP 位址。 從您的 iFolder 管理員取得此資訊。

例如： svr21.example.com 或 192.168.1.1

您可以修改此位址，不過必須在進行變更時中斷 iFolder 伺服器的連接。

帳戶內容 > 伺服器索引標籤

是

密碼

可讓您指定使用者名稱的密碼。

帳戶內容 > 識別索引標籤

是

記住密碼

代表當 iFolder 開啟時，iFolder 便會自動登入。如果帳戶的「自動連接」 參數已啟用 (預設值)，則帳戶便會嘗試使用儲存的密碼連接。

選取「記住密碼」可以在每次登入電腦時都自動登入到此 iFolder 帳戶。

如果您的網路密碼發生了變更，則下一次 iFolder 嘗試登入到 iFolder 帳戶時自動驗證理所當然就會失敗。iFolder 會提示您使用新密碼登入並且允許您指定並儲存這個新密碼。

帳戶內容 > 識別索引標籤

是

容量

回報企業伺服器上配置給您的 iFolder 帳戶的總空間量。此值可能是為所有使用者設定的預設容量，或是使用者指定的容量。如果未設定容量，該值就為空或 N/A (不適用)。您也可以在 iFolder 主視窗中檢視每個已連接伺服器的可用空間。

帳戶內容 > 磁碟空間索引標籤

否

已使用

回報此帳戶上所擁有之所有 iFolder 目前已消耗的總空間量。其並未包含 iFolder 與您共用但由其他人擁有所使用的空間。

帳戶內容 > 磁碟空間索引標籤

否

Available

回報目前可用於此 iFolder 帳戶之 iFolder 的總空間量。該值等於容量減去已用空間。

帳戶內容 > 磁碟空間索引標籤

否

自動連線

如果選取此值 (啟用)，iFolder 會在開啟時自動連接至此帳戶。不選該設定會停止帳戶的連接。

此設定僅控制啟用或停用此帳戶的用戶端功能。

當您使用 Novell® iFolder® 3.7 伺服器的 iFolder 時，iFolder 系統管理員就可以在伺服器端關閉您的 iFolder 帳戶 (通常是暫時地)。當此情況發生時，您可以預期下列的行為：

  • 如果您在該帳戶從伺服器端上關閉時登入，就可以管理 iFolder，而您的 iFolder 資料會繼續與伺服器同步，直到您通過登出或者關閉同步來結束會期，或者是伺服器切斷與會期的連接為止 (視哪一個先發生而定)。

  • 如果帳戶是在伺服器端關閉的，您就會接收到一個立現式訊息，告知您每次嘗試登入時帳戶都是關閉的。請聯絡您的 iFolder 系統管理員尋求協助。

帳戶內容，所有索引標籤上

是

帳戶為預設

當您擁有多個帳戶時，請選取「帳戶為預設」核取方塊，讓此選取的帳戶在您建立和上傳 iFolder 時，在帳戶的下拉式清單中為預設。

如果您只有單一帳戶，則可修改此值。

帳戶內容，所有索引標籤上

是，如果您有多個帳戶

加密設定

您可以在「安全性」功能表底下，設定和重設加密設定。如需有關加密設定的詳細資訊，請參閱 管理已加密 iFolder 的密碼片語

保全性功能表

使用者可重設通關密語

  1. 以滑鼠右鍵按一下通知區域中的「iFolder 服務」圖示，然後選取「帳戶設定」開啟「iFolder 優先設定」至「帳戶」索引標籤。

    在「帳戶」報告中，您可以管理下列每個帳戶的參數：

    • 線上 (選取以連接；不選以中斷連接)

    • 伺服器名稱 (僅能檢視)

    • 使用者名稱 (僅檢視)

  2. 選取您要管理的 iFolder 帳戶，然後按一下「內容」開啟帳戶內容對話方塊至「伺服器」索引標籤。

    您可以在「伺服器」索引標籤上，管理下列選取帳戶的參數：

    • 名稱 (僅檢視)

    • 位址 (若連接則為僅檢視；若中斷連接則可修改)

    • 伺服器描述 (僅檢視；捲動可檢視所有資訊)

    • 自動連接 (選取可啟用；不選則停用)

    • 帳戶為預設 (選取可啟用；僅存在多個帳戶時可使用)

  3. 選取「識別」索引標籤以管理下列選取帳戶的參數：

    • 使用者名稱 (僅檢視)

    • 密碼 (指定密碼)

    • 記住密碼 (選取可啟用；不選則停用))

    • 自動連接 (選取可啟用；不選則停用)

    • 帳戶為預設 (選取可啟用；僅存在多個帳戶時可使用)

  4. 選取「磁碟空間」索引標籤以檢視下列選取帳戶的參數：

    • 配額 (僅檢視)

    • 已使用 (僅檢視)

    • 可用 (僅檢視)

    • 自動連接 (選取可啟用；不選則停用)

    • 帳戶為預設 (選取可啟用；僅存在多個帳戶時可使用)

  5. 當您完成時，請關閉「帳戶內容」對話方塊，然後關閉「iFolder 優先設定」對話方塊。

有關更詳細的資訊，請參閱 法律聲明.
