Pesquisar Itens Marcados Para

Para pesquisar itens específicos no Conteúdo desta documentação

  • Verifique se a caixa perto de cada item que deseja pesquisar está marcada.
    •

  • Digite as palavras-chave a serem pesquisadas (são permitidos curingas ? e *) na caixa de texto. (Mais)

  • Clique no ícone de seta, "Pesquisar os Itens Marcados Para," ou pressione enter.

Por padrão, todos os itens são marcados automaticamente para você. Clique na caixa perto de cada item que não deseja pesquisar para desmarcá-lo.

Se nenhum item for marcado, um aviso será emitido.




Imprimir Itens Marcados

Para concatenar e imprimir itens específicos desta documentação

  • Verifique se a caixa perto de cada item que deseja imprimir está marcada.
    •

  • Clique em "Imprimir Itens Marcados".

Por padrão, todos os itens são marcados automaticamente para você. Clique na caixa perto de cada item que não deseja imprimir para desmarcá-lo.

Se nenhum item for marcado, um aviso será emitido.
