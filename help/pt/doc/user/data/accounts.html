Configurando uma conta do iFolder

Você deve configurar pelo menos uma conta do iFolder para poder criar pastas iFolder ou compartilhar pastas iFolder no sistema. É possível criar somente uma conta para qualquer computador ou servidor host, mas você pode ter várias contas.

Um usuário com um login local e uma conta de usuário diferentes pode efetuar login no mesmo computador várias vezes e utilizar o iFolder para criar seu próprio conjunto de pastas iFolder sob tal identidade. O iFolder sincroniza somente pastas iFolder que pertençam ao usuário ativo.

  1. Use um dos métodos a seguir para acessar a configuração de conta:

    • Na primeira vez em que você iniciar o iFolder após a instalação, o iFolder solicitará que você configure uma conta do iFolder. Clique em Sim para ir diretamente para as Preferências do iFolder > guia Contas.
    • Clique o botão direito do mouse no ícone na área de Notificação e, em seguida, selecione Contas para abrir a caixa de diálogo Preferências do iFolder na guia Contas.

  2. Clique em Novo para acessar a área Configurações de Conta.

  3. Especifique os seguintes valores:

    Parâmetro Descrição

    Servidor

    O nome DNS ou endereço IP do servidor empresarial onde você tem uma conta. Por exemplo: svr21.example.com ou 192.168.1.1.

    Nome de Usuário

    Informe sua identidade de usuário para esta conta. O seu administrador do iFolder decide se o formato que você deve usar é o seu ID de usuário na rede (nome comum LDAP) ou o seu endereço de e-mail.

    Por exemplo, se o seu nome for John Smith com o ID de usuário jsmith e o endereço de e-mail john.smith@example.com, o administrador configura uma definição para o servidor que determinará qual dos dois formatos deve ser aceito no campo Nome de Usuário para propósitos de autenticação.

    Senha

    A senha para o seu nome de usuário.

    Lembrar Senha

    Selecione Lembrar Senha para efetuar login automaticamente nesta conta do iFolder sempre que efetuar login no computador. Para obter informações, consulte Efetuando login em uma conta do iFolder.

    Se a sua senha de rede mudar, a autenticação automática falhará normalmente na próxima vez em que o iFolder tentar efetuar login na conta do iFolder. O iFolder solicitará que você efetue login com a nova senha e permitirá que você especifique e grave a nova senha.

    Habilitar Conta

    Selecione Habilitar Conta para permitir a sincronização em segundo plano dos convites do iFolder, da lista de usuários e das pastas iFolder para esta conta.

    Conta Padrão

    Selecione Conta Padrão para tornar esta conta a conta selecionada por padrão na lista suspensa de contas quando você criar pastas iFolder.

  4. Clique em OK ou efetue login.

    Se um certificado for solicitado, verifique o certificado e clique em OK para aceitá-lo. A conta é mostrada na lista de contas com status Conectado. O iFolder começa a sincronização de quaisquer pastas iFolder existentes, notificações do iFolder e da lista de usuários para aquela conta.

  5. Clique em OK para fechar a caixa de diálogo Preferências do iFolder.

  6. Configure pastas iFolder para a conta utilizando um dos seguintes métodos:

    • Se as pastas iFolder estiverem disponíveis para a conta, aparecerá uma mensagem de Notificação do iFolder. Clique na mensagem para abrir a caixa de diálogo iFolder. Continue com Configurando um iFolder disponível.
    • Se não existirem pastas iFolder para a conta ou se você quiser começar a adicionar pastas iFolder imediatamente, clique o botão direito do mouse no ícone do aplicativo iFolder e, em seguida, selecione Pastas iFolder para abrir a janela Pastas iFolder. Continue com Criando um iFolder.
