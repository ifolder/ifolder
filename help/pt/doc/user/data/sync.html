Sincronizando arquivos


Sincronizando arquivos sob solicitação

  1. Na área de Notificação, clique o botão direito do mouse no ícone do aplicativo iFolder e, em seguida, clique em Pastas iFolder para abrir o browser de pastas iFolder.

  2. Na lista de pastas iFolder, selecione a pasta iFolder.

  3. Execute uma das seguintes ações:

    • Clique em Sincronizar .
    • Selecione iFolder > Sincronizar Agora.


Configurando o intervalo de sincronização

  1. Na área de Notificação, clique o botão direito do mouse no ícone do aplicativo iFolder e, em seguida, clique em Preferências para abrir a caixa de diálogo Preferências do iFolder na guia Geral.

  2. Execute uma das seguintes ações:

    • Selecione Sincronizar para habilitar a sincronização em segundo plano.
    • Desmarque Sincronizar para desabilitar a sincronização.

  3. Se você habilitar a sincronização, especifique o intervalo de sincronização dos arquivos, como a cada uma hora ou a cada dois dias.

    Quando o cliente do iFolder é usado com o Novell iFolder 3.x, o intervalo de sincronização mínimo do sistema, normalmente a cada cinco minutos, é configurado pelo administrador do iFolder. Você não pode anular o mínimo do sistema. O mínimo absoluto do sistema é 1 minuto.
