Configurando um iFolder disponível

Quando outras pessoas compartilham pastas iFolder com você, as pastas iFolder aparecem com um ícone iFolder Disponível no browser de pastas iFolder. Para participar do iFolder compartilhado em seu computador local atual, configure o iFolder para torná-lo ativo no computador.

A localização do iFolder local deve satisfazer as seguintes diretrizes:

  • Diretrizes para localização de pastas iFolder
  • Convenções de nomeação para um iFolder e seus arquivos e pastas

Para configurar um iFolder:

  1. Na área de Notificação, clique o botão direito do mouse no ícone do iFolder e, em seguida, clique em Pastas iFolder para abrir o browser de pastas iFolder.

  2. Na lista de pastas iFolder, selecione o iFolder disponível e, em seguida, clique em Configurar .

  3. Se você não estiver conectado na conta, efetue login com seu nome de usuário e senha e, em seguida, clique em OK.

  4. Navegue até a localização onde deseja criar um iFolder local.

  5. Clique em OK duas vezes.
