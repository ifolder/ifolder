Configurando as preferências do iFolder para o cliente

  1. Clique o botão direito do mouse no ícone do aplicativo iFolder na área de Notificação e, em seguida, selecione Preferências para abrir a caixa de diálogo Preferências do iFolder na guia Geral.


  2. Especifique os seguintes valores:

    Parâmetro Descrição

    Iniciar iFolder Quando Efetuar Login na Área de Trabalho

    Selecione essa opção para iniciar o iFolder sempre que efetuar login com o nome de usuário local onde você configurar a conta do iFolder. Naquelas contas onde você habilitou a opção Lembrar Senha, a sincronização acontecerá em segundo plano no intervalo de sincronização especificado.

    Desmarque esta opção para iniciar o iFolder manualmente.

    Mostrar Caixa de Diálogo de Confirmação ao Criar Pastas iFolder

    Selecione essa opção para receber uma mensagem de confirmação sempre que criar um iFolder, além das dicas visuais normais, como o iFolder na lista de pastas iFolder.

    Desmarque essa opção para interromper o recebimento da mensagem de confirmação.

    Notificar Sobre Pastas iFolder Compartilhadas

    Selecione essa opção para habilitar a mensagem pop-up na área de Notificação sempre que você receber um novo convite para compartilhar uma pasta iFolder.

    Os convites são recuperados com base no intervalo de sincronização especificado nas Preferências do iFolder. O intervalo controla a sincronização de arquivos, os convites de compartilhamento do iFolder e a lista de usuários. Você também pode clicar em Atualizar para atualizar os convites sob solicitação.

    Desmarque essa opção para desabilitar o comportamento da mensagem pop-up para convites.

    Notificar Sobre Conflitos

    Selecione essa opção para habilitar uma mensagem pop-up na área de Notificação sempre que ocorrer um conflito durante a sincronização de arquivos.

    Desmarque esta opção para desabilitar o comportamento da mensagem pop-up para conflitos de sincronização.

    Notificar da Entrada de um Usuário

    Selecione essa opção para habilitar a mensagem pop-up na área de Notificação sempre que um usuário aceitar um convite para compartilhar um iFolder. A aceitação é indexada para quando o usuário realmente configurar o iFolder em seu computador.

    Desmarque essa opção para desabilitar o comportamento da mensagem pop-up para confirmação de compartilhamento de usuário.

    Sincronização

    Sincronizar Pastas iFolder Automaticamente: Marque essa caixa de seleção para habilitar a sincronização para todas as pastas iFolder no computador sob a sua identidade de login/logon local. Desmarque a caixa de seleção para desabilitar a sincronização.

    Sincronizar Pastas iFolder a Cada: Especifique o intervalo mínimo a ser usado para a sincronização de pastas iFolder no computador sob a sua identidade de login/logon local. Especifique o valor e as unidades.

    Se você usar o cliente iFolder com o Novell iFolder 3.x, o intervalo de sincronização mínimo efetivo será sempre o maior valor das seguintes configurações:

    • A política do sistema (padrão igual a zero), a não ser que exista uma política do usuário definida. Se uma política do usuário estiver definida, sendo ela maior ou menor no valor, ela anulará a política do sistema.
    • A política da máquina local ou a configuração na máquina do cliente sincronizando com o servidor.
    • A política do iFolder (coleção).

  3. Ao terminar, clique em Aplicar para aplicar suas mudanças.

  4. Clique em OK para fechar a caixa de diálogo Preferências do iFolder.
