Löschen von iFolder-Konten

Sie können angeben, ob ein Konto nur vom aktuellen Computer oder auch aus Ihren iFolder-Ordnern vom Server entfernt werden soll.

Wenn Sie ein iFolder-Konto nur vom Computer entfernen:

  • Die lokalen iFolder-Ordner für dieses Konto werden auf normale Ordner zurückgesetzt. Wenn die lokale Kopie der Daten nicht benötigt wird, machen Sie sie mithilfe eines Datei-Managers ausfindig und löschen Sie sie.

  • Von anderen Computern aus bzw. über den iFolder-Client können Sie weiterhin auf die iFolder-Ordner zugreifen .

  • Wenn der iFolder-Client mit einem Novell iFolder-Enterprise-Server der Version 3.7 oder höher verwendet wird, ist außerdem der Zugriff mit Novell iFolder 3.7 Web Access (oder höher) von allen Standorten aus möglich.

Wenn Sie Ihr iFolder-Konto vom Server entfernen, geschieht Folgendes:

  • Die Freigabe sämtlicher iFolder-Ordner, deren Eigentümer Sie sind, wird aufgehoben.

  • Die lokalen Kopien von iFolder-Ordnern auf Mitgliedercomputern werden auf normale Ordner zurückgesetzt. Wenn die lokale Kopie der Daten nicht benötigt wird, machen Sie sie mithilfe eines Datei-Managers ausfindig und löschen Sie sie.

  • Der iFolder-Ordner und sein Inhalt werden vom Server entfernt.

  • Sie werden als Mitglied von iFolder-Ordnern entfernt, die andere Benutzer zuvor für Sie freigegeben haben.

  • Das Konto wird vom lokalen Computer entfernt.

Wenn Sie dieses Konto auf diesem oder einem anderen Computer erneut verwenden möchten, müssen Sie hinsichtlich der Einrichtung des Kontos und der zugehörigen iFolder-Ordner genauso vorgehen wie bei der Neuerstellung.

So löschen Sie ein iFolder-Konto:

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Dienstesymbol und wählen Sie dann Kontoeinstellungen aus, um das Dialogfeld mit den iFolder-Einstellungen auf der Registerkarte Konten zu öffnen.

  2. Wählen Sie das zu löschende iFolder-Konto aus und klicken Sie dann auf Entfernen.

  3. In einer Meldung werden Sie aufgefordert, den Umfang der Entfernen-Aktion zu bestimmen.

    Führen Sie einen der folgenden Vorgänge aus:

    • Entfernen des iFolder-Kontos nur vom aktuellen Computer: Vergewissern Sie sich, dass Meine iFolder-Ordner und Dateien vom Server entfernen deaktiviert ist, und klicken Sie anschließend auf Ja.

    • Entfernen des iFolder-Kontos und der iFolder-Ordner und -Dateien vom Server: Wählen Sie Meine iFolder-Ordner und Dateien vom Server entfernen aus und klicken Sie anschließend auf Ja.

    • Abbrechen des Löschvorgangs: Klicken Sie auf Nein, um die Löschaktion abzubrechen. Ihr Konto und die zugehörigen iFolder-Ordner bleiben unverändert.
