Anzeigen und Bearbeiten von iFolder-Konto-Einstellungen

Die folgenden Parameter können in den Dialogfeldern Kontoeinstellungen und Kontoeigenschaften angezeigt werden:

Parameter

Beschreibung

Standort

Vom Benutzer zu ändern (Ja/Nein)

Online

Zeigt an, ob Sie aktuell für ein Konto mit dem iFolder-Server verbunden sind.

Aktivieren Sie das Kontrollkästchen Online neben dem Konto, um für dieses Konto eine Verbindung zum iFolder-Server herzustellen.

Deaktivieren Sie das Kontrollkästchen Online neben dem Konto, um die Verbindung zum iFolder-Server zu beenden.

Registerkarte iFolder-Einstellungen > Konten

Ja

Servername

Zeigt den beschreibenden Namen des iFolder-Servers an, auf dem Ihre iFolders für das Konto gespeichert sind. Dieser Name wird vom Server bereitgestellt und vom iFolder-Administrator gesteuert.

Diese Einstellung kann von Benutzern nicht geändert werden.

Registerkarte iFolder-Einstellungen > Konten

Registerkarte Kontoeinstellungen > Server

Nein

Benutzername

Zeigt die Benutzeridentität für dieses Konto an, z. B. Ihre Netzwerk-Benutzer-ID (LDAP-Eigenname) oder Email-Adresse. Diese Einstellung kann nicht geändert werden.

Der Benutzername wird angegeben, wenn Sie das Konto im iFolder-Kontoassistenten einrichten. Für ein vorhandenes Konto kann er nicht geändert werden.

Um einen anderen Benutzernamen auf dem Server zu verwenden, müssen Sie das vorhandene Konto löschen und dann ein neues Konto erstellen.

Registerkarte iFolder-Einstellungen > Konten

Registerkarte Kontoeinstellungen > Identität

Nein

Adresse

Zeigt den DNS-Namen oder die IP-Adresse des iFolder-Enterprise-Servers an, auf dem Sie ein Konto haben. Fragen Sie Ihren iFolder-Administrator nach diesen Informationen.

Beispiel: svr21.beispiel.com oder 192.168.1.1

Sie können die Adresse ändern, müssen aber vom iFolder-Server getrennt sein, wenn Sie die Änderung vornehmen.

Registerkarte Kontoeinstellungen > Server

Ja

Passwort

Ermöglicht Ihnen die Angabe des Passworts für Ihren Benutzernamen.

Registerkarte Kontoeinstellungen > Identität

Ja

Passwort speichern

Zeigt an, ob iFolder Sie automatisch anmeldet, wenn iFolder gestartet wird. Wenn der Parameter Automatisch verbinden für das Konto aktiviert ist (Standard), wird versucht, eine Verbindung mit dem gespeicherten Passwort herzustellen.

Aktivieren Sie die Option Passwort speichern, um bei jeder Anmeldung an Ihrem Computer automatisch bei diesem iFolder-Konto angemeldet zu werden.

Wenn sich Ihr Netzwerkpasswort ändert, tritt ein Fehler auf, wenn iFolder das nächste Mal versucht, sich bei dem iFolder-Konto anzumelden. iFolder fordert Sie zur Anmeldung mit Ihrem neuen Passwort auf und ermöglicht Ihnen die Angabe und Speicherung des neuen Passworts.

Registerkarte Kontoeinstellungen > Identität

Ja

Quote

Zeigt den Gesamtspeicherplatz an, der Ihrem iFolder-Konto auf dem Enterprise-Server zugeordnet wurde. Bei diesem Wert kann es sich um die für alle Benutzer festgelegten Standardquote oder eine vom Benutzer angegebene Quote handeln. Wenn keine Quote festgelegt wurde, ist kein Wert angegeben oder lautet N/Z (Nicht zutreffend). Sie können im iFolder-Hauptfenster auch für jeden Server, mit dem Sie verbunden sind, den verfügbaren Speicherplatz anzeigen.

Registerkarte Kontoeigenschaften > Speicherplatz

Nein

Belegt

Zeigt den Gesamtspeicherplatz an, der zurzeit von allen iFolder-Ordnern verwendet wird, die Sie für dieses Konto besitzen. Speicherplatz, der von iFolder-Ordnern verwendet wird, die Sie nutzen, sich aber im Besitz anderer Benutzer befinden, sind darin nicht enthalten.

Registerkarte Kontoeigenschaften > Speicherplatz

Nein

Verfügbar

Zeigt den Gesamtspeicherplatz an, der Ihnen zurzeit in diesem iFolder-Konto für iFolder-Ordner zur Verfügung steht. Der Wert entspricht der Speicherquote abzüglich des belegten Speichers.

Registerkarte Kontoeigenschaften > Speicherplatz

Nein

Automatisch verbinden

Wenn diese Einstellung ausgewählt (aktiviert) wird, stellt iFolder automatisch eine Verbindung zu diesem Konto her, wenn iFolder gestartet wird. Deaktivieren Sie die Einstellung, damit keine Verbindung zum Konto hergestellt wird.

Diese Einstellung gilt nur für die clientseitige Fähigkeit, das Konto zu aktivieren oder zu deaktivieren.

Wenn Sie iFolder mit einem Novell® iFolder® 3.7-Server verwenden, kann der iFolder-Systemadministrator Ihr iFolder-Konto auf der Serverseite deaktivieren. In diesem Fall können Sie mit dem folgenden Verhalten rechnen:

  • Wenn Sie angemeldet sind und Ihr Konto auf der Serverseite deaktiviert wird, können Sie Ihre iFolder-Ordner verwalten und Ihre iFolder-Daten werden weiterhin mit dem Server synchronisiert, bis Sie Ihre Sitzung beenden, indem Sie sich abmelden, bis Sie die Synchronisierung deaktivieren oder bis der Server Ihre Sitzungsverbindung trennt, je nachdem, welcher Fall zuerst eintritt.

  • Wenn das Konto auf der Serverseite deaktiviert wird, werden Sie bei jedem Anmeldeversuch in einer Popup-Meldung darauf hingewiesen, dass Ihr Konto deaktiviert ist. Wenden Sie sich an den für Sie zuständigen iFolder-Systemadministrator.

Kontoeigenschaften, auf allen Registerkarten

Ja

Konto ist Standard

Wenn Sie über mehrere Konten verfügen, aktivieren Sie das Kontrollkästchen für Konto ist Standard, damit dieses Konto in der Dropdown-Liste der Konten automatisch ausgewählt wird, wenn Sie iFolder-Ordner erstellen und hochladen.

Dieser Wert kann nicht geändert werden, wenn Sie nur über ein Konto verfügen.

Kontoeigenschaften, auf allen Registerkarten

Ja, wenn Sie über mehrere Konten verfügen

Verschlüsselungseinstellungen

Ihre Verschlüsselungseinstellungen können Sie im Menü Sicherheit festlegen und zurücksetzen. Weitere Informationen zu den Verschlüsselungseinstellungen finden Sie unter Verwalten von Passwortsätzen für verschlüsselte iFolder-Ordner

Menü Sicherheit

Benutzer kann den Passwortsatz zurücksetzen

  1. Klicken Sie mit der rechten Maustaste auf das iFolder-Dienstesymbol im Benachrichtigungsbereich und wählen Sie Kontoeinstellungen, um die Registerkarte Konten der iFolder-Einstellungen zu öffnen.

    Im Kontenbericht können Sie die folgenden Parameter für die einzelnen Konten verwalten:

    • Online (zum Verbinden aktivieren, zum Trennen der Verbindung deaktivieren)

    • Servername (nur anzeigen)

    • Benutzername (nur anzeigen)

  2. Wählen Sie das zu verwaltende iFolder-Konto aus und klicken Sie dann auf Eigenschaften, um das Dialogfeld Kontoeigenschaften auf der Registerkarte Server zu öffnen.

    Auf der Registerkarte Server können Sie die folgenden Parameter für das ausgewählte Konto verwalten:

    • Name (nur anzeigen)

    • Adresse (wird nur bei Verbindung angezeigt; kann ohne Verbindung bearbeitet werden)

    • Serverbeschreibung (nur anzeigen; blättern, um alle Informationen anzuzeigen)

    • Automatisch verbinden (zum Aktivieren auswählen, zum Deaktivieren nicht auswählen)

    • Konto ist Standard (zum Aktivieren auswählen; nur verfügbar, wenn mehrere Konten vorhanden sind)

  3. Wählen Sie die Registerkarte Identität aus, um folgende Parameter für das ausgewählte Konto zu verwalten:

    • Benutzername (nur anzeigen)

    • Passwort (geben Sie das Passwort an)

    • Passwort speichern (zum Aktivieren auswählen, zum Deaktivieren nicht auswählen))

    • Automatisch verbinden (zum Aktivieren auswählen, zum Deaktivieren nicht auswählen)

    • Konto ist Standard (zum Aktivieren auswählen; nur verfügbar, wenn mehrere Konten vorhanden sind)

  4. Wählen Sie die Registerkarte Speicherplatz aus, um folgende Parameter für das ausgewählte Konto anzuzeigen:

    • Quote (nur anzeigen)

    • Verwendet (nur anzeigen)

    • Verfügbar (nur anzeigen)

    • Automatisch verbinden (zum Aktivieren auswählen, zum Deaktivieren nicht auswählen)

    • Konto ist Standard (zum Aktivieren auswählen; nur verfügbar, wenn mehrere Konten vorhanden sind)

  5. Wenn Sie fertig sind, schließen Sie das Dialogfeld Kontoeigenschaften und schließen Sie dann das Dialogfeld iFolder-Einstellungen.
