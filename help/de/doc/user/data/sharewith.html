Freigeben von iFolder-Ordnern

Sie können einen iFolder-Ordner gemeinsam mit mehreren Benutzern verwenden. In diesem Abschnitt werden die folgenden Aufgaben behandelt:

  • Benutzerzugriffsrechte
  • Hinzufügen von Benutzern zu iFolder-Ordnern
  • Bearbeiten von Benutzerzugriffsrechten
  • Entfernen von Benutzern aus iFolder-Ordnern

Hinzufügen von Benutzern zu iFolder-Ordnern


Hinzufügen eines Benutzers mit einem Datei-Manager

  1. Navigieren Sie im Datei-Manager zum iFolder-Ordner und wählen Sie ihn dann aus.

  2. Klicken Sie mit der rechten Maustaste auf den iFolder-Ordner und klicken Sie dann auf "iFolder" > "Eigenschaften".

  3. Klicken Sie auf "Gemeinsam verwenden mit" .

  4. Wählen Sie auf der Liste der Benutzer einen oder mehrere Benutzer aus.

  5. Wählen Sie das Zugriffsrecht aus: "Vollständige Kontrolle", "Lesen/Schreiben" oder "Nur Lesen".

  6. Klicken Sie auf "Anwenden".


Hinzufügen eines Benutzers mit dem Browser für iFolder-Ordner

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Symbol () und klicken Sie dann auf "iFolder-Ordner", um den Browser für iFolder-Ordner zu öffnen.

  2. Wählen Sie in der Liste der iFolder-Ordner den iFolder-Ordner aus und klicken Sie dann auf "Freigeben" , um das Fenster "iFolder-Eigenschaften" mit der Registerkarte "Freigabe" zu öffnen.

  3. Klicken Sie zum Öffnen des Dialogfelds "Benutzer auswählen" auf "Hinzufügen".

  4. Wählen Sie aus der Liste der Benutzer einen oder mehrere Benutzer aus, indem Sie sie zur Liste "Ausgewählte Benutzer" hinzufügen, und klicken Sie anschließend auf "OK".

    Es kann einige Sekunden dauern, bis die Einladungen auf dem Server synchronisiert werden.

  5. Wählen Sie einen oder mehrere Benutzer aus, klicken Sie auf "Zugriff", wählen Sie das Zugriffsrecht "Vollständige Kontrolle", "Lesen/Schreiben" oder "Nur Lesen" aus und klicken Sie dann auf "OK".

  6. Wiederholen Sie die Schritte 4 und 5, bis Sie alle gewünschten Benutzer zur gemeinsamen Verwendung des iFolder-Ordners eingeladen haben.

  7. Klicken Sie auf "Anwenden" und klicken Sie danach auf "OK", um das Fenster "iFolder-Eigenschaften" zu schließen.


Bearbeiten von Benutzerzugriffsrechten

Da sich die Arbeitsbeziehungen und der Status der Mitglieder eines iFolder-Ordners ändern, müssen Sie möglicherweise das Zugriffsrecht eines Benutzers für den iFolder-Ordner ändern. Sie können beispielsweise einem vertrauenswürdigen Benutzer das Recht der Dateikontrolle für den iFolder-Ordner gewähren.


Festlegen von Benutzerrechten mit einem Datei-Manager

  1. Navigieren Sie im Datei-Manager zum iFolder-Ordner und wählen Sie ihn dann aus.

  2. Klicken Sie mit der rechten Maustaste auf den iFolder-Ordner und klicken Sie dann auf "iFolder" > "Eigenschaften".

  3. Klicken Sie auf "Gemeinsam verwenden mit" .

  4. Wählen Sie in der Liste der Benutzer einen oder mehrere Benutzer aus.

  5. Wählen Sie "Zugriff" aus und geben Sie dann das Recht "Vollständige Kontrolle", "Lesen/Schreiben" oder "Nur Lesen" an.

  6. Klicken Sie auf "Anwenden" und klicken Sie danach auf "OK", um das Fenster "iFolder-Eigenschaften" zu schließen.


Festlegen von Benutzerrechten mit dem Browser für iFolder-Ordner

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Symbol () und klicken Sie dann auf "iFolder-Ordner", um den Browser für iFolder-Ordner zu öffnen.

  2. Wählen Sie in der Liste der iFolder-Ordner den iFolder-Ordner aus und klicken Sie anschließend auf "Gemeinsam verwenden mit" .

  3. Wählen Sie in der Liste der Benutzer einen oder mehrere Benutzer aus.

  4. Wählen Sie "Zugriff" aus und geben Sie dann das Recht "Vollständige Kontrolle", "Lesen/Schreiben" oder "Nur Lesen" an.

  5. Klicken Sie auf "Anwenden" und klicken Sie danach auf "OK", um das Fenster "iFolder-Eigenschaften" zu schließen.


Entfernen von Benutzern aus iFolder-Ordnern

Wenn Sie die Freigabe eines iFolder-Ordners aufheben möchten, müssen Sie einen Benutzer als Mitglied eines iFolder-Ordners entfernen. Der iFolder-Ordner auf der Arbeitsstation des Benutzers wird ein normaler Ordner. Eine Kopie der Daten verbleibt auf der Arbeitsstation des ehemaligen Mitglieds, die Dateien werden jedoch nicht mehr mit dem freigegebenen iFolder-Ordner synchronisiert.


Entfernen von Benutzern mit einem Datei-Manager

  1. Navigieren Sie im Datei-Manager zum iFolder-Ordner und wählen Sie ihn dann aus.

  2. Klicken Sie mit der rechten Maustaste auf den iFolder-Ordner und klicken Sie dann auf "iFolder" > "Eigenschaften".

  3. Klicken Sie auf "Gemeinsam verwenden mit" .

  4. Wählen Sie in der Liste der Benutzer einen oder mehrere Benutzer aus.

  5. Klicken Sie auf "Entfernen", klicken Sie auf "Anwenden" und klicken Sie dann auf "OK".


Entfernen von Benutzern mit dem Browser für iFolder-Ordner

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Symbol () und klicken Sie dann auf "iFolder-Ordner", um den Browser für iFolder-Ordner zu öffnen.

  2. Wählen Sie in der Liste der iFolder-Ordner den iFolder-Ordner aus und klicken Sie anschließend auf "Gemeinsam verwenden mit" .

  3. Wählen Sie in der Liste der Benutzer einen oder mehrere Benutzer aus.

  4. Klicken Sie auf "Entfernen", klicken Sie auf "Anwenden" und klicken Sie dann auf "OK".
