Verschieben von iFolder-Ordnern

Nach dem Erstellen eines iFolder-Ordners ist der Pfad zu diesem iFolder-Ordner statisch. Sie können einen iFolder-Ordner nicht von seinem ursprünglichen Standort an einen anderen Standort auf Ihrem lokalen Computer verschieben und ihn gleichzeitig als iFolder-Ordner beibehalten. Durch das Verschieben eines iFolder-Ordners wird die Verbindung zwischen der Synchronisierungsdatenbank und dem in einen iFolder-Ordner konvertierten Ordner unterbrochen.

So verschieben Sie einen iFolder-Ordner auf dem Computer, auf dem er erstellt wurde:

  1. Konvertieren Sie den iFolder-Ordner in einen normalen Ordner.

  2. Verschieben Sie den Ordner.

  3. Konvertieren Sie den Ordner in einen iFolder-Ordner.

  4. (Optional) Geben Sie den iFolder-Ordner frei.

So verschieben Sie einen iFolder-Ordner auf einen teilnehmenden Computer:

  1. Setzen Sie unter "iFolder-Ordner" den iFolder-Ordner zurück, sodass er nicht mehr am iFolder-System beteiligt ist, Ihnen jedoch weiterhin zur Verfügung steht.

  2. Wählen Sie den verfügbaren iFolder-Ordner aus und klicken Sie dann auf "Einrichtung".

  3. Geben Sie einen neuen lokalen Pfad an, unter dem der iFolder-Ordner gespeichert werden soll.

  4. Lassen Sie zu, dass der iFolder-Ordner Dateien mit dem Enterprise-Server synchronisiert.
