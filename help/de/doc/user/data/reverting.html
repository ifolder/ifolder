Zurücksetzen von iFolder-Ordnern auf normale Ordner

Alle Mitglieder können iFolder-Ordner zurücksetzen. Nach dem Zurücksetzen wird die Kooperationsfreigabe-Beziehung für alle iFolder-Mitglieder beibehalten, der iFolder-Ordner wird jedoch nicht mehr auf diesem Computer synchronisiert. Die iFolder-Ordner und seine Inhalte werden nicht von den Mitgliederarbeitsstationen gelöscht.

  1. Suchen Sie den iFolder-Ordner, den Sie auf einen normalen Ordner zurücksetzen möchten, in einem Datei-Manager oder auf dem Desktop.

  2. Verwenden Sie eine der folgenden Vorgehensweisen:

    • Klicken Sie mit der rechten Maustaste auf den iFolder-Ordner und klicken Sie dann auf "iFolder-Ordner" > "Auf normalen Ordner zurücksetzen".
    • Wählen Sie den iFolder-Ordner aus und klicken Sie dann in der Symbolleiste des Datei-Managers auf "Datei" > "iFolder-Ordner" > "Auf normalen Ordner zurücksetzen".

    Das iFolder-Symbol wird zu einem normalen Ordnersymbol. Die Daten auf allen Mitgliedercomputern bleiben unverändert.
