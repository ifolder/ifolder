Vorteile von iFolder

iFolder bietet folgende Vorteile:

  • iFolder kann in Ihre systemeigene Desktop-Umgebung integriert werden und ermöglicht so die einfache Erstellung und Verwaltung mehrerer iFolder-Ordner.

  • iFolder weist ein hohes Maß an Skalierbarkeit und Flexibilität auf, d. h., die Anzahl der iFolder-Ordner pro Benutzer bzw. die Anzahl der Mitglieder pro iFolder-Ordner ist nahezu unbegrenzt.

  • Wenn die Verschlüsselung aktiviert ist, schützt iFolder Ihre Daten bei der Übertragung und bei der Speicherung auf dem iFolder-Server.

  • Mit der verbesserten Webzugriffskonsole können Sie von einer Arbeitsstation aus ohne iFolder-Client auf Ihre Dateien auf dem iFolder-Server zugreifen und mithilfe eines Webbrowsers alle Operationen auf dem iFolder-Client durchführen.

  • Sie können auf einfache Weise ausgewählte persönliche und berufliche Dateien freigeben.

  • iFolder kann Ihre iFolder-Daten zur aktuellen Version migrieren und upgraden.

  • iFolder unterstützt LDAP-Gruppen. Sie können iFolder-Ordner für LDAP-Gruppen freigeben. Mitglieder der LDAP-Gruppen können iFolder-Ordner erstellen, besitzen und für andere Einzelbenutzer oder andere LDAP-Gruppen freigeben.

  • Sie können die Zugriffsstufe von Benutzern steuern, die Mitglieder der iFolder-Ordner sind, deren Eigentümer Sie sind bzw. für die Sie über die erforderlichen Rechte zur Steuerung der Zugriffsrechte von Mitgliedern verfügen.

  • iFolder führt transparente Aktualisierungen Ihrer Dateien für iFolder-Mitgliederordner auf mehreren Arbeitsstationen durch.

  • Wenn iFolder ausgeführt wird, werden Änderungen, die Sie offline vornehmen, überwacht und protokolliert. Außerdem werden diese Änderungen synchronisiert, wenn Sie eine Verbindung zum Server herstellen.

  • iFolder stellt eine sichere Authentifizierung für Mitglieder bereit, die zum Synchronisieren der Daten auf einen iFolder-Ordner zugreifen.

  • iFolder bietet eine Alternative zum Austausch von Dateien per Email, auf diese Weise müssen für Email-Anlagen geltende Größenbeschränkungen nicht mehr berücksichtigt und weniger Speicherplatz für Emails bereitgestellt werden.
