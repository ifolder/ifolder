使用可能なiFolderのセットアップ

他のユーザとiFolderを共用する場合、iFolderはiFolderブラウザ上で使用可能なiFolderアイコンとして表示されます。現在のローカルコンピュータで共有iFolderに参加するには、コンピュータ上でiFolderがアクティブになるようにiFolderを設定します。

ローカルのiFolderの配置先は次のガイドラインを満たしていなければなりません。

  • iFolderの配置のガイドライン
  • iFolderとそのフォルダおよびファイルの命名規則

iFolderをセットアップするには次のことを実行します。

  1. ［通知］エリアでiFolderアイコンをクリックし、［iFolder］をクリックしてiFolderブラウザを開きます。

  2. iFolderのリストで使用可能なiFolderを選択し、［セットアップ］をクリックします。

  3. アカウントにログインしていない場合はユーザ名とパスワードを入力し、［OK］をクリックします。

  4. ローカルのiFolderを作成する場所を参照します。

  5. ［OK］を2回クリックします。
