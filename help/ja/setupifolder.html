使用可能なiFolderのダウンロード

他のユーザがiFolderを共有すると、そのiFolderは、iFolderブラウザの［iFolders on iFolder Server Name (<iFolderサーバ名>上のiFolder)］に、iFolderダウンロードアイコンが付いた状態で表示されます。<iFolderサーバ名>の部分はそのアカウントのあるサーバの名前です。コンピュータ上の共有iFolderに参加する場合は、そのiFolderをダウンロードできます。

新しいユーザが共有iFolderのリストに参加すると、iFolderは［通知］スペースにポップアップメッセージを表示して、新しい共有ユーザが参加したことを知らせます。

共有に参加したユーザも、共有iFolderのダウンロードが終了するとすぐに、［通知］スペースに吹き出しのポップアップメッセージを受信します。メッセージの内容は、「<iFolderを共有するユーザのユーザ名>がiFolder<iFolderの名前>へ参加しました」のようになります。新しいユーザが共有メンバーに参加すると、すべての既存のメンバーユーザに、その新しい共有メンバーのことが通知されます。

  1. iFolderサービスアイコンを右クリックし、［iFolder］をクリックしてiFolderブラウザを開きます。

  2. iFolderの一覧から、利用可能なiFolderを選択し、［Download an iFolder (iFolderのダウンロード)］をクリックします。

  3. アカウントにログインしていない場合はユーザ名とパスワードを入力し、［接続］をクリックします。

  4. ローカルiFolderを作成する場所をブラウズし、このiFolderをローカルで使用するための名前を入力します。

    場所とフォルダの名前は、次のガイドラインを満たす必要があります。

    • iFolderの場所と利用法のガイドライン

    • iFolderとそのフォルダおよびファイルの命名規則

  5. ［OK］を2回クリックします。

詳細については、「 保証と著作権.
