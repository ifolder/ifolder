Klient programu iFolder

Klient iFolder integruje się z systemem operacyjnym, oferując usługi iFolder w macierzystym środowisku roboczym. Klient programu iFolder umożliwia zarządzanie własnymi iFolderami oraz określenie współużytkowanych iFolderów do skonfigurowania na każdym komputerze. Program iFolder współpracuje z następującymi systemami operacyjnymi:

  • SUSE® Linux Enterprise Desktop 10 SP1 i SP2

  • SUSE® Linux Enterprise Desktop 11

  • Windows Vista SP1

  • Windows XP SP2

  • OpenSUSE 11.1

  • Apple Macintosh® wersja 10.4 i nowsze

Aby uzyskać więcej informacji, patrz sekcja Informacje prawne.
