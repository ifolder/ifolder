iFolder mappák kezelése

Az iFolderTM olyan helyi könyvtár, amely fájloknak a felhasználó által megadott tagokkal való szelektív megosztására és szinkronizálására szolgál. Az iFolder fájljait minden tag elérheti. Akik arra fel vannak jogosítva, olvashatják, illetve módosíthatják is a fájlokat.
