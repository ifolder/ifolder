Навигатор iFolder

Навигатор iFolder предназначен для управления папками iFolder. В таблице ниже вкратце описаны элементы этого навигатора.

Название элемента

Описание

Создать iFolder

Позволяет преобразовать папку на локальном компьютере в папку iFolder и отправить ее на сервер iFolder, с которым установлено соединение.

Дополнительные сведения см. в разделе Создание и отправка папки iFolder

Синхронизировать сейчас

Позволяет немедленно запустить синхронизацию папок iFolder на локальном компьютере и на сервере.

Совместное использование

Позволяет использовать папку iFolder совместно с другими пользователями.

Дополнительные сведения см. в разделе Совместное использование папки iFolder

Преобразовать в стандартную папку

Позволяет преобразовать папку iFolder в обычную папку. Кроме того, можно удалить папку iFolder с сервера. Дополнительные сведения см. в разделе Преобразование папки iFolder в стандартную папку

Сервер

IP-адрес сервера iFolder, с которым установлено соединение.

Имя

Имя выбранной папки iFolder.

Владелец

Имя владельца выбранной папки iFolder.

Права

Права владельца на папку iFolder.

Файлы и папки для синхронизации

Количество файлов или папок, ожидающих синхронизации.

Время последней успешной синхронизации

Дата и время последней успешной синхронизации.

Загрузить iFolder

Позволяет загрузить папку iFolder с сервера iFolder на локальный компьютер.

Дополнительные сведения см. в разделе Загрузка доступной папки iFolder

Тип iFolder

Указывает, является ли выбранная папка iFolder зашифрованной или обычной.

Размер iFolder

Размер выбранной папки iFolder.

Объединить

Функция слияния позволяет консолидировать данные iFolder и уменьшить объем данных, передаваемый между сервером и клиентом iFolder. Можно выполнить слияние содержимого папки iFolder на сервере с содержимым папки, существующей на локальном компьютере. Дополнительные сведения см. в разделе Слияние папок

Удалить с сервера

Позволяет удалить выбранную папку iFolder с сервера.

Список доменов

Список серверов, на которых настроены учетные записи.

Установить/разорвать соединение

Позволяет установить и разорвать соединение с сервером iFolder. Если с сервером iFolder установлено соединение, отображается кнопка Разорвать соединение, в противном случае — Установить соединение.

Исправить

Предназначена для разрешения конфликтов с файлами. Чтобы разрешить конфликт версий файла, выберите правильную версию в диалоговом окне Разрешение конфликтов. Дополнительные сведения см. в разделе Разрешение конфликтов файлов

Пользователь

Имя пользователя, зарегистрированного на выбранном сервере iFolder.

Количество папок iFolder

Общее число папок iFolder на сервере и на локальном компьютере для выбранного сервера iFolder.

Квота диска

Значения полей "Используется" и "Доступен".

Используется

Общий объем дискового пространства, используемого всеми папками и каталогами на выбранном сервере iFolder в настоящий момент.

Доступен

Общий объем дискового пространства, доступного для файлов и каталогов на выбранном сервере iFolder в настоящий момент.

Автоматическая синхронизация

Интервал между двумя последовательными автоматическими циклическими синхронизациями

Дополнительную информацию см. в Юридическая информация.
