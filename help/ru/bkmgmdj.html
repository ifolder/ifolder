Смена пароля

Пароль к учетной записи iFolder можно изменить.

  1. В навигаторе iFolder щелкните Безопасность > Смена пароля.

  2. В окне Смена пароля выберите учетную запись iFolder, для которой необходимо изменить пароль, и укажите перечисленные ниже данные.

    Текущий пароль Введите текущий пароль.

    Новый пароль Введите новый пароль для текущей учетной записи iFolder.

    Подтверждение нового пароля Для подтверждения введите новый пароль еще раз.

    Запомнить пароль Установите флажок Запомнить пароль, чтобы каждый раз автоматически регистрироваться на компьютере с данной учетной записью.

  3. Щелкните Сброс.

    ПРИМЕЧАНИЕ.Пароль к учетной записи iFolder меняется непосредственно на сервере LDAP. Если один сервер LDAP служит центром управления пользователями для нескольких приложений, новый пароль будет действителен для всех этих приложений. После смены пароля сеанс iFolder заканчивается и выполняется выход. Чтобы установить соединение со службами iFolder, необходимо снова зарегистрироваться в системе.

Дополнительную информацию см. в Юридическая информация.
