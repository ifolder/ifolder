Общие сведения о значках iFolder

В следующей таблице приводится описание значков iFolder и соответствующих им задач.

Значок iFolder

Описание

Значок Сервисы iFolder отображается для клиента iFolder 3 в меню приложений на рабочей станции.

Если сервис iFolder запущен, значок Сервисы iFolder также появляется в области оповещений на панели задач рабочей станции. Щелкните его правой кнопкой мыши для отображения меню доступных задач, включая навигатор iFolder, настройки учетной записи, журнал синхронизации, настройки и справку.

В навигаторе iFolder отображается значок Создать iFolder.

Можно создать новую папку iFolder или преобразовать существующую папку в папку iFolder. При наличии нескольких учетных записей необходимо указать, какую учетную запись нужно использовать; папка iFolder может принадлежать только одной учетной записи. См. дополнительные сведения: Создание и отправка папки iFolder.

Значок Загрузка iFolder отображается в случае, если папка iFolder доступна на указанном сервере, но еще не загружена на локальный компьютер.

Используя различные учетные записи, можно загружать папки iFolder на локальный компьютер в любых сочетаниях. См. дополнительные сведения: Загрузка доступной папки iFolder.

Значок зашифрованной папки iFolder (заблокированная папка iFolder) в навигаторе iFolder означает, что папка iFolder зашифрована.

Указываемая при регистрации контрольная фраза открывает доступ к файлам в зашифрованной папке iFolder. Для получения дополнительной информации о шифровании см. раздел Управление контрольными фразами для зашифрованных папок iFolder.

Значок iFolder (зеленая буква "i") появляется на папках iFolder, если запущено приложение сервисов iFolder для папок iFolder, просматриваемых в менеджере файлов, на рабочем столе или в навигаторе iFolder.

Если приложение iFolder не запущено, папки iFolder отображаются в виде обычных папок в менеджере файлов или на рабочем столе. Если приложение iFolder не запущено, навигатор iFolder недоступен.

Рекомендуется хранить папки iFolder на локальном компьютере в отдельной папке, содержащей только папки iFolder. Благодаря этому можно легко определить, какая из папок является папкой iFolder, даже если приложение iFolder не запущено, и папки не отмечены значком папок iFolder.

Значок предупреждения iFolder обозначает, что сервер недоступен. Приложение iFolder запущено, но соединение с сервером iFolder не установлено. Возможны следующие причины:

  • Регистрация с помощью учетной записи iFolder не выполнена. См. дополнительные сведения: Регистрация с помощью учетной записи iFolder.

  • Учетная запись отключена на компьютере клиента, то есть, параметр Подключаться автоматически отключен. Для получения информации см. подраздел Автоматическое подключение в разделе Просмотр и изменение настроек учетной записи iFolder.

  • Сервер закрыт, или учетная запись отключена на сервере. Обратитесь за помощью к системному администратору.

  • Трафик iFolder заблокирован брандмауэром. Для получения информации об устранении этой проблемы см. раздел Конфигурирование параметров локального брандмауэра для трафика iFolder.

  • Сетевое соединение неработоспособно. Проверьте локальные сетевые параметры.

Значок Конфликт iFolder указывает на наличие конфликта между копиями файла на локальном компьютере и на сервере. Устраните конфликт, указав версию файла, которую необходимо сохранить. Дополнительную информацию о конфликте файлов см. в разделе Разрешение конфликтов файлов.

Значок ожидания синхронизации iFolder обозначает, что в папке iFolder содержатся файлы, которые необходимо синхронизировать с сервером, и процесс синхронизации еще не начался.

Приложение iFolder одновременно выполняет синхронизацию только одной папки iFolder, задачи синхронизации помещаются в очередь. Чтобы убедиться что синхронизация файлов в локальной папке iFolder с файлами на сервере выполнена, не закрывайте приложение iFolder до успешного завершения синхронизации, о чем будет свидетельствовать изменение значка.

Значок синхронизации iFolder обозначает, что файлы в папке iFolder в настоящий момент синхронизируются с сервером.

Значок Ошибка iFolder указывает на то, что в папке iFolder не синхронизированы один или несколько файлов из-за конфликта политик: ограничения размера файла или количества папок iFolder.

В диалоговом окне "Совместное использование" значок владельца папки iFolder обозначает, что пользователь является владельцем совместно используемой папки iFolder.

В диалоговом окне "Совместное использование" значок текущего пользователя iFolder обозначает, что пользователь в настоящий момент управляет папкой iFolder. Если текущий пользователь является владельцем, вместо этого значка отображается значок владельца.

Дополнительную информацию см. в Юридическая информация.
