Перемещение папки iFolder

После создания папки iFolder путь папки iFolder становится постоянным. Чтобы папка осталась папкой iFolder, ее нельзя перемещать с исходного места в другое место локальной рабочей станции. При перемещении папки iFolder теряется связь между базой данных синхронизации и папкой, которую Вы преобразовали в iFolder.

Для переноса папки iFolder на компьютере, на котором она была создана, выполните следующие действия:

  1. В менеджере файлов или в навигаторе iFolder выберите папку iFolder, которую необходимо переместить.

  2. Преобразуйте папку iFolder в обычную папку.

  3. В менеджере файлов переместите папку в другое местоположение на компьютере.

  4. Преобразуйте папку обратно в папку iFolder и передайте папку iFolder на сервер.

  5. Установите совместное использование для папки iFolder (необязательно).

  6. Разрешите синхронизацию файлов папки iFolder с сервером iFolder.

Для переноса папки iFolder на компьютер другого пользователя выполните следующие действия:

  1. В папках iFolder преобразуйте папку iFolder для останова участия в iFolder на текущем компьютере.

  2. Выберите доступную папку iFolder, затем щелкните Загрузка папки iFolder.

  3. Укажите новый локальный путь, куда нужно поместить папку iFolder.

  4. Разрешите синхронизацию файлов папки iFolder с сервером iFolder.

Дополнительную информацию см. в Юридическая информация.
