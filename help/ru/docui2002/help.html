Поиск в отмеченных разделах

Поиск в определенных разделах содержания данной документации

  • Убедитесь в том, что рядом с каждым разделом, поиск в котором необходимо выполнить, установлен флажок.
    •

  • В текстовое поле введите ключевые слова для поиска (можно использовать универсальные символы ? и *). (Дополнительная информация)

  • Щелкните значок со стрелкой, ссылку "Искать в отмеченных разделах" или нажмите клавишу Enter.

По умолчанию отмечены все разделы. Щелкните флажки рядом с разделами, поиск в которых выполнять не требуется, для снятия флажков.

Если не отмечен ни один из разделов, появится предупреждающее сообщение.




Печать отмеченных разделов

Добавление и печать определенных разделов данной документации

  • Убедитесь в том, что рядом с каждым разделом, печать которого необходимо выполнить, установлен флажок.
    •

  • Щелкните "Печать отмеченных разделов".

По умолчанию отмечены все разделы. Щелкните флажки рядом с разделами, печать которых выполнять не требуется, для снятия флажков.

Если не отмечен ни один из разделов, появится предупреждающее сообщение.
