Просмотр и конфигурирование свойств iFolder

Используйте страницу "Свойства iFolder" для просмотра и конфигурирования приведенной ниже информации о выбранной папке iFolder.


Общие сведения о свойствах iFolder

Следующие параметры относятся к дисковому пространству, используемому на сервере:

Параметр Описание

Доступно

Общий объем дискового пространства, доступного в данный момент на сервере для файлов или каталогов в выбранной папке iFolder. Это значение равно квоте за вычетом уже использованного пространства.

Используется

Общий объем дискового пространства на сервере, занятого в данный момент всеми файлами и каталогами выбранной папки iFolder.

Одно из следующих:

  • Установить квоту
  • Квота

  • Установить квоту. Если Вы являетесь владельцем папки iFolder, Вы можете установить квоту для папки iFolder.

  • Квота. Если Вы являетесь членом, это значение показывает общий объем дискового пространства, выделенный владельцем папки iFolder для выбранной папки iFolder. Если квота не установлена, значение является пустым или обозначается как Н/Д (недоступно).

Следующие параметры отражают состояние синхронизации локальной копии папки iFolder:

Параметр Описание

Последняя успешная синхронизация

Время завершения последней успешной синхронизации локальной копии папки iFolder с сервером iFolder.

Файлов/папок для синхронизации

Общее количество элементов хранения (файлов и папок) iFolder, которые необходимо синхронизировать.

Автоматическая синхронизация через каждые

Интервал синхронизации iFolder. Укажите значение в минутах, часах или днях.

Если клиент iFolder используется с сервером Novell iFolder 3.x, этот интервал не может переопределить минимальный интервал, указанный администратором iFolder для сервера или четной записи на сервере, но он может переопределить интервал синхронизации на уровне клиента.

Синхронизировать сейчас

Щелкните эту кнопку для выполнения синхронизации по требованию.


Доступ к свойствам iFolder из менеджера файлов

  1. В менеджере файлов перейдите к папке iFolder и выберите ее.

  2. Щелкните iFolder правой кнопкой, затем щелкните iFolder > Свойства.


Доступ к свойствам iFolder из навигатора iFolder

  1. В области оповещения щелкните правой кнопкой значок приложения iFolder , затем щелкните "Папки iFolder" для открытия навигатора iFolder.

  2. В списке папок iFolder выберите папку iFolder и щелкните "Свойства".
