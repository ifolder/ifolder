Отображение и скрытие доступных папок iFolder

Можно выбрать, какие папки iFolder, необходимо загружать на каждый компьютер. Если папки iFolder, которые не были загружены, доступны на сервере, они отображаются в навигаторе iFolder в списке Имя сервера, относящемуся к учетной записи. Имя сервера - указанное администратором описательное имя сервера, для которого существует учетная запись iFolder.

Чтобы просмотреть доступные папки iFolder в навигаторе iFolder, щелкните Просмотреть доступные папки iFolder.

Чтобы скрыть доступные папки iFolder в навигаторе iFolder, щелкните Скрыть доступные папки iFolder.

Дополнительную информацию см. в Юридическая информация.
