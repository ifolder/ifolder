Управление папками iFolder

iFolder™ - это локальный каталог, который предназначен для ограниченного совместного использования и синхронизации файлов с определенными пользователями-членами. Файлы iFolder доступны для всех членов через сервис хоста, например сервер Novell® iFolder® 3.7. Файлы iFolder могут быть изменены и прочитаны пользователями, обладающими правами на выполнение этих действий.
