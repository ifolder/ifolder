Обзор iFolder

Novell® iFolder® - это приложение для совместного использования файлов клиентами Linux, Windows и Macintosh. Вы можете совместно использовать файлы в нескольких папках iFolder, каждая из которых предназначена для различных пользователей или групп LDAPGroup. Вы можете управлять пользователями папок iFolder и устанавливать уровень доступа для каждого члена. Кроме того, Вы можете стать пользователем папок iFolder, доступ к которым был предоставлен коллегами.

Приложение iFolder 3.7 предоставляет удобный пользовательский интерфейс для миграции клиента iFolder 2.x в iFolder 3.6 и более поздней версии, а также для обновления клиента iFolder 3.x на iFolder 3.6 и более позднюю версию.
