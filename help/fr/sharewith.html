Partage d'un iFolder

Vous pouvez partager un dossier iFolder avec plusieurs utilisateurs. Cette section traite des tâches suivantes :

  • Présentation des droits d'accès des utilisateurs

  • Accès à l'onglet Partage

  • Ajout d'un utilisateur à un dossier iFolder

  • Modification des droits d'accès des utilisateurs

  • Suppression d'un utilisateur d'un dossier iFolder

  • Transfert de propriété à un utilisateur iFolder

Accès à l'onglet Partage

Vous pouvez gérer l'adhésion pour le dossier iFolder à partir de l'onglet Partage dans la boîte de dialogue Propriétés iFolder.

IMPORTANT :Vous ne pouvez partager des dossiers iFolder que si l'administrateur iFolder a activé la stratégie de partage pour votre compte.

  1. Si iFolder n'est pas exécuté, lancez-le.

  2. Effectuez l'une des opérations suivantes pour ouvrir la boîte de dialogue Propriétés iFolder sous l'onglet Partage.

    • Dans un gestionnaire de fichiers, cliquez avec le bouton droit sur le dossier iFolder que vous souhaitez partager, puis cliquez sur Partager avec.

    • Dans le navigateur iFolder, sélectionnez le dossier iFolder, puis sélectionnez Partager avec.

Ajout d'un utilisateur à un dossier iFolder

  1. Sous l'onglet Partage, cliquez sur Ajouter pour ouvrir la boîte de dialogue Sélectionner des utilisateurs.

  2. Dans la liste des utilisateurs, sélectionnez un ou plusieurs utilisateurs ou groupes LDAP en les ajoutant à la liste Utilisateurs à ajouter, puis cliquez sur OK.

    La synchronisation avec le serveur peut prendre quelques secondes. Immédiatement après la synchronisation, une notification apparaît dans la partie inférieure droite de la page.

  3. Sous l'onglet Partage, sélectionnez un ou plusieurs utilisateurs auxquels vous souhaitez assigner le même droit d'accès.

  4. Cliquez sur Droits, sélectionnez le droit d'accès Contrôle total, Lecture/écriture ou Lecture seule, puis cliquez sur OK.

  5. Répétez les étapes précédentes jusqu'à ce que vous ayez invité tous les utilisateurs ou groupes LDAP souhaités à partager le dossier iFolder et défini leurs droits d'accès.

  6. Cliquez sur Fermer pour fermer la fenêtre Propriétés iFolder.

Modification des droits d'accès des utilisateurs

Étant donné que les relations de travail et l'état des membres d'un dossier iFolder changent, il peut être nécessaire de modifier le droit d'accès d'un utilisateur au dossier iFolder. Ainsi, il se peut que vous souhaitiez accorder à un utilisateur de confiance le droit Contrôle total pour le dossier iFolder.

  1. Sous l'onglet Partage, sélectionnez un ou plusieurs utilisateurs auxquels vous souhaitez assigner le même droit d'accès.

  2. Sélectionnez Droits, puis spécifiez le droit Contrôle total, Lecture/écriture ou Lecture seule.

    Lorsque vous invitez pour la première fois un utilisateur ou un groupe LDAP à participer, l'option Faire de cet utilisateur le propriétaire du dossier iFolder est désactivée (grisée) dans la boîte de dialogue Accès. Si vous prévoyez de faire du nouvel utilisateur le propriétaire du dossier, vous devez attendre que l'état de l'utilisateur passe de Utilisateur invité à Utilisateur iFolder. Vous pouvez ensuite transférer la propriété. Lorsque vous transférez la propriété à un groupe LDAP, tous les membres de ce groupe LDAP disposent des droits de possession.

  3. Cliquez sur OK, puis cliquez sur OK pour fermer la fenêtre Propriétés iFolder.

Suppression d'un utilisateur d'un dossier iFolder

Pour annuler le partage d'un dossier iFolder, vous devez supprimer un utilisateur ou un groupe LDAP qui est membre de ce dossier. Le dossier iFolder sur le poste de travail de l'utilisateur ou du membre du groupe LDAP devient un dossier normal. Une copie des données est conservée sur le poste de travail de l'ancien membre, mais les fichiers ne sont plus synchronisés avec le dossier iFolder partagé.

  1. Sous l'onglet Partage, sélectionnez un ou plusieurs utilisateurs dans la liste.

  2. Cliquez sur Supprimer, puis sur Oui.

Transfert de propriété à un utilisateur iFolder

Le propriétaire d'un dossier iFolder peut transférer la propriété de ce dernier à un autre membre individuel ou groupe LDAP dont l'état est Utilisateur iFolder. Après le transfert, le propriétaire d'origine devient un membre disposant du droit Lecture-écriture, et l'espace occupé par le dossier iFolder est attribué au nouveau propriétaire. Lorsqu'un groupe LDAP devient le propriétaire d'un dossier iFolder, tous les membres du groupe disposent des droits de possession.

  1. Dans un gestionnaire de fichiers ou le navigateur iFolder, cliquez avec le bouton droit sur le dossier iFolder que vous souhaitez transférer, puis sélectionnez Partager avec pour ouvrir la boîte de dialogue Propriétés du dossier iFolder dans l'onglet Partage

  2. Dans la liste des utilisateurs, sélectionnez l'utilisateur iFolder que vous souhaitez définir comme nouveau propriétaire, puis cliquez sur Droits.

  3. Sélectionnez Désigner cet utilisateur comme propriétaire du dossier iFolder.

    L'option Désigner cet utilisateur comme propriétaire du dossier iFolder est désactivée (grisée) dans la boîte de dialogue Accès si l'état de l'utilisateur est Utilisateur invité. Dans ce cas, quittez la boîte de dialogue, attendez que l'état de l'utilisateur passe de Utilisateur invité à Utilisateur iFolder, puis répétez les étapes précédentes pour transférer la propriété.

  4. Cliquez sur OK, puis cliquez sur OK pour fermer la fenêtre Propriétés iFolder.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
