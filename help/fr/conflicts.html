Résolution des conflits de fichiers

Des conflits peuvent survenir entre les versions des fichiers enregistrées simultanément sur des ordinateurs différents ou lorsqu'un nom de fichier diffère par la casse uniquement. Lors de conflits, le fichier est stocké dans la zone conflictuelle et le fichier est signalé par l'état Conflit de nom. L'icône Erreur iFolder s'affiche sur le dossier iFolder. Si l'option Signaler les conflits est activée dans vos Préférences iFolder, une alerte est également envoyée à la zone Notification. Vous pouvez contrôler le comportement contextuel des notifications en définissant votre préférence dans Préférences iFolder > sous l'onglet Général.

Pour résoudre les conflits de fichiers, reportez-vous aux sections suivantes :

  • Résolution des conflits des versions de fichier

  • Résolution des conflits de noms de fichier

Résolution des conflits des versions de fichier

Pour résoudre les conflits de version de fichier, vous devez sélectionner la version correcte dans la boîte de dialogue Résolution des conflits.

  1. Dans la zone Notification, cliquez avec le bouton droit sur l'icône Services iFolder , puis cliquez sur les dossiers iFolder pour ouvrir le navigateur iFolder.

  2. Sélectionnez le dossier iFolder en conflit , puis cliquez sur Résoudre pour ouvrir la boîte de dialogue Résoudre les conflits.

    Vous pouvez afficher deux versions du même fichier :

    • Version locale : Fichier enregistré sur votre machine locale.

    • Version du serveur: Fichier enregistré sur le serveur.

    Vous pouvez afficher les informations sur le fichier sur votre ordinateur local et le fichier sur le serveur pour déterminer la version du fichier à conserver.

  3. Cliquez sur Ouvrir pour chaque fichier pour les afficher.

  4. Sélectionnez une des options suivantes pour Enregistrer dans le dossier iFolder en fonction de votre préférence.

    Version locale : Sélectionnez cette option pour enregistrer la version locale dans le dossier iFolder.

    Version du serveur: Sélectionnez cette option pour enregistrer la version du serveur dans le dossier iFolder.

  5. Sélectionnez Activer pour activer l'option Enregistrer dans la liste des conflits. Lorsque vous activez cette option, le fichier en conflit est sélectionné automatiquement pour être enregistré dans la liste des conflits. Votre sélection détermine quelle version enregistrer dans la liste des conflits pour usage ultérieur.

  6. Une fois le conflit résolu, cliquez sur Enregistrer.

Résolution des conflits de noms de fichier

Les conflits de noms de fichier se produisent lorsque la version locale du fichier et celle du serveur présentent des différences au niveau des majuscules/minuscules. Ces fichiers peuvent avoir le même contenu ou non. Vous pouvez résoudre ce type de conflit en renommant les fichiers tout en veillant à l'endroit où vous renommez le fichier.

  • Si le contenu est identique, vous pouvez renommer l'un des fichiers à l'aide de la boîte de dialogue Résoudre les conflits.

  • S'il est différent, veillez à créer un copie du fichier local en lui attribuant un nom différent, puis revenez à la boîte de dialogue Résoudre les conflits et enregistrez la version serveur du fichier. Les deux fichiers peuvent alors exister dans iFolder sous des noms différents. Si vous renommez une version de fichier dans la boîte de dialogue Résoudre les conflits, cette version remplace l'autre et le contenu de l'autre fichier est supprimé.

  1. Dans la zone Notification, cliquez avec le bouton droit sur l'icône Services iFolder , puis cliquez sur les dossiers iFolder pour ouvrir le navigateur iFolder.

  2. Sélectionnez le fichier en conflit , puis cliquez sur Résoudre pour ouvrir la boîte de dialogue Résoudre les conflits.

  3. Vérifiez qu'il s'agit bien d'un conflit de nom de fichier et déterminez si le contenu est identique ou différent.

  4. Effectuez l'une des opérations suivantes :

    • Si le contenu est identique, renommez l'un des fichiers, cliquez sur Enregistrer cette version, puis cliquez sur OK pour fermer la boîte de dialogue Résoudre les conflits. Vous avez terminé.

    • Si le contenu est différent, passez à l'Étape 5.

  5. Fermez la boîte de dialogue Résoudre les conflits.

  6. Ouvrez un gestionnaire de fichiers, puis accédez à la copie locale du fichier à l'origine du conflit.

  7. Copiez le fichier et enregistrez-le sous un nom distinct dans iFolder.

    iFolder effectue la synchronisation du nouveau fichier avec le serveur et les ordinateurs membres.

  8. Retournez à la boîte de dialogue Résoudre les conflits, sélectionnez la version serveur du fichier, cliquez sur Enregistrer cette version, puis cliquez sur OK.

    La version serveur du fichier est téléchargée sur l'ordinateur local et remplace la copie locale par le fichier du même nom.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
