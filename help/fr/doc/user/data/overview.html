Présentation de iFolder

iFolderTM est une application de partage de fichiers pour clients Linux, Windows* et Macintosh*. Vous pouvez partager des fichiers dans plusieurs dossiers iFolder, chacun d'eux étant destiné à un groupe d'utilisateurs. Vous déterminez qui peut accéder à un dossier iFolder et le niveau d'accès de chaque membre. Vous pouvez également accéder aux dossiers iFolder que d'autres membres partagent avec vous.
