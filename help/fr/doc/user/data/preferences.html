Configuration des préférences iFolder du Client

  1. Cliquez avec le bouton droit de la souris sur l'icône iFolder dans la zone Notification, puis sélectionnez Préférences pour ouvrir la boîte de dialogue Préférences iFolder à l'onglet Général.


  2. Spécifiez les valeurs suivantes :

    Paramètre Description

    Démarrer iFolder lors de la connexion au bureau

    Sélectionnez cette option pour lancer iFolder lorsque vous vous connectez avec le nom d'utilisateur local que vous avez défini lors de la création du compte iFolder. Dans le cas des comptes pour lesquels vous avez activé Mémoriser le mot de passe, la synchronisation se produit en arrière-plan à l'intervalle de synchronisation spécifié.

    Désélectionnez cette option pour démarrer iFolder manuellement.

    Afficher la boîte de dialogue de confirmation lors de la création de dossiers iFolder

    Sélectionnez cette option pour recevoir un message de confirmation à chaque fois que vous créez un dossier iFolder, en plus des signes visuels normaux comme l'apparition du dossier iFolder dans la liste des dossiers iFolder.

    Désélectionnez cette option pour ne plus recevoir le message de confirmation.

    Notification de dossiers iFolder partagés

    Sélectionnez cette option pour activer un message contextuel dans la zone de notification lorsque vous recevez une invitation à partager un dossier iFolder.

    Les invitations sont présentées en fonction de l'intervalle de synchronisation que vous avez spécifié dans la boîte de dialogue Préférences iFolder. Cet intervalle contrôle la synchronisation des fichiers, les invitations de partage de dossiers iFolder et la liste des utilisateurs. Vous pouvez également cliquer sur Rafraîchir pour mettre à jour les invitations sur demande.

    Désélectionnez cette option pour désactiver le message contextuel pour les invitations.

    Notification des conflits

    Sélectionnez cette option pour activer un message contextuel dans la zone de notification lorsqu'un conflit se produit lors de la synchronisation des fichiers.

    Désélectionnez cette option pour désactiver le message contextuel lors de conflits de synchronisation.

    Notification en cas d'adhésion d'un utilisateur

    Sélectionnez cette option pour activer un message contextuel dans la zone de notification lorsqu'un utilisateur accepte une invitation à partager un dossier iFolder. L'acceptation de l'utilisateur est enregistrée lorsque celui-ci configure le dossier iFolder sur son ordinateur.

    Désélectionnez cette option pour désactiver le message contextuel lorsque les utilisateurs confirment le partage de leurs dossiers.

    Synchronisation

    Synchroniser automatiquement les dossiers iFolder : Cochez cette case pour synchroniser tous les dossiers iFolder sur cet ordinateur en ayant recours à votre identité de login locale. Supprimez la coche de cette case pour désactiver la synchronisation.

    Synchroniser les dossiers iFolder toutes les : Spécifiez l'intervalle minimum à utiliser pour synchroniser les dossiers iFolder sur cet ordinateur en ayant recours à votre identité de login locale. Spécifiez la valeur et les unités.

    Si vous utilisez le client iFolder avec Novell iFolder 3.x, l'intervalle de synchronisation minimum pris en compte est toujours la plus grande valeur des paramètres suivants :

    • La règle du système (sa valeur est par défaut 0), sauf si une règle d'utilisateur est définie. Si c'est le cas, la règle d'utilisateur prévaut sur la règle du système, que la valeur de la règle d'utilisateur soit plus grande ou plus petite que la règle du système.
    • La règle de machine locale ou le paramètre sur la machine cliente synchronisée avec le serveur.
    • La règle (collection) de dossier iFolder.

  3. Une fois que vous avez terminé, cliquez sur Appliquer pour que les modifications soient prises en compte.

  4. Cliquez sur OK pour fermer la boîte de dialogue Préférences iFolder.
