Connexion à un compte iFolder

Après avoir configuré un compte iFolder, vous devez vous loguer chaque fois que vous souhaitez synchroniser les données de votre dossier iFolder local, la liste des utilisateurs iFolder et vos notifications de dossiers iFolder partagés. Vous pouvez vous loguer à plusieurs comptes en même temps. Pour vous loguer à un compte, procédez de l'une des façons suivantes :

  • Activez l'option Mémoriser le mot de passe pour que la connexion au compte iFolder s'effectue automatiquement chaque fois que vous vous connectez depuis votre bureau ou que vous démarrez le client iFolder.

    Lorsque votre mot de passe change, l'authentification automatique échoue la prochaine fois qu'iFolder essaie de vous loguer au compte iFolder. iFolder vous invite alors à vous connecter avec votre nouveau mot de passe et vous permet de le spécifier et de l'enregistrer.

  • Désactivez l'option Mémoriser le mot de passe et loguez-vous à un compte iFolder uniquement lorsque vous êtes prêt à synchroniser les fichiers du dossier iFolder ou à en assurer la gestion.

Pour activer ou désactiver l'option Mémoriser le mot de passe pour un compte iFolder existant :

  1. Cliquez avec le bouton droit de la souris sur l'icône iFolder dans la zone Notification, puis sélectionnez Comptes.

  2. Sélectionnez le compte iFolder dont vous voulez afficher les paramètres.

  3. Cochez (pour activer) ou décochez (pour désactiver) la case Mémoriser le mot de passe.

  4. Indiquez un mot de passe, puis cliquez sur OK ou Appliquer.

  5. Cliquez sur OK pour vérifier que les paramètres de compte ont été modifiés ou sur Annuler pour annuler les modifications.

Pour se loguer à un compte iFolder :

  1. Cliquez avec le bouton droit de la souris sur l'icône iFolder dans la zone Notification, puis sélectionnez Comptes.

  2. Sélectionnez le compte iFolder dont vous voulez afficher les paramètres.

  3. Tapez le mot de passe, puis cliquez sur Loguer.

    Si la connexion réussit, l'état du compte passe à Logué.

    Si elle échoue, un message d'erreur indique la cause, par exemple, la saisie d'un mot de passe erroné.

  4. Cliquez sur OK pour fermer la boîte de dialogue Comptes.
