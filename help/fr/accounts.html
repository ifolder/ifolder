Configuration d'un compte iFolder

Utilisez l'assistant de création d'un compte iFolder pour ajouter et configurer un nouveau compte. Vous devez configurer au moins un compte iFolder pour pouvoir créer ou partager des dossiers iFolders sur votre système. Vous ne pouvez vous connecter au serveur iFolder qu'avec un seul compte à la fois. En revanche, pour vous connecter à plusieurs serveurs iFolder, vous pouvez configurer plusieurs comptes. Vous pouvez vous loguer séparément à chaque compte et être logué simultanément à plusieurs comptes.

Plusieurs utilisateurs ayant des identités de login local différentes peuvent posséder des dossiers iFolder sur le même ordinateur. Au premier démarrage de iFolder sous votre identité de login local actuelle, iFolder vous invite à configurer un compte iFolder.

IMPORTANT :iFolder 3.7 introduit une nouveauté, la création automatique de comptes, qui permet à l'administrateur iFolder de vous créer un compte et de le configurer. Si votre administrateur iFolder administrator a activé la création automatique de compte, vous ne devez pas vous créer de compte. Au premier démarrage d'iFolder, vous êtes invité à introduire votre mot de passe. Entrez votre mot de passe, puis configurez le compte.

Pour configurer un nouveau compte :

  1. Accédez à l'assistant de création d'un compte iFolder, à l'aide d'une des méthodes suivantes :

    • Au premier démarrage de iFolder sous votre identité de login local actuelle, iFolder vous invite à configurer un compte iFolder. Cliquez sur Suivant pour accéder directement à l'assistant de création d'un compte iFolder.

    • Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, sélectionnez Comptes pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Comptes, puis cliquez sur Ajouter pour ouvrir l'assistant de création d'un compte iFolder.

  2. Suivez les instructions qui s'affichent à l'écran pour spécifier les valeurs suivantes :

    Paramètre

    Description

    Adresse du serveur

    Nom DNS ou adresse IP du serveur d'entreprise iFolder sur lequel se trouve votre compte. Demandez ces informations à votre administrateur iFolder.

    Par exemple : svr21.exemple.com ou 192.168.1.1

    Le nom de serveur s'affiche toujours avec son adresse IP dans l'interface utilisateur iFolder. Par exemple : iFolder - 192.168.1.1

    IMPORTANT :Vous ne pouvez pas utiliser un client iFolder 3.7 pour vous connecter à un serveur iFolder 3.2.

    Définir ce serveur comme celui par défaut

    Si plusieurs serveurs sont connectés à votre compte, cochez la case Définir ce serveur comme celui par défaut pour définir le serveur sélectionné dans la liste déroulante en tant que serveur par défaut lorsque vous créez ou téléchargez un dossier iFolder.

    Nom d'utilisateur

    Spécifiez votre identité utilisateur pour ce compte. Votre administrateur iFolder décide si le format que vous devez utiliser correspond à votre adresse de messagerie ou à votre ID d'utilisateur réseau (nom commun LDAP).

    Par exemple, si votre nom est Jean Deschamp, votre ID d'utilisateur jdeschamp et votre adresse de messagerie jean.deschamp@exemple.com, votre administrateur configure un paramètre pour le serveur qui détermine lequel des deux formats accepter dans le champ Nom d'utilisateur à des fins d'authentification.

    Mot de passe

    Spécifiez le mot de passe pour votre nom d'utilisateur.

    L'authentification s'effectue via un canal sécurisé (SSL). Vous êtes invité à valider et à accepter le certificat de serveur.

    Mémoriser le mot de passe

    Cochez la case Mémoriser le mot de passe sur cet ordinateur pour vous loguer automatiquement à ce compte iFolder lorsque vous vous loguez à votre ordinateur.

    Si votre mot de passe de réseau change, l'authentification automatique échoue la prochaine fois qu'iFolder essaie de vous loguer au compte iFolder. iFolder vous invite à vous loguer avec votre nouveau mot de passe et vous permet de le spécifier et de l'enregistrer.

  3. Vérifiez les informations saisies. Pour les modifier, cliquez sur le bouton Précédent et revenez aux pages précédentes. Cliquez sur Se connecter pour continuer et valider votre connexion au serveur.

  4. Si vous êtes invité à accepter le certificat de ce serveur, cliquez sur Détails pour afficher les informations sur le certificat, puis sur OK pour les accepter.

    REMARQUE :Lors de la création du compte, le login s'effectue via SSL. Lorsque vous êtes invité à accepter le certificat de serveur, vous devez le valider et l'accepter pour pouvoir établir une connexion.

  5. Si l'administrateur a activé la stratégie de codage, iFolder vous invite à entrer votre phrase secrète.

    Entrez votre phrase secrète dans le champ Phrase secrète.

    Entrez de nouveau la phrase secrète dans le champ Retaper la phrase secrète.

    Cochez la case Mémoriser la phrase secrète pour accéder automatiquement aux données iFolder codées pendant la session actuelle. Si vous cochez cette case, iFolder peut mémoriser votre phrase secrète lors du prochain processus de login. Cet échange de phrase secrète est effectué automatiquement.

  6. Sélectionnez l'agent de récupération désiré dans la liste et cliquez sur OK.

  7. Spécifiez l'emplacement de destination du dossier iFolder par défaut.

    Par défaut, le dossier iFolder est créé à l'emplacement suivant :<répertoire privé utilisateur>/ifolder/nom_serveur/nom_utilisateur.

  8. Sélectionnez le type de dossier iFolder par défaut à créer. Si vous choisissez Codage, vous serez invité à spécifier la phrase secrète.

    IMPORTANT :Vous ne pouvez partager que les dossiers iFolder ordinaires. Les dossiers iFolder codés ne peuvent pas être partagés.

    Si votre administrateur iFolder n'a pas activé le codage pour votre compte actuel, vous ne pouvez créer que des dossiers iFolder ordinaires et, si le codage est activé, l'option Normal est sélectionnée par défaut.

  9. Sélectionnez Synchronisation sécurisée pour activer la synchronisation sécurisée des fichiers dans le dossier iFolder que vous créez.

    Vous pouvez activer un canal sécurisé pour les dossiers iFolder ordinaires et codés. Lorsque cette option est sélectionnée, iFolder vous permet d'utiliser un canal sécurisé pour effectuer la synchronisation entre iFolder sur le serveur et sur la machine locale.

    Si vous supprimez un dossier iFolder par défaut sur le serveur, lors de la prochaine configuration du compte à partir de n'importe quel client lourd, iFolder vous invite à créer un nouveau dossier iFolder par défaut.

    Toutefois, si vous supprimez le dossier iFolder par défaut de la machine locale, la prochaine fois que vous configurerez le compte, iFolder vous invitera à télécharger le dossier iFolder par défaut.

  10. Si vous avez déjà configuré un dossier iFolder par défaut, vous pouvez le télécharger sur votre machine locale.

    Téléchargez le dossier iFolder par défaut et cliquez sur Suivant pour continuer.

  11. Si un dossier du même nom que le dossier iFolder par défaut existe déjà sur votre système, vous êtes invité à fusionner les deux dossiers ou à télécharger le dossier iFolder par défaut vers un autre emplacement.

    Cliquez sur OK pour fusionner les deux dossiers ou cliquez sur Annuler pour télécharger le dossier iFolder par défaut vers un autre emplacement.

  12. Cliquez sur Terminer pour fermer l'assistant de création d'un compte iFolder.

    iFolder synchronise la liste d'utilisateurs pour le compte spécifié et identifie les dossiers iFolder disponibles pour le téléchargement. Ce téléchargement initial peut prendre de quelques secondes à quelques minutes, en fonction de la taille de la liste d'utilisateurs pour le compte.

  13. Vérifiez que le compte que vous avez configuré figure dans la liste des comptes.

    1. Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, puis sélectionnez Préférences dans le menu pour ouvrir la boîte de dialogue Préférences iFolder dans l'onglet Comptes.

      Lorsque vous êtes connecté au serveur iFolder, la case En ligne est cochée.

    2. Fermez la boîte de dialogue Préférences iFolder.

  14. Configurez des dossiers iFolders pour ce compte en faisant appel à l'une de ces méthodes :

    • Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, puis sélectionnez Dossiers iFolder pour ouvrir le navigateur de dossiers iFolder. Passez à la section Création et téléchargement d'un dossier iFolder.

    • Si des dossiers iFolder pour ce compte sont téléchargeables à partir du serveur, ils sont listés dans la fenêtre iFolder sous Dossiers iFolder sur <nom serveur iFolder>. L'icône de téléchargement indique que le dossier iFolder peut être téléchargé. Passez à la section Téléchargement d'un dossier iFolder disponible.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
