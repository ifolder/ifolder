Recherche d'éléments cochés

Pour rechercher des éléments dans le sommaire de cette documentation

  • Cochez la case de chaque élément à rechercher.
    •

  • Dans la zone de texte, entrez les mots-clés à rechercher (vous pouvez utiliser les caractères génériques ? et *). (Suite)

  • Cliquez sur l'icône de flèche « Rechercher les éléments cochés » ou appuyez sur Entrée.

Par défaut, tous les éléments sont cochés. Pour ne pas rechercher un élément, supprimez la coche de sa case en cliquant sur cette dernière.

Si aucun élément n'est coché, un avertissement s'affiche.




Impression d'éléments cochés

Pour concaténer et imprimer des éléments de cette documentation

  • Cochez la case de chaque élément à imprimer.
    •

  • Cliquez sur « Imprimer les éléments cochés ».

Par défaut, tous les éléments sont cochés. Pour ne pas imprimer un élément, supprimez la coche de sa case en cliquant sur cette dernière.

Si aucun élément n'est coché, un avertissement s'affiche.
