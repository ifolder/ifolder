Présentation des icônes iFolder

Le tableau suivant décrit les icônes iFolder et les tâches s'y rapportant.

Icône iFolder

Description

L'icône Services iFolder s'affiche pour le Client iFolder 3 dans le menu Applications se trouvant sur le Bureau.

Lorsque iFolder s'exécute, l'icône Services iFolder apparaît également dans la zone Notification de la barre des tâches du Bureau. Cliquez dessus avec le bouton droit pour afficher un menu des tâches possibles, incluant le navigateur iFolder, les paramètres de compte, le journal de synchronisation, les préférences et l'aide.

L'icône Créer un dossier iFolder s'affiche dans le navigateur iFolder.

Vous pouvez créer un nouveau dossier iFolder ou convertir un dossier existant en dossier iFolder. Si vous possédez plusieurs comptes, vous devez spécifier le compte à utiliser car un dossier iFolder ne peut appartenir qu'à un seul compte. Pour plus d'informations, reportez-vous à Création et téléchargement d'un dossier iFolder.

L'icône Télécharger le dossier iFolder s'affiche lorsqu'un dossier iFolder est disponible sur le serveur spécifié, mais que vous ne l'avez pas téléchargé sur votre machine locale.

Vous pouvez télécharger n'importe quelle combinaison de dossiers iFolder à partir de différents comptes vers l'ordinateur local. Pour plus d'informations, reportez-vous à Téléchargement d'un dossier iFolder disponible.

L'icône Dossier iFolder codé (dossier iFolder verrouillé) indique que le dossier iFolder est codé et s'affiche dans le navigateur iFolder.

La phrase secrète proposée au moment du login vous permet de récupérer les fichiers du dossier iFolder codé. Pour plus d'informations sur le codage, reportez-vous à la section Gestion des phrases secrètes pour les dossiers iFolder codés.

Le symbole iFolder (un « i » vert) apparaît sur les dossiers iFolder lorsque l'application Services iFolder est exécutée pour les dossiers iFolder affichés dans un gestionnaire de fichiers, sur le Bureau ou dans le navigateur iFolder.

Lorsque iFolder n'est pas exécuté, les dossiers iFolder apparaissent comme des dossiers normaux lorsqu'ils sont affichés dans le gestionnaire de fichiers ou sur le Bureau. Le navigateur iFolder n'est pas disponible lorsque iFolder n'est pas exécuté.

Il est recommandé d'organiser votre collection locale de dossiers iFolder dans un répertoire distinct qui ne contient que des dossiers iFolder. Cela vous permet de vous souvenir facilement quels dossiers sont des dossiers iFolder même lorsque iFolder n'est pas exécuté et qu'il n'affiche pas d'icônes pour identifier ces dossiers comme dossiers iFolder.

L'icône iFolder Avertissement indique que le serveur n'est pas disponible. iFolder est exécuté mais vous n'êtes pas connecté au serveur iFolder. Les causes possibles sont les suivantes :

  • Vous n'êtes pas logué au compte iFolder. Pour plus d'informations, reportez-vous à Connexion à un compte iFolder.

  • Le compte est désactivé côté client car le paramètre Connexion automatique est désactivé. Pour plus d'informations, reportez-vous à la rubrique Connexion automatique de la section Affichage et modification des paramètres du compte iFolder.

  • Le serveur est inactif ou votre compte est désactivé côté serveur. Contactez votre administrateur iFolder pour obtenir une assistance.

  • Votre pare-feu bloque le trafic iFolder. Pour plus d'informations sur la résolution de ce problème, reportez-vous à la Configuration des paramètres du pare-feu local pour le trafic iFolder.

  • Votre connexion réseau est inactive. Vérifiez vos paramètres de réseau local.

L'icône Conflit iFolder indique un conflit de fichiers entre la copie locale et celle du serveur. Indiquez la version du fichier que vous souhaitez garder pour résoudre le conflit. Pour plus d'informations sur les conflits de fichiers, reportez-vous à la section Résolution des conflits de fichiers.

L'icône iFolder En attente de synchronisation indique que le dossier iFolder contient des fichiers à synchroniser avec le serveur et que le processus de synchronisation n'a pas encore commencé.

iFolder synchronise un dossier iFolder à la fois et met les tâches de synchronisation en file d'attente. Pour vous assurer que les fichiers dans ce dossier iFolder local sont synchronisés avec les fichiers sur le serveur, ne fermez pas iFolder avant que la synchronisation ne commence et ne soit complètement terminée, comme l'indiquent les modifications de l'icône.

L'icône iFolder Synchronisation indique que les fichiers du dossier iFolder sont en cours de synchronisation avec le serveur.

L'icône Erreur iFolder indique qu'un ou plusieurs fichiers du dossier iFolder n'ont pas été synchronisés en raison d'un conflit de stratégie lié à la limite de taille des fichiers ou du nombre de dossiers iFolder.

Dans la boîte de dialogue Partage, l'icône iFolder Utilisateur propriétaire indique que l'utilisateur possède le dossier iFolder partagé.

Dans la boîte de dialogue Partage, l'icône iFolder Utilisateur actuel indique que l'utilisateur est en train de gérer le dossier iFolder. Si l'utilisateur actuel est le propriétaire, c'est l'icône Utilisateur propriétaire qui s'affiche.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
