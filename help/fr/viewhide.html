Affichage et masquage des dossiers iFolder disponibles

Vous pouvez choisir les dossiers iFolder que vous souhaitez télécharger sur chacun de vos ordinateurs. Si des dossiers iFolder que vous n'avez pas téléchargés sont disponibles sur le serveur, ils sont listés dans le navigateur iFolder sous le Nom du serveur du compte associé. L'option Nom du serveur correspond au nom descriptif spécifié par l'administrateur du serveur sur lequel vous possédez un compte iFolder.

Pour afficher les dossiers iFolder disponibles dans le navigateur iFolder, cliquez sur Afficher les dossiers iFolder disponibles.

Pour masquer les dossiers iFolder disponibles dans le navigateur iFolder, cliquez sur Masquer les dossiers iFolder disponibles.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
