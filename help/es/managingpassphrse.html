Gestión de la contraseña codificada para carpetas iFolder cifradas

Cada carpeta iFolder cifrada posee una clave de cifrado de datos exclusiva que se genera de forma automática durante la creación de la carpeta iFolder. La clave se cifra mediante una contraseña codificada que ha proporcionado el usuario y también con la clave pública del agente de recuperación.

El agente de recuperación se selecciona durante el proceso de creación de la cuenta iFolder si el administrador de la carpeta iFolder ha habilitado la directiva de cifrado en el servidor.

Si el administrador de la carpeta iFolder ha habilitado el cifrado, aparecerá un recuadro de diálogo durante la creación de iFolder para que introduzca la contraseña codificada. iFolder utiliza la contraseña codificada para generar una clave de cifrado exclusiva para cifrar o descifrar los datos de la carpeta iFolder en el servidor de iFolder. Si olvida la contraseña codificada, no podrá acceder a los datos de iFolder. Debe restaurar la contraseña codificada para obtener acceso a los datos de iFolder.

Recuperación de una carpeta iFolder cifrada

Utilice el asistente de recuperación de contraseña codificada para restaurar la contraseña cifrada y recuperar una carpeta iFolder cifrada. Siga los pasos que figuran a continuación para lanzar el asistente de recuperación de contraseña codificada:

  1. En el navegador de iFolder, haga clic en Seguridad > Contraseña codificada olvidada para lanzar el asistente.

  2. En la lista Cuenta iFolder, seleccione el dominio para el que debe restablecerse la contraseña codificada y haga clic en Remitir.

    Durante el proceso de creación de la cuenta iFolder, si se seleccionó la opción Server_Default como agente de recuperación, aparecerá la página Establecer nueva contraseña codificada. Para restaurar la nueva contraseña codificada, consulte Restauración de la contraseña codificada cuando el agente de recuperación es Server_Default.

    Por otra parte, si seleccionó un agente de recuperación distinto de Server_Default, debe restaurar la contraseña codificada mediante Restauración de la contraseña codificada cuando el agente de recuperación no es Server_Default.

Restauración de la contraseña codificada cuando el agente de recuperación es Server_Default

Para establecer la nueva contraseña codificada, realice los pasos siguientes:

  1. En la página Establecer nueva contraseña codificada, especifique la nueva contraseña codificada en el campo Nueva contraseña codificada .

  2. Vuelva a introducir la nueva contraseña codificada en el campo Confirmar contraseña codificada > .

  3. Especifique la contraseña en el campo Contraseña, a continuación, haga clic en Remitir.

  4. Una vez restaurada con éxito la contraseña codificada, haga clic en Finalizar para cerrar el asistente.

Restauración de la contraseña codificada cuando el agente de recuperación no es Server_Default

Para restaurar la contraseña codificada cuando el agente de recuperación seleccionado no es Server_Default, siga los pasos indicados a continuación:

  1. La página de bienvenida del asistente muestra información relacionada con la información que necesita para restaurar la contraseña codificada. Haga clic en Remitir para ver la página del mecanismo de recuperación de contraseña codificada.

  2. Hay tres métodos que puede utilizar dentro del asistente para restaurar la contraseña codificada:

    • Mediante el archivo secreto y su contraseña: el archivo secreto y su contraseña se utilizan para restaurar la contraseña codificada. Si ha conservado el archivo secreto y su contraseña, puede utilizar el asistente para especificar la ubicación del archivo secreto y proporcionar su contraseña. Para restaurar la contraseña codificada mediante el archivo secreto, consulte Utilización del archivo secreto para restaurar la contraseña codificada.

      NOTA:obtiene el archivo secreto y su contraseña del administrador de la carpeta iFolder.

    • Mediante el nuevo archivo de datos: el nuevo archivo de datos también se puede utilizar para restaurar la contraseña codificada. Puede obtenerlo del administrador tras realizar una operación de exportación del archivo de datos anterior. Para restaurar la contraseña codificada mediante el nuevo archivo de datos, consulte Utilización del nuevo archivo de datos para restaurar la contraseña codificada.

    • Exportación del archivo de datos anterior: si no tiene el archivo secreto ni el nuevo archivo de datos, puede utilizar el asistente para exportar el archivo de datos anterior y, a continuación, enviar el archivo al administrador por correo electrónico. Cuando el administrador devuelva el nuevo archivo de datos, puede utilizar el asistente para restaurar la contraseña codificada. Para exportar el archivo de datos anterior, consulte Utilización del archivo exportado para restaurar la contraseña codificada.

Valores de configuración

Descripción

Lista de cuentas iFolder

Muestra la lista de cuentas iFolder configuradas. Por defecto, se encuentra seleccionada la primera cuenta con la que se inicia sesión.

Ubicación del archivo secreto

Indica la ubicación del archivo de clave privado del certificado que se puede obtener del administrador de la carpeta iFolder.

Contraseña secreta

Representa la contraseña del archivo de clave secreto.

Contraseña para una vez

Representa la contraseña que el administrador utiliza para cifrar el nuevo archivo de datos.

Ubicación del nuevo archivo de datos

Indica la ubicación del archivo de clave descifrado. El nuevo archivo de datos se obtiene del administrador de la carpeta iFolder tras realizar una operación de exportación del archivo de datos anterior.

Ubicación para guardar el archivo de datos anterior

Indica la ubicación para guardar el archivo de clave exportado. Una vez exportado el archivo de datos anterior, puede enviarlo por correo electrónico al administrador. A continuación, el administrador le enviará el nuevo archivo de datos.

Nueva contraseña codificada

Representa la nueva contraseña codificada que introduce, la cual se utiliza para cifrar la clave de cifrado de datos exclusiva que se autogenera durante la creación de la carpeta iFolder.

Confirmar contraseña codificada

Representa la nueva contraseña codificada que debe volver a introducir.

Utilización del archivo secreto para restaurar la contraseña codificada

Para restaurar la contraseña codificada mediante el archivo secreto, realice los siguientes pasos:

  1. Seleccione la opción Tengo el archivo secreto y su contraseña y, a continuación, haga clic en Remitir.

  2. Cumplimente los siguientes campos:

    Ubicación del archivo secreto: especifique la ubicación del archivo secreto o haga clic en el botón Examinar para localizar y seleccionar el mismo.

    Contraseña secreta: especifique la contraseña del archivo secreto.

    Nueva contraseña codificada: especifique la nueva contraseña codificada.

    Confirmar contraseña codificada Vuelva a introducir la nueva contraseña codificada.

  3. Haga clic en Remitir para restaurar la contraseña codificada.

  4. Haga clic en Finalizar para cerrar el asistente.

NOTA:Obtiene el archivo secreto y su contraseña del administrador de la carpeta iFolder.

Utilización del nuevo archivo de datos para restaurar la contraseña codificada

Para restaurar la contraseña codificada mediante el nuevo archivo de datos, realice los siguientes pasos:

  1. Seleccione la opción Tengo el nuevo archivo de datos enviado por el administrador y, a continuación, haga clic en Remitir.

  2. En el campo Ubicación del nuevo archivo de datos, especifique la ubicación del nuevo archivo de datos o haga clic en el botón Examinar para localizar y seleccionar el mismo.

  3. Si el nuevo archivo de datos que recibe del administrador está cifrado, seleccione ¿Está cifrado el archivo anterior? Si selecciona este recuadro de verificación, se habilitará el campo Contraseña para una vez.

  4. En el campoContraseña para una vez, especifique la contraseña que el administrador utiliza para cifrar el nuevo archivo de datos.

  5. En el campo Nueva contraseña codificada, especifique la nueva contraseña codificada.

  6. En el campo Confirmar contraseña codificada, vuelva a introducir la nueva contraseña codificada.

  7. Haga clic en Remitir para restaurar la contraseña codificada.

  8. Haga clic en Finalizar para cerrar el asistente.

Utilización del archivo exportado para restaurar la contraseña codificada

Para restaurar la contraseña codificada cuando no tenga el archivo secreto o el nuevo archivo de datos:

  1. Seleccione la opción No tengo archivo secreto ni archivo de datos y, a continuación, haga clic en Remitir.

  2. En el campo Ubicación para guardar archivo de datos anterior, especifique la ubicación donde debe guardarse el archivo de datos anterior o haga clic en el botón Examinar para especificar la ubicación en donde guardar el mismo.

  3. Haga clic en Remitir para obtener el archivo de datos anterior. Aparecerá la ubicación del archivo de datos anterior y la dirección de correo electrónico del administrador de iFolder. Puede utilizar esta información para obtener el nuevo archivo de datos.

  4. Haga clic en Finalizar para cerrar el asistente.

  5. Envíe por correo electrónico el archivo de datos anterior al administrador de la carpeta iFolder solicitándole que le envíe el nuevo archivo de datos.

  6. Cuando reciba el nuevo archivo de datos, consulte Utilización del nuevo archivo de datos para restaurar la contraseña codificada para restaurar la contraseña codificada.

Cambio de contraseña codificada

Por motivos de seguridad, debe cambiar la contraseña codificada de forma periódica. Puede restaurar la contraseña codificada mediante la contraseña codificada para una vez enviada por el agente de recuperación.

  1. En el navegador de iFolder, seleccione Seguridad > Cambiar contraseña codificada.

  2. En la ventana Cambiar contraseña codificada, seleccione la cuenta de iFolder cuya contraseña codificada desee restaurar y especifique los siguientes datos:

    Introduzca la contraseña codificada: especifique la contraseña codificada actual.

    Introduzca la nueva contraseña codificada: especifique la nueva contraseña codificada que desee establecer para la cuenta de iFolder actual.

    Reescriba la contraseña codificada: especifique la nueva contraseña codificada para confirmarla.

    Agente de recuperación: Seleccione el agente de recuperación que ha establecido para la cuenta de iFolder actual.

  3. Haga clic en Reiniciar.
