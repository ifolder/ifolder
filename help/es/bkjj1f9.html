Navegador de iFolder

El navegador de iFolder permite gestionar las carpetas iFolder. La tabla proporcionada a continuación resume los elementos del navegador de iFolder:

Nombre de elemento

Descripción

Crear carpeta iFolder

Permite convertir una carpeta del equipo local a una carpeta iFolder y cargarla en el servidor iFolder al que está conectado.

Para obtener más información, consulte la Creación y carga de una carpeta iFolder.

Sincronizar ahora

Permite iniciar inmediatamente una sincronización de carpetas iFolder en el equipo local y el servidor.

Compartir con

Permite compartir las carpetas iFolder con otros usuarios.

Para obtener más información, consulte la Uso compartido de carpetas iFolder.

Revertir a una carpeta normal

Permite revertir una carpeta iFolder a una carpeta normal. También puede suprimir la carpeta iFolder del servidor. Para obtener más información, consulte la Conversión de una carpeta iFolder en una carpeta normal.

Servidor

Muestra la dirección IP del servidor de iFolder al que está conectado.

Nombre

Muestra el nombre de la carpeta iFolder seleccionada.

Propietario

Muestra el nombre del propietario de la carpeta iFolder seleccionada.

Derechos

Muestra los derechos que el propietario de la carpeta iFolder posee.

Archivos/carpeta para sincronizar

Indica el número de archivos o carpetas que están esperando una sincronización.

Última sincronización satisfactoria

Indica la hora y la fecha de la última sincronización satisfactoria.

Descarga de una carpeta iFolder

Permite descargar una carpeta iFolder del servidor de iFolder en el equipo local.

Para obtener más información, consulte la Descarga de una carpeta iFolder disponible.

Tipo de carpeta iFolder

Indica si la carpeta iFolder seleccionada está cifrada o es normal.

Tamaño de la carpeta iFolder

Muestra el tamaño de la carpeta iFolder seleccionada.

Fusionar

Esta opción permite consolidar los datos de iFolder y minimizar la transferencia de datos entre el Cliente y el servidor iFolder.. Puede fusionar el contenido de una carpeta iFolder del servidor con el contenido de una carpeta de un equipo local. Para obtener más información, consulte la Fusión de carpetas iFolder.

Eliminar de servidor

Permite suprimir la carpeta iFolder seleccionada del servidor.

Lista de dominios

Muestra la lista de servidores donde tiene una cuenta configurada.

Conectar/desconectar

Permite conectar o desconectar de un servidor de iFolder. Si está conectado al servidor de iFolder, aparecerá el botón Desconectar y si no está conectado a ningún servidor de iFolder, aparecerá el botón Conectar.

Resolver

Permite resolver conflictos de archivos. Para resolver conflictos de versiones de archivos, debe seleccionar la versión correcta en el recuadro de diálogo Resolución del conflicto. Para obtener más información, consulte la Resolución de conflictos con los archivos.

Usuario

Muestra el nombre del usuario que inició sesión en el servidor de iFolder seleccionado.

N.º de carpetas iFolder

Muestra el número total de carpetas iFolder en el servidor de iFolder y el equipo local para el servidor de iFolder seleccionado.

Cuota de disco

Muestra los valores de los campos Usado y Disponible.

Usado

Muestra la cantidad total de espacio en disco del servidor actualmente consumido por todos los archivos y directorios del servidor de iFolder seleccionado.

Disponible

Muestra la cantidad total de espacio en disco del servidor actualmente disponible para archivos o directorios del servidor de iFolder seleccionado.

Sincronización automática

Muestra el intervalo de tiempo entre dos ciclos de sincronización automática consecutivos.

Para obtener más información, consulte Información legal.
