Configuración de las preferencias de iFolder del cliente

  1. Haga clic con el botón derecho en el icono de aplicación iFolder en el área de notificaciones y, a continuación, seleccione Preferencias para abrir el recuadro de diálogo Preferencias de iFolder en la pestaña General.


  2. Especifique los valores siguientes:

    Parámetro Descripción

    Start iFolder When Logging In to the Desktop (Iniciar iFolder al entrar en el escritorio)

    Seleccione esta opción para iniciar iFolder siempre que entre con el nombre de usuario local en el equipo donde ha configurado la cuenta de iFolder. En las cuentas en las que ha habilitado la opción Recordar contraseña, la sincronización se producirá en segundo plano según el intervalo de sincronización especificado.

    Deseleccione esta opción para iniciar iFolder manualmente.

    Show Confirmation Dialog When Creating iFolders (Mostrar recuadro de diálogo de confirmación al crear carpetas iFolder)

    Seleccione esta opción para recibir un mensaje de confirmación cada vez que cree una carpeta iFolder, además de las indicaciones visuales habituales como, por ejemplo, la aparición de la carpeta iFolder en la lista de carpetas iFolder.

    Deseleccione esta opción para dejar de recibir el mensaje de confirmación.

    Notify of Shared iFolders (Notificar existencia de carpetas iFolder compartidas)

    Seleccione esta opción para habilitar un mensaje emergente en el área de notificaciones siempre que reciba una nueva invitación para compartir una carpeta iFolder.

    Las invitaciones se recuperan según el intervalo de sincronización especificado en Preferencias de iFolder. El intervalo controla la sincronización de archivos, las invitaciones para compartir carpetas iFolder y la lista de usuarios. También puede hacer clic en Actualizar para actualizar las invitaciones a pedido.

    Deseleccione esta opción para inhabilitar el comportamiento de mensaje emergente para las invitaciones.

    Notify of Conflicts (Notificar conflictos)

    Seleccione esta opción para habilitar un mensaje emergente en el área de notificaciones siempre que se produzca un conflicto al sincronizar archivos.

    Deseleccione esta opción para inhabilitar el comportamiento de mensaje emergente para los conflictos de sincronización.

    Notify When a User Joins (Notificar la unión de un usuario)

    Seleccione esta opción para habilitar un mensaje emergente en el área de notificaciones siempre que un usuario acepte una invitación para compartir una carpeta iFolder. La aceptación tiene lugar cuando el usuario configura realmente la carpeta iFolder en su equipo.

    Deseleccione esta opción para inhabilitar el comportamiento de mensaje emergente para la confirmación de compartimiento del usuario.

    Sincronización

    Automatically Synchronize iFolders (Sincronizar carpetas iFolder automáticamente): Seleccione esta casilla de verificación para habilitar la sincronización de todas las carpetas iFolder en este equipo con su identidad de entrada local activa. Deseleccione la casilla de verificación para inhabilitar la sincronización.

    Synchronize iFolders Every (Sincronizar carpetas iFolder cada): Especifique el intervalo mínimo que desea utilizar para sincronizar carpetas iFolder en este equipo con su identidad de entrada local activa. Especifique el valor y las unidades.

    Si utiliza el Cliente iFolder con iFolder 3.x de Novell, el intervalo de sincronización mínimo efectivo siempre es el valor más alto de los parámetros siguientes:

    • La directiva del sistema (por defecto, cero), salvo que exista una directiva de usuario establecida. Si existe una directiva de usuario establecida, ésta anulará la directiva del sistema, tanto si es superior o inferior en valor.
    • La directiva del equipo local o el ajuste del equipo cliente que se sincroniza con el servidor.
    • La directiva de iFolder (recopilación).

  3. Cuando haya finalizado, haga clic en Aplicar para que los cambios surtan efecto.

  4. Haga clic en Aceptar para cerrar el recuadro de diálogo Preferencias de iFolder.
