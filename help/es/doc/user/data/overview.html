Descripción general de iFolder

iFolderTM es una aplicación para el uso compartido de archivos destinada a clientes Linux, Windows* y Macintosh*. Es posible compartir archivos en varias carpetas iFolder, cada una con un grupo de usuarios diferente. El propietario controla quién puede participar en una carpeta iFolder y el nivel de acceso de cada miembro. También puede participar en carpetas iFolder que otros comparten con usted.
