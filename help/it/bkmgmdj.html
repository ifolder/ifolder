Modifica della password

È possibile cambiare la password del conto iFolder.

  1. Nel browser delle cartelle iFolder, selezionare Sicurezza > Cambia password.

  2. Nella finestra Cambia password selezionare il conto iFolder per cui si desidera cambiare la password e specificare i seguenti dettagli:

    Password attuale: specificare la password attuale.

    Nuova password: specificare la nuova password che si desidera impostare per il conto iFolder attuale.

    Conferma nuova password: specificare di nuovo la nuova password per conferma.

    Memorizza password: selezionare la casella di controllo Memorizza password per eseguire automaticamente il login al conto iFolder ogni volta che si esegue il login al computer.

  3. Fare clic su Reimposta.

    NOTA:iFolder cambia la password direttamente nel server LDAP. Se lo stesso server LDAP è un'origine di gestione utenti per altre applicazioni, la nuova password sarà applicabile anche a tali applicazioni. Una volta modificata la password, la sessione di iFolder scadrà e si verrà scollegati. È necessario eseguire di nuovo il login per connettersi ai servizi iFolder.

Per ulteriori informazioni, vedere Note legali.
