Spostamento di una cartella iFolder

Il percorso di una cartella iFolder è statico. Non è possibile spostare una cartella iFolder dall'ubicazione originale in un'altra nel computer locale preservandone le caratteristiche. Lo spostamento di una cartella iFolder determina infatti l'interruzione del collegamento tra il database di sincronizzazione e la cartella convertita per iFolder.

Per spostare una cartella iFolder all'interno del computer in cui è stata creata:

  1. Selezionare la cartella iFolder che si desidera spostare in un manager di file o nel browser delle cartelle iFolder.

  2. Convertire la cartella iFolder in una cartella normale.

  3. In un manager di file, spostare la cartella in un'altra ubicazione nel computer.

  4. Convertire nuovamente la cartella in cartella iFolder e caricare la cartella iFolder nel server.

  5. (Facoltativo) Condividere la cartella iFolder.

  6. Eseguire la sincronizzazione dei file della cartella iFolder con il server iFolder.

Per spostare una cartella iFolder in un computer partecipante:

  1. In Cartelle iFolder, convertire la cartella iFolder in cartella normale per impedire la partecipazione nel computer attuale.

  2. Selezionare la cartella iFolder disponibile e fare clic su Scarica

  3. Specificare un nuovo percorso locale in cui collocare la cartella iFolder.

  4. Eseguire la sincronizzazione dei file della cartella iFolder con il server iFolder.

Per ulteriori informazioni, vedere Note legali.
