Configurazione di una cartella iFolder disponibile

Quando altri condividono cartelle iFolder con l’utente, queste cartelle vengono visualizzate con un’icona di cartella iFolder disponibile nel browser delle cartelle iFolder. Per partecipare alla cartella iFolder condivisa sul computer locale in uso, è necessario configurarla in modo da renderla attiva sul computer.

L’ubicazione locale della cartella deve soddisfare le seguenti indicazioni:

  • Indicazioni per l'ubicazione delle cartelle iFolder
  • Convenzioni di denominazione per le cartelle iFolder e i relativi file e cartelle

Per configurare una cartella iFolder:

  1. Nell'area di notifica fare clic con il pulsante destro del mouse sull'icona iFolder e scegliere Cartelle iFolder per aprire il browser.

  2. Selezionare la cartella iFolder disponibile nell'apposito elenco, quindi fare clic su Configura .

  3. Se non è ancora stato eseguito, effettuare il login al conto utilizzando nome utente e parola d’ordine, quindi fare clic su OK.

  4. Selezionare l’ubicazione in cui si desidera creare una cartella iFolder locale.

  5. Fare clic su OK due volte.
