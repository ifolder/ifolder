Cancellazione di un conto iFolder

È possibile specificare se la rimozione del conto deve avvenire solo nel computer in uso o se si desidera rimuovere anche le cartelle iFolder di cui si è proprietari dal server, di fatto interrompendo la condivisione delle cartelle iFolder configurate per il conto.

Se si rimuove un conto iFolder dal computer, le copie locali delle cartelle iFolder ad esso correlate verranno convertite in cartelle normali. Sarà possibile continuare ad eccedere alle cartelle iFolder da altri computer o dal Web.

Se si rimuove un conto iFolder dal server:

  • Verrà annullata la condivisione di tutte le cartelle iFolder di cui si è proprietari.
  • Le copie locali delle cartelle iFolder presenti sui computer membri verranno convertite in cartelle normali.
  • La cartella iFolder e i relativi contenuti verranno rimossi dal server.
  • Lo stato di membro delle cartelle iFolder condivise di altri utenti verrà revocato.

  1. Fare clic con il pulsante destro del mouse sull'icona dell'applicazione iFolder nell'area di notifica e scegliere Conti per aprire la scheda Conti della finestra di dialogo Preferenze di iFolder.

  2. Selezionare il conto iFolder che si desidera cancellare, quindi fare clic su Cancella.

  3. Verrà visualizzato un messaggio in cui viene chiesto di definire il limite dell’azione di cancellazione. Eseguire una delle seguenti operazioni:

    • Rimozione del conto iFolder solo dal computer in uso: deselezionare Rimuovi cartelle iFolder e file personali dal server, quindi fare clic su Sì. Il conto selezionato verrà rimosso solo dalla workstation locale, mentre le cartelle iFolder locali associate a tale conto verranno convertite in cartelle normali. Se la copia locale dei dati non è più necessaria, utilizzare un manager di file per individuarne l’ubicazione e cancellarla.

      È possibile continuare ad accedere alle cartelle iFolder associate a questo conto da altri computer con il client iFolder. Quando il client iFolder è utilizzato con un server aziendale Novell® iFolder® 3.x, l’accesso ai file può essere eseguito da qualsiasi postazione provvista di accesso Web Novell iFolder 3.x.

    • Rimozione del conto iFolder, delle cartelle iFolder e dei rispettivi file personali dal sever: selezionare Rimuovi cartelle iFolders e file personali dal server, quindi fare clic su Sì. Il conto selezionato verrà rimosso dalla workstation locale. Inoltre, tutte le cartelle iFolder di cui si è proprietari associate a tale conto verranno rimosse dal server e convertite in cartelle normali sul computer in uso e su tutti i computer membri. Infine, verrà revocato lo stato di membro delle cartelle iFolder condivise relative a tale conto. Le cartelle iFolder locali verranno tutte convertite in cartelle normali. Se la copia locale dei dati non è più necessaria, utilizzare un manager di file per individuarne l’ubicazione e cancellarla.

      Se si decide di utilizzare questo conto su questo o su un altro computer, è necessario configurare il conto e le relative cartelle iFolder come all’inizio.

    • Annullamento della cancellazione: fare clic su No per annullare l’operazione di cancellazione. Il conto e le relative cartelle iFolder rimarranno invariati.
