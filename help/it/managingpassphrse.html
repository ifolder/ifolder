Gestione delle stringhe di cifratura per le cartelle iFolder cifrate

Una cartella iFolder cifrata dispone di una chiave di cifratura dei dati univoca generata automaticamente durante la creazione della cartella iFolder. La chiave viene cifrata utilizzando una stringa di cifratura fornita dall'utente e utilizzando una chiave pubblica dell'agente di recupero.

L'agente di recupero viene selezionato durante il processo di creazione del conto iFolder se l'amministratore iFolder ha abilitato la norma di cifratura sul server.

Se l'amministratore iFolder ha abilitato la cifratura, viene visualizzata una finestra di dialogo durante la creazione della cartella iFolder in cui immettere la stringa di cifratura. La stringa di cifratura viene utilizzata per generare una chiave di cifratura univoca per cifrare o decifrare i dati iFolder sul server iFolder. Se si dimentica la stringa di cifratura, non è possibile accedere a tali dati. È necessario reimpostare la stringa di cifratura per ottenere l'accesso ai dati iFolder.

Recupero di una cartella iFolder cifrata

Utilizzare Recupero guidato della stringa di cifratura per reimpostare la stringa di cifratura e recuperare una cartella iFolder cifrata. Per avviare Recupero guidato della stringa di cifratura, seguire la procedura riportata di seguito.

  1. Nel browser delle cartelle iFolder, fare clic su Sicurezza > Stringa di cifratura dimenticata per avviare la procedura guidata.

  2. Dall'elenco Conto iFolder selezionare il dominio per cui è necessario reimpostare la stringa di cifratura e fare clic su Avanti.

    Durante il processo di creazione del conto iFolder, se l'opzione Default _server è stata selezionata come agente di recupero, viene visualizzata la pagina Imposta nuova stringa di cifratura. Per reimpostare la nuova stringa di cifratura, vedere Reimpostazione della stringa di cifratura quando l'agente di recupero è Default_server..

    Dall'altra parte, se è stato selezionato un agente di recupero diverso da Default_server, è necessario reimpostare la stringa di cifratura utilizzando Reimpostazione della stringa di cifratura quando l'agente di recupero non è Default_server.

Reimpostazione della stringa di cifratura quando l'agente di recupero è Default_server.

Per reimpostare la nuova stringa di cifratura, seguire la procedura riportata di seguito:

  1. Nella pagina Imposta nuova stringa di cifratura, specificare la nuova stringa di cifratura nel campo Nuova stringa di cifratura.

  2. Immettere di nuovo la stringa di cifratura nel campo Conferma stringa di cifratura > .

  3. Specificare la password nel campo Password e fare clic su Avanti.

  4. Dopo che la stringa di cifratura è stata reimpostata, fare clic su Fine per chiudere la procedura guidata.

Reimpostazione della stringa di cifratura quando l'agente di recupero non è Default_server

Per reimpostare la stringa di cifratura quando l'agente di recupero non è Default_server, seguire la procedura riportata di seguito.

  1. Nella pagina iniziale della procedura guidata vengono visualizzate informazioni riguardanti i dati necessari per reimpostare la stringa di cifratura. Fare clic su Avanti per visualizzare la pagina relativa al meccanismo di recupero della stringa di cifratura.

  2. Nella procedura guidata sono disponibili tre metodi per reimpostare la stringa di cifratura:

    • Utilizzando il file secreto e la relativa password: Il file secreto e la relativa password vengono utilizzati per reimpostare la stringa di cifratura. Se sono stati mantenuti il file segreto e la relativa password, è possibile utilizzare la procedura guidata per specificare l'ubicazione del file secreto e fornire la relativa password. Per reimpostare la stringa di cifratura utilizzando il file secreto, vedere Uso del file segreto per la reimpostazione della stringa di cifratura.

      NOTA:Richiedere il file secreto e la relativa password all'amministratore iFolder.

    • Utilizzando il nuovo file di dati: Il nuovo file di dati può essere utilizzato per reimpostare la stringa di cifratura. È possibile ottenere il nuovo file di dati dall'amministratore dopo avere eseguito un'operazione di esportazione del vecchio file di dati. Per reimpostare la stringa di cifratura utilizzando il nuovo file di dati, vedere Utilizzando il nuovo file di dati per reimpostare la stringa di cifratura.

    • Esportazione del vecchio file di dati: Se non si dispone del file secreto o del nuovo file di dati, è possibile utilizzare la procedura guidata per esportare il vecchio file di dati, quindi inviare via e-mail il file all'amministratore. Quando l'amministratore invia il nuovo file di dati, è possibile utilizzare la procedura guidata per reimpostare la stringa di cifratura. Per esportare il vecchio file di dati, vedere Uso del file esportato per la reimpostazione della stringa di cifratura.

Rappresenta la nuova stringa di cifratura che è necessario reimmettere.

Uso del file segreto per la reimpostazione della stringa di cifratura

Per reimpostare la stringa di cifratura utilizzando il file segreto, seguire la procedura riportata di seguito.

  1. Selezionare l'opzione È disponibile il file segreto e la password corrispondente, quindi fare clic su Avanti.

  2. Immettere le informazioni nei campi:

    Ubicazione del file segreto: Specificare l'ubicazione del file segreto oppure fare clic sul pulsante Sfoglia per individuarlo e selezionarlo.

    Password segreta: Specificare la password del file segreto.

    Nuova stringa di cifratura: Specificare la nuova stringa di cifratura.

    Conferma stringa di cifratura Reimmettere la nuova stringa di cifratura.

  3. Per reimpostare la stringa di cifratura, fare clic su Avanti.

  4. Fare clic su Fine per chiudere la procedura guidata.

NOTA:Richiedere il file secreto e la relativa password all'amministratore iFolder.

Utilizzando il nuovo file di dati per reimpostare la stringa di cifratura

Per reimpostare la stringa di cifratura utilizzando il nuovo file di dati, seguire la procedura riportata di seguito.

  1. Selezionare l'opzione È disponibile il nuovo file di dati inviato dall'amministratore, quindi fare clic su Avanti.

  2. Nel campo Ubicazione del nuovo file di dati specificare l'ubicazione del nuovo file di dati o fare clic sul pulsante Sfoglia per individuare e selezionare il nuovo file di dati.

  3. Se il nuovo file di dati ricevuto dall'amministratore è cifrato, selezionare la casella di controllo Il file precedente è cifrato?. Quando si seleziona questa casella di controllo, il campo Singola password è abilitato.

  4. Nel campo Singola password, specificare la password utilizzata dall'amministratore per cifrare il nuovo file di dati.

  5. Nel campo Nuova stringa di cifratura, specificare la nuova stringa di cifratura.

  6. Nel campo Conferma stringa di cifratura, reimmettere la nuova stringa di cifratura.

  7. Per reimpostare la stringa di cifratura, fare clic su Avanti.

  8. Fare clic su Fine per chiudere la procedura guidata.

Uso del file esportato per la reimpostazione della stringa di cifratura

Per reimpostare la stringa di cifratura quando non si dispone del file segreto o del nuovo file di dati:

  1. Selezionare l'opzione Non è disponibile il file segreto o di dati, quindi fare clic su Avanti.

  2. Nel campo Ubicazione in cui salvare il vecchio file di dati, specificare l'ubicazione in cui deve essere salvato il vecchio file di dati oppure fare clic sul pulsante Sfoglia per specificare l'ubicazione in cui salvare il vecchio file di dati.

  3. Fare clic su Avanti per ottenere il vecchio file di dati. Vengono visualizzati l'ubicazione del vecchio file di dati e l'indirizzo email dell'amministratore iFolder. È possibile utilizzare queste informazioni per ottenere il nuovo file di dati.

  4. Fare clic su Fine per chiudere la procedura guidata.

  5. Inviare via e-mail il vecchio file di dati all'amministratore iFolder con una richiesta di ricevere in cambio il nuovo file di dati.

  6. Quando si riceve il nuovo file di dati, vedere Utilizzando il nuovo file di dati per reimpostare la stringa di cifratura per reimpostare la stringa di cifratura.

Modifica della stringa di cifratura

Per motivi di sicurezza, è necessario cambiare periodicamente la stringa di cifratura. È possibile reimpostare la stringa di cifratura utilizzando la cifratura singola inviata dall'agente di recupero.

  1. Nel browser delle cartelle iFolder, selezionare Sicurezza > Cambia stringa di cifratura.

  2. Nella finestra Cambia stringa di cifratura selezionare il conto iFolder per cui si desidera reimpostare la stringa di cifratura e specificare i seguenti dettagli:

    Immettere la stringa di cifratura: Specificare la stringa di cifratura attuale.

    Immettere la nuova stringa di cifratura: Specificare la nuova stringa di cifratura che si desidera impostare per il conto iFolder attuale.

    Ridigitare la stringa di cifratura: Specificare di nuovo la nuova stringa di cifratura per conferma.

    Agente di recupero: Selezionare l'agente di recupero impostato per il conto iFolder attuale.

  3. Fare clic su Reimposta.
