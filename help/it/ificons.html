Informazioni sulle icone iFolder

Nella tabella seguente vengono descritte le icone iFolder e i task a queste associati.

Icona iFolder

Descrizione

L'icona Servizi iFolder viene visualizzata per il client iFolder 3 nel menu delle applicazioni del desktop.

Quando iFolder è in esecuzione questa icona viene visualizzata anche nell'area di notifica della barra delle applicazioni. Fare clic con il pulsante destro del mouse sull'icona per visualizzare il menu dei task disponibili, che comprendono Browser iFolder, Impostazioni conto, Log di sincronizzazione, Preferenze e Guida.

L'icona Crea cartella iFolder viene visualizzata nel browser delle cartelle iFolder.

È possibile creare una nuova cartella iFolder o convertire una cartella esistente in cartella iFolder. Se si dispone di più conti, è necessario specificare il conto da utilizzare. Una cartella iFolder può appartenere a un solo conto. Per ulteriori informazioni, vedere il Creazione e caricamento di una cartella iFolder.

L'icona relativa allo scaricamento di iFolder viene visualizzata quando una cartella iFolder è disponibile sul server specificato, ma non è stata scaricata nel computer locale.

È possibile scaricare nel computer locale qualsiasi combinazione di cartelle iFolder di diversi conti. Per ulteriori informazioni, vedere il Scaricamento di una cartella iFolder disponibile.

L'icona relativa alle cartelle iFolder cifrate (cartella iFolder bloccata) indica che la cartella iFolder è cifrata e viene visualizzata nel browser delle cartelle iFolder.

La stringa di cifratura fornita al momento del login consente di recuperare i file della cartella iFolder cifrata. Per ulteriori informazioni sulla cifratura, vedere Gestione delle stringhe di cifratura per le cartelle iFolder cifrate.

L'icona iFolder ("i" verde) viene visualizzata sulle cartelle iFolder quando l'applicazione Servizi iFolder è in esecuzione per le cartelle visualizzate in un manager di file, nel desktop o nel browser delle cartelle iFolder.

Quando iFolder non è in esecuzione, le cartelle iFolder vengono visualizzate come cartelle normali nel manager di file o nel desktop. Il browser delle cartelle iFolder non è disponibile se iFolder non è in esecuzione.

È consigliabile organizzare le cartelle iFolder locali in una directory separata contenente solo tali tipi di cartelle. In questo modo è possibile identificare facilmente le cartelle iFolder anche quando iFolder non è in esecuzione e non vengono visualizzate icone che contraddistinguono tali cartelle.

L'icona di avviso di iFolder indica che il server non è disponibile. iFolder è in esecuzione ma non si è connessi al server iFolder. Le cause possibili sono:

  • Non si è eseguito il login al conto iFolder. Per ulteriori informazioni, vedere il Login a un conto iFolder.

  • Il conto è disabilitato sul lato client, ovvero il parametro Connessione automatica è disabilitato. Per informazioni, vedere la sezione relativa alla connessione automatica in Visualizzazione e modifica delle impostazioni del conto iFolder.

  • Il server è disattivato o il conto è disabilitato sul lato server. Per assistenza, rivolgersi all'amministratore iFolder.

  • Il firewall blocca il traffico iFolder. Per informazioni sulla risoluzione del problema, vedere Configurazione delle impostazioni del firewall locale per il traffico iFolder.

  • La connessione di rete è disattivata. Verificare le impostazioni della rete locale.

L'icona Conflitto iFolder indica che è presente un conflitto tra la copia locale e la copia sul server di un file. Risolvere il conflitto indicando la versione del file da mantenere. Per ulteriori informazioni sui conflitti di file, vedere Risoluzione dei conflitti tra file.

L'icona In attesa di sincronizzazione di iFolder indica che la cartella iFolder contiene file da sincronizzare con il server e il processo di sincronizzazione non è ancora iniziato.

La sincronizzazione delle cartelle iFolder viene eseguita una cartella per volta e viene creata una coda dei task di sincronizzazione. Per garantire la corretta sincronizzazione dei file nella cartella iFolder locale con i file nel server, prima di chiudere iFolder attendere l'inizio e il completamento della sincronizzazione, indicati dai cambiamenti dell'icona.

L'icona Sincronizzazione in corso di iFolder indica che è in corso la sincronizzazione dei file della cartella iFolder con il server.

L'icona Errore iFolder indica che non è stata eseguita la sincronizzazione di uno o più file nella cartella iFolder a causa di un conflitto di norme dovuto al limite delle dimensioni dei file o del numero massimo di cartelle iFolder.

Nella finestra di dialogo Condivisione, l'icona Utente proprietario di iFolder indica che la cartella iFolder condivisa è di proprietà dell'utente.

Nella finestra di dialogo Condivisione, l'icona Utente attuale di iFolder indica che la cartella iFolder è attualmente gestita dall'utente. Se l'utente attuale è il proprietario, viene visualizzata l'icona Utente proprietario.

Per ulteriori informazioni, vedere Note legali.
