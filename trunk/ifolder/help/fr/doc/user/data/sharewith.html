Partage d'un dossier iFolder

Vous pouvez partager un dossier iFolder avec plusieurs utilisateurs. Cette section traite des tâches suivantes :

  • Présentation des droits d'accès des utilisateurs
  • Ajout d'un utilisateur à un dossier iFolder
  • Modification des droits d'accès des utilisateurs
  • Suppression d'un utilisateur d'un dossier iFolder

Ajout d'un utilisateur à un dossier iFolder


Ajout d'un utilisateur avec un gestionnaire de fichiers

  1. Dans le gestionnaire de fichiers, accédez au dossier iFolder et sélectionnez-le.

  2. Cliquez sur le bouton droit, puis sélectionnez iFolder > Propriétés.

  3. Cliquez sur Partager avec .

  4. Sélectionnez un ou plusieurs utilisateurs dans la liste des utilisateurs.

  5. Sélectionnez le droit d'accès approprié : Contrôle total, Lecture/écriture ou Lecture seule.

  6. Cliquez sur Appliquer.


Ajout d'un utilisateur avec le navigateur de dossiers iFolder

  1. Dans la zone de notification, cliquez avec le bouton droit sur l'icône iFolder , puis sélectionnez Dossiers iFolder pour ouvrir le navigateur de dossiers iFolder.

  2. Dans la liste des dossiers iFolder, sélectionnez le dossier iFolder voulu, puis cliquez sur Partager pour ouvrir la fenêtre Propriétés iFolder à l'onglet Partage.

  3. Cliquez sur Ajouter pour ouvrir la boîte de dialogue Sélectionner des utilisateurs.

  4. Dans la liste des utilisateurs, sélectionnez un ou plusieurs utilisateurs en les ajoutant à la liste Utilisateurs sélectionnés, puis cliquez sur OK.

    La synchronisation des invitations avec le serveur peut prendre quelques secondes.

  5. Sélectionnez un ou plusieurs utilisateurs, cliquez sur Accès, sélectionnez le droit d'accès Contrôle total, Lecture/écriture ou Lecture seule, puis cliquez sur OK.

  6. Répétez les étapes 4 et 5 jusqu'à ce que vous ayez invité les utilisateurs voulus à partager le dossier iFolder.

  7. Cliquez sur Appliquer, puis sur OK pour fermer la fenêtre Propriétés iFolder.


Modification des droits d'accès des utilisateurs

Étant donné que les relations de travail et l'état des membres d'un dossier iFolder changent, il peut être nécessaire de changer le droit d'accès d'un utilisateur à ce dossier. Vous pouvez, par exemple, accorder le droit d'accès Contrôle total à un utilisateur approuvé.


Configuration des droits d'accès des utilisateurs avec un gestionnaire de fichiers

  1. Dans le gestionnaire de fichiers, accédez au dossier iFolder et sélectionnez-le.

  2. Cliquez sur le bouton droit, puis sélectionnez iFolder > Propriétés.

  3. Cliquez sur Partager avec .

  4. Sélectionnez un ou plusieurs utilisateurs dans la liste des utilisateurs.

  5. Sélectionnez Accès, puis spécifiez le droit Contrôle total, Lecture/écriture ou Lecture seule.

  6. Cliquez sur Appliquer, puis sur OK pour fermer la fenêtre Propriétés iFolder.


Configuration des droits d'accès des utilisateurs avec le navigateur de dossiers iFolder

  1. Dans la zone de notification, cliquez avec le bouton droit sur l'icône iFolder , puis cliquez sur Dossiers iFolder pour ouvrir le navigateur de dossiers iFolder.

  2. Dans la liste des dossiers iFolder, sélectionnez le dossier voulu, puis cliquez sur Partager avec .

  3. Sélectionnez un ou plusieurs utilisateurs dans la liste des utilisateurs.

  4. Sélectionnez Accès, puis spécifiez le droit Contrôle total, Lecture/écriture ou Lecture seule.

  5. Cliquez sur Appliquer, puis sur OK pour fermer la fenêtre Propriétés iFolder.


Suppression d'un utilisateur d'un dossier iFolder

Pour annuler le partage d'un dossier iFolder, vous devez supprimer un utilisateur qui est membre de ce dossier. Le dossier iFolder sur le poste de travail de l'utilisateur devient un dossier normal. Une copie des données reste sur son poste de travail, mais les fichiers ne sont plus synchronisés avec le dossier iFolder partagé.


Suppression d'utilisateurs avec un gestionnaire de fichiers

  1. Dans le gestionnaire de fichiers, accédez au dossier iFolder et sélectionnez-le.

  2. Cliquez avec le bouton droit, puis sélectionnez iFolder > Propriétés.

  3. Cliquez sur Partager avec .

  4. Sélectionnez un ou plusieurs utilisateurs dans la liste des utilisateurs.

  5. Cliquez sur Retirer, sur Appliquer, puis sur OK.


Suppression d'utilisateurs avec le navigateur de dossiers iFolder

  1. Dans la zone de notification, cliquez avec le bouton droit sur l'icône iFolder , puis sélectionnez Dossiers iFolder pour ouvrir le navigateur de dossiers iFolder.

  2. Dans la liste des dossiers iFolder, sélectionnez le dossier iFolder voulu, puis cliquez sur Partager avec .

  3. Sélectionnez un ou plusieurs utilisateurs dans la liste des utilisateurs.

  4. Cliquez sur Retirer, sur Appliquer, puis sur OK.
