設定可用的 iFolder

若其他人與您共用 iFolder，則這些 iFolder 會在 iFolder 瀏覽器中以「可用的 iFolder」圖示顯示。若要在目前本機電腦上參與共用 iFolder，請設定 iFolder，以使其在電腦上的狀態為使用中。

本地 iFolder 位置必須符合下列指示：

  • 放置 iFolder 的指示
  • iFolder 及其資料夾和檔案的命名慣例

若要設定 iFolder：

  1. 在「通知」區域中，以滑鼠右鍵按一下 iFolder 圖示 ，然後按一下「iFolder」，以開啟 iFolder 瀏覽器。

  2. 在 iFolder 清單中，選取可用的 iFolder，然後按一下「設定」。

  3. 如果您尚未登入帳戶，請以使用者名稱和密碼登入，然後按一下「確定」。

  4. 瀏覽到您要建立本地 iFolder 的位置。

  5. 按一下「確定」兩次。
