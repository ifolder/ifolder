設定 iFolder 交通的本地病毒掃描器設定

如果要在電腦上使用病毒掃描軟體，請從病毒掃描中排除 ..\simias\WorkArea\ 目錄。例如，Windows 上的 ..\simias\WorkArea\ 目錄位於 c:\documents and settings\username\local settings\application data\simias\WorkArea\.simias.wf.guid 中。

此 ..\simias\WorkArea\ 目錄是 iFolder 將檔案進行分階段處理的位置，以便從伺服器下載。如果 iFolder 檔案附帶病毒，且您的病毒掃描軟體掃描到該病毒，而該檔案位於 ..\simias\WorkArea 目錄，則病毒掃描器會立現找到病毒的訊息，然後將該檔案移至隔離區，接著 iFolder 會再次嘗試同步該檔案。

不掃描 WorkArea 便會允許完成檔案的同步化；受感染的檔案會成功下載，並移至本地 iFolder。隨後，病毒掃描器會在 iFolder 中偵測到受感染的檔案，並將其從資料夾移至隔離區，以保護電腦免受感染。iFolder 偵測到檔案被刪除後，便會在新的同步開始時，從伺服器及成員電腦中移除該檔案。

但是，如果從中下載檔案的 iFolder 是「唯讀」的，則不會對伺服器及其他成員電腦強制執行本地刪除。如果許可權允許同步化變更，則必須在其他位置偵測病毒。使用者的病毒掃描器很可能會警告使用者發現病毒，因此該使用者應該警告其他 iFolder 成員關於此病毒的訊息。
