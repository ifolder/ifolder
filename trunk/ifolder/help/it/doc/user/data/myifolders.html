Gestione delle cartelle iFolder

Una cartella iFolderTM è una directory locale utilizzata per condividere e sincronizzare in modo selettivo file con membri specificati dall'utente. Tutti i membri possono accedere ai file memorizzati nella cartella iFolder e, se in possesso dei privilegi appropriati, leggerli e modificarli.

  • Convenzioni di denominazione per le cartelle iFolder e i relativi file e cartelle
  • Creazione di una cartella iFolder

  • Configurazione di una cartella iFolder disponibile
  • Visualizzazione e configurazione delle proprietà di una cartella iFolder
  • Sincronizzazione dei file
