Visualizzazione e configurazione delle proprietà di una cartella iFolder

La pagina delle proprietà delle cartelle iFolder consente di visualizzare e configurare le seguenti informazioni relative alla cartella iFolder selezionata.


Descrizione della finestra Proprietà iFolder

I seguenti parametri indicano lo spazio su disco utilizzato sul server:

Parametro Descrizione

Disponibile

Quantità totale di spazio su disco del server attualmente disponibile per i file e le directory nella cartella iFolder selezionata. Il valore corrisponde alla quota meno lo spazio utilizzato.

Usato

Quantità totale di spazio su disco del server attualmente utilizzato da tutti i file e le directory della cartella iFolder selezionata.

Una delle seguenti voci:

  • Imposta quota
  • Quota

  • Imposta quota: se l’utente è il proprietario della cartella iFolder, è possibile specificare una quota per la cartella.

  • Quota: se l’utente è un membro, il valore indica la quantità totale di spazio su disco del server allocata dal proprietario alla cartella iFolder selezionata. Se non è configurata alcuna quota, il valore è assente o non applicabile (N/A).

I seguenti parametri indicano lo stato di sincronizzazione della copia locale della cartella iFolder:

Parametro Descrizione

Ultima sincronizzazione completata

Data di completamento della sincronizzazione più recente della copia locale della cartella iFolder con il server iFolder.

File/cartelle da sincronizzare

Numero totale di elementi di memorizzazione (file e cartelle) nella cartella iFolder che devono essere sincronizzati.

Sincronizza automaticamente ogni

Intervallo di sincronizzazione per la cartella iFolder. Specificare il valore in minuti, ore o giorni.

Quando il client iFolder viene utilizzato con un server aziendale Novell iFolder 3.x , l’intervallo non può sostituire l’intervallo minimo specificato dall’amministratore di iFolder per il server o per il conto sul server, ma può sostituire l’intervallo di sincronizzazione a livello di client.

Sincronizza ora

Facendo clic sul pulsante, viene avviata una sincronizzazione su richiesta.


Accesso alla finestra Proprietà iFolder da un manager di file

  1. Nel manager di file individuare la cartella iFolder e selezionarla.

  2. Fare clic con il pulsante destro del mouse sulla cartella iFolder e scegliere iFolder > Proprietà.


Accesso alla finestra Proprietà iFolder dal browser delle cartelle iFolder

  1. Nell'area di notifica fare clic con il pulsante destro del mouse sull'icona dell’applicazione iFolder e scegliere Cartelle iFolder per aprire il browser.

  2. Selezionare la cartella iFolder nell’apposito elenco e fare clic su Proprietà.
