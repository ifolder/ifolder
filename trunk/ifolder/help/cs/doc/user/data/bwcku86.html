Pokyny pro různé platformy

Uživatelská rozhraní verzí klienta iFolder pro systémy Linux, Windows a Macintosh odpovídají zásadám a zvyklostem rozhraní na jednotlivých platformách. Způsob provádění běžných úloh je u každé platformy obdobný, avšak možná postupně zjistíte drobné rozdíly pokud jde o tlačítka, oblast zobrazení, popisky funkcí, umístění oblasti upozornění na pracovní ploše, předvolby spuštění aplikace iFolder atd. S výjimkou postupů, které se výslovně liší, tato příručka uvádí pokyny pouze pro jednu ze tří platforem.
