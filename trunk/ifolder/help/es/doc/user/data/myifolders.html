Gestión de carpetas iFolder

Una carpeta iFolderTM es un directorio local que se utiliza para compartir y sincronizar archivos de forma selectiva con miembros especificados por el usuario. Todos los miembros pueden acceder a los archivos de iFolder. Asimismo, pueden modificarlos y leerlos los miembros que dispongan de los privilegios necesarios.

En esta sección se tratan las siguientes tareas de iFolder:

  • Directrices de localización de carpetas iFolder
  • Directrices para tipos y tamaños de archivos que no se van a sincronizar
  • Convenciones de denominación de una carpeta iFolder y de las carpetas y archivos que contiene
  • Creación de una carpeta iFolder
  • Compartimiento de una carpeta iFolder
  • Configuración de una carpeta iFolder disponible
  • Visualización y configuración de propiedades de una carpeta iFolder
  • Sincronización de archivos
  • Resolución de conflictos con los archivos
  • Reversión de una carpeta iFolder a una carpeta normal
  • Movimiento de una carpeta iFolder
  • Supresión de una carpeta iFolder
