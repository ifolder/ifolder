Synchronisieren von Dateien


Synchronisieren von Dateien bei Bedarf

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das iFolder-Symbol () und klicken Sie dann auf "iFolder-Ordner", um den Browser für iFolder-Ordner zu öffnen.

  2. Wählen Sie in der Liste der iFolder-Ordner den iFolder-Ordner aus.

  3. Führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf "Synchronisieren" .
    • Wählen Sie "iFolder" > "Jetzt synchronisieren" aus.


Konfigurieren des Synchronisierungsintervalls

  1. Klicken Sie im Benachrichtigungsbereich mit der rechten Maustaste auf das Symbol für die iFolder-Anwendung und wählen Sie dann "Einstellungen" aus, um das Dialogfeld "iFolder-Einstellungen" mit der Registerkarte "Allgemein" aufzurufen.

  2. Führen Sie einen der folgenden Schritte aus:

    • Wählen Sie "Synchronisieren" aus, um die Synchronisierung im Hintergrund zu aktivieren.
    • Heben Sie die Auswahl von "Synchronisieren" auf, um die Synchronisierung zu deaktivieren.

  3. Wenn Sie die Synchronisierung aktivieren, geben Sie das Intervall für die Synchronisierung von Dateien an, beispielsweise einmal pro Stunde oder alle 2 Tage.

    Wenn der iFolder-Client mit Novell iFolder 3.x verwendet wird, wird das minimale Synchronisierungsintervall für das System (im Allgemeinen alle 5 Minuten) durch den iFolder-Administrator konfiguriert. Das Minimum für das System kann nicht überschrieben werden. Das absolute Minimum für das System beträgt 1 Minute.
