Iniciando o cliente iFolder


Iniciando o iFolder para Novell Linux Desktop 9

  1. Efetue o login no computador com a identidade de usuário que deseja usar ao criar pastas iFolder.

  2. Use um dos seguintes métodos para iniciar o iFolder:

    • Abra um shell de terminal e digite

      /opt/novell/ifolder3/bin/ifolder

    • Se desejar que o iFolder seja iniciado automaticamente quando você efetuar login, verifique se o iFolder está sendo executado quando efetuar o logout de sua sessão GNOME e, em seguida, grave a sessão. Você também pode abrir uma janela de terminal, efetuar login como usuário root e depois executar /opt/gnome/bin/gnome-session-properties para adicionar o iFolder como um programa de inicialização quando você efetuar login no computador.

  3. Se o recurso Lembrar Senha estiver desabilitado para a sua conta do iFolder, você deverá efetuar login na conta para sincronizar a lista de usuários e as pastas iFolder e gerenciar as pastas iFolder.

    Para obter informações, consulte Efetuando login em uma conta do iFolder.


Iniciando o iFolder para Windows 2000/XP

  1. Efetue o login no computador com a identidade de usuário do Windows que deseja usar quando criar pastas iFolder.

  2. Se o iFolder não for iniciado automaticamente em um logon bem-sucedido, inicie o iFolder clicando em Iniciar > Programas > iFolder e, em seguida, selecione o aplicativo iFolder.

    O iFolder não será iniciado automaticamente se você desabilitar a opção Iniciar iFolder Quando Efetuar Login na Configuração da Área de Trabalho nas Preferências do iFolder.

  3. Se o recurso Lembrar Senha estiver desabilitado para a sua conta do iFolder, você deverá efetuar login na conta para sincronizar a lista de usuários e as pastas iFolder e gerenciar as pastas iFolder.

    Para obter informações, consulte Efetuando login em uma conta do iFolder.


Iniciando o iFolder para Macintosh OS X v10.3 e posterior

  1. Efetue o login no computador com a identidade de usuário que deseja usar ao criar pastas iFolder.

  2. Use um dos seguintes métodos para iniciar o iFolder:

    • Abra o menu de aplicativos, selecione o submenu Internet e, em seguida, selecione iFolder.
    • Para iniciar o iFolder da linha de comando, abra um shell de terminal e, em seguida, digite

      /opt/novell/ifolder3/bin/ifolder

    • Se deseja que o iFolder seja iniciado automaticamente quando efetuar login no computador Macintosh, adicione o iFolder aos itens de inicialização.

  3. Se o recurso Lembrar Senha estiver desabilitado para a sua conta do iFolder, você deverá efetuar login na conta para sincronizar a lista de usuários e as pastas iFolder e gerenciar as pastas iFolder.

    Para obter informações, consulte Efetuando login em uma conta do iFolder.
