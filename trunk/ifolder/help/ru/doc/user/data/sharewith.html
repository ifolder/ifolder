Совместное использование папки iFolder

Можно использовать папку iFolder совместно с несколькими пользователями. В этой главе рассматриваются следующие задачи:

  • Общие сведения о правах доступа пользователей
  • Добавление пользователя папки iFolder
  • Изменение прав доступа пользователя
  • Удаление пользователя папки iFolder

Добавление пользователя папки iFolder


Добавление пользователя с помощью менеджера файлов

  1. В менеджере файлов перейдите к папке iFolder и выберите ее.

  2. Щелкните iFolder правой кнопкой, затем щелкните iFolder > Свойства.

  3. Щелкните "Совместное использование" .

  4. Выберите одного или нескольких пользователей из списка.

  5. Выберите право доступа: "Полный контроль", "Чтение/запись" или "Только чтение".

  6. Щелкните кнопку "Применить".


Добавление пользователя с помощью средства просмотра iFolder

  1. В области оповещения щелкните правой кнопкой значок iFolder , затем щелкните "Папки iFolder" для открытия средства просмотра iFolder.

  2. Выберите папку iFolder из списка и щелкните "Совм. исп." для открытия окна "Свойства iFolder" на вкладке "Совместное использование".

  3. Щелкните кнопку "Добавить" для открытия диалогового окна "Выбор пользователей".

  4. Выберите одного или нескольких пользователей из списка путем добавления их в список "Выбранные пользователи" и щелкните кнопку "ОК".

    Для синхронизации приглашений на сервере потребуется несколько секунд.

  5. Выберите одного или нескольких пользователей, щелкните "Доступ", выберите право доступа: "Полный контроль", "Чтение/запись" или "Только чтение", затем щелкните кнопку "ОК".

  6. Повторяйте действия 4 и 5, пока все нужные пользователи не получат приглашение на совместное использование папки iFolder.

  7. Щелкните кнопку "Применить", затем кнопку "ОК" для закрытия окна "Свойства iFolder".


Изменение прав доступа пользователя

Поскольку рабочие взаимосвязи и состояния членов iFolder меняются, может потребоваться изменение права доступа пользователя к папке iFolder. Например, Вы можете захотеть дать право полного контроля в папке iFolder пользователю, которому Вы доверяете.


Настройка прав пользователя с помощью менеджера файлов

  1. В менеджере файлов перейдите к папке iFolder и выберите ее.

  2. Щелкните iFolder правой кнопкой, затем щелкните iFolder > Свойства.

  3. Щелкните "Совместное использование" .

  4. Выберите одного или нескольких пользователей из списка.

  5. Выберите "Доступ", затем укажите право: "Полный контроль", "Чтение/запись" или "Только чтение".

  6. Щелкните кнопку "Применить", затем кнопку "ОК" для закрытия окна "Свойства iFolder".


Настройка прав пользователя с помощью средства просмотра iFolder

  1. В области оповещения щелкните правой кнопкой значок iFolder , затем щелкните "Папки iFolder" для открытия средства просмотра iFolder.

  2. Выберите папку iFolder из списка, затем щелкните "Совместное использование" .

  3. Выберите одного или нескольких пользователей из списка.

  4. Выберите "Доступ", затем укажите право: "Полный контроль", "Чтение/запись" или "Только чтение".

  5. Щелкните кнопку "Применить", затем кнопку "ОК" для закрытия окна "Свойства iFolder".


Удаление пользователя папки iFolder

Для отмены совместного использования папки iFolder необходимо удалить пользователя как члена папки iFolder. На рабочей станции пользователя папка iFolder становится стандартной папкой. На рабочей станции бывшего участника остается копия данных, однако файлы больше не синхронизируются с совместно используемой папкой iFolder.


Удаление пользователей с помощью менеджера файлов

  1. В менеджере файлов перейдите к папке iFolder и выберите ее.

  2. Щелкните iFolder правой кнопкой, затем щелкните iFolder > Свойства.

  3. Щелкните "Совместное использование" .

  4. Выберите одного или нескольких пользователей из списка.

  5. Щелкните кнопку "Удалить", затем кнопку "Применить", а затем "OK".


Удаление пользователей с помощью средства просмотра iFolder

  1. В области оповещения щелкните правой кнопкой значок iFolder , затем щелкните "Папки iFolder" для открытия средства просмотра iFolder.

  2. Выберите папку iFolder из списка, затем щелкните "Совместное использование" .

  3. Выберите одного или нескольких пользователей из списка.

  4. Щелкните кнопку "Удалить", затем кнопку "Применить", а затем "OK".
